# 附錄 D：流程圖與決策樹

---

## 概述

這些決策樹和流程圖為加州開發中最常見的決策提供逐步指導。在項目評估和許可審批過程中，將它們作為快速參考工具使用。

---

## D.1 CEQA 決策樹

```
開始：您的項目是否受 CEQA 管轄?
│
├── 該項目是否為行政性質（無裁量權）？
│   ├── 是 → 無需 CEQA
│   │   範例：SB 35、SB 9、ADU 依權利建造、
│   │   無裁量行為的建築許可
│   │
│   └── 否 → 繼續 ↓
│
├── 是否適用法定豁免？
│   │
│   ├── SB 35/423、SB 9、SB 684、AB 2011？
│   │   └── 是 → 無需 CEQA（行政性質）
│   │
│   ├── AB 130 填充式住宅豁免？（2025）
│   │   要求：
│   │   □ 住宅（非旅館/臨時住宿）
│   │   □ 城市填充，≤20 英畝
│   │   □ ≥15 單位/英畝（都會區）或 ≥10 單位/英畝（郊區）
│   │   □ ≥2/3 住宅用途
│   │   □ 符合總體規劃或分區（僅需其一）
│   │   □ 不在保護/危險土地上
│   │   └── 是 → CEQA 豁免
│   │       需要第一階段環境評估 + 部落諮詢
│   │       時間表：104–178 天
│   │       完整標準見第 3 章 §3.6.2
│   │
│   ├── 其他法定豁免？
│   │   └── 是 → 無需 CEQA
│   │       提交豁免通知（建議）
│   │
│   └── 否 → 繼續 ↓
│
├── 是否適用類別豁免？
│   │
│   ├── Class 1：現有設施
│   │   （現有結構的輕微改造）
│   │
│   ├── Class 3：新建、小型結構
│   │   （都市化區域內最多 6 個住宅單位）
│   │
│   ├── Class 32：填充式開發
│   │   要求：
│   │   □ 符合總體規劃和分區
│   │   □ 市界內，≤5 英畝
│   │   □ 周圍基本為城市用途
│   │   □ 無瀕危物種棲息地
│   │   □ 無顯著交通/噪音/空氣/水影響
│   │   □ 公用設施充足
│   │
│   ├── 是（豁免適用）→
│   │   │
│   │   └── 是否有任何例外適用？（§15300.2）
│   │       □ 敏感環境（濕地、棲息地）
│   │       □ 累積影響
│   │       □ 異常情況 → 顯著影響
│   │       □ 景觀公路
│   │       □ 有害廢物場地（已列出）
│   │       □ 歷史資源
│   │       │
│   │       ├── 無例外 → 豁免
│   │       │   提交豁免通知
│   │       │
│   │       └── 有例外 → 不豁免 → 繼續 ↓
│   │
│   └── 否 → 繼續 ↓
│
├── SB 131「臨界」檢查（2025）
│   項目是否僅因一個條件
│   而未能獲得豁免？
│   │
│   ├── 是 → 有限聚焦 CEQA 審查
│   │   僅分析該單一
│   │   不合格條件的影響
│   │   無需替代方案分析
│   │   無需增長誘發影響分析
│   │   詳見第 3 章 §3.6.2
│   │
│   └── 否 → 繼續 ↓
│
├── 需要初步研究
│   評估所有環境影響類別
│   │
│   ├── 所有影響能否減緩至
│   │   低於顯著水平？
│   │   │
│   │   ├── 是 → 減緩負面聲明（MND）
│   │   │   時間表：12–18 個月
│   │   │   成本：$50K–$150K
│   │   │   30 天公眾審查期
│   │   │
│   │   └── 否 → 繼續 ↓
│   │
│   └── 是否有潛在的顯著
│       無法減緩的影響？
│       │
│       └── 是 → 環境影響報告（EIR）
│           時間表：18–36 個月
│           成本：$200K–$1M+
│           45 天公眾審查期
│           需要公開聽證會
│
└── 公平論證警告
    如果存在任何潛在顯著影響的
    實質性證據（即使只有一個有事實依據的
    公眾評論），無論初步研究結論如何，
    都可能需要 EIR。
```

## D.2 許可途徑決策樹 — 住宅

```
開始：加州住宅項目
│
├── 1. 是 ADU 或 JADU 嗎？
│   ├── 是 → ADU 依權利途徑
│   │   • Gov. Code §65852.2
│   │   • 60 天批准時間表
│   │   • 無 CEQA，無公開聽證會
│   │   • <750 SF 無影響費
│   │   → 完成（最快途徑）
│   │
│   └── 否 → 繼續 ↓
│
├── 2. 是單戶住宅地塊上的
│      雙拼或地塊分割嗎？
│   ├── 是 → SB 9 途徑
│   │   要求：
│   │   □ 現有單戶住宅分區
│   │   □ 城市地塊（非鄉村/農業）
│   │   □ 非歷史街區
│   │   □ 不在洪水/火災/斷層區*
│   │   □ 符合客觀標準
│   │   │
│   │   ├── 全部符合 → 行政批准
│   │   │   • 60 天時間表
│   │   │   • 無 CEQA，無聽證會
│   │   │   • 最多 4 個單位（地塊分割 + 雙拼）
│   │   │
│   │   └── 未全部符合 → 繼續 ↓
│   │
│   └── 否 → 繼續 ↓
│
├── 3. 該管轄區是否符合 SB 35 資格？
│      （查看 HCD 決定）
│   │
│   ├── 是 → 項目是否符合 SB 35 要求？
│   │   □ 多戶（2+ 單位）
│   │   □ 地塊上 ≥75% 住宅
│   │   □ 符合所有客觀標準
│   │   □ 包含可負擔組件：
│   │     • 10% 可負擔（若城市未達中上收入 RHNA）
│   │     • 50% 可負擔（若城市未達低收入 RHNA）
│   │   □ 支付現行工資
│   │   □ 不在環境敏感地點
│   │   □ 不在農田或火災危險區
│   │   │
│   │   ├── 全部符合 → SB 35 行政批准
│   │   │   • <150 單位：90 天審查
│   │   │   • 150+ 單位：180 天審查
│   │   │   • 無 CEQA，無公開聽證會
│   │   │   • 無主觀設計審查
│   │   │
│   │   └── 未全部符合 → 繼續 ↓
│   │
│   └── 否 → 繼續 ↓
│
├── 4. 裁量途徑
│   │
│   ├── 項目是否與提議的
│   │   總體規劃和分區一致？
│   │   │
│   │   ├── 是 → 僅需場地規劃/設計審查
│   │   │   └── CEQA 決定（見 D.1）
│   │   │
│   │   └── 否 → 需要哪些批准？
│   │       □ 總體規劃修訂
│   │       □ 重新分區
│   │       □ 變更
│   │       □ 條件使用許可（CUP）
│   │       □ 具體規劃修訂
│   │       └── 每項都需要 CEQA + 公開聽證會
│   │
│   └── 應用 SB 330 保護：
│       □ 立即提交初步申請
│       □ 最多 5 次公開聽證會
│       □ 審查期間不得降低分區
│       □ 費用鎖定在申請前日期
│       □ 適用 HAA 拒絕保護
│
└── 時間表摘要
    ADU：              60 天
    SB 9：             60 天
    SB 35（<150）：     90 天
    SB 35（150+）：     180 天
    裁量：            6–36 個月
```

## D.3 許可途徑決策樹 — 旅館

```
開始：加州旅館項目
│
├── 注意：旅館不符合以下資格：
│   • SB 35（僅限住宅）
│   • SB 9（僅限住宅）
│   • SB 330 保護（僅限住宅）
│   • ADU 法（僅限住宅）
│   旅館遵循完整的裁量途徑。
│
├── 1. 分區檢查
│   │
│   ├── 旅館在該區是否為允許用途？
│   │   ├── 是（依權利）→ 僅需場地規劃審查
│   │   │   仍需 CEQA 決定
│   │   │
│   │   ├── 是（需 CUP）→ 需要 CUP 申請
│   │   │   旅館最常見途徑
│   │   │
│   │   └── 否 → 選項：
│   │       □ 需要重新分區
│   │       □ 需要總體規劃修訂
│   │       □ 尋找替代場地
│   │       → 每項增加 12–24 個月
│   │
│   └── 繼續 ↓
│
├── 2. 品牌與加盟（平行軌道）
│   │
│   ├── 市場研究完成了嗎？
│   │   └── 加盟意向書所需
│   │
│   ├── 品牌選定了嗎？
│   │   └── 決定建築方案
│   │
│   ├── 加盟意向書簽署了嗎？
│   │   └── 關鍵：意向書有到期日
│   │       典型：12–18 個月
│   │       必須與許可時間表對齊
│   │
│   └── 繼續 ↓
│
├── 3. CEQA 途徑（見 D.1）
│   │
│   ├── 旅館項目很少符合：
│   │   □ Class 3 豁免（太大）
│   │   □ Class 32 填充（交通/噪音通常顯著）
│   │
│   ├── 最常見：MND 或 EIR
│   │   │
│   │   ├── 旅館的主要影響領域：
│   │   │   □ 交通（高出行生成）
│   │   │   □ 噪音（HVAC、裝卸、活動）
│   │   │   □ 空氣質量（運營、備用發電機）
│   │   │   □ 美學（高度、照明、標識）
│   │   │   □ 停車
│   │   │   □ 溫室氣體排放
│   │   │
│   │   └── 例外：來自規劃 EIR 的分層
│   │       如果場地位於具有認證規劃 EIR 的
│   │       具體規劃或總體規劃區域內，
│   │       旅館可以分層 → 節省 $500K+ 和 12+ 個月
│   │
│   └── 繼續 ↓
│
├── 4. ABC 許可（平行軌道 — 第 6 個月開始）
│   │
│   ├── 需要的許可類型？
│   │   □ Type 47（現場銷售一般 — 全套酒吧）
│   │   □ Type 41（現場銷售啤酒/葡萄酒）
│   │   □ Type 57（特殊現場銷售一般 — 餐廳）
│   │
│   ├── 轉讓 vs. 新許可？
│   │   轉讓：更快，可能避免普查限制
│   │   新：受普查限制，可能被拒絕
│   │
│   └── 時間表：6–12 個月
│       提早開始以避免延遲開業
│
├── 5. 批准與聽證會
│   │
│   ├── 設計審查 → 規劃委員會
│   ├── CUP 聽證會 → 規劃委員會
│   ├── EIR 認證 → 規劃委員會或市議會
│   ├── 上訴（如提交）→ 市議會
│   │
│   └── 總聽證會：通常 3–5 次
│
└── 6. 建築許可
    │
    ├── 提交施工文件
    ├── 加盟品牌審查（平行）
    ├── 消防局審查
    ├── 衛生部審查（食品服務）
    │
    └── 時間表：計劃檢查 3–6 個月

旅館總時間表：通常 24–36 個月
```

## D.4 場地篩選流程圖

```
開始：已識別潛在開發場地
│
├── 階段 1：3 分鐘篩選（在線 GIS）
│   │
│   ├── 檢查地塊大小 → 項目足夠嗎？
│   │   └── 否 → 跳過
│   │
│   ├── 檢查分區 → 與項目兼容嗎？
│   │   └── 否 → 需要總體規劃修訂？
│   │       └── 是 → 增加 12–24 個月 + 成本
│   │
│   ├── 檢查 FEMA 洪水區 → Zone AE/VE？
│   │   └── 是 → 致命缺陷，除非價格反映
│   │
│   ├── 檢查地震斷層區 → Alquist-Priolo？
│   │   └── 是 → 致命缺陷（無可居住結構）
│   │
│   └── 通過 3 分鐘篩選？→ 繼續 ↓
│
├── 階段 2：10 分鐘深入調查
│   │
│   ├── CGS 液化區？
│   │   └── 是 → 預算額外 $8K–$15K/單位
│   │
│   ├── CAL FIRE FHSZ（高/極高）？
│   │   └── 是 → WUI 預算 $40K–$80K/單位
│   │
│   ├── EnviroStor（1,000 英尺內污染）？
│   │   └── 是 → 評估預算 $100K–$200K
│   │
│   ├── GeoTracker（地下水問題）？
│   │   └── 是 → 評估蒸汽侵入風險
│   │
│   ├── 濕地清單？
│   │   └── 是 → 需要 USACE Section 404 許可
│   │
│   └── 通過 10 分鐘篩選？→ 繼續 ↓
│
├── 階段 3：30 分鐘分析
│   │
│   ├── CalEnviroScreen 百分位？
│   │   └── 高 → 額外社區參與
│   │
│   ├── 場地上/附近有歷史資源？
│   │   └── 是 → CEQA 歷史評估
│   │
│   ├── SB 35 資格（住宅）？
│   │   └── 是 → 行政途徑可用！
│   │
│   ├── 管轄區的 RHNA 狀態？
│   │   └── 落後 → 城市更可能批准
│   │
│   └── 初步財務模型可行？
│       └── 否 → 跳過
│
└── 決定
    │
    ├── 綠燈：無致命缺陷，分區兼容，
    │   經濟可行 → 進行意向書/合同
    │
    ├── 黃燈：可管理的限制，經濟
    │   邊際 → 附條件進行
    │   （價格調整、附加條件）
    │
    └── 紅燈：致命缺陷或經濟不可行 → 跳過
```

## D.5 密度獎勵決策樹

```
開始：考慮密度獎勵？
│
├── 1. 您可以提供什麼可負擔水平？
│   │
│   ├── 極低收入（≤50% AMI）
│   │   5% VLI → 20% 獎勵
│   │   10% VLI → 33.25% 獎勵
│   │   15% VLI → 50% 獎勵（最大）
│   │
│   ├── 低收入（≤80% AMI）
│   │   10% LI → 20% 獎勵
│   │   20% LI → 35% 獎勵
│   │   24% LI → 43.75% 獎勵
│   │
│   ├── 中等收入（≤120% AMI，僅限出售）
│   │   10% Mod → 5% 獎勵
│   │   40% Mod → 35% 獎勵
│   │
│   └── 100% 可負擔 → 80% 獎勵
│
├── 2. 優惠/激勵
│   │
│   ├── 您可以申請多少？
│   │   5% VLI 或 10% LI → 1 項優惠
│   │   10% VLI 或 17% LI → 2 項優惠
│   │   15% VLI 或 24% LI → 3 項優惠
│   │
│   └── 常見優惠：
│       □ 減少退線距離
│       □ 減少停車（最低 0.5/單位）
│       □ 增加高度
│       □ 增加容積率
│       □ 減少開放空間
│
├── 3. 豁免（無限）
│   │
│   └── 任何實際阻止實現
│       獎勵密度的標準都可以豁免。
│       城市必須批准，除非會有
│       特定的不利健康/安全影響。
│       │
│       └── 範例：高度限制阻止建造
│           獎勵單位 → 批准高度豁免
│
└── 4. 組合策略
    │
    ├── 密度獎勵 + SB 35 = 最大威力
    │   獎勵密度 + 行政批准
    │
    ├── 密度獎勵 + ADU = 額外單位
    │   獎勵單位 + ADU 單位（不計入）
    │
    └── 密度獎勵 + SB 330 = 費用保護
        獎勵密度 + 鎖定費用
```

## D.6 模組化 vs. 傳統決策樹

```
開始：該項目應使用模組化建造嗎？
│
├── 1. 項目類型檢查
│   │
│   ├── 旅館（精選服務，重複房間）？
│   │   └── 模組化強候選
│   │
│   ├── 多戶（開間/1BR/2BR 重複）？
│   │   └── 模組化強候選
│   │
│   ├── 學生/老年住宅（重複單位）？
│   │   └── 模組化強候選
│   │
│   ├── 定制單戶住宅？
│   │   └── 模組化弱候選（考慮面板化替代）
│   │
│   └── 不規則樓板的混合用途？
│       └── 仔細評估（混合方法可能有效）
│
├── 2. 場地通道檢查
│   │
│   ├── 模組能到達場地嗎？
│   │   □ 道路寬度足夠（最小 14 英尺淨空）？
│   │   □ 轉彎半徑足夠？
│   │   □ 橋樑重量限制可以？
│   │   □ 無頭頂障礙物？
│   │   │
│   │   ├── 全部是 → 繼續 ↓
│   │   └── 任何否 → 傳統
│   │       （除非可以減小模組尺寸）
│   │
│   ├── 起重機放置可行嗎？
│   │   □ 起重機墊區域可用？
│   │   □ 相鄰建築允許吊臂擺動？
│   │   │
│   │   └── 否 → 傳統或面板化
│   │
│   └── 繼續 ↓
│
├── 3. 結構檢查
│   │
│   ├── 地震設計類別？
│   │   □ SDC A–D → 標準模組化連接可以
│   │   □ SDC E–F → 需要特殊抗震連接
│   │       （BRB 框架、彎矩連接）
│   │       → 增加成本但仍可行
│   │
│   ├── 建築高度？
│   │   □ 1–5 層 → 標準模組化
│   │   □ 6–12 層 → 增強結構的模組化
│   │   □ 12+ 層 → 仔細評估
│   │       （混合：混凝土裙樓上的模組化單元）
│   │
│   └── 繼續 ↓
│
├── 4. 經濟檢查
│   │
│   ├── 該市場勞動力稀缺/昂貴嗎？
│   │   └── 是 → 模組化優勢增加
│   │
│   ├── 時間表關鍵嗎？
│   │   └── 是 → 模組化節省 40–50% 時間
│   │
│   ├── 模組運輸距離？
│   │   □ <200 英里 → 經濟有利
│   │   □ 200–500 英里 → 評估運輸成本
│   │   □ >500 英里 → 運輸成本可能抵消節省
│   │   □ 海外（集裝箱）→ 4–8 週海運
│   │
│   ├── 有產能的製造商嗎？
│   │   └── 否 → 交貨期長；評估替代方案
│   │
│   └── 繼續 ↓
│
└── 決策矩陣
    │
    ├── 推薦體積模組化如果：
    │   □ 重複房間/單位類型
    │   □ 場地通道足夠
    │   □ 時間表優先
    │   □ 製造商有產能
    │   → 預期節省：10–20% 成本，40–50% 時間表
    │
    ├── 推薦面板化如果：
    │   □ 中等重複性
    │   □ 場地通道受限（無起重機）
    │   □ 需要設計靈活性
    │   □ 本地面板製造商可用
    │   → 預期節省：5–15% 成本，20–30% 時間表
    │
    ├── 推薦混合如果：
    │   □ 高層建築（裙樓 + 模組）
    │   □ 混合用途（商業底層 + 模組上層）
    │   □ 複雜場地（結合傳統 + 模組化）
    │   → 預期節省：因配置而異
    │
    └── 推薦傳統如果：
        □ 獨特/定制設計
        □ 場地通道嚴重受限
        □ 無製造商可用
        □ 小項目（<10 單位）
        □ 複雜結構要求
```

## D.9 財務決策門檻流程圖

```
開始：新開發機會
│
├── 級別 1：餐巾紙測試（5 分鐘）
│   │
│   ├── 土地成本 / 可建單位 = 單位成本
│   │   └── > 市場比較？ → 終止（土地溢價）
│   │
│   ├── 估計單位總成本
│   │   └── > ARV 的 80%？ → 終止（利潤不足）
│   │
│   ├── 估計毛利率
│   │   └── < 15%？ → 終止（利潤太薄）
│   │
│   └── 通過？ → 繼續級別 2
│
├── 級別 2：開發財務模型
│   │
│   ├── 來源與用途
│   │   □ 土地 + 交易成本
│   │   □ 硬成本（場地 + 基礎 + 建築 + FF&E）
│   │   □ 軟成本（A&E + CEQA + 法律 + 許可 + 費用）
│   │   □ 融資成本
│   │   □ 應急費（10–15%）
│   │   └── 總開發成本（TDC）
│   │
│   ├── 收入
│   │   □ 住宅：租金表或銷售比較
│   │   □ 旅館：ADR x 入住率 x 鑰匙數 x 365
│   │   └── 穩定 NOI
│   │
│   ├── 回報
│   │   □ 開發利潤（利潤 / TDC）
│   │   □ 現金對現金（NOI / 股權）
│   │   □ IRR（5 年持有）
│   │   □ 退出價值（NOI / cap rate）
│   │
│   ├── 死亡點分析（附錄 G §2D）
│   │   月度持有成本 = $_______
│   │   總利潤 = $_______
│   │   死亡點 = 利潤 ÷ 月度持有成本 = _____ 個月
│   │   └── 死亡點 < 融資期限？ → 終止
│   │
│   └── 通過？ → 繼續級別 3
│
├── 級別 3：壓力測試與終止標準
│   │
│   ├── 壓力測試（每個輸入變化 +/- 10-15%）
│   │   □ 建造成本敏感性
│   │   □ 時間表敏感性
│   │   □ 收入/租金敏感性
│   │   □ 利率敏感性
│   │   □ 退出 cap rate 敏感性
│   │
│   ├── 12 項終止標準（任何一項 = 終止）
│   │   完整列表見附錄 G §3C
│   │   關鍵觸發因素：
│   │   □ #3：TDC 超過實際退出價值
│   │   □ #5：監管拒絕風險 > 30%
│   │   □ #8：時間表超過融資期限
│   │   □ #11：市場變化消除需求論點
│   │
│   └── 全部通過？ → 執行/投入資金
│
└── 決定
    │
    ├── 執行 — 全部 3 個級別通過
    │   → 批准設計（在附錄 E 預設計分析之後）
    │   → 提交 SB 330 初步申請（如果是住宅）
    │   → 獲得加盟意向書（如果是旅館）
    │
    ├── 終止 — 任何級別失敗
    │   → 記錄原因
    │   → 放棄場地
    │   → 轉向下一個機會
    │
    └── 有條件 — 級別 2 或 3 邊緣
        → 重新談判購買價格
        → 修改方案（密度、產品類型）
        → 用新假設重新運行級別 1–3
```
