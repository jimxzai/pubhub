# 附錄 J:住宅快速開發專案——新手指南

**每位加州開發商必須了解的 6 種住宅專案類型**

---

## 如何使用本指南

本附錄去除法律複雜性,為您提供加州六種最常見住宅專案類型的**逐步執行手冊**。每個部分回答五個問題:

1. **這是什麼?** — 白話定義
2. **流程是什麼?** — 逐步說明,按順序
3. **需要多長時間?** — 實際時間表
4. **成本多少?** — 概略範圍(灣區)
5. **什麼會終止專案?** — 讓新手跌倒的陷阱

使用末尾的**決策流程圖**來確定哪種專案類型適合您的情況。

## 住宅改建 (Residential Remodel)

### 這是什麼?

您已經擁有一棟房屋。您想改變內部——新廚房、新浴室、打開牆壁、更換窗戶、升級電氣/管道。房屋的占地面積保持不變。沒有新增平方英尺。

### 何時使用

- 廚房或浴室全面翻新
- 拆除或添加內牆
- 升級配電箱、管道或 HVAC
- 更換窗戶、門或屋頂
- 抗震加固 (Seismic retrofit)

### 逐步流程

```text
步驟 1:設計 (2–6 週)
    聘請建築師或設計師繪製平面圖
    對於表面工程(油漆、地板、固定裝置):無需許可證
    對於結構/MEP 工程:需要平面圖
    │
步驟 2:許可證申請 (1 天)
    向市建築部門提交平面圖
    支付平面審查費用(通常 $500–$3,000)
    │
步驟 3:平面審查 (2–8 週)
    市政府審查規範合規性
    可能需要更正(增加 1–3 週)
    │
步驟 4:許可證核發
    在現場張貼許可卡
    │
步驟 5:施工 (4–16 週)
    聘請持照承包商(或非結構工程可自建)
    在每個階段安排檢查:
      □ 粗框架(如果移動牆壁)
      □ 粗電氣
      □ 粗管道
      □ 粗機械(HVAC)
      □ 隔熱
      □ 最終檢查
    │
步驟 6:最終簽核
    建築檢查員批准最終檢查
    結案許可證
    完成。
```

### 時間表

| 範圍 | 設計 | 許可證 | 施工 | 總計 |
| --- | --- | --- | --- | --- |
| **表面工程**(無需許可證) | 1–2 週 | 無 | 2–6 週 | **3–8 週** |
| **廚房/浴室改建** | 2–4 週 | 3–6 週 | 6–12 週 | **3–5 個月** |
| **全屋改建** | 4–8 週 | 4–8 週 | 12–24 週 | **5–10 個月** |

### 成本範圍(灣區)

| 範圍 | 每平方英尺成本 | 典型總成本 |
| --- | --- | --- |
| **表面工程** | $50–$100/SF | $25K–$75K |
| **廚房改建** | $250–$500/SF | $50K–$150K |
| **浴室改建** | $300–$600/SF | $25K–$75K |
| **全屋拆除重建** | $200–$400/SF | $200K–$600K |

### 什麼會終止專案

| 陷阱 | 會發生什麼 | 如何避免 |
| --- | --- | --- |
| **未經許可的先前工程** | 檢查員發現前業主未經許可的增建;必須在新許可證結案前合法化 | 在購買**之前**調閱財產記錄和許可證歷史 |
| **石棉/鉛漆** | 1978 年前的房屋需要測試;清除費用增加 $5K–$30K | 在任何 1978 年前的房屋中為測試編列預算 |
| **結構意外** | 打開牆壁,發現乾腐、白蟻或框架不足 | 始終預留 15–20% 應急費用 |
| **HOA 限制** | 外部變更可能需要 HOA 批准 | 在簽署任何設計合約前檢查 CC&Rs |

### 主要法律

- 結構、電氣、管道或機械工程**需要建築許可證**
- **Title 24 能源法規**適用於任何需許可的工程(高效窗戶、隔熱等)
- **無 CEQA** — 改建不屬於 CEQA 下的「專案」
- **無規劃批准** — 僅需建築許可證(除非在歷史街區進行外部變更)

## 增建 (Addition——擴建您的住宅)

### 這是什麼?

您為現有房屋添加**新的平方英尺**——新房間、二樓、凸出部分或車庫改建為生活空間。建築占地面積改變(或增加新樓層)。

### 何時使用

- 添加臥室、家庭活動室或家庭辦公室
- 在單層房屋上建造二樓
- 延伸房屋的後部或側面
- 將車庫改建為生活空間(不是 ADU——這是主要住宅的一部分)

### 逐步流程

```text
步驟 1:分區檢查 (1 天)
    驗證您的地塊能夠支持增建:
      □ Setbacks(退縮距離):您可以距離地界線多近建造?
      □ Lot coverage(建蔽率):可建造的地塊最大百分比
      □ FAR(容積率):最大總平方英尺
      □ Height limit(高度限制):您可以向上建造嗎?
    市規劃櫃檯或線上分區查詢
    │
步驟 2:設計 (4–10 週)
    聘請建築師
    如果是二樓或重大荷載變化需要結構工程師
    能源計算(Title 24)
    │
步驟 3:規劃審查——如果需要 (4–12 週)
    一些城市要求對從街道可見的增建進行「設計審查」
    其他城市僅需建築許可證(無需規劃審查)
    向您所在城市的規劃部門查詢
    │
步驟 4:建築許可證申請
    提交建築平面圖 + 結構圖 + Title 24 計算
    平面審查費用:$2,000–$8,000
    │
步驟 5:平面審查 (4–12 週)
    建築部門審查結構、電氣、管道、機械
    通常需要更正——增加 2–4 週
    │
步驟 6:許可證核發
    支付許可費 + 學校費 + 任何影響費
    │
步驟 7:施工 (3–9 個月)
    地基 → 框架 → 粗 MEP → 隔熱 →
    石膏板 → 完工 → 最終檢查
    │
步驟 8:入住許可證 (Certificate of Occupancy) 或最終簽核
    所有檢查通過
    許可證結案
    完成。
```

### 時間表

| 範圍 | 設計 + 許可證 | 施工 | 總計 |
| --- | --- | --- | --- |
| **小型增建**(200–400 SF) | 2–4 個月 | 3–5 個月 | **5–9 個月** |
| **主要增建**(400–1,000 SF) | 3–6 個月 | 5–8 個月 | **8–14 個月** |
| **二樓** | 4–8 個月 | 6–12 個月 | **10–20 個月** |

### 成本範圍(灣區)

| 類型 | 每平方英尺成本 | 典型總成本 |
| --- | --- | --- |
| **一樓增建** | $400–$700/SF | $150K–$500K |
| **二樓增建** | $500–$800/SF | $300K–$800K |
| **車庫改建**(為生活空間) | $200–$400/SF | $60K–$160K |

### 什麼會終止專案

| 陷阱 | 會發生什麼 | 如何避免 |
| --- | --- | --- |
| **超過 FAR 或建蔽率** | 市政府拒絕許可證——您的地塊無法支持增建 | **首先**運行分區數字,在聘請建築師之前 |
| **退縮距離侵佔** | 增建距離地界線太近;需要 variance(6–12 個月,$5K+ 費用,無保證) | 在退縮距離內設計——不要指望獲得 variance |
| **地基問題** | 現有地基無法支持二樓;需要全新地基($50K–$150K) | 在承諾二樓之前獲得結構工程師評估 |
| **鄰居反對** | 如果需要設計審查,鄰居可以反對並延遲 | 預先通知鄰居;主動修改設計以解決問題 |
| **樹木移除** | 建築占地面積內有受保護的樹木;需要樹木學家報告 + 更換 | 在設計增建占地面積前調查樹木 |

### 主要法律

- **分區法規 (Zoning code)**:退縮距離、建蔽率、FAR、高度限制控制您可以建造什麼
- **建築法規 (CBC)**:結構、防火、能源(Title 24)要求
- **設計審查 (Design review)**:一些城市對可見的增建要求;增加 1–3 個月
- 單戶增建**無 CEQA**(類別豁免 Class 1 或 3)
- **SB 330 費用鎖定**:如果為現有財產申請多單元增建,請儘早鎖定費用

## 定製住宅建造 (Custom Home Build——新建)

### 這是什麼?

在空地上建造**全新房屋**或拆除現有房屋並建造新房。您從頭開始——設計、地基、框架,一切。

### 何時使用

- 您購買或擁有的空地
- 拆除重建(現有房屋無法修復或相對地塊潛力太小)
- 您想要根據自己需求定製的設計

### 逐步流程

```text
步驟 1:場地盡職調查 (2–4 週)
    在購買之前(或設計之前):
      □ 分區:您可以建造什麼?(密度、高度、退縮距離、FAR)
      □ 產權報告:有任何地役權、留置權、契約限制嗎?
      □ 地質技術:土壤類型、承載能力、液化風險?
      □ 測量:地塊尺寸、地形、現有公用設施
      □ 環境:如果有商業歷史需要 Phase I ESA
      □ 樹木:受保護的樹木?Heritage oaks?
      □ 洪水區/火災區?(FEMA、CAL FIRE 地圖)
    │
步驟 2:設計 (8–16 週)
    建築師 → 示意設計 → 設計發展 → 施工文件
    結構工程師(必需)
    土木工程師(如果需要平整、排水或雨水管理)
    Title 24 能源計算
    │
步驟 3:規劃審查 (4–16 週)
    大多數城市要求對新住宅進行規劃/設計審查
    一些城市(Palo Alto、Los Altos)出名地慢
    其他僅需行政審查(2–4 週)
    │
步驟 4:建築許可證 (6–16 週)
    提交完整的施工文件
    平面審查:建築、結構、防火、MEP、Title 24
    更正:通常 1–2 輪(增加 2–6 週)
    │
步驟 5:支付費用
    學校費:$4–$5/SF
    影響費:因城市而異(新建 $10K–$50K+)
    公用設施連接費:$5K–$20K
    許可費:$5K–$20K
    │
步驟 6:施工 (8–14 個月)
    拆除(如果拆除重建) → 平整/地基 → 框架 →
    屋頂 → 窗戶/門 → 粗 MEP → 隔熱 →
    石膏板 → 精細木工 → 固定裝置 → 景觀 →
    最終檢查
    │
步驟 7:入住許可證 (Certificate of Occupancy)
    所有最終檢查通過
    公用設施連接啟用
    入住。
```

### 時間表

| 方法 | 盡職調查 + 設計 | 許可證 | 施工 | 總計 |
| --- | --- | --- | --- | --- |
| **生產建商**(預設計平面圖) | 1–2 個月 | 2–4 個月 | 8–12 個月 | **12–18 個月** |
| **定製建造**(建築師設計) | 3–6 個月 | 3–6 個月 | 10–14 個月 | **16–26 個月** |
| **拆除重建** | 3–6 個月 | 3–6 個月 + 拆除許可證 | 10–14 個月 | **16–26 個月** |

### 成本範圍(灣區)

| 品質等級 | 每平方英尺成本 | 2,500 SF 住宅 |
| --- | --- | --- |
| **標準** | $400–$600/SF | $1.0M–$1.5M |
| **高端** | $600–$900/SF | $1.5M–$2.25M |
| **奢華** | $900–$1,200+/SF | $2.25M–$3.0M+ |

*不包括土地成本。灣區地塊:$500K–$3M+ 取決於位置。*

### 什麼會終止專案

| 陷阱 | 會發生什麼 | 如何避免 |
| --- | --- | --- |
| **糟糕的地質** | 液化、膨脹性黏土或高地下水位 = $50K–$200K 額外地基成本 | **始終**在購買地塊前獲得地質技術報告 |
| **隱藏的地役權** | 公用設施或通行地役權侵入可建造區域 | 在成交**之前**獲得初步產權報告 + ALTA 測量 |
| **鄰居 CEQA 挑戰** | 鄰居對您的新住宅提出 CEQA 反對(在 Palo Alto 罕見但真實) | 使用 Class 3 或 Class 32 類別豁免;記錄合規性 |
| **緩慢的城市** | 一些灣區城市僅平面審查就需要 6–12 個月 | 在選擇地塊**之前**研究城市處理時間 |
| **預算超支** | 定製住宅平均比初始預算超出 15–25% | 獲得固定價格合約或詳細的 GMP(保證最高價格) |

### 主要法律

- **分區法規 (Zoning code)**:控制一切——密度、高度、退縮距離、FAR、停車
- **建築法規 (CBC)**:結構、防火、能源、無障礙
- **Title 24 2025**:熱泵、電動車充電、電池儲能(2026+ 年申請的許可證)
- **CEQA**:Class 3(新建 ≤3 單元)或 Class 32(填充)豁免——通常無 CEQA 問題
- **Chapter 7A**:如果在火災危險區,需要 WUI 防火加固
- **SB 330**:如果您正在建造住房,請提交初步申請以鎖定費用

> **專業建議——TJH 模式**:Thomas James Homes(第 4 章 Frontier Profile)每年使用**預設計平面圖庫**適應每個地塊建造 350+ 棟定製住宅。這將設計時間從 4–6 個月縮短到 2–4 週並保證定價。如果您正在進行拆除重建,請研究 TJH 手冊(附錄 F,§F.1)。

## ADU (輔助住宅單元——Accessory Dwelling Unit)

### 這是什麼?

在您現有住宅地塊上的**第二個較小的住宅**——後院小屋、車庫改建或您現有房屋內的單元。加州法律保證您建造一個的權利。任何城市都不能說不。

### 何時使用

- 您想從財產中獲得租金收入
- 您需要為年邁父母或成年子女提供住所
- 您想增加財產價值(HCD 認證的 ADU 在灣區增加 $400K–$700K)
- 您想將 ADU 作為獨立公寓出售(AB 1033——如果您的城市已選擇加入)

### 四種 ADU 類型

| 類型 | 這是什麼 | 最大尺寸 | 退縮距離 | 關鍵規則 |
| --- | --- | --- | --- | --- |
| **獨立式 ADU (Detached ADU)** | 您庭院中的獨立結構 | 1,200 SF | 側面/後方 4 英尺 | 無論建蔽率如何,城市必須至少允許 800 SF |
| **附建式 ADU (Attached ADU)** | 您現有房屋的增建 | 1,200 SF 或主屋的 50% | 側面/後方 4 英尺 | 如果車庫改建無需更換停車位 |
| **JADU (Junior ADU)** | 您現有房屋內的房間 | 500 SF | 無 | 效率廚房可以;需要業主居住 |
| **車庫改建 (Garage Conversion)** | 將現有車庫改建為生活空間 | 無尺寸限制 | 無需退縮距離 | 無需更換停車位 |

### 逐步流程

```text
步驟 1:檢查您的地塊 (1 天)
    您的地塊是否分區為住宅或混合用途? → 您可以建造 ADU。就是這樣。
    州法律保證。任何城市都不能拒絕您。
    │
步驟 2:選擇您的路徑 (1 天)
    ┌─────────────────────────────────────────────┐
    │  選項 A:預先批准的平面圖(最快)              │
    │  檢查您所在城市的預先批准 ADU 目錄           │
    │  選擇平面圖 → 30 天許可證(AB 1332)          │
    │  成本:$0–$5K 平面圖                         │
    ├─────────────────────────────────────────────┤
    │  選項 B:預製/模組化(快速 + 簡單)            │
    │  選擇供應商:AIAadu、Abodu、Villa 等          │
    │  供應商處理設計 + 許可證 + 建造              │
    │  成本:$120K–$450K 全包(交鑰匙)              │
    ├─────────────────────────────────────────────┤
    │  選項 C:定製設計(最靈活)                    │
    │  聘請建築師 → 定製平面圖                    │
    │  60 天許可證(州法律)                        │
    │  成本:$8K–$20K 設計 + 施工                  │
    └─────────────────────────────────────────────┘
    │
步驟 3:許可證 (30–60 天)
    向建築部門提交平面圖
    預先批准的平面圖:城市必須在 30 天內處理
    定製平面圖:城市必須在 60 天內處理
    如果城市錯過截止日期:許可證被視為批准
    │
步驟 4:施工
    預製/模組化:1 天安裝 + 2–4 週場地工程
    現場建造:3–6 個月
    │
步驟 5:最終檢查 + 入住
    檢查員簽核
    開始收取租金(或您的家人搬入)
    │
    可選步驟 6:作為公寓出售(AB 1033)
    如果您的城市已採用 AB 1033(San Jose、San Diego 等)
    將財產轉換為公寓 → 單獨出售 ADU
```

### 時間表

| 路徑 | 設計 | 許可證 | 施工 | 總計 |
| --- | --- | --- | --- | --- |
| **預先批准的平面圖 + 預製** | 0 週 | 4 週 | 4–8 週 | **2–4 個月** |
| **預製(交鑰匙供應商)** | 0–2 週 | 4–8 週 | 8–12 週 | **3–6 個月** |
| **定製設計 + 現場建造** | 4–8 週 | 8–12 週 | 16–24 週 | **7–12 個月** |

### 成本範圍(灣區)

| 類型 | 成本範圍 | 典型月租金 |
| --- | --- | --- |
| **車庫改建** | $80K–$180K | $2,500–$3,500 |
| **預製 ADU(套房)** | $120K–$230K | $2,000–$3,000 |
| **預製 ADU(1BR)** | $180K–$300K | $2,800–$3,800 |
| **預製 ADU(2BR)** | $240K–$450K | $3,200–$4,500 |
| **定製現場建造** | $250K–$500K | $3,000–$4,500 |

### 什麼會終止專案

| 陷阱 | 會發生什麼 | 如何避免 |
| --- | --- | --- |
| **城市說不** | 非法。州法律保證您建造 ADU 的權利。 | 引用 Government Code §65852.2。如果城市阻撓,向 HCD 報告。 |
| **未經 HCD 認證(預製)** | 您的 ADU 被歸類為個人財產——像汽車一樣貶值 | **始終**在與預製供應商簽約前驗證 HCD 認證 |
| **下水道容量** | 舊的下水道管線無法處理第二個單元;城市要求升級($10K–$30K) | 為下水道管線檢查和可能的更換編列預算 |
| **現場無法使用起重機** | 預製模組無法透過起重機吊到房屋後院 | 在簽約**之前**與您的預製供應商驗證起重機範圍和通道 |
| **公用設施連接延遲** | PG&E、水或下水道連接需要 2–4 個月 | **同時**申請公用設施連接和許可證——不要等待 |

### 主要法律

| 法律 | 為您做什麼 |
| --- | --- |
| **Gov. Code §65852.2** | 您建造 ADU 的權利——依權利、行政性、無 CEQA |
| **AB 1332 / AB 434** | 預先批准的平面圖 + 30 天許可證時間表 |
| **AB 976** | 無需業主居住(永久)——出租兩個單元 |
| **AB 1033** | 將您的 ADU 作為獨立公寓出售(如果城市選擇加入) |
| **SB 13** | 750 SF 以下的 ADU 無影響費 |
| **SB 897** | 獨立式 ADU 高度 16–18 英尺 |
| **AB 2533** | 合法化 2020 年前建造的未經許可的 ADU |

## 細分 (Subdivision——將土地分割為地塊)

### 這是什麼?

您取**一塊大地塊**並將其分為**兩個或更多較小的地塊**,每個地塊都有自己的法律描述、APN(Assessor's Parcel Number,評估員地塊號)和可建造區域。然後每個新地塊可以獨立出售或開發。

### 何時使用

- 您擁有一塊大地塊並想出售部分
- 您想在獨立地塊上建造多個住宅以單獨出售
- 您正在創建聯排別墅或小地塊開發
- 您想通過向定製建商出售個別地塊來最大化土地價值

### 細分類型

| 類型 | 創建的地塊 | 批准 | 需要地圖 | 典型時間表 |
| --- | --- | --- | --- | --- |
| **地界線調整 (Lot Line Adjustment)** | 0(僅移動 2 個現有地塊之間的邊界) | 行政性 | 無地圖 | 2–4 個月 |
| **地塊圖 (Parcel Map)** | 2–4 個新地塊 | Planning Commission | 地塊圖(記錄) | 6–12 個月 |
| **暫定區段圖 (Tentative Tract Map)** | 5+ 個新地塊 | Planning Commission + 通常市議會 | 區段圖(記錄) | 12–24 個月 |
| **既得暫定地圖 (Vesting Tentative Map)** | 5+ 個新地塊 | 與區段圖相同 | 相同 | 12–24 個月(但**鎖定費用**——見主要法律) |
| **SB 9 地塊分割 (SB 9 Lot Split)** | 從 1 個單戶地塊分為 2 個 | **行政性**(無聽證會) | 城市地塊分割地圖 | **2–4 個月** |

### 逐步流程(傳統地塊/區段圖)

```text
步驟 1:可行性 (2–4 週)
    □ 分區:地塊允許細分嗎?最小地塊面積?
    □ 總體規劃:細分是否與土地使用指定一致?
    □ 公用設施:城市能為額外地塊提供服務嗎?(下水道、水、雨水)
    □ 通道:每個新地塊能有合法的街道通道嗎?
    □ 產權:有任何契約限制禁止細分嗎?
    │
步驟 2:申請前會議 (1–2 週)
    與市規劃人員會面
    討論地塊布局、通道、公用設施要求
    獲取所需研究和申請材料清單
    │
步驟 3:設計暫定地圖 (4–8 週)
    土木工程師準備暫定地圖顯示:
      □ 擬議的地塊邊界
      □ 街道通道和車道
      □ 公用設施位置(下水道、水、雨水、電)
      □ 平整和排水
      □ 地役權
    │
步驟 4:CEQA 審查(變化)
    ≤4 個地塊:通常類別豁免(Class 15)
    5+ 個地塊:可能需要初步研究 → ND/MND
    大型專案:可能需要 EIR(12–24 個月)
    SB 684(MF 區≤10 個地塊):CEQA 豁免,行政性
    │
步驟 5:申請 + 公聽會 (4–12 週)
    提交暫定地圖 + CEQA 文件 + 申請
    人員審查 → 人員報告 → Planning Commission 聽證會
    施加批准條件(道路改善、公用設施、費用)
    │
步驟 6:最終地圖(批准後 4–8 週)
    持照測量師準備最終地圖
    向縣記錄員記錄
    每個地塊獲得自己的 APN
    │
步驟 7:改善 + 建造
    安裝所需的改善(道路、公用設施、人行道)
    為每個地塊申請建築許可證
    建造並出售。
```

### 時間表

| 類型 | 總計(設計到記錄地圖) |
| --- | --- |
| **地界線調整** | 2–4 個月 |
| **地塊圖(2–4 個地塊)** | 6–12 個月 |
| **區段圖(5–20 個地塊)** | 12–24 個月 |
| **SB 9 地塊分割** | **2–4 個月**(見下面第 6 節) |
| **SB 684(MF 區≤10 個地塊)** | **2–4 個月**(行政性,60 天批准) |

### 成本範圍

| 項目 | 成本範圍 |
| --- | --- |
| **土木工程師(暫定地圖)** | $15K–$50K |
| **持照測量師(最終地圖)** | $10K–$30K |
| **申請 + 申報費** | $5K–$25K |
| **CEQA(如果需要)** | $0(豁免)到 $200K+(EIR) |
| **所需改善**(道路、公用設施) | $50K–$500K+(高度可變) |
| **每個地塊的影響費** | 每個地塊 $20K–$100K(灣區) |

### 什麼會終止專案

| 陷阱 | 會發生什麼 | 如何避免 |
| --- | --- | --- |
| **最小地塊面積** | 城市要求最小 5,000 SF;您的地塊只能產生 2 個地塊而不是 3 個 | 在購買**之前**運行分區數學——地塊面積 ÷ 最小地塊面積 = 最大地塊數 |
| **CEQA 挑戰** | 鄰居或活動團體提出 CEQA 反對 | 如果符合條件,使用 SB 684(行政性,無 CEQA);或針對單戶地塊使用 SB 9 |
| **所需改善** | 城市批准條件為您未編列預算的 $200K 道路/公用設施工程 | 在承諾前獲得**書面**申請前會議條件 |
| **公用設施容量** | 下水道或供水系統達到容量;無新連接可用 | 在申請**之前**與供應商驗證公用設施容量 |
| **無通道** | 新地塊被陸地包圍——無合法街道通道 | 每個地塊必須有公共街道臨街或批准的通行地役權 |

### 主要法律

| 法律 | 做什麼 |
| --- | --- |
| **Subdivision Map Act**(Gov. Code §66410+) | 主框架——每個細分都遵循此法律 |
| **Vesting Tentative Map**(§66498.1+) | 在申請日期鎖定所有費用和標準——對灣區專案**至關重要** |
| **SB 9**(§66452.6) | 單戶地塊的行政性地塊分割——繞過整個傳統流程 |
| **SB 684**(§66499.40) | MF 分區填充地塊≤10 個地塊的行政性批准——60 天批准 |
| **SB 330**(§65941.1) | 初步申請在您甚至設計專案之前鎖定費用 |
| **CEQA Class 15** | 小型土地分割的類別豁免(城市地區≤4 個地塊) |

> **交叉參考**:見第 7 章完整 Subdivision Map Act 框架、既得地圖策略和 SB 9 與傳統地塊圖比較。

## SB 9 (California HOME Act)——單戶地塊快速通道

### 這是什麼?

SB 9 讓您取**任何單戶分區地塊**並創建最多**4 個住宅單元**——**無公聽會、無 CEQA 和無自由裁量審查**。這是在加州單戶地塊上增加密度的最快、最便宜的方式。

### 兩個工具

| 工具 | 做什麼 | 結果 |
| --- | --- | --- |
| **SB 9 複式 (Duplex)** | 在您現有地塊上建造 2 個單元 | 1 個地塊 → 2 個單元 |
| **SB 9 地塊分割 (Lot Split)** | 將您的地塊分為 2 個地塊 | 1 個地塊 → 2 個地塊 |
| **兩者結合** | 分割 + 每個地塊上的複式 | 1 個地塊 → **4 個單元** |

### 逐步流程

```text
步驟 1:檢查資格 (1 天)
    您的地塊必須:
      □ 分區為單戶住宅
      □ 在城市地區(incorporated city 或 Census urbanized area)
      □ 不在:歷史街區、洪水區、火災危險嚴重區、
        優質農田、保護區或受保護棲息地
      □ 無現有契約限制阻止它
      □ 以前未根據 SB 9 分割過
    │
步驟 2:決定您的策略
    ┌─────────────────────────────────────────────────┐
    │  策略 A:僅複式(最簡單)                          │
    │  在您現有地塊上建造 2 個單元                     │
    │  無需地塊分割                                    │
    │  結果:2 個單元                                   │
    ├─────────────────────────────────────────────────┤
    │  策略 B:僅地塊分割                              │
    │  分為 2 個地塊(每個≥原始的 40%,                 │
    │  每個≥1,200 SF)                                  │
    │  在每個地塊上建造 1 個住宅                       │
    │  結果:2 個地塊上的 2 個住宅(可單獨出售)         │
    ├─────────────────────────────────────────────────┤
    │  策略 C:地塊分割 + 複式(最大)                   │
    │  分為 2 個地塊                                   │
    │  在每個地塊上建造複式                            │
    │  結果:2 個地塊上的 4 個單元                      │
    ├─────────────────────────────────────────────────┤
    │  策略 D:地塊分割 + 複式 + ADU + JADU            │
    │  (Sunnyvale 8 單元組合——第 4 章,案例 4)        │
    │  分割 → 每個上的複式 → 每個上的 ADU →           │
    │  每個複式中的 JADU                               │
    │  結果:最多 8 個單元                              │
    └─────────────────────────────────────────────────┘
    │
步驟 3:設計 (2–6 週)
    建築師為複式和/或地塊分割地圖準備平面圖
    最小退縮距離:距離側面和後方地界線 4 英尺
    城市必須允許每個單元至少 800 SF
    │
步驟 4:提交申請 (1 天)
    行政性申請——無 CUP、無 variance、無聽證會
    市規劃部門行政處理
    │
步驟 5:城市審查(最多 60 天)
    城市僅應用客觀標準——無主觀設計審查
    城市不能將密度降低到每個地塊 2 個單元以下
    如果城市在 60 天內不處理 → 視為批准
    │
步驟 6:許可證 + 建造
    申請建築許可證
    建造單元
    │
步驟 7:入住
    對於地塊分割:3 年需要業主居住(其中一個單元)
    任何單元都不得短期租賃(Airbnb)——需要契約限制
    3 年後:業主居住要求到期
```

### 時間表

| 策略 | 設計 | 批准 | 施工 | 總計 |
| --- | --- | --- | --- | --- |
| **僅複式** | 2–4 週 | 60 天 | 4–8 個月 | **6–10 個月** |
| **僅地塊分割** | 2–4 週 | 60 天 | N/A(出售地塊) | **3–4 個月** |
| **地塊分割 + 複式** | 4–8 週 | 60 天 | 6–10 個月 | **8–14 個月** |
| **完整組合(8 個單元)** | 6–12 週 | 60 天(每個組件) | 8–14 個月 | **10–18 個月** |

### 成本範圍(灣區)

| 項目 | 成本 |
| --- | --- |
| **SB 9 地塊分割申請** | $1K–$3K(城市費用) |
| **測量師(地塊分割地圖)** | $5K–$10K |
| **建築平面圖(複式)** | $10K–$25K |
| **施工(每個單元,800–1,200 SF)** | $250K–$500K |
| **地塊分割 + 2 個複式(4 個單元)總計** | $1.2M–$2.5M(不包括土地) |

### 使 SB 9 有效的數學

**範例:Sunnyvale 8,000 SF 地塊(第 4 章,案例 4)**

| 情境 | 單元 | 每單元土地成本 | 月租金 | 年收入 |
| --- | --- | --- | --- | --- |
| **保持單戶** | 1 | $1.5M(整個地塊) | $4,500 | $54K |
| **SB 9 複式** | 2 | $750K | $3,800 × 2 | $91K |
| **SB 9 分割 + 複式** | 4 | $375K | $3,200 × 4 | $154K |
| **完整組合(+ ADU + JADU)** | 8 | $188K | 平均 $2,500 × 8 | $240K |

每單元土地成本從 $1.5M 降至 $188K——**降低 87.5%**。這就是為什麼 SB 9 改變了整個財務模型。

### 什麼會終止專案

| 陷阱 | 會發生什麼 | 如何避免 |
| --- | --- | --- |
| **火災危險區** | SB 9 不適用於 VHFHSZ 或 SHFHSZ | 在規劃前檢查 CAL FIRE FHSZ 地圖 |
| **歷史街區** | SB 9 不適用於當地指定的歷史街區 | 向市規劃部門驗證 |
| **城市阻撓** | 城市施加限制性「客觀」標準,有效阻止 2 個單元 | 向 HCD 報告——州積極執行 SB 9(見 Woodside、Menlo Park 案例) |
| **業主居住(地塊分割)** | 地塊分割後您必須在一個單元中居住 3 年 | 將此納入您的投資時間表;不適用於僅複式策略 |
| **不允許 STR** | 契約限制禁止所有 SB 9 單元的短期租賃 | 僅計劃長期租賃——無 Airbnb |
| **同一地塊上以前的 SB 9** | 每個原始地塊只能使用 SB 9 地塊分割一次 | 檢查地塊歷史 |

### 主要法律

| 法律 | 引用 | 做什麼 |
| --- | --- | --- |
| **SB 9 Duplex** | Gov. Code §65852.21 | 任何單戶地塊上的 2 個單元——行政性 |
| **SB 9 Lot Split** | Gov. Code §66452.6 | 將單戶地塊分為 2——行政性 |
| **結合** | 兩個部分 | 總共 4 個單元——無 CEQA、無聽證會 |
| **ADU 疊加** | Gov. Code §65852.2 | 向每個 SB 9 地塊添加 ADU + JADU,最多 8 個單元 |
| **AB 1033** | Gov. Code §65852.2 | 將 ADU 作為公寓出售(城市選擇加入的地方) |

> **交叉參考**:見第 3 章(§3.2)SB 9 法定細節。見第 4 章(案例 4)Sunnyvale 8 單元組合案例研究。見第 7 章(§7.4,案例 2)SB 9 地塊分割與傳統地塊圖比較。

## 新手 5 分鐘測試:我應該做這個專案嗎?

在花一分錢在設計上之前,運行這個快速健全性檢查:

### 1. 分區檢查(5 分鐘)
前往您所在城市的 GIS/分區地圖(通常免費線上)。查找您的地塊。寫下:
- **區域**:_______________
- **允許用途**:_______________
- **最大 FAR**:_______________
- **最大建蔽率**:_______________
- **最大高度**:_______________
- **退縮距離**(前方/側面/後方):_______________

如果您的專案不符合這些數字,您將需要 variance(困難)、區域變更(更困難)或不同的專案。

### 2. 預算檢查(5 分鐘)
您能負擔**總成本**包括:
- [ ] 設計費(建築師、工程師、測量師)
- [ ] 許可費(城市 + 學校 + 影響)
- [ ] 施工成本(至少獲得 2 個承包商報價)
- [ ] 20% 應急費用(新手不可協商)
- [ ] 施工期間的持有成本(抵押貸款、稅收、保險)
- [ ] 公用設施連接

### 3. 時間表檢查(5 分鐘)
您能處理從開始到結束的**總時間**嗎?記住:
- 許可證總是比您想像的時間長
- 施工總是比承包商說的時間長
- 在您被告知的任何時間表上增加 30%

### 4. 退出策略檢查(5 分鐘)
您如何從這個專案中賺錢(或獲得價值)?
- [ ] **住在裡面**——價值在於生活質量
- [ ] **出租**——計算數字:月租金 × 12 ÷ 總成本 = 總收益(目標:8%+)
- [ ] **出售**——可比銷售(comps)支持您預計的售價?
- [ ] **再融資**——新的評估價值足夠高以提取您的投資?

如果所有四項檢查都通過,則繼續進行上述相關部分的步驟 1。

## 新手詞彙表

| 術語 | 白話 |
| --- | --- |
| **APN** | Assessor's Parcel Number——您的地塊在縣的 ID 號碼 |
| **By-right** | 如果您符合規則,城市必須批准——無投票、無聽證會 |
| **CBC** | California Building Code——建築物如何建造的規則 |
| **CEQA** | 環境審查,可能為您的專案增加數月或數年 |
| **CUP** | Conditional Use Permit——來自城市的特殊許可(需要聽證會) |
| **FAR** | Floor Area Ratio——總建築 SF ÷ 地塊 SF(控制您能建造多大) |
| **GMP** | Guaranteed Maximum Price——承包商的最高價格(超支由他們承擔) |
| **HCD** | Housing & Community Development——執行住房法的州機構 |
| **LOC / Lot Coverage** | 建蔽率——建築物覆蓋的地塊百分比,城市限制此項 |
| **MEP** | Mechanical, Electrical, Plumbing——建築物的內部系統 |
| **Ministerial** | 由工作人員批准,而非民選官員——最快途徑 |
| **MND** | Mitigated Negative Declaration——中級 CEQA 審查 |
| **Phase I ESA** | Environmental Site Assessment——檢查污染 |
| **Plan Check** | 城市審查您的施工平面圖是否符合規範 |
| **Setback** | 退縮距離——您的建築物與地界線之間的要求距離 |
| **Title 24** | 加州能源法規——要求效率、太陽能、熱泵等 |
| **Variance** | 打破分區規則的許可——很難獲得,不要指望它 |
| **Zoning** | 分區法規——控制您可以在地塊上建造什麼的規則 |
