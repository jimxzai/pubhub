# 附錄 C: 模板與清單

---

## 概述

這些模板為加州開發的關鍵活動提供標準化框架 — 從初步地點篩選到權利取得與建設。請根據您的具體項目和管轄區域進行調整。

---

   □ Class 1 — 現有設施 (輕微變更)
   □ Class 3 — 新建築, 小型結構 (≤6單位於城市)
   □ Class 32 — Infill 開發 (城市, 符合 GP/zoning,
     無重大影響, 充足公用設施)
   □ 常識豁免 (§15061(b)(3))
   □ 其他豁免: _______________________________
   → 如果是: 提交豁免通知。此處停止。
   → 例外檢查: 是否適用任何例外? (§15300.2)
     □ 位置 (敏感環境)
     □ 累積影響
     □ 重大影響 (異常情況)
     □ Scenic highway
     □ 危險廢物場址
     □ 歷史資源
     → 如果任何例外適用: 豁免不適用。繼續。

3. 初步研究 — 影響評估
   所有影響能否緩解至不顯著?
   □ 是 → Mitigated Negative Declaration (MND)
     時間線: 12–18個月
     成本: $50K–$150K
   □ 否 → 需要 Environmental Impact Report (EIR)
     時間線: 18–36個月
     成本: $200K–$1M+

4. EIR 類型 (如果需要)

   □ Program EIR (general plan, specific plan)
   □ Supplemental/Subsequent EIR (變更條件)
   □ Addendum (已批准項目的輕微修改)
   □ Tiered EIR (項目從 program EIR 分層)
