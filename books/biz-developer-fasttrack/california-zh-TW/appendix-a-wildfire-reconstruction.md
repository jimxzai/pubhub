# 附錄 A：加州野火重建 — 案例與經驗教訓

**Pacific Palisades | Santa Monica | Paradise | Santa Rosa**

---

## 概述

加州毀滅性的野火在全州創造了**緊急重建需求**。野火重建在獨特的監管框架下運作——簡化許可、在某些情況下放寬分區規定，以及對建築勞動力和材料的大量需求。對於開發商而言，野火區既帶來機會（快速許可、模組化需求），也帶來挑戰（污染、基礎設施缺口、保險危機）。

---

## A.1 Los Angeles — Pacific Palisades 與 Eaton 野火（2025 年 1 月）

### 事件經過

Palisades 野火在 2025 年 1 月摧毀了 Pacific Palisades、Malibu 及周邊地區約**5,300+ 棟建築**。Eaton 野火摧毀了 Altadena 和 Pasadena 約**7,000+ 棟建築**。這兩場野火合計造成估計**$250B+ 的損失**——美國歷史上最昂貴的自然災害。

### 監管應對

| 行動 | 權責機構 | 效果 |
| --- | --- | --- |
| **行政命令 N-4-25** | 州長 Newsom | 暫停 CEQA 用於重建；豁免許可費用；簡化圖紙審查 |
| **緊急許可** | LA 市/縣 | 同地基重建獲得快速圖紙審查的權利許可 |
| **FEMA 清理碎片** | 聯邦/FEMA | 政府資助在重建前清除有害碎片 |
| **SB 9/ADU 疊加** | 州法律 | 重建可根據 SB 9 和 ADU 法律包含額外單元 |
| **保險緊急狀態** | CDI | 暫停火災區保單取消 |

### 重建挑戰

| 挑戰 | 影響 | 對開發商的意義 |
| --- | --- | --- |
| **有害碎片** | 灰燼含有鉛、石棉、重金屬、家用化學品 | 重建前需要 Phase II 級別的環境清除 |
| **土壤污染** | 火災碎片污染表土；某些場地需要修復 | 每個地塊需預算 $10K–$50K 用於土壤測試和修復 |
| **基礎設施破壞** | 水管、下水道、燃氣管線、電網受損 | 重建取決於公用事業恢復時間表 |
| **勞動力短缺** | 5,000+ 套房屋同時競爭相同的承包商 | 建築成本在重建高峰期上漲 20–40% |
| **保險危機** | 許多房主保險不足；保險公司離開 CA 市場 | 重建成本可能超過保險賠付 |
| **海岸法** | Palisades 位於海岸區；海岸委員會管轄 | 即使有 CEQA 豁免，仍可能需要海岸許可 |

### 模組化機會

Palisades/Eaton 重建代表了加州**有史以來最大的**模組化和預製建築機會：

- **5,000+ 套房屋**需要在壓縮的時間表內重建
- 傳統建築勞動力嚴重受限
- 工廠建造的房屋可以繞過現場勞動力短缺
- 面板式和體積式系統可在 4–6 個月內交付房屋，而傳統方式需要 12–18 個月
- 州長的行政命令明確鼓勵預製/模組化方法

### 經驗教訓

> **教訓 1**：野火重建是**速度遊戲**——部署模組化/預製的房主比等待傳統建築的房主提前 6–12 個月回到家中。將模組化定位為勞動力稀缺的解決方案，而不僅僅是成本節約。

> **教訓 2**：即使有 CEQA 豁免，**海岸法仍適用於** Pacific Palisades。開發商必須就任何改變建築地基、增加高度或修改地塊覆蓋的重建事項應對海岸委員會的要求。CEQA 豁免不會取代海岸法管轄權。

> **教訓 3**：**火災碎片造成的土壤污染**是野火重建的隱藏成本。每個地塊在基礎工程前都需要環境清除。在您的財務模型中為此預算——這不是可選項。

## A.2 Paradise（Camp Fire — 2018 年 11 月）

### 事件經過

Camp Fire 摧毀了 Paradise 及周邊地區的**18,804 棟建築**——當時加州歷史上最致命、最具破壞性的野火。Paradise 鎮（人口 26,000）基本上被抹去了。

### 重建時間表（7+ 年）

| 階段 | 時間表 | 狀態 |
| --- | --- | --- |
| **清理碎片** | 2019–2020 | 完成（FEMA/CalOES 資助） |
| **基礎設施恢復** | 2019–2023 | 供水系統重建；部分道路仍在等待 |
| **許可申請** | 2019–至今 | 約 4,500 份許可已發放（共 11,000+ 套被毀房屋） |
| **房屋重建** | 2019–2026 | 約 3,500 套完成（火災前住房存量的不到 20%） |
| **人口恢復** | 2019–2026 | 約 10,000 名居民返回（火災前的 40%） |

### 為何重建如此緩慢

1. **保險缺口**：平均保險賠付僅覆蓋 60–70% 的重建成本；許多居民無法負擔差額
2. **基礎設施瓶頸**：在發放建築許可之前，供水系統必須完全重建
3. **勞動力稀缺**：偏遠位置難以吸引建築隊伍
4. **承包商欺詐**：多起針對脆弱房主的承包商欺詐案件
5. **規範升級**：按現行規範重建（WUI 防火標準、Title 24 能源）比原始建築昂貴得多
6. **情感/財務疲憊**：許多居民選擇搬遷而不是重建

### Paradise 的模組化

幾家模組化/製造房屋公司進入了 Paradise 市場。工廠建造的房屋是**最快的入住途徑**——一些房主在 6 個月內就住進了模組化房屋，而鄰居等待傳統建築需要 2 年以上。然而，模組化房屋面臨一些居民的污名化，他們將其與「臨時」住房聯繫起來。

### 經驗教訓

> **教訓 1**：**基礎設施恢復是門檻因素**，而不是許可。在 Paradise，建築許可在數月內就可獲得，但供水系統花了 3 年以上才完全恢復。開發商在承諾野火重建項目前必須評估基礎設施時間表。

> **教訓 2**：**保險缺口**是重建的最大障礙。許多房主面臨實際重建成本 30–40% 的短缺。能夠提供成本效益解決方案（模組化、面板化、標準化設計）的開發商將佔領市場。高端定製房屋不是這個市場所需要的。

> **教訓 3**：**社區規模**（數千套房屋）的野火重建需要 7–10+ 年。Paradise 在 7 年後仍不到 20% 的重建。將這一現實納入任何開發戰略——需求是真實的，但吸收緩慢。

## A.3 Santa Rosa（Tubbs Fire — 2017 年 10 月）

### 事件經過

Tubbs Fire 摧毀了 Santa Rosa 約**5,300 棟建築**，包括 **Coffey Park** 和 **Fountaingrove** 社區。這些是成熟的郊區社區，房屋價值 $400K–$1.5M。

### 重建：成功案例

Santa Rosa 的重建是加州歷史上**最成功的**大規模野火重建：

| 指標 | 結果 |
| --- | --- |
| **發放的許可** | 約 3,800 份（共約 5,300 套被毀） |
| **重建的房屋** | 到 2024 年約 3,200 套完成 |
| **平均重建時間** | 每套房屋 18–24 個月 |
| **模組化/預製份額** | 約 15% 的重建 |
| **人口恢復** | 約火災前水平的 85% |

### 為何 Santa Rosa 成功而 Paradise 掙扎

| 因素 | Santa Rosa | Paradise |
| --- | --- | --- |
| **位置** | 城市/郊區；靠近 SF Bay Area 勞動力市場 | 農村；距 Sacramento 90 分鐘 |
| **基礎設施** | 基本完好（地下公用事業倖存） | 完全摧毀 |
| **保險覆蓋** | 更高的保單限額；房主保險更好 | 許多保險不足 |
| **房產價值** | 更高價值證明重建成本合理 | 較低價值使某些重建不經濟 |
| **市政應對** | 快速許可；專門的重建中心 | 小鎮資源有限 |
| **勞動力獲取** | Bay Area 建築勞動力池 | 偏遠；勞動力必須進口 |

### Coffey Park — 大規模重建模式

Coffey Park（1,500+ 套房屋被毀）主要作為**大規模住宅**重建，由批量購買燒毀地塊的生產建商完成。這種方法提供了：
- **標準化設計**，將圖紙審查時間縮短至 2 週
- **規模經濟**，在採購和分包方面
- **更快許可**，因為重複計劃只需審查一次
- **更低的每平方英尺成本**，相比定製重建

幾家生產建商使用**面板式建築**用於牆壁和屋頂組件，將框架時間從每套房屋 3 週減少到 1 週。

### Fountaingrove — 定製重建挑戰

Fountaingrove（較高價值房屋，$800K–$1.5M）面臨更長的重建時間表，因為：
- 定製房屋需要單獨的建築設計和圖紙審查
- 陡峭的山坡場地需要新的岩土報告和擋土牆
- WUI 防火硬化要求（A 級屋頂、防燼通風口、不可燃甲板）每套房屋增加 $40K–$80K
- 一些地塊存在通道問題，使建築分期複雜化

### 經驗教訓

> **教訓 1**：**大規模生產建築**是社區恢復的最快途徑。Coffey Park 的大規模重建模式——標準化設計、批量地塊收購、重複圖紙審查——比定製重建每套房屋快 6–12 個月。能夠提供標準化設計和有限定製選項的開發商將獲得野火重建需求的最大份額。

> **教訓 2**：**WUI（野地-城市界面）防火硬化**現在對所有火災區的重建都是強制性的。每套房屋預算 $40K–$80K 用於：A 級屋頂、防燼通風口、鋼化玻璃窗、不可燃 5 英尺防禦區、防火壁板。這些要求與原始建築相比顯著增加了重建成本。

> **教訓 3**：Santa Rosa 成功是因為它擁有 Paradise 缺乏的**城市優勢**：完好的基礎設施、勞動力的接近性、更高的房產價值，以及一個投入資源用於快速許可的市政府。在評估野火重建機會時，在承諾前評估這些結構性因素——並非所有火災區都提供相同的重建經濟性。

## A.4 野火重建的監管框架

### 快速許可

| 條款 | 作用 | 限制 |
| --- | --- | --- |
| **州長行政命令** | 暫停 CEQA，豁免費用，簡化審查 | 臨時；通常持續 1–3 年 |
| **同地基重建** | 按權利重建至相同尺寸/位置 | 如果地塊已改變（侵蝕、碎片流）可能不適用 |
| **GC §65852.25** | 災區快速 ADU 許可 | 鼓勵重建期間增加密度 |
| **SB 9 疊加** | 重建期間可進行地塊分割/雙拼 | 可將前單戶地塊增加至 2–4 單元 |
| **WUI 防火標準** | CBC Chapter 7A 要求 | 強制性；不能豁免 |

### 環境考慮

- **碎片清除**：FEMA 資助在重建前清除有害材料
- **土壤測試**：火災後地塊可能有升高的鉛、砷、石棉水平
- **侵蝕/碎片流**：燒毀的山坡在火災後 3–5 年內易發生泥石流
- **雨水**：火災後徑流攜帶污染物；可能需要增強雨水處理
- **生物**：火災可能摧毀受保護棲息地，觸發重建的生物資源審查

## A.5 野火重建的模組化建築

### 為何模組化適合

| 優勢 | 野火背景 |
| --- | --- |
| **速度** | 房屋在 4–6 個月內交付，而傳統方式需要 12–18 個月 |
| **勞動力獨立** | 工廠工人不與現場勞動力池競爭 |
| **質量控制** | 工廠環境中一致的防火硬化標準 |
| **成本可預測性** | 固定價格合同降低房主的財務風險 |
| **可擴展性** | 工廠每月可生產數十套房屋 |

### 火災區的模組化挑戰

| 挑戰 | 緩解措施 |
| --- | --- |
| **場地通道** | 火災受損道路可能無法容納模組運輸；驗證路線 |
| **基礎準備** | 碎片清除和土壤修復必須在基礎澆築前完成 |
| **公用事業連接** | 必須與公用事業恢復時間表協調 |
| **社區認知** | 一些居民將模組化與「臨時」聯繫；使用優質飾面和營銷 |
| **WUI 合規** | 模組必須以 CBC Chapter 7A 防火硬化特性製造 |

## A.6 前沿開發商檔案：Thomas James Homes (TJH) — Palisades 重建

**Thomas James Homes** (tjh.com) 是 Pacific Palisades 野火重建中最重要的生產規模參與者。TJH 擁有**每年 350+ 套房屋的生產量**，在加州、太平洋西北、科羅拉多和亞利桑那州設有分部，已將其嵌入式拆除重建專業知識轉向大規模野火重建。

### TJH 商業模式

TJH 在一個獨特適合野火重建的利基市場運營：**成熟社區中的單地塊嵌入式拆除重建**。其核心模式——拆除老舊房屋並在同一地塊上建造新房屋——與火災後地塊重建在運營上相同。

| 指標 | 詳情 |
| --- | --- |
| **年度產量** | 350+ 套房屋 |
| **市場** | SoCal、NorCal（矽谷）、太平洋西北、科羅拉多、亞利桑那 |
| **Palisades 價格保證** | 合同簽訂時 $650/SF 固定價格 |
| **交付保證** | 從許可批准起 12 個月，否則 TJH 向房主支付每日罰金 |
| **建築債務融資** | $400M（來自 Ascent Developer Solutions） |
| **平面圖庫** | 約 50 個平面圖、3-4 個立面、4-5 個結構選項 |

### TJH Palisades 重建計劃

TJH 在 2025 年 1 月野火後完成了 **Pacific Palisades 的第一個展示重建房屋**。其計劃提供：

- **保證固定價格** $650/SF——消除火災倖存者的成本超支風險
- **12 個月完工保證**，從許可批准起——延遲向房主支付每日罰金
- **BUILD by TJH 計劃**——在房主現有地塊上的一站式設計-建造解決方案
- **預先設計的平面圖**——避免 6-12 個月的定製設計過程

### 為何 TJH 對野火重建很重要

傳統的定製重建模式在野火區的大規模應用中失敗，因為：

1. 定製建築師僅設計就需要 6-12 個月
2. 定製投標沒有價格確定性——在勞動力稀缺市場中成本不斷上升
3. 個人房主與分包商沒有談判籌碼

TJH 解決了所有三個問題：預先設計的計劃（無設計延遲）、保證定價（無成本上升）和生產規模採購（分包商籌碼）。其 $400M 建築債務融資使其擁有同時承擔數十個項目的資產負債表。

**經驗教訓**：TJH 的模式是**嵌入式/重建背景下生產房屋建造**的最接近形式。對於評估野火重建機會的開發商，TJH 的劇本——標準化設計、保證定價、生產規模運營——是值得研究的模式。其願意接受交付時間表上的罰金風險表明真正的運營信心。

## 交叉引用：野火重建的財務框架

野火重建需要標準開發財務模型的修改版本。對附錄 G 框架的關鍵調整：

| 標準財務模型項目 | 野火修改 | App G 級別 |
| --- | --- | --- |
| **土地收購** | 通常 $0（房主擁有地塊）或火災甩賣定價 | 級別 1：餐巾紙測試土地基礎根本不同 |
| **場地工作** | 每個地塊增加 $10K–$50K 用於碎片清除 + 土壤修復 | 級別 2：新項目 |
| **建築成本** | 在重建高峰勞動力短缺期間 +20–40% 通脹 | 級別 3：壓力測試——模擬勞動力成本上升 |
| **保險賠付** | 實際重建成本的 60–70%（Paradise 數據） | 級別 2：來源——保險缺口必須資助 |
| **時間表** | 模組化：4–6 個月 vs. 傳統：12–18 個月 | 級別 2：死亡點——模組化優勢在此最為明顯 |
| **WUI 防火硬化** | 每套房屋 $40K–$80K 強制性（CBC Chapter 7A） | 級別 2：添加到硬成本——不可協商 |

> **前沿建商連接**：TJH 在附錄 F（§F.1）中的完整案例研究提供了其 Palisades 重建模式的深度財務分析——$650/SF 保證定價、$400M 建築融資和罰金支持的 12 個月交付保證。對於評估大規模野火重建的開發商，這是基準商業模式。Aro 的體積預製方法（附錄 F，§F.2）為豪華市場提供了 $1,400/SF 的替代方案。這兩種模式都解決了勞動力稀缺約束，這是野火區的約束性限制。

## 關鍵要點

> 加州野火重建是一個由氣候變化、老化基礎設施和 WUI 開發模式驅動的**大規模、重複性市場**。成功的開發商是那些提供**速度、成本可預測性和防火硬化建築**的人——這些都是模組化建築的優勢。研究 Santa Rosa 的 Coffey Park 模式以獲得生產規模方法，並將模組化定位為限制每次重建的勞動力短缺的解決方案。
