# 第十三章:產權、測量與大地工程

**法規依據**:Insurance Code §12340+(產權保險);Public Resources Code §2690+(Seismic Hazards Mapping Act);California Building Code §1613(地震荷載)

---

## 執行摘要、關鍵要點與行動檢查清單

---

::: execsummary

產權、測量和大地工程調查構成了交易評估的三位一體,將有能力的開發商與那些被隱藏責任蒙蔽的開發商區分開來。本章教您批判性地閱讀和解釋每個專業領域,在資金投入前捕捉場地限制。產權報告揭示可能從根本上限制開發潛力的負擔和 CC&Rs;測量將這些限制物理地映射到您的場地規劃上;大地工程報告量化地下條件和基礎成本。灣區呈現特定挑戰:Santa Clara Valley 的沖積土壤具有液化危害,需要每單位 $8K-$15K 用於地面改良和增強基礎;靠近小溪的高地下水位觸發 $200K-$500K 的降水和防水措施;South Bay 的膨脹性黏土需要後張預力樓板和每單位 $6K-$15K 的基礎額外費用。真實案例研究——Menlo Park 的 $1.4M 液化成本衝擊、Palo Alto 的 14 個月地役權談判、San Jose 的膨脹性黏土驚喜——證明跳過盡職調查的成本遠高於 $3K-$6K 的 Phase I ESA。具有適當 Table A 項目的 ALTA 測量可以捕捉產權報告遺漏的地役權衝突。揭示液化風險、高地下水位或膨脹性土壤的大地工程調查應直接驅動設計決策(高架裙樓、地面改良、替代基礎),而不是在圖審期間才被發現。**核心原則**:將盡職調查結果視為設計輸入,而非需要繞過的障礙。

:::

---

::: keytakeaways

- **一起閱讀產權報告和測量,永遠不要分開**:產權報告通過文件參考描述地役權;ALTA 測量顯示它們實際位於地面的位置。Palo Alto 案例研究:產權中的 10 英尺 PG&E 地役權在測量上變成了 15 英尺對角線地役權,減少了 18% 的可建築範圍。在成交前始終與 Table A Items 6、11、17 交叉參考。

- **灣區液化風險在 Menlo Park 和 San Jose 不可妥協**:CGS Seismic Hazard Zone 地圖識別 Santa Clara Valley 沖積土壤中的標定液化區。預算每單位 $8K-$15K 用於地面改良(石柱、噴射灌漿)和增強基礎(筏板基礎、側向擴展設計)。Menlo Park 案例:由於跳過 GIS 篩查,成交後發現 $1.4M 額外成本(約每單位 $12K)。

- **大地工程承載力直接控制基礎經濟性**:1,500+ psf 的允許承載壓力允許獨立基礎;小於 1,500 psf 或高度變化的條件觸發深基礎(樁/墩),每單位增加 $3K-$8K。液化安全係數(FOS)小於 1.2 需要緩解。在大地工程階段而非基礎設計時解決這些問題。

- **高地下水位需要防水和降水規劃**:Santa Clara Valley 靠近小溪和濕地的區域可能有季節性高地下水位,距離地面 10 英尺以內。預算每單位 $3K-$8K 用於防水;如需地下室則需 $100K-$300K 用於臨時降水系統。ICF 地下室項目尤其脆弱——提前 6 個月規劃降水許可。

- **膨脹性黏土需要後張預力樓板和過度開挖**:South Bay(Willow Glen 等)經常有 CH 分類的膨脹性黏土(PI > 40)。預算後張預力樓板基礎(每單位 $4K-$6K 額外費用)加上 18 英吋過度開挖與進口非膨脹性回填(增加每單位 $5K-$10K 場地工作)。San Jose 案例:每單位 $6K 基礎成本 + 膨脹性土壤導致的 25% 場地平整成本增加。

- **Phase I ESA 在成交前是強制性的;REC 識別後 Phase II 成本可控**:Phase I 成本 $3K-$6K 且不可妥協——$5K 的 Phase I 可以節省 $5M 的修復責任。如果 Phase I 識別出 RECs,預算 Phase II 為 $15K-$100K 用於土壤/地下水採樣和測試。檢查 DTSC EnviroStor 和 SWRCB GeoTracker 1,000 英尺內的鄰近污染(不僅僅是現場)。

- **CREC(受控 REC)或 HREC(歷史 REC)與工程控制措施仍可能允許開發**:具有制度控制(契約通知、工程評估)的前加油站可能繼續進行;需要記錄控制措施保持有效,並將持續監測納入運營預算。與需要 Phase II 調查和可能修復的新 REC 區分。

- **大地工程結果驅動場地規劃設計決策**:Menlo Park 的液化 + 洪水區導致高架裙樓設計——大地工程 + 測量結果的直接設計結果。不要將大地工程視為需要克服的限制;將其視為塑造建築佔地面積、基礎類型和成本結構的設計參數。

:::

## 概述

技術盡職調查將有經驗的開發商與新手區分開來。流利閱讀產權報告、測量和大地工程報告的能力——並知道要尋找什麼——直接影響您準確評估交易和避免昂貴驚喜的能力。

## 第 A 部分:產權報告分析

### 13.1 初步產權報告

初步產權報告是您**首次了解物業負擔**的機會。Schedule B 中的每個項目(承保除外)都必須審查和分類。

| 例外類別 | 典型項目 | 風險級別 | 所需行動 |
| --- | --- | --- | --- |
| **CC&Rs** | 使用限制、建築控制、HOA 要求 | 中-高 | 審查與擬議開發的衝突 |
| **地役權** | 公用事業、通行、排水、景觀、保護 | 高 | 繪製在場地規劃上;檢查建築範圍衝突 |
| **留置權** | 稅務留置權、機械師留置權、判決留置權 | 關鍵 | 必須在成交前清除或使其次順位 |
| **契約限制** | 建築高度限制、使用限制、退縮要求 | 高 | 可能需要安靜產權訴訟或修改 |
| **礦產權** | 地下權利保留 | 低-中 | 評估對基礎和地下工程的影響 |

### 產權報告中的危險信號

- 測量期間發現的未記錄地役權
- 與計劃場地循環不一致的通行地役權
- 限制密度、高度或使用類型的 CC&Rs
- 前業主的未清留置權
- 對於酒店:商業使用、酒精銷售或 24/7 運營的限制

## 第 B 部分:ALTA/NSPS 土地產權測量

### 13.2 ALTA 測量

ALTA 測量是開發場地的**黃金標準**。它將邊界測量、地形信息和產權例外映射合併為單一文件。

### 必要的 Table A 可選項目

對於開發項目,指定這些 Table A 項目:

| 項目 | 內容 | 為何重要 |
| --- | --- | --- |
| **Item 1** | 界碑 | 確認邊界標記存在 |
| **Item 2** | 地址和地塊 ID | 與稅務評估記錄交叉參考 |
| **Item 3** | 洪水區 | FEMA 確定;保險要求 |
| **Item 4** | 總土地面積 | 確認用於密度計算的地塊大小 |
| **Item 6** | 退縮/建築線 | 定義可建築範圍 |
| **Item 8** | 實質特徵 | 邊界附近的樹木、牆壁、結構 |
| **Item 11** | 公用設施 | 所有公用設施連接和服務的位置 |
| **Item 13** | 鄰近業主姓名 | 識別外展的鄰居 |
| **Item 16** | 通行證據 | 確認合法和物理通行 |
| **Item 17** | 場外地役權 | 使物業受益的地役權 |
| **Item 19** | 地形等高線 | 場地平整設計基礎;排水分析 |

## 第 C 部分:大地工程

### 13.3 閱讀大地工程報告

大地工程報告將地下條件轉化為直接影響您的**基礎設計、施工方法和項目成本**的工程建議。

| 部分 | 要尋找什麼 | 危險信號 |
| --- | --- | --- |
| **土壤剖面** | USCS 分類、基岩深度、稠度 | 膨脹性黏土(CH)、有機土壤(OH/OL)、填方土壤 |
| **地下水** | 季節性高地下水位深度 | 基礎 10 英尺內的地下水位;棲息水 |
| **承載力** | 允許承載壓力(psf) | 小於 1,500 psf;整個場地的顯著變化 |
| **液化** | 安全係數、沉降估計 | FOS 小於 1.2;沉降大於 1 英吋;側向擴展 |
| **地震設計** | 場地等級、譜加速度值 | 場地等級 E 或 F;近斷層效應 |
| **基礎建議** | 類型、深度、鋼筋、排水 | 需要深基礎(樁/墩);廣泛的土壤改良 |
| **邊坡穩定性** | 挖填邊坡的安全係數 | 靜態 FOS 小於 1.5;地震 FOS 小於 1.0 |

### 酒店特定的大地工程關注點

| 關注點 | 為何對酒店重要 |
| --- | --- |
| **更大的佔地面積** | 整個場地土壤變化更大 |
| **地下停車場** | 降水、側向土壓力、防水 |
| **模組化堆疊荷載** | 模組承載點的更高點荷載 |
| **整個街區場地** | 可能需要地面改良計劃 |
| **模組連接** | 必須通過連接傳遞地震荷載 |

---

### 13.4 灣區大地工程考慮因素

San Francisco Bay Area 呈現特定挑戰:

- Santa Clara Valley 的**沖積土壤**容易液化
- 靠近小溪和前沼澤地的**高地下水位**
- Seismic Hazards Mapping Act 下的**標定液化危害區**
- **Menlo Park 和 San Jose** 的部分地區有記錄的液化風險

### 基礎影響

對於有地下室的項目(例如,ICF 地下室建造):
- 基於地下水深度的**防水要求**
- **施工期間的降水**——許可和臨時系統
- 地下室牆的**側向土壓力**設計
- 筏板基礎上的**靜水上舉**潛力

## 灣區案例研究與經驗教訓

### 案例 1:Menlo Park——前沼澤地的液化風險

一位開發商在 Menlo Park 靠近 Bayfront Expressway 收購了一塊 1.5 英畝的地塊,用於 120 單位的多戶項目。初步產權報告很清晰,ALTA 測量顯示沒有異常條件。然而,大地工程調查顯示該場地位於**前沼澤地,地面下 4 英尺處有季節性高地下水位**,設計地震的液化安全係數低於 1.0。

大地工程工程師建議:(1)使用石柱進行地面改良($800K),(2)筏板基礎而非獨立基礎($400K 額外費用),(3)為側向擴展力設計的結構樓板($200K 額外費用)。總大地工程驅動的成本增加:**$1.4M**——約每單位 $12K,這不在原始評估中。

**經驗教訓**:在 Menlo Park 和其他灣區鄰近地區,在簽訂合同前始終檢查 CGS Seismic Hazard Zone 地圖的液化情況。如果場地位於標定液化區,預算每單位 $8K-$15K 用於地面改良和增強基礎。此成本應反映在您的報價中,而不是在成交後發現。

---

### 案例 2:Palo Alto——公用設施地役權破壞建築範圍

一位開發商在 Palo Alto University Avenue 購買了一塊 12,000 平方英尺的地塊,用於混合使用項目。初步產權報告顯示沿後方物業線有一條 10 英尺的 PG&E 燃氣地役權——看似常規。然而,當 ALTA 測量繪製地役權時,發現該地役權實際上是**15 英尺地役權**(產權報告參考了錯誤的記錄文件),並且是對角線而非平行於後方地塊線。地役權切入可建築範圍,將建築佔地面積減少了 18%,並使計劃的地下停車場布局不可行。

開發商花了 14 個月與 PG&E 談判地役權搬遷,最終支付 $180K 讓 PG&E 搬遷燃氣主管並重新記錄地役權。

**經驗教訓**:永遠不要單獨信任產權報告——始終與 ALTA 測量交叉參考。產權報告通過文件參考描述地役權;測量顯示它們實際在地面的位置。訂購具有 Table A Items 6(退縮線)、11(公用設施)和 17(場外地役權)的 ALTA 測量,以在成交前捕捉這些衝突。

---

### 案例 3:San Jose——膨脹性黏土驚喜

San Jose Willow Glen 地區的一位住宅開發商在整個場地的淺層深度遇到了**高度膨脹性黏土(CH 分類,PI > 40)**。大地工程報告建議在所有側面使用至少 30 英吋濕氣屏障的後張預力樓板基礎——每單位增加 $6K 基礎成本。

更關鍵的是,膨脹性土壤條件需要更深的公用設施溝渠和進口的非膨脹性回填,使場地工作成本增加 25%。場地平整計劃必須修訂以容納整個墊層區域 18 英吋的過度開挖和替換。

**經驗教訓**:Santa Clara Valley 土壤變化極大。即使在單一社區內,土壤條件也可能從級配良好的礫石到高度膨脹性黏土。在最終確定財務分析前始終訂購大地工程調查——South Bay 的基礎成本僅基於土壤條件就可能每單位變化 $5K-$15K。

## 財務決策門檻

盡職調查結果直接輸入財務分析決策框架:

| 發現 | 成本影響 | App G 連接 |
| --- | --- | --- |
| **地役權減少可建築面積** | 更少單位→收入減少 | Level 1:如果單位數下降,Napkin Test 可能失敗 |
| **液化區** | 每單位 $8K-$15K 用於地面改良 | Level 2:添加到 Sources——場地工作行 |
| **膨脹性黏土** | 每單位 $5K-$15K 用於增強基礎 | Level 2:添加到 Sources——基礎行 |
| **高地下水位** | 降水 + 防水 = $200K-$500K | Level 3:Stress Test——大地工程應急 |
| **產權瑕疵/留置權** | 可變——可能是交易終結者 | Level 3:Kill Criteria #2(產權無法保險) |
