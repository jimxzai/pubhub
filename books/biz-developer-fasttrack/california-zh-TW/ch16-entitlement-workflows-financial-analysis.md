# 第十六章:許可審批流程與財務分析

**法定參考**: Government Code §65920–65964 (Permit Streamlining Act); §65941.1 (SB 330 Preliminary Application); §65864–65869.5 (Development Agreements); §66000–66025 (Mitigation Fee Act)

---

## 執行摘要、關鍵要點與行動檢查清單

---

::: execsummary

許可審批是開發風險最高的階段,因為在收入確定性存在之前就部署了大量資本。本章提供住宅和酒店項目的端到端工作流程,以及與附錄G的三級決策方法整合的財務分析框架。住宅路徑範圍從部長級(SB 35多戶住宅在RHNA不足的城市 = 3–6個月,無CEQA)到自由裁量級(需要EIR = 24–36個月,大量持有成本)。酒店許可審批本質上屬於自由裁量,需要Conditional Use Permit (CUP)配合設計審查、交通/停車/噪音研究,時間表為20–28個月。Permit Streamlining Act (Government Code §65920–65964)規定了嚴格的法定期限,這是你的執行槓桿:完整性審查(30天)、EIR項目決策(180天)、SB 35批准(90–180天,取決於單位數量)。如果城市錯過期限,項目將依法被"視為批准"。SB 330初步申請(在場地控制時提交)在許可審批中期增加$5K–$10K/單位之前鎖定費用和開發標準——單一的一頁申請可以在多個項目中節省$600K–$935K。影響費用因司法管轄區和項目類型而異($30K–$100K/單位住宅,$10K–$35K/鑰匙酒店);在許可審批進行期間同時提交建築許可可節省關鍵路徑上的4–8個月,在典型Bay Area項目上價值$340K–$600K的持有成本削減。Palo Alto/Redwood City案例研究展示了在相同的80單位建築上,自由裁量(28個月)和部長級(5個月)路徑之間$2.8M的許可審批溢價差異。**核心原則**:司法管轄區選擇比任何其他因素更能決定許可審批時間表和成本。在場地收購之前了解你的路徑。

:::

---

::: keytakeaways

- **SB 35部長級路徑將許可審批從28個月壓縮到5–6個月**: RHNA不足司法管轄區(大多數Bay Area城市)的多戶住宅項目繞過CEQA、公聽會和自由裁量否決,轉而採用客觀標準合規。在承諾司法管轄區之前檢查HCD的年度確定函。如果符合條件,時間表為90–180天,而非自由裁量的24–36個月——相當於在典型Bay Area多戶住宅上節省$2.44M的持有成本。

- **SB 330初步申請在許可審批中期增加之前鎖定費用**: 一頁申請(場地地址、單位數量、佔地面積草圖)在場地控制時凍結開發費用、徵費、建築標準,期限3年。典型Bay Area城市在許可審批期間增加$4K–$8K/單位的費用。在San Jose、Mountain View、Sunnyvale提交的三個項目 = 總共節省$2.375M費用。住宅開發中ROI最高的活動——需要1小時,每個項目節省$600K–$935K。

- **司法管轄區選擇是$2.8M的問題**: Palo Alto相同的80單位多戶住宅(自由裁量,28個月,$3.42M許可審批成本)與Redwood City(SB 35部長級,5個月,$615K許可審批成本)。$2.8M的差異純粹是司法管轄區選擇,而非場地質量或建築類型。在場地收購之前,按司法管轄區規劃許可審批路徑。限制性城市中的"更便宜"土地在總開發成本上花費更多。

- **同時提交建築許可節省關鍵路徑上的4–8個月**: 在許可審批程序繼續時提交建築許可計劃(而非批准後)。風險:如果條件改變需要修改計劃。優勢:$340K–$600K持有成本節省。預期價值計算在大多數情況下強烈支持同時提交。需要許可審批和設計團隊之間的密切協調,但顯著壓縮Death Point(§附錄G)。

- **Permit Streamlining Act期限是執行槓桿**: 城市有30天完整性審查、180天EIR項目決策、90–180天SB 35批准(依法)。如果城市錯過期限,項目依法被"視為批准"。在項目日曆中仔細跟踪每個期限;如果城市錯過則發送正式通知。這是你加速遲緩審查的槓桿。

- **司法管轄區之間的影響費用差異造成$1M+的波動**: 住宅影響費用:$30K–$100K/單位,取決於城市、經濟適用房組成部分、密度獎勵資格。酒店影響費用:$10K–$35K/鑰匙加上持續的TOT(每年8–14%客房收入)。不同城市的相同建築類型有截然不同的費用負擔。在Level 1 Napkin Test中包括費用估計;如果是住宅,通過SB 330鎖定。

- **Development agreements鎖定10–25年的監管確定性**: 對於監管風險高的大型分期項目,與城市協商development agreement(需要公聽會、自由裁量、受CEQA約束)。開發商提供增強的公共利益(經濟適用房、公園、基礎設施);城市提供許可審批條件不會改變的確定性。當當前許可審批條件有利且你想在城市規劃變更之前鎖定它們時很有用。

- **酒店許可審批需要20–28個月和完整的自由裁量程序**: CUP申請配合交通、停車、噪音研究;設計審查聽證會;規劃委員會CUP聽證會;ABC酒類執照(平行軌道)。酒店不存在部長級路徑。預算完整的CEQA(MND或EIR)、設計審查程序和政治風險。每鑰匙的影響費用低於住宅每單位,但TOT義務(188鑰匙酒店每年$1.4M)是住宅不面臨的持續負擔。

::: checklist

**住宅許可審批路徑:**

- [ ] 在場地收購前確定部長級與自由裁量路徑
- [ ] 如果是多戶住宅:檢查HCD SB 35年度確定函(確認RHNA不足狀態)
- [ ] 如果符合SB 35:90–180天時間表,無CEQA,僅客觀標準合規
- [ ] 如果自由裁量:預算24–36個月,CEQA(可能MND或EIR),設計審查程序,政治風險
- [ ] 對於SB 9(單戶住宅地塊分割 + ADUs):60–90天部長級時間表,無CEQA

**場地控制與初步申請(第1–2週):**

- [ ] 一旦場地控制立即提交SB 330初步申請(如果可能第1天)
- [ ] 初步申請僅需要:場地地址、單位數量、大致佔地面積草圖
- [ ] SB 330鎖定費用和開發標準3年
- [ ] 估計費用節省:$600K–$935K,取決於司法管轄區
- [ ] 提交所需時間估計:30分鐘
- [ ] 讓城市規劃人員確認收到初步申請和既得日期

**預申請會議(第2–3週):**

- [ ] 向規劃主任或許可審批規劃師申請預申請會議
- [ ] 簡報:項目摘要、土地使用、單位數量、停車、高度/體量、經濟適用房組成部分(如適用)
- [ ] 交付成果:城市書面摘要確認要求、已識別問題、建議路徑
- [ ] 對於酒店:確認CUP路徑和所需研究(交通、停車、噪音、設計審查)
- [ ] 在設計進行之前以書面形式記錄任何口頭反饋

**環境與技術研究(第2–4個月):**

- [ ] 確定CEQA路徑:豁免、分類豁免、ND、MND或EIR
- [ ] 如果豁免或分類豁免:預算6–12個月
- [ ] 如果需要MND:預算12–18個月(環境評估 + 公眾審查)
- [ ] 如果需要EIR:預算18–36個月(完整環境審查 + 替代方案分析)
- [ ] 對於酒店:訂購交通影響研究、停車研究、噪音研究(CUP所需)

**申請提交(第4–6個月):**

- [ ] 提交完整申請,包括所有所需研究、建築圖紙、技術文檔
- [ ] Permit Streamlining Act:城市有30天完整性期限
- [ ] 在項目日曆中跟踪完整性審查期限
- [ ] 如果城市在第25天要求額外信息:期限重置另外30天(快速回應以避免延誤)

**公聽會(第8–18個月,取決於路徑):**

- [ ] 部長級(SB 35):不需要公聽會
- [ ] 自由裁量住宅:設計審查 + 規劃委員會聽證會(1–3次聽證會)
- [ ] 酒店:設計審查 + 規劃委員會CUP聽證會(至少2–3次聽證會)
- [ ] SB 330將住宅項目限制為最多5次聽證會
- [ ] 為上訴期預算應急(通常10個日曆日)

**批准條件談判:**

- [ ] 批准條件在聽證會上協商(有時之前,如果城市願意)
- [ ] 典型住宅條件:設計修訂、交通平靜化、停車管理、經濟適用房監測
- [ ] 典型酒店條件:營業時間、噪音限制、送貨時間、代客泊車、社區關係
- [ ] 紅旗條件:改變項目經濟性的條件(高度降低、單位損失、增加停車要求)
- [ ] 離開聽證會前要求書面條件;澄清任何模糊語言

**Permit Streamlining Act日曆:**

- [ ] 第0天:申請被視為完整(或城市要求額外信息)
- [ ] 第30天:完整性審查期限(PSA §65943)
- [ ] 第60–180天:項目決策期限(因CEQA路徑而異,§65950):
  - 分類豁免或ND/MND項目第60天
  - EIR項目第180天
  - SB 35第90–180天(<150單位為90天,150+單位為180天)
- [ ] 如果城市錯過期限:項目依法被"視為批准"
- [ ] 以書面形式記錄所有錯過的期限;如果出現模式,上報給城市經理

**同時提交建築許可(許可審批期間第8–12個月):**

- [ ] 在許可審批仍在進行時開始建築許可設計(而非批准後)
- [ ] 與許可審批批准同時提交建築許可(或之前2–4個月,如果對許可審批結果有信心)
- [ ] 風險緩解:與許可審批律師協調條件變更風險
- [ ] 預期價值:$340K–$600K持有成本節省與$30K–$60K計劃修訂成本
- [ ] 對於許可審批路徑明確的項目:同時提交幾乎總是ROI為正

**建築許可協調(第12–18個月):**

- [ ] 與建築部門進行計劃檢查(4–8週)
- [ ] 與消防局、公共工程、雨水、水/下水道區域並行提交
- [ ] 如果可用,使用分階段許可:在完整建築許可之前獲得分級/基礎許可(提前2–3個月開始場地工作)
- [ ] 跟踪AB 1114計劃檢查期限(<25單位30天,26+單位60天)
- [ ] 如果城市錯過AB 1114期限:申請人可以使用城市費用的第三方計劃檢查

**酒店許可審批細節:**

- [ ] 酒店沒有部長級路徑(需要完整的自由裁量程序)
- [ ] 時間表:從申請到建築許可20–28個月(與SB 35住宅5–6個月相比)
- [ ] 所需研究:市場研究、交通影響、停車分析、噪音研究、設計審查
- [ ] 平行軌道:ABC酒類執照申請(第6個月,與許可審批同時)
- [ ] ABC執照時間表:6–12個月(單獨程序,通常比建築許可快)
- [ ] CUP條件通常包括:營業時間、送貨窗口、代客泊車要求、社區關係

**費用與財務跟踪:**

- [ ] 影響費用(住宅):預算$40K–$100K/單位,取決於司法管轄區和包容性條例
- [ ] 影響費用(酒店):預算$10K–$35K/鑰匙
- [ ] SB 330初步申請節省:記錄在申請日期鎖定的費用
- [ ] TOT(酒店):預算總客房收入的8–14%作為持續運營費用
- [ ] 學校費用:法定上限($4–$5/SF),無協商空間
- [ ] 公園/Quimby費用:通過土地奉獻替代方案可能可以協商

**財務壓力測試(與附錄G整合):**

- [ ] Level 1 Napkin Test:使用§16.3的影響費用估計;確認項目在最大分區建設量時能夠盈利
- [ ] Level 2 Development Pro Forma:在來源與使用中包括影響費用作為行項目;包括SB 330費用既得日期
- [ ] Level 3 Stress Test:建模時間表延遲情景(6、12、18個月);計算持有成本影響
- [ ] 敏感性分析:變化影響費用±20%、許可審批時間表±6個月、利率±100基點
- [ ] 如果使用同時提交策略:如果條件改變,壓力測試計劃修訂成本的影響

## 概述

許可審批是獲得開發財產所需的所有政府批准的過程。它通常是開發的**最高風險階段**,因為在存在任何收入確定性之前部署了大量資本。本章涵蓋住宅和酒店項目的完整工作流程,以及財務分析框架。

## A部分:住宅許可審批工作流程

### 16.1 分步流程

| 步驟 | 行動 | 時間表 |
| --- | --- | --- |
| 1 | **預申請會議** | 第1–2週 |
| 2 | **提交SB 330初步申請** | 第2–3週(鎖定費用/標準) |
| 3 | **確定路徑**:部長級與自由裁量 | 第3–4週 |
| 4 | **提交申請**及所有所需研究 | 第4–8週 |
| 5 | **完整性審查**(根據PSA 30天) | 30天 |
| 6 | **CEQA確定** | 0–24個月(取決於路徑) |
| 7 | **公聽會**(根據SB 330最多5次) | 1–3個月 |
| 8 | **協商批准條件** | 聽證會時 |
| 9 | **建築許可**提交 | 許可審批後 |

### 路徑決策矩陣

| 路徑 | 何時適用 | 時間表 | CEQA |
| --- | --- | --- | --- |
| **SB 35** | 多戶住宅、RHNA不足司法管轄區、經濟適用房組成部分 | 3–6個月 | 無 |
| **SB 9** | 單戶住宅地塊、雙拼/地塊分割 | 2–4個月 | 無 |
| **ADU** | 住宅/混合使用地塊的法定權利 | 60天 | 無 |
| **分類豁免** | Class 3(最多6單位)或Class 32(填充) | 6–12個月 | 豁免 |
| **MND** | 可緩解影響 | 12–18個月 | MND |
| **EIR** | 重大無法緩解影響 | 24–36個月 | 完整EIR |

## D部分:Development Agreements

**法定參考**: Government Code §65864–65869.5

Development agreement是**開發商與地方政府之間的合同**,在指定期限(通常10–25年)內鎖定開發商根據當前法規進行開發的權利。

### 何時使用

- 監管確定性至關重要的大型分期項目
- 需要大量基礎設施投資的項目
- 當前許可審批條件有利的情況

### 結構

- 開發商提供超出正常條件的**增強公共利益**
- 政府提供協議期限內的**監管確定性**
- 協議對財產進行登記並隨土地轉移
- 需要公聽會、自由裁量並受CEQA約束

## E部分:影響費用與財務分析

### 16.3 影響費用比較:住宅與酒店

| 費用類別 | 住宅 | 酒店 | 法律依據 |
| --- | --- | --- | --- |
| **學校費用** | $4–$5/SF | $0.66/SF | Gov. Code §65995 |
| **公園/Quimby** | $5K–$30K/單位 | N/A | Gov. Code §66477 |
| **交通影響** | $5K–$25K/單位 | $2K–$8K/鑰匙 | 地方條例 |
| **經濟適用房替代** | $20–$75/SF | 變化 | 地方包容性條例 |
| **下水道/水連接** | $5K–$20K/單位 | $3K–$10K/鑰匙 | 公用事業區域 |
| **公共藝術** | 建設成本的1% | 建設成本的1% | 地方條例 |
| **TOT** | N/A | 客房收入的8–14% | 地方(持續) |

### 費用協商策略

| 費用類型 | 可協商性 | 策略 |
| --- | --- | --- |
| **學校費用** | 無 — 法定上限 | 許可發放時支付 |
| **公園/Quimby** | 中等 | 土地奉獻可替代費用 |

## G部分:同時提交與建築許可協調

### 16.5 壓縮關鍵路徑

許可審批到建設時間表有一個隱藏的瓶頸:**規劃批准和建築許可發放之間的順序交接**。大多數開發商在提交建築許可計劃之前等待最終許可審批。這造成了4–8個月的死區,項目產生零進展但持有成本全額。

### 同時提交策略

在**許可審批仍在進行時**提交建築許可計劃可以在關鍵路徑上節省3–6個月。風險是如果批准條件改變需要修改計劃——但在大多數情況下,預期價值計算強烈支持同時提交。

| 階段 | 順序方法 | 同時方法 | 節省時間 |
| --- | --- | --- | --- |
| 許可審批審查 | 第1–12個月 | 第1–12個月 | — |
| 建築許可設計 | 第13–16個月 | 第8–12個月(重疊) | 4個月 |
| 計劃檢查週期1 | 第17–19個月 | 第13–15個月 | 2個月 |
| 計劃檢查更正 | 第20–22個月 | 第16–18個月 | 2個月 |
| 許可發放 | 第23個月 | 第19個月 | **4個月** |

### 建築許可協調檢查清單

建築許可階段涉及多個必須在發放前簽字的部門。並行協調這些——而非順序——至關重要。

| 部門 | 提交要求 | 典型審查時間 | 並行? |
| --- | --- | --- | --- |
| **建築部門** | 完整計劃集、Title 24能源計算、結構計算 | 4–8週 | 主軌道 |
| **消防局** | 消防噴淋計劃、消防通道、警報系統 | 3–6週 | 是 — 與建築一起提交 |
| **公共工程** | 分級計劃、侵占許可、公用事業連接 | 4–8週 | 是 — 與建築一起提交 |
| **雨水(C.3)** | 雨水管理計劃、BMP規模 | 3–6週 | 是 — 與建築一起提交 |
| **水務區域** | 供水確認函、連接申請 | 2–4週 | 是 — 可在許可審批期間提交 |
| **衛生區域** | 下水道連接許可、容量驗證 | 2–4週 | 是 — 可在許可審批期間提交 |
| **學區** | 學校費用計算 | 1–2週 | 是 — 許可發放時支付 |
| **綠色建築** | CalGreen合規、能源建模(如需要) | 2–4週 | 是 — 與計劃一起提交 |

### 16.6 計劃檢查加速策略

| 策略 | 如何運作 | 預期節省 |
| --- | --- | --- |
| **預提交會議** | 提交前與計劃檢查主管會面;逐步討論複雜項目 | 減少1輪更正週期 |
| **櫃台更正** | 對於小更正,要求OTC審查而非正式重新提交 | 每次更正節省2–4週 |
| **加急計劃檢查** | 某些司法管轄區提供付費加急審查(1.5–2倍費用) | 計劃檢查時間縮短50% |
| **第三方計劃檢查** | 使用城市批准的第三方計劃檢查公司 | 與城市審查相比節省4–8週 |
| **分階段許可** | 在完整建築許可之前獲得分級/基礎許可 | 允許場地工作提前2–3個月開始 |

### AB 1114計劃檢查期限(2025年生效)

加州現在要求建築部門在法定期限內返回**全面的計劃檢查意見**:

| 項目規模 | 首次審查期限 | 後續審查 |
| --- | --- | --- |
| **25單位或更少** | 30個工作日 | 30個工作日 |
| **26+單位** | 60個工作日 | 60個工作日 |

> **執行**: 如果司法管轄區錯過期限,申請人可以使用司法管轄區費用的第三方計劃檢查公司。像跟踪PSA期限一樣積極地跟踪這些期限。

### 16.7 進度壓縮的財務影響

| 變量 | 80單位MF項目 | 188鑰匙酒店 |
| --- | --- | --- |
| 每月持有成本(土地 + 軟成本) | ~$85K/月 | ~$120K/月 |
| 順序時間表(許可審批 → 許可) | 23個月 | 32個月 |
| 同時時間表(並行提交) | 19個月 | 27個月 |
| **持有成本節省** | **$340K** | **$600K** |
| 機會成本(更早收入) | ~$200K | ~$350K |
| **壓縮總價值** | **~$540K** | **~$950K** |

> 同時提交策略的下行風險是如果條件改變,$30K–$60K的計劃修訂,對比預期上行的$340K–$600K持有成本節省。對於許可審批路徑明確的項目,預期價值是壓倒性的正值。

## Bay Area案例研究與經驗教訓

### 案例1:Palo Alto與Redwood City — 許可審批時間表比較

兩個相同的項目——80單位多戶住宅建築——由同一開發團隊同時在Palo Alto和Redwood City提出。兩個項目都在類似分區的地塊(多戶住宅)上。

#### 並排比較

| 因素 | Palo Alto(自由裁量) | Redwood City(SB 35部長級) |
| --- | --- | --- |
| **許可審批路徑** | 自由裁量 — 完整設計審查 | SB 35部長級 — 僅客觀標準 |
| **預申請** | 鄰居關切標記;需要建築審查 | 確認SB 35資格(城市落後於RHNA) |
| **環境審查** | 鄰居挑戰MND → 升級為重點EIR(交通、美學) | 無 — 根據SB 35豁免CEQA |
| **公聽會** | ARB/規劃委員會前3次聽證會 | 無 — 無公聽會 |
| **批准條件** | 施加14項條件 | 僅客觀標準合規 |
| **經濟適用房組成部分** | 根據城市包容性條例 | 10%經濟適用房(SB 35最低) |
| **總時間表** | **28個月**(申請 → 建築許可) | **5個月**(申請 → 建築許可) |
| **總軟成本** | **$450K**(環境、法律、設計修訂、顧問) | **$85K**(設計、申請費用) |

#### 財務影響:$2.8M問題

| 成本組成部分 | Palo Alto | Redwood City | 差異 |
| --- | --- | --- | --- |
| 土地收購 | $13M | $13M | — |
| 每月持有成本 | ~$106K/月(貸款利息 + 稅 + 保險) | ~$106K/月 | — |
| 許可審批時間表 | 28個月 | 5個月 | **23個月** |
| **許可審批期間持有成本** | **$2.97M** | **$530K** | **$2.44M** |
| **軟成本** | **$450K** | **$85K** | **$365K** |
| **總許可審批成本** | **$3.42M** | **$615K** | **$2.8M** |

相同的開發商、相同的建築類型、相同的土地成本——選擇錯誤司法管轄區的**$2.8M許可審批溢價**(見引言:$2.8M問題)。

**經驗教訓**: 司法管轄區選擇是Bay Area開發中影響最大的決策。在Redwood City需要5個月的項目在Palo Alto需要28個月——相同建築類型上$2.8M的許可審批溢價。在承諾場地之前,按司法管轄區規劃許可審批路徑。限制性城市中的"更便宜"土地在總開發成本上可能比流線型城市中的"昂貴"土地花費更多。

### 案例2:Hotel Indigo San Carlos — 影響費用分析

501 Industrial Road的Hotel Indigo(188鑰匙)說明了Bay Area酒店費用結構。基於San Carlos的費用時間表和項目的119,036 SF建築:

| 費用類別 | 估計金額 | 依據 |
| --- | --- | --- |
| 學校影響費用 | ~$78K | $0.66/SF x 119,036 SF |
| 交通影響費用 | ~$564K | ~$3K/鑰匙 x 188鑰匙 |
| 下水道/水連接 | ~$940K | ~$5K/鑰匙 x 188鑰匙 |
| 建築許可費用 | ~$350K | 計劃檢查時估計 |
| **一次性費用總計** | **~$1.93M** | **~$10.3K/鑰匙** |
| TOT(持續) | ~$1.4M/年 | 12% x 估計$12M總客房收入 |

相比之下,同一場地上的188單位多戶住宅項目將面臨約$40K–$60K/單位的影響費用(總計$7.5M–$11.3M)——每單位成本顯著更高。但是,住宅項目將受益於SB 330費用既得和潛在的密度獎勵優惠。

**經驗教訓**: Bay Area司法管轄區的酒店每鑰匙影響費用($10K–$25K)通常遠低於住宅每單位費用($40K–$100K)。但是,酒店面臨住宅項目沒有的持續TOT義務(每年客房收入的8–14%)。在比較同一場地的酒店與住宅時,建模10年內的總稅/費負擔(包括TOT),以獲得準確的比較。

### 案例3:Bay Area — SB 330初步申請在司法管轄區之間的費用節省

一位開發商同時在Bay Area司法管轄區為三個項目提交了SB 330初步申請:

| 司法管轄區 | 單位 | 許可審批期間費用增加 | SB 330既得節省 |
| --- | --- | --- | --- |
| **San Jose** | 120單位 | 公園:+$4K/單位;交通:+$2K/單位 | **節省$720K** |
| **Mountain View** | 85單位 | 公園:+$8K/單位;下水道:+$3K/單位 | **節省$935K** |
| **Sunnyvale** | 60單位 | 低於市場價替代:+$12K/單位 | **節省$720K** |

三個項目的總節省:**$2.375M** — 來自提交三個一頁初步申請。

**經驗教訓**: SB 330初步申請是加州住宅開發中ROI最高的活動。Bay Area城市定期且通常大幅增加費用。提交初步申請基本上不花費任何東西,可以節省六位或七位數。養成在每個住宅項目的場地控制第一週內提交SB 330初步申請的習慣。

第16章和附錄G形成互補對——本章提供許可審批和費用框架;附錄G提供使用本章數據作為輸入的3級財務決策方法。
