# Appendix C: Templates & Checklists

---

## Overview

These templates provide standardized frameworks for the key activities in California development — from initial site screening through entitlement and construction. Adapt them to your specific project and jurisdiction.

---

   □ SB 9 eligible (duplex/lot split on SFR lot)
   □ ADU by-right (Gov. Code §65852.2)
   □ Other ministerial pathway: _____________________
   → If YES to any: NO CEQA REQUIRED. Stop here.

2. IS THE PROJECT EXEMPT?
   □ Class 1 — Existing facilities (minor alteration)
   □ Class 3 — New construction, small structures (≤6 units in urban)
   □ Class 32 — Infill development (urban, consistent with GP/zoning,
     no significant effect, adequate utilities)
   □ Common sense exemption (§15061(b)(3))
   □ Other exemption: _______________________________
   → If YES: File Notice of Exemption. Stop here.
   → EXCEPTION CHECK: Does any exception apply? (§15300.2)
     □ Location (sensitive environment)
     □ Cumulative impact
     □ Significant effect (unusual circumstances)
     □ Scenic highway
     □ Hazardous waste site
     □ Historical resource
     → If any exception applies: exemption does NOT apply. Continue.

3. INITIAL STUDY — IMPACT ASSESSMENT
   Can all impacts be mitigated to less than significant?
   □ Yes → Mitigated Negative Declaration (MND)
     Timeline: 12–18 months
     Cost: $50K–$150K
   □ No → Environmental Impact Report (EIR) required
     Timeline: 18–36 months
     Cost: $200K–$1M+

4. EIR TYPE (if required)
   □ Project EIR (single project)
   □ Program EIR (general plan, specific plan)
   □ Supplemental/Subsequent EIR (changed conditions)
   □ Addendum (minor modifications to approved project)
   □ Tiered EIR (project tiers from program EIR)

## C.11 Financial Decision Framework Quick Reference

For the complete 3-level financial decision framework, see **Appendix G**. Below is a quick-reference summary:

```
FINANCIAL DECISION GATEWAY — QUICK REFERENCE
═══════════════════════════════════════════════════

LEVEL 1: 5-MINUTE NAPKIN TEST
□ Land cost per unit/key: $_______________
□ Construction cost per SF: $_______________
□ Total estimated cost: $_______________
□ Comparable sales/revenue: $_______________
□ Gross margin > 15%?  □ Yes → PROCEED  □ No → KILL

LEVEL 2: DEVELOPMENT PRO FORMA (see App G for full template)
□ Sources & Uses complete
□ Revenue projections supported by comps
□ Returns meet minimum thresholds:
  □ Development margin > 20%
  □ Cash-on-cash > 8%
  □ IRR (5-year) > 15%
□ DEATH POINT CALCULATED: Month ___
□ Death Point > financing term?  □ Yes → KILL  □ No → PROCEED

LEVEL 3: STRESS TEST & KILL CRITERIA (see App G for 12 criteria)
□ Construction cost +15%: still viable?  □ Yes  □ No
□ Timeline +6 months: still viable?  □ Yes  □ No
□ Revenue -10%: still viable?  □ Yes  □ No
□ Interest rate +100 bps: still viable?  □ Yes  □ No
□ Any of 12 Kill Criteria triggered?  □ Yes → KILL  □ No → GO

DECISION:  □ GO  □ KILL  □ CONDITIONAL
═══════════════════════════════════════════════════
```
