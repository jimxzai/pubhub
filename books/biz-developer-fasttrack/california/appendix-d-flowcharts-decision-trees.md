# Appendix D: Flowcharts & Decision Trees

---

## Overview

These decision trees and flowcharts provide step-by-step guidance for the most common decisions in California development. Use them as quick-reference tools during project evaluation and entitlement.

---

## D.1 CEQA Decision Tree

```
START: Is Your Project Subject to CEQA?
│
├── Is the project ministerial (no discretion)?
│   ├── YES → NO CEQA REQUIRED
│   │   Examples: SB 35, SB 9, ADU by-right,
│   │   building permits with no discretionary action
│   │
│   └── NO → Continue ↓
│
├── Does a statutory exemption apply?
│   │
│   ├── SB 35/423, SB 9, SB 684, AB 2011?
│   │   └── YES → NO CEQA REQUIRED (ministerial)
│   │
│   ├── AB 130 infill housing exemption? (2025)
│   │   Requirements:
│   │   □ Housing (not hotel/transient lodging)
│   │   □ Urban infill, ≤20 acres
│   │   □ ≥15 du/ac (metro) or ≥10 du/ac (suburban)
│   │   □ ≥2/3 residential
│   │   □ GP OR zoning consistent (only one needed)
│   │   □ Not on protected/hazardous lands
│   │   └── YES → CEQA EXEMPT
│   │       Phase I ESA + tribal consultation required
│   │       Timeline: 104–178 days
│   │       See Ch 3, §3.6.2 for full criteria
│   │
│   ├── Other statutory exemption?
│   │   └── YES → NO CEQA REQUIRED
│   │       File Notice of Exemption (recommended)
│   │
│   └── NO → Continue ↓
│
├── Does a categorical exemption apply?
│   │
│   ├── Class 1: Existing Facilities
│   │   (Minor alteration of existing structures)
│   │
│   ├── Class 3: New Construction, Small Structures
│   │   (Up to 6 residential units in urbanized area)
│   │
│   ├── Class 32: Infill Development
│   │   Requirements:
│   │   □ Consistent with GP and zoning
│   │   □ Within city limits, on ≤5 acres
│   │   □ Substantially surrounded by urban uses
│   │   □ No habitat for endangered species
│   │   □ No significant traffic/noise/air/water effects
│   │   □ Adequately served by utilities
│   │
│   ├── YES (exemption applies) →
│   │   │
│   │   └── Does any EXCEPTION apply? (§15300.2)
│   │       □ Sensitive environment (wetlands, habitat)
│   │       □ Cumulative impacts
│   │       □ Unusual circumstances → significant effect
│   │       □ Scenic highway
│   │       □ Hazardous waste site (listed)
│   │       □ Historical resource
│   │       │
│   │       ├── NO exceptions → EXEMPT
│   │       │   File Notice of Exemption
│   │       │
│   │       └── YES exception → NOT EXEMPT → Continue ↓
│   │
│   └── NO → Continue ↓
│
├── SB 131 "NEAR-MISS" CHECK (2025)
│   Does the project miss an exemption by
│   ONLY ONE condition?
│   │
│   ├── YES → LIMITED FOCUSED CEQA REVIEW
│   │   Analyze ONLY the impacts from that
│   │   single disqualifying condition
│   │   No alternatives analysis required
│   │   No growth-inducing impacts required
│   │   See Ch 3, §3.6.2 for details
│   │
│   └── NO → Continue ↓
│
├── INITIAL STUDY REQUIRED
│   Evaluate all environmental impact categories
│   │
│   ├── Can ALL impacts be mitigated to
│   │   less than significant?
│   │   │
│   │   ├── YES → MITIGATED NEGATIVE DECLARATION (MND)
│   │   │   Timeline: 12–18 months
│   │   │   Cost: $50K–$150K
│   │   │   30-day public review period
│   │   │
│   │   └── NO → Continue ↓
│   │
│   └── Are there potentially significant
│       unmitigable impacts?
│       │
│       └── YES → ENVIRONMENTAL IMPACT REPORT (EIR)
│           Timeline: 18–36 months
│           Cost: $200K–$1M+
│           45-day public review period
│           Public hearings required
│
└── FAIR ARGUMENT WARNING
    If ANY substantial evidence of a potentially
    significant effect exists (even one public comment
    with factual basis), an EIR may be required
    regardless of your Initial Study conclusions.
```

## D.2 Entitlement Pathway Decision Tree — Residential

```
START: Residential Project in California
│
├── 1. Is it an ADU or JADU?
│   ├── YES → ADU BY-RIGHT PATHWAY
│   │   • Gov. Code §65852.2
│   │   • 60-day approval timeline
│   │   • No CEQA, no public hearing
│   │   • No impact fees for <750 SF
│   │   → DONE (fastest pathway)
│   │
│   └── NO → Continue ↓
│
├── 2. Is it a duplex or lot split on a
│      single-family lot?
│   ├── YES → SB 9 PATHWAY
│   │   Requirements:
│   │   □ Existing SFR zoning
│   │   □ Urban lot (not rural/ag)
│   │   □ Not historic district
│   │   □ Not in flood/fire/fault zone*
│   │   □ Meets objective standards
│   │   │
│   │   ├── All met → MINISTERIAL APPROVAL
│   │   │   • 60-day timeline
│   │   │   • No CEQA, no hearing
│   │   │   • Up to 4 units (lot split + duplexes)
│   │   │
│   │   └── Not all met → Continue ↓
│   │
│   └── NO → Continue ↓
│
├── 3. Is the jurisdiction SB 35-eligible?
│      (Check HCD determination)
│   │
│   ├── YES → Does project meet SB 35 requirements?
│   │   □ Multifamily (2+ units)
│   │   □ On site with ≥75% residential
│   │   □ Complies with ALL objective standards
│   │   □ Includes affordable component:
│   │     • 10% affordable (if city missed above-moderate RHNA)
│   │     • 50% affordable (if city missed low-income RHNA)
│   │   □ Pays prevailing wages
│   │   □ Not on environmentally sensitive site
│   │   □ Not on farmland or fire hazard zone
│   │   │
│   │   ├── All met → SB 35 MINISTERIAL APPROVAL
│   │   │   • <150 units: 90-day review
│   │   │   • 150+ units: 180-day review
│   │   │   • No CEQA, no public hearing
│   │   │   • No subjective design review
│   │   │
│   │   └── Not all met → Continue ↓
│   │
│   └── NO → Continue ↓
│
├── 4. DISCRETIONARY PATHWAY
│   │
│   ├── Is the project consistent with
│   │   GP and zoning as proposed?
│   │   │
│   │   ├── YES → Site plan / design review only
│   │   │   └── CEQA determination (see D.1)
│   │   │
│   │   └── NO → What approvals needed?
│   │       □ General Plan amendment
│   │       □ Rezoning
│   │       □ Variance
│   │       □ CUP
│   │       □ Specific plan amendment
│   │       └── Each requires CEQA + public hearing
│   │
│   └── Apply SB 330 protections:
│       □ File preliminary application IMMEDIATELY
│       □ Maximum 5 public hearings
│       □ No downzoning during review
│       □ Fees locked at pre-app date
│       □ HAA denial protections apply
│
└── TIMELINE SUMMARY
    ADU:              60 days
    SB 9:             60 days
    SB 35 (<150):     90 days
    SB 35 (150+):     180 days
    Discretionary:    6–36 months
```

## D.3 Entitlement Pathway Decision Tree — Hotel

```
START: Hotel Project in California
│
├── NOTE: Hotels do NOT qualify for:
│   • SB 35 (residential only)
│   • SB 9 (residential only)
│   • SB 330 protections (residential only)
│   • ADU law (residential only)
│   Hotels follow the FULL DISCRETIONARY pathway.
│
├── 1. ZONING CHECK
│   │
│   ├── Is hotel a permitted use in the zone?
│   │   ├── YES (by right) → Site plan review only
│   │   │   Still requires CEQA determination
│   │   │
│   │   ├── YES (with CUP) → CUP APPLICATION REQUIRED
│   │   │   Most common pathway for hotels
│   │   │
│   │   └── NO → Options:
│   │       □ Rezoning required
│   │       □ General Plan amendment required
│   │       □ Find alternative site
│   │       → Each adds 12–24 months
│   │
│   └── Continue ↓
│
├── 2. BRAND & FRANCHISE (Parallel Track)
│   │
│   ├── Market study complete?
│   │   └── Required for franchise LOI
│   │
│   ├── Brand selected?
│   │   └── Determines building program
│   │
│   ├── Franchise LOI executed?
│   │   └── Critical: LOI has expiration date
│   │       Typical: 12–18 months
│   │       Must align with entitlement timeline
│   │
│   └── Continue ↓
│
├── 3. CEQA PATHWAY (see D.1)
│   │
│   ├── Hotel projects RARELY qualify for:
│   │   □ Class 3 exemption (too large)
│   │   □ Class 32 infill (traffic/noise usually significant)
│   │
│   ├── MOST COMMON: MND or EIR
│   │   │
│   │   ├── Key impact areas for hotels:
│   │   │   □ Traffic (high trip generation)
│   │   │   □ Noise (HVAC, loading, events)
│   │   │   □ Air quality (operations, backup generators)
│   │   │   □ Aesthetics (height, lighting, signage)
│   │   │   □ Parking
│   │   │   □ GHG emissions
│   │   │
│   │   └── EXCEPTION: Tiering from Program EIR
│   │       If site is within a specific plan or
│   │       master plan area with certified Program EIR,
│   │       hotel may tier → saves $500K+ and 12+ months
│   │
│   └── Continue ↓
│
├── 4. ABC LICENSING (Parallel Track — Start Month 6)
│   │
│   ├── License type needed?
│   │   □ Type 47 (On-sale general — full bar)
│   │   □ Type 41 (On-sale beer/wine)
│   │   □ Type 57 (Special on-sale general — restaurants)
│   │
│   ├── Transfer vs. new license?
│   │   Transfer: faster, may avoid census restriction
│   │   New: subject to census, may be denied
│   │
│   └── Timeline: 6–12 months
│       Start early to avoid delaying opening
│
├── 5. APPROVALS & HEARINGS
│   │
│   ├── Design Review → Planning Commission
│   ├── CUP Hearing → Planning Commission
│   ├── EIR Certification → Planning Commission or City Council
│   ├── Appeal (if filed) → City Council
│   │
│   └── Total hearings: typically 3–5
│
└── 6. BUILDING PERMIT
    │
    ├── Submit construction documents
    ├── Franchise brand review (parallel)
    ├── Fire Marshal review
    ├── Health Department review (food service)
    │
    └── Timeline: 3–6 months for plan check

TOTAL HOTEL TIMELINE: 24–36 months typical
```

## D.4 Site Screening Flowchart

```
START: Potential Development Site Identified
│
├── PHASE 1: 3-MINUTE SCREEN (Online GIS)
│   │
│   ├── Check parcel size → Adequate for project?
│   │   └── NO → PASS
│   │
│   ├── Check zoning → Compatible with project?
│   │   └── NO → GP amendment needed?
│   │       └── YES → Add 12–24 months + cost
│   │
│   ├── Check FEMA flood zone → Zone AE/VE?
│   │   └── YES → FATAL FLAW unless price reflects
│   │
│   ├── Check earthquake fault zone → Alquist-Priolo?
│   │   └── YES → FATAL FLAW (no habitable structures)
│   │
│   └── PASS 3-MINUTE SCREEN? → Continue ↓
│
├── PHASE 2: 10-MINUTE DEEP DIVE
│   │
│   ├── CGS liquefaction zone?
│   │   └── YES → Budget $8K–$15K/unit extra
│   │
│   ├── CAL FIRE FHSZ (high/very high)?
│   │   └── YES → Budget $40K–$80K/unit for WUI
│   │
│   ├── EnviroStor (contamination within 1,000 ft)?
│   │   └── YES → Budget $100K–$200K for assessment
│   │
│   ├── GeoTracker (groundwater issues)?
│   │   └── YES → Evaluate vapor intrusion risk
│   │
│   ├── Wetlands inventory?
│   │   └── YES → USACE Section 404 permit needed
│   │
│   └── PASS 10-MINUTE SCREEN? → Continue ↓
│
├── PHASE 3: 30-MINUTE ANALYSIS
│   │
│   ├── CalEnviroScreen percentile?
│   │   └── High → Additional community engagement
│   │
│   ├── Historic resources on/near site?
│   │   └── YES → CEQA historic evaluation
│   │
│   ├── SB 35 eligibility (residential)?
│   │   └── YES → Ministerial pathway available!
│   │
│   ├── RHNA status of jurisdiction?
│   │   └── Behind → City more likely to approve
│   │
│   └── Preliminary pro forma pencils?
│       └── NO → PASS
│
└── DECISION
    │
    ├── GREEN LIGHT: No fatal flaws, zoning compatible,
    │   economics work → PROCEED TO LOI/CONTRACT
    │
    ├── YELLOW: Manageable constraints, economics
    │   marginal → PROCEED WITH CONDITIONS
    │   (price adjustment, contingencies)
    │
    └── RED: Fatal flaws or economics don't work → PASS
```

## D.5 Density Bonus Decision Tree

```
START: Considering Density Bonus?
│
├── 1. What affordability level can you provide?
│   │
│   ├── Very Low Income (≤50% AMI)
│   │   5% VLI → 20% bonus
│   │   10% VLI → 33.25% bonus
│   │   15% VLI → 50% bonus (max)
│   │
│   ├── Low Income (≤80% AMI)
│   │   10% LI → 20% bonus
│   │   20% LI → 35% bonus
│   │   24% LI → 43.75% bonus
│   │
│   ├── Moderate Income (≤120% AMI, for-sale only)
│   │   10% Mod → 5% bonus
│   │   40% Mod → 35% bonus
│   │
│   └── 100% Affordable → 80% bonus
│
├── 2. CONCESSIONS / INCENTIVES
│   │
│   ├── How many can you request?
│   │   5% VLI or 10% LI → 1 concession
│   │   10% VLI or 17% LI → 2 concessions
│   │   15% VLI or 24% LI → 3 concessions
│   │
│   └── Common concessions:
│       □ Reduced setbacks
│       □ Reduced parking (0.5/unit minimum)
│       □ Increased height
│       □ Increased FAR
│       □ Reduced open space
│
├── 3. WAIVERS (Unlimited)
│   │
│   └── Any standard that physically prevents
│       achieving the bonus density can be waived.
│       City must approve unless it would have
│       a specific adverse health/safety impact.
│       │
│       └── Example: Height limit prevents building
│           the bonus units → height waiver granted
│
└── 4. COMBINED STRATEGY
    │
    ├── Density Bonus + SB 35 = Maximum power
    │   Bonus density + ministerial approval
    │
    ├── Density Bonus + ADU = Additional units
    │   Bonus units + ADU units (not counted)
    │
    └── Density Bonus + SB 330 = Fee protection
        Bonus density + locked-in fees
```

## D.6 Modular vs. Conventional Decision Tree

```
START: Should This Project Use Modular Construction?
│
├── 1. PROJECT TYPE CHECK
│   │
│   ├── Hotel (select-service, repetitive rooms)?
│   │   └── STRONG CANDIDATE for modular
│   │
│   ├── Multifamily (studio/1BR/2BR repetitive)?
│   │   └── STRONG CANDIDATE for modular
│   │
│   ├── Student/senior housing (repetitive units)?
│   │   └── STRONG CANDIDATE for modular
│   │
│   ├── Custom single-family homes?
│   │   └── WEAK CANDIDATE (consider panelized instead)
│   │
│   └── Mixed-use with irregular floor plates?
│       └── EVALUATE CAREFULLY (hybrid approach may work)
│
├── 2. SITE ACCESS CHECK
│   │
│   ├── Can modules reach the site?
│   │   □ Road width adequate (min 14 ft clear)?
│   │   □ Turning radius adequate?
│   │   □ Bridge weight limits OK?
│   │   □ No overhead obstructions?
│   │   │
│   │   ├── YES to all → Continue ↓
│   │   └── NO to any → CONVENTIONAL
│   │       (unless modules can be reduced in size)
│   │
│   ├── Is crane placement feasible?
│   │   □ Crane pad area available?
│   │   □ Adjacent buildings allow boom swing?
│   │   │
│   │   └── NO → CONVENTIONAL or PANELIZED
│   │
│   └── Continue ↓
│
├── 3. STRUCTURAL CHECK
│   │
│   ├── Seismic Design Category?
│   │   □ SDC A–D → Standard modular connections OK
│   │   □ SDC E–F → Special seismic connections required
│   │       (BRB frames, moment connections)
│   │       → adds cost but still feasible
│   │
│   ├── Building height?
│   │   □ 1–5 stories → Standard modular
│   │   □ 6–12 stories → Modular with enhanced structure
│   │   □ 12+ stories → Evaluate carefully
│   │       (hybrid: modular units on concrete podium)
│   │
│   └── Continue ↓
│
├── 4. ECONOMICS CHECK
│   │
│   ├── Is labor scarce/expensive in this market?
│   │   └── YES → Modular advantage increases
│   │
│   ├── Is schedule critical?
│   │   └── YES → Modular saves 40–50% time
│   │
│   ├── Module transport distance?
│   │   □ <200 miles → Favorable economics
│   │   □ 200–500 miles → Evaluate transport cost
│   │   □ >500 miles → Transport cost may offset savings
│   │   □ Overseas (shipping container) → 4–8 week ocean transit
│   │
│   ├── Is there a manufacturer with capacity?
│   │   └── NO → Long lead time; evaluate alternatives
│   │
│   └── Continue ↓
│
└── DECISION MATRIX
    │
    ├── VOLUMETRIC MODULAR recommended if:
    │   □ Repetitive room/unit types
    │   □ Site access adequate
    │   □ Schedule is priority
    │   □ Manufacturer available with capacity
    │   → Expected savings: 10–20% cost, 40–50% schedule
    │
    ├── PANELIZED recommended if:
    │   □ Moderate repetition
    │   □ Site access constrained (no crane)
    │   □ Design flexibility needed
    │   □ Local panel manufacturer available
    │   → Expected savings: 5–15% cost, 20–30% schedule
    │
    ├── HYBRID recommended if:
    │   □ Tall building (podium + modules)
    │   □ Mixed uses (commercial ground floor + modules above)
    │   □ Complex site (combine conventional + modular)
    │   → Expected savings: varies by configuration
    │
    └── CONVENTIONAL recommended if:
        □ Unique/custom design
        □ Site access severely constrained
        □ No manufacturer available
        □ Small project (<10 units)
        □ Complex structural requirements
```

## D.9 Financial Decision Gateway Flowchart

```
START: New Development Opportunity
│
├── LEVEL 1: NAPKIN TEST (5 Minutes)
│   │
│   ├── Land cost / buildable units = Cost per unit
│   │   └── > Market comp?  → KILL (overpaying for land)
│   │
│   ├── Estimated total cost per unit
│   │   └── > 80% of ARV?  → KILL (insufficient margin)
```
