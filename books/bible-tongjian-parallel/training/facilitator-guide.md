## 一、培训师的角色

### 核心职责

作为培训师，你不只是传递知识，更是：

```

│                    培训师的五重角色                           │
├─────────────────────────────────────────────────────────────┤
```

### 培训师资格

**必备条件**：
- [ ] 完成进阶培训课程
- [ ] 实际使用本材料超过6个月
- [ ] 有教导成人的经验
- [ ] 对中国历史文化有基本了解
- [ ] 牧者或教会领袖推荐

**加分条件**：
- [ ] 神学院学位或同等训练
- [ ] 曾在文化处境中护教
- [ ] 有跨文化服事经验
- [ ] 写作或教材开发经验

## 二、培训前预备

### 4周预备时间表

```
第4周前  │ 确认场地、设备、人数
         │ 发送报名确认和预习材料
         │
第3周前  │ 完成PPT和讲义准备
         │ 准备分组活动材料
         │
第2周前  │ 自己演练一遍全部内容
         │ 确认所有行政安排
         │
第1周前  │ 与同工团队祷告
         │ 最终确认学员名单
         │
培训当天 │ 提前1小时到场
         │ 检查设备、布置场地
```

### 场地与设备清单

**场地要求**：
- 可容纳预计人数的空间
- 可以分成小组的灵活座位
- 安静、少干扰
- 有白板或投影设备

**设备清单**：
```
□ 投影仪/电视
□ 电脑
□ 白板/白纸
□ 马克笔
□ 名牌
□ 学员手册（人数+10%）
□ 反馈表格
□ 签到表
□ 茶点（如有）
```

### 学员预习材料

培训前一周发送：

1. **欢迎信**
   - 培训时间地点
   - 预习要求
   - 需要携带的物品

2. **预习阅读**
   - README.md 总纲
   - 00-readers-guide.md 读者指南
   - 第一卷第一章

3. **预习问题**
   - 你为什么想参加这个培训？
   - 你计划如何使用这套材料？
   - 你有什么问题想在培训中得到解答？

## 三、培训带领技巧

### 成人学习原则

成人学习者的特点：

| 特点 | 应对策略 |
|------|---------|
| 有丰富经验 | 邀请分享，尊重经验 |
| 需要知道"为什么" | 清楚说明目的和价值 |
| 自主性强 | 提供选择，减少命令 |
| 关注实用性 | 强调应用，举实例 |
| 时间宝贵 | 高效、不拖延 |

### 教学方法组合

**不要只用讲授！**

```
一堂2小时的培训建议比例：

讲授      │████████░░░░░░░│ 30%
讨论      │██████░░░░░░░░░│ 25%
活动      │██████░░░░░░░░░│ 25%
实习      │████░░░░░░░░░░░│ 15%
问答      │█░░░░░░░░░░░░░░│ 5%
```

**各种方法的使用**：

| 方法 | 适用场合 | 注意事项 |
|------|---------|---------|
| 讲授 | 传递新知识 | 不超过15分钟一段 |
| 讨论 | 深化理解 | 需要好问题 |
| 小组 | 应用练习 | 明确任务和时间 |
| 角色扮演 | 技巧训练 | 创造安全环境 |
| 案例分析 | 情境学习 | 贴近实际 |
| 反思 | 内化吸收 | 给足时间 |

### 提问技巧

**好问题的特征**：
- 开放式（不能只答是/否）
- 有思考深度
- 与学员经验相关
- 指向学习目标

**问题类型**：

```
知识型：你记得...是什么吗？
理解型：你如何解释...？
应用型：你会如何在...情况下使用...？
分析型：为什么...会发生？
评价型：你认为...的优缺点是什么？
创造型：你会如何设计...？
```

**提问技巧**：
1. 问完后停顿，给时间思考
2. 不要自己回答自己的问题
3. 肯定回答，即使不完美
4. 用追问深化："你能多说一点吗？"
5. 连接不同人的回答

### 处理困难情况

**情况一：有人主导讨论**

```
策略：
• "感谢你的分享。我们也想听听其他人的看法。"
• 在小组讨论时，分配他与不太发言的人一组
• 课后私下感谢他的热情，请他帮助带动其他人
```

**情况二：无人发言**

```
策略：
• 先问较简单的问题建立信心
• "先跟旁边的人讨论2分钟"
• 点名邀请（选看起来有想法的人）
• 分享你自己的经历作为示范
```

**情况三：有人提出挑战性问题**

```
策略：
• "这是个很好的问题，谢谢你提出"
• 如果你知道答案，简洁回应
• 如果不确定，"让我想想，之后回复你"
• 如果是争议性问题，"这个问题基督徒有不同看法..."
• 不要防御性反应
```

**情况四：偏离主题**

```
策略：
• "这个话题很有趣，我们课后可以继续。现在先回到..."
• 把偏离的话题写在"停车场"白板上
• 温和但坚定地引导回来
```

**情况五：学员之间意见冲突**

```
策略：
• "两位的观点都很有价值，让我们看看圣经怎么说"
• 找出双方的共同点
• 强调目标是学习，不是争辩
• 必要时课后分别跟进
```

## 四、分模块带领指南

### 模块1：为什么要对话？（2小时）

**学习目标**：
学员能够解释为什么需要圣经与资治通鉴的对话

**带领要点**：

| 时间 | 内容 | 方法 | 注意 |
|------|------|------|------|
| 0-10 | 开场、祷告、破冰 | 互动 | 建立轻松氛围 |
| 10-30 | 华人信徒的处境 | 讨论 | 引出真实经历 |
| 30-50 | 资治通鉴的价值 | 讲授 | 不要太学术 |
| 50-60 | 休息 | - | - |
| 60-90 | 四大主题框架 | 讲授+讨论 | 用实例说明 |
| 90-110 | 不是什么vs是什么 | 小组讨论 | 澄清误解 |
| 110-120 | 总结、问答、祷告 | 互动 | 鼓励提问 |

**关键问答**：

Q：这会不会变成混合主义？
A：不会。我们坚持圣经权威，资治通鉴是对话伙伴不是同等权威。

Q：为什么选资治通鉴而不是其他经典？
A：因为它是公认的历史权威，时间跨度与圣经重叠，提供丰富的人性案例。

### 模块2：神学根基（2小时）

**学习目标**：
学员能够说明比较研究的神学原则

**带领要点**：

| 时间 | 内容 | 方法 | 注意 |
|------|------|------|------|
| 0-10 | 复习上节、祷告 | 问答 | 连接前后 |
| 10-40 | 圣经权威的坚守 | 讲授 | 强调核心 |
| 40-70 | 正确的比较方法 | 讲授+案例 | 用具体例子 |
| 70-80 | 休息 | - | - |
| 80-100 | 救赎历史视角 | 讲授 | 可能较难 |
| 100-115 | 小组讨论：应用原则 | 小组 | 给具体案例 |
| 115-120 | 总结、祷告 | - | - |

**案例讨论**：

给学员以下场景，请他们判断是否正确使用：

1. "孔子的仁和耶稣的爱本质上是一样的" ✗
2. "孔子的仁反映了神放在人心中的道德意识" ✓
3. "资治通鉴证明中国人也有神的启示" ✗
4. "资治通鉴的历史印证了圣经所说的人性堕落" ✓

### 模块3：实际应用（2小时）

**学习目标**：
学员能够设计使用本材料的讲道/教学

**带领要点**：

| 时间 | 内容 | 方法 | 注意 |
|------|------|------|------|
| 0-10 | 复习、祷告 | 问答 | - |
| 10-35 | 讲道应用 | 讲授+示范 | 给讲道大纲模板 |
| 35-60 | 小组带领 | 讲授+示范 | 实际演示 |
| 60-70 | 休息 | - | - |
| 70-90 | 主日学教学 | 讲授 | 根据学员背景调整 |
| 90-115 | 实习：设计一个小组查经 | 小组 | 给具体主题 |
| 115-120 | 分享、总结 | 互动 | 鼓励具体化 |

**实习任务**：

4-5人一组，选择以下一个主题，设计30分钟小组查经：

1. 人性的堕落：亚当vs商纣王
2. 嫉妒的毁灭：该隐vs曹丕
3. 情欲的陷阱：大卫vs唐玄宗

设计内容包括：
- 开场问题
- 圣经段落阅读与讨论
- 历史对照介绍
- 反思问题
- 应用祷告

### 模块4：回应常见问题（3小时）

**学习目标**：
学员能够回应常见的文化/护教问题

**带领要点**：

| 时间 | 内容 | 方法 | 注意 |
|------|------|------|------|
| 0-10 | 开场、祷告 | - | - |
| 10-50 | "基督教是西方宗教" | 讲授+角色扮演 | 多次演练 |
| 50-90 | "信耶稣就是不孝" | 讲授+讨论 | 敏感处理 |
| 90-100 | 休息 | - | - |
| 100-140 | "宗教都差不多" | 讲授+案例 | 避免论断 |
| 140-170 | 其他问题（学员提问） | 问答 | 灵活应变 |
| 170-180 | 总结、祷告 | - | - |

**角色扮演设置**：

两人一组：
- A扮演提问者
- B扮演基督徒回应者
- 3分钟后交换

提供不同程度的场景卡：
- 初级：友善询问
- 中级：有些质疑
- 高级：敌意挑战

## 五、培训后跟进

### 立即跟进（培训后1周内）

1. **发送感谢信**
   - 感谢参与
   - 总结重点
   - 附上电子资源

2. **收集反馈**
   - 发送评估问卷
   - 汇总分析结果
   - 记录改进建议

3. **建立联系**
   - 创建学员群组
   - 提供后续支持渠道

### 短期跟进（1-3个月）

1. **定期联系**
   - 每月一次简短问候
   - 询问使用情况
   - 回应问题

2. **资源支持**
   - 提供更新材料
   - 分享成功案例
   - 解答疑难

3. **同伴支持**
   - 促进学员间联系
   - 组织分享会
   - 建立学习小组

### 长期跟进（3个月以上）

1. **进阶培训**
   - 识别有潜力的学员
   - 邀请参加进阶培训
   - 培养成为新培训师

2. **持续评估**
   - 追踪长期影响
   - 收集使用案例
   - 更新培训内容

3. **社群建设**
   - 年度聚会
   - 在线社群
   - 资源共享平台

## 六、培训师自我成长

### 持续学习

- 定期重读材料，保持熟悉
- 关注相关学术发展
- 参加更高阶培训
- 与其他培训师交流

### 反思实践

每次培训后反思：
- 什么做得好？
- 什么需要改进？
- 学员的反馈如何？
- 下次如何调整？

### 灵命更新

- 保持个人灵修
- 在教会中服事
- 谦卑学习的心
- 为培训事工祷告
