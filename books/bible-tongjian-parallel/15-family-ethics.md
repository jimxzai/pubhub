# 第十五卷：家庭与伦理——圣经家庭观与中国家庭观

> 当孝敬父母，使你的日子在耶和华你神所赐你的地上得以长久。——出埃及记 20:12
> 身体发肤，受之父母，不敢毁伤，孝之始也。——《孝经》

---

## 卷首：家庭的意义

家庭是社会的基本单位。

在圣经中，家庭是**神设立的第一个制度**——先于国家、先于教会。
在中国文化中，家庭是**伦理的根基**——"修身齐家治国平天下"。

本卷将对照圣经与中国传统的家庭观，探讨：
- 婚姻的本质
- 亲子关系
- 孝道的理解
- 家庭伦理
- 家庭与社会

---

## 一、婚姻的本质

### 1. 圣经的婚姻观

#### 创造秩序中的婚姻

**创世记的设计**：

> "耶和华神说：'那人独居不好，我要为他造一个配偶帮助他。'"（创2:18）
>
> "因此，人要离开父母，与妻子连合，二人成为一体。"（创2:24）

**婚姻的要素**：

| 要素 | 经文 | 意义 |
|------|------|------|
| 神设立 | 创2:18 | 不是人的发明 |
| 一男一女 | 创2:24 | 性别区分 |
| 离开父母 | 创2:24 | 建立新家庭 |
| 连合 | 创2:24 | 盟约关系 |
| 一体 | 创2:24 | 亲密合一 |

#### 新约的深化

**以弗所书5章**：

> "丈夫是妻子的头，如同基督是教会的头……你们作丈夫的，要爱你们的妻子，正如基督爱教会，为教会舍己。"（弗5:23,25）

婚姻是**基督与教会关系的图像**。

#### 婚姻的目的

| 目的 | 经文 | 说明 |
|------|------|------|
| 相伴 | 创2:18 | "独居不好" |
| 生养 | 创1:28 | "生养众多" |
| 圣洁 | 林前7:2 | 避免淫乱 |
| 见证 | 弗5:32 | 基督与教会 |

### 2. 中国传统的婚姻观

#### 婚姻的定义

**《礼记·昏义》**：

> "昏礼者，将合二姓之好，上以事宗庙，而下以继后世也。故君子重之。"

婚姻的核心是：
- **合二姓之好**——家族联姻
- **事宗庙**——祭祀祖先
- **继后世**——延续香火

#### 婚姻的礼仪（六礼）

| 礼 | 内容 |
|---|------|
| 纳采 | 男方送礼求婚 |
| 问名 | 询问女方姓名 |
| 纳吉 | 占卜吉凶 |
| 纳征 | 送聘礼 |
| 请期 | 确定婚期 |
| 亲迎 | 新郎迎娶 |

#### 婚姻中的角色

**《礼记》**：

> "夫妇之道，参配阴阳。"
> "妻者，齐也。"

传统观念：
- 夫为妻纲
- 男主外女主内
- 妻子从属于丈夫

### 3. 对照与反思

| 维度 | 圣经婚姻观 | 中国传统婚姻观 |
|------|-----------|---------------|
| **设立者** | 神 | 社会/祖先 |
| **核心目的** | 相伴、见证基督 | 传宗接代、祭祀祖先 |
| **关系本质** | 盟约 | 社会契约 |
| **夫妻关系** | 相爱、顺服 | 夫为妻纲 |
| **神学意义** | 基督与教会 | 阴阳配合 |
| **可否离婚** | 原则上不可（太19:6） | 有条件可以（七出） |

**神学反思**：

圣经的婚姻观强调：
1. **盟约性**——不只是契约，是神面前的盟约
2. **平等性**——"你们作丈夫的，要爱妻子"（弗5:25）
3. **基督论**——婚姻指向基督与教会

中国传统婚姻观强调：
1. **家族性**——婚姻是两个家族的事
2. **功能性**——传宗接代是核心
3. **等级性**——男尊女卑

## 二、亲子关系

### 1. 圣经的亲子观

#### 父母的责任

**申命记6章**：

> "我今日所吩咐你的话都要记在心上，也要殷勤教训你的儿女。"（申6:6-7）

| 责任 | 经文 | 内容 |
|------|------|------|
| 教导 | 申6:7 | 传递信仰 |
| 管教 | 箴13:24 | 爱的管教 |
| 供养 | 林后12:14 | 物质供应 |
| 不惹气 | 弗6:4 | 不激怒儿女 |

#### 儿女的责任

**十诫**：

> "当孝敬父母。"（出20:12）

| 责任 | 经文 | 内容 |
|------|------|------|
| 孝敬 | 出20:12 | 尊重父母 |
| 听从 | 弗6:1 | 顺服父母 |
| 赡养 | 提前5:4 | 照顾父母 |

#### 平衡与限度

> "人到我这里来，若不爱我胜过爱自己的父母……就不能作我的门徒。"（路14:26）

孝敬父母有限度——当父母的要求与神的命令冲突时，应当顺从神。

### 2. 中国传统的亲子观

#### 孝道的核心地位

**《孝经》**：

> "夫孝，德之本也，教之所由生也。"

孝是**一切美德的根本**。

#### 孝的层次

**《孝经》**：

| 层次 | 内容 |
|------|------|
| 始于事亲 | 服侍父母 |
| 中于事君 | 忠于君主 |
| 终于立身 | 成就自己 |

#### 孝的具体表现

| 表现 | 内容 |
|------|------|
| 养亲 | 供养父母 |
| 敬亲 | 尊重父母 |
| 顺亲 | 顺从父母 |
| 谏亲 | 规劝父母（不可愚孝） |
| 继志 | 继承父母志向 |
| 祭亲 | 祭祀先人 |

#### 极端的孝

**二十四孝**中的一些故事：

- 郭巨埋儿——为养母而埋子
- 卧冰求鲤——卧冰为母求鱼
- 尝粪忧心——尝父亲粪便判断病情

这些故事反映了孝道的极端化。

### 3. 对照与反思

| 维度 | 圣经亲子观 | 中国传统亲子观 |
|------|-----------|---------------|
| **孝的地位** | 十诫之一，但不是最高 | 德之本，百善孝为先 |
| **孝的限度** | 神的命令高于父母 | 几乎无限度（愚孝） |
| **孝的内容** | 孝敬、赡养 | 顺从、祭祀 |
| **父母责任** | 教导、管教、不惹气 | 教养、传承 |
| **宗教意义** | 敬畏神的延伸 | 孝本身是宗教 |

**神学反思**：

圣经肯定孝敬父母，但：
1. **神高于父母**——"爱父母过于爱我的，不配作我的门徒"
2. **孝敬≠盲从**——当父母要求违背神时，不能顺从
3. **不祭祖**——父母死后不需要祭祀

中国孝道的问题：
1. **孝的绝对化**——将孝提升为最高美德
2. **祭祖**——与敬拜独一神冲突
3. **愚孝**——某些极端做法违背人性

## 三、家庭伦理

### 1. 夫妻伦理

#### 圣经的夫妻伦理

**以弗所书5章**：

| 角色 | 责任 | 榜样 |
|------|------|------|
| 丈夫 | 爱妻子 | 如基督爱教会 |
| 丈夫 | 舍己 | 为妻子舍命 |
| 妻子 | 顺服丈夫 | 如教会顺服基督 |
| 妻子 | 敬重丈夫 | 弗5:33 |

**注意**：
- 丈夫的"头"是服务性的领导，不是压制
- 妻子的顺服是对爱的回应
- 彼此顺服（弗5:21）

#### 中国传统的夫妻伦理

**三从四德**：

三从：
- 未嫁从父
- 既嫁从夫
- 夫死从子

四德：
- 妇德
- 妇言
- 妇容
- 妇功

#### 对照

| 维度 | 圣经 | 中国传统 |
|------|------|---------|
| **丈夫角色** | 爱、舍己、服务 | 主导、权威 |
| **妻子角色** | 顺服、敬重 | 三从四德 |
| **相互关系** | 彼此顺服 | 等级分明 |
| **基础** | 基督的爱 | 阴阳秩序 |

### 2. 亲子伦理

#### 圣经的亲子伦理

**父母对子女**：

> "你们作父亲的，不要惹儿女的气，只要照着主的教训和警戒养育他们。"（弗6:4）

**子女对父母**：

> "你们作儿女的，要在主里听从父母，这是理所当然的。"（弗6:1）

#### 中国传统的亲子伦理

**父慈子孝**：

- 父要慈爱
- 子要孝顺

**《三字经》**：

> "养不教，父之过。"

#### 对照

| 维度 | 圣经 | 中国传统 |
|------|------|---------|
| **父母责任** | 教导信仰、不惹气 | 教养、传家 |
| **子女责任** | 在主里听从 | 孝顺（几乎无条件） |
| **关系基调** | 爱与尊重 | 等级与服从 |
| **终极目标** | 认识神 | 成人成才 |

### 3. 兄弟伦理

#### 圣经的兄弟观

正面：
> "看哪，弟兄和睦同居是何等地善，何等地美！"（诗133:1）

负面案例：
- 该隐杀亚伯
- 约瑟被兄弟卖
- 押沙龙杀暗嫩

#### 中国的兄弟观

**悌**：

> "弟子入则孝，出则悌。"（《论语》）

**兄友弟恭**：

- 兄长要友爱
- 弟弟要恭敬

#### 对照

两者都强调兄弟和睦，但：
- 圣经更多呈现兄弟冲突的真实（不回避）
- 中国更强调理想的"兄友弟恭"

## 四、家庭与社会

### 1. 家国关系

#### 圣经的观点

家庭是社会的基本单位，但：
- 国家不能取代家庭
- 家庭不是最高效忠对象
- 神高于家庭和国家

> "人的仇敌就是自己家里的人。"（太10:36）——当家庭阻挡人跟随神时

#### 中国的观点

**齐家治国**：

> "修身、齐家、治国、平天下"（《大学》）

家是治国的基础，国是家的延伸：
- 君如父
- 臣如子
- 家国同构

#### 对照

| 维度 | 圣经 | 中国传统 |
|------|------|---------|
| **家国关系** | 家庭重要但不是最高 | 家国同构 |
| **忠孝关系** | 神高于家、国 | 忠孝一体 |
| **个人定位** | 神的儿女 | 家的成员 |

### 2. 家庭与信仰传承

#### 圣经的信仰传承

**申命记6章**：

> "我今日所吩咐你的话都要记在心上，也要殷勤教训你的儿女。无论你坐在家里，行在路上，躺下，起来，都要谈论。"（申6:6-7）

家庭是**信仰传承的主要场所**。

#### 中国的文化传承

家庭也是文化传承的场所：
- 家训
- 家风
- 家学

例：《颜氏家训》《朱子家训》

#### 对照

两者都强调家庭的教育功能，但内容不同：
- 圣经：传承对神的认识
- 中国：传承道德文化

## 五、当代应用

### 1. 基督徒的家庭挑战

| 挑战 | 圣经原则 |
|------|---------|
| 祭祖问题 | 敬重祖先可以，祭拜祖先不可 |
| 婚姻对象 | 信与不信不可同负一轭 |
| 孝道与信仰 | 孝敬父母，但神第一 |
| 家庭敬拜 | 建立家庭祭坛 |

### 2. 肯定与批判

**可以肯定的**：

| 中国传统 | 圣经支持 |
|---------|---------|
| 孝敬父母 | 十诫之一 |
| 家庭和谐 | 圣经也强调 |
| 教育子女 | 圣经也命令 |
| 婚姻忠诚 | 圣经更强调 |

**需要批判的**：

| 中国传统 | 圣经观点 |
|---------|---------|
| 祭祖 | 不可敬拜受造物 |
| 愚孝 | 神高于父母 |
| 男尊女卑极端化 | 在基督里平等 |
| 家高于神 | 神高于家 |

### 3. 建立基督化家庭

| 要素 | 内容 |
|------|------|
| 家庭敬拜 | 一起读经、祷告 |
| 信仰传承 | 教导儿女认识神 |
| 夫妻相爱 | 如基督与教会 |
| 孝敬父母 | 在主里孝敬 |
| 福音见证 | 家庭是福音的见证 |

## 六、家庭伦理总结

### 核心对照表

| 主题 | 圣经 | 中国传统 | 核心差异 |
|------|------|---------|---------|
| 婚姻本质 | 盟约，基督与教会 | 契约，传宗接代 | 神学vs功能 |
| 孝道 | 敬畏神的延伸 | 德之本 | 有限vs绝对 |
| 祭祖 | 不可 | 核心义务 | 冲突 |
| 夫妻关系 | 爱与顺服 | 夫为妻纲 | 相互vs等级 |
| 亲子关系 | 在主里 | 几乎绝对 | 有限vs无限 |
| 家国关系 | 神>家>国 | 家国同构 | 超越vs同构 |

### 神学总结

1. **神设立家庭**——家庭不是人的发明，是神的设计
2. **神高于家庭**——家庭不是偶像
3. **家庭指向基督**——婚姻是基督与教会的图像
4. **爱是核心**——不是等级，是爱
5. **传承信仰**——家庭是信仰传承的主要场所
