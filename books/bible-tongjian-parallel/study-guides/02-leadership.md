# 查经材料：领导力（12周）

> 基于《圣经与资治通鉴对照》第二、三卷

---

## 课程简介

**适合对象**：职场基督徒、教会领袖、小组长
**时长**：12周，每周1-2小时
**目标**：
1. 从圣经和历史学习领导原则
2. 理解权力的试探与陷阱
3. 建立仆人式领导观
4. 在职场和教会实践圣经领导力

---

## 第一单元：权力的本质（第1-3周）

### 第一周：权力从何而来？

#### 本周经文
- 罗马书 13:1-7（权柄出于神）
- 但以理书 2:20-21（神废王立王）

#### 核心内容

**一、权力的来源**

> "在上有权柄的，人人当顺服他，因为没有权柄不是出于神的。凡掌权的都是神所命的。"（罗13:1）

| 观点 | 内容 |
|------|------|
| 圣经观点 | 权柄来自神 |
| 中国观点 | 天命所归 |
| 世俗观点 | 社会契约 |

**二、资治通鉴的权力观**

> "秦失其鹿，天下共逐之，于是高材疾足者先得焉。"——《史记》

中国历史中，权力常被视为"可争夺之物"。

**三、基督徒的权力观**

权力是**托付**，不是**拥有**：
- 神是权力的真正来源
- 领袖是管家，不是主人
- 权力是为服务，不是为私利

#### 讨论问题
1. 权力本身是好是坏？
2. 如何理解"顺服掌权者"与"抵抗暴政"的张力？
3. 你在工作中如何看待权力？

### 第二周：权力的腐化

#### 本周经文
- 撒母耳记上 8章（以色列求王）
- 列王纪上 21章（亚哈夺拿伯葡萄园）

#### 核心内容

**一、权力的危险**

撒母耳警告以色列人立王的后果：
> "管辖你们的王必这样行：他必派你们的儿子……你们的女儿……他必取你们最好的田地……"（撒上8:11-14）

**二、亚哈与拿伯**

权力的腐化过程：
1. 贪欲——想要拿伯的葡萄园
2. 受挫——拿伯拒绝
3. 纵容恶——耶洗别出谋
4. 谋杀——杀害无辜
5. 侵占——夺取产业

**三、资治通鉴的权力腐化案例**

- 赵高指鹿为马
- 王莽篡汉
- 杨国忠弄权

**四、权力腐化的模式**

```
权力 → 特权 → 骄傲 → 滥用 → 败亡
```

#### 讨论问题
1. 为什么权力会腐化人？
2. 你见过哪些权力腐化的例子？
3. 如何防止自己被权力腐化？

### 第三周：用权的原则

#### 本周经文
- 申命记 17:14-20（君王的律法）
- 马可福音 10:42-45（非要受人服事）

#### 核心内容

**一、圣经对君王的要求（申17章）**

| 要求 | 经文 | 意义 |
|------|------|------|
| 不可为自己加添马匹 | 17:16 | 不倚靠武力 |
| 不可多立妃嫔 | 17:17 | 不沉溺情欲 |
| 不可为自己多积金银 | 17:17 | 不贪财 |
| 要抄录律法书 | 17:18 | 以神的话为准则 |
| 诵读律法 | 17:19 | 时刻学习 |
| 不可心高气傲 | 17:20 | 保持谦卑 |

**二、耶稣的领导观**

> "你们知道，外邦人有尊为君王的，治理他们，有大臣操权管束他们。只是在你们中间，不是这样。你们中间，谁愿为大，就必作你们的用人；谁愿为首，就必作众人的仆人。"（可10:42-44）

**三、仆人式领导**

| 世俗领导 | 仆人式领导 |
|---------|-----------|
| 被服事 | 服事人 |
| 追求地位 | 追求贡献 |
| 权力控制 | 权力赋能 |
| 自我中心 | 他人中心 |

#### 讨论问题
1. "仆人式领导"在现代职场可行吗？
2. 服事与领导如何平衡？
3. 你如何在工作中实践仆人式领导？

## 第二单元：领袖的品格（第4-6周）

### 第四周：忠心

#### 本周经文
- 但以理书 6章（但以理在狮子坑）
- 马太福音 25:14-30（忠心的仆人）

#### 核心内容

**一、但以理的忠心**

| 处境 | 回应 | 结果 |
|------|------|------|
| 不吃王膳 | 坚持信仰 | 神赐恩 |
| 释梦 | 归荣耀给神 | 被提升 |
| 禁止祷告 | 继续祷告 | 下狮子坑 |
| 狮子坑 | 信靠神 | 神拯救 |

**二、忠心的含义**

> "好，你这又良善又忠心的仆人，你在不多的事上有忠心，我要把许多事派你管理。"（太25:21）

忠心 = 在小事上可靠 + 长期持守 + 对主人（神）负责

**三、资治通鉴中的忠臣**

| 人物 | 忠诚对象 | 结局 |
|------|---------|------|
| 诸葛亮 | 刘备、刘禅 | 鞠躬尽瘁 |
| 魏征 | 唐太宗 | 成就明君 |
| 文天祥 | 宋朝 | 殉国 |

**四、忠心的优先序**

但以理的忠诚有优先序：**忠于神 > 忠于王**

> "王啊，你的禁令不能改变我对神的忠诚。"（但6章的实质）

#### 讨论问题
1. 忠心于公司与忠心于神会冲突吗？
2. 如何在职场保持信仰的忠诚？
3. 忠心的界限在哪里？

### 第五周：正直

#### 本周经文
- 撒母耳记下 12:1-14（拿单责备大卫）
- 箴言 11:3（正直人的纯正必引导自己）

#### 核心内容

**一、拿单的正直**

面对君王的罪，先知拿单：
1. 用比喻引入（智慧）
2. 直接指出罪（勇气）
3. 宣告神的审判（忠于神）
4. 也传达神的赦免（恩典）

**二、正直的代价**

| 圣经人物 | 正直的行为 | 代价 |
|---------|-----------|------|
| 拿单 | 责备大卫 | 冒生命危险 |
| 以利亚 | 对抗亚哈 | 被追杀 |
| 耶利米 | 预言审判 | 被囚禁 |
| 施洗约翰 | 指责希律 | 被杀 |

**三、中国的直臣**

| 人物 | 正直的行为 | 代价 |
|------|-----------|------|
| 比干 | 谏商纣 | 被剖心 |
| 魏征 | 谏太宗 | 被尊重 |
| 海瑞 | 上疏嘉靖 | 被囚禁 |

**四、正直的智慧**

正直不等于鲁莽：
- 拿单用比喻
- 以斯帖选择时机
- 尼希米先祷告

#### 讨论问题
1. 面对上级的错误，你会怎么做？

### 第六周：谦卑

#### 本周经文
- 民数记 12:3（摩西为人极其谦和）
- 腓立比书 2:3-11（基督的谦卑）

#### 核心内容

**一、摩西的谦卑**

> "摩西为人极其谦和，胜过世上的众人。"（民12:3）

摩西的谦卑表现：
- 不愿接受领导任务（出3-4章）
- 听从岳父建议（出18章）
- 为百姓代求（出32章）
- 被米利暗亚伦攻击时不自辩（民12章）

**二、基督的谦卑**

> "他本有神的形像，不以自己与神同等为强夺的；反倒虚己，取了奴仆的形像，成为人的样式；既有人的样子，就自己卑微，存心顺服，以至于死，且死在十字架上。"（腓2:6-8）

**三、谦卑与自卑的区别**

| 谦卑 | 自卑 |
|------|------|
| 正确看自己 | 过低看自己 |
| 归荣耀给神 | 否定神的创造 |
| 能服务他人 | 无法服务 |
| 有安全感 | 没有安全感 |

**四、资治通鉴中的骄傲与败亡**

| 人物 | 骄傲表现 | 结局 |
|------|---------|------|
| 项羽 | 不听劝谏 | 乌江自刎 |
| 王莽 | 自命不凡 | 身死国灭 |
| 隋炀帝 | 好大喜功 | 国破身亡 |

#### 讨论问题
1. 领袖如何保持谦卑？
2. 谦卑会不会被人看轻？
3. 你见过谦卑的领袖吗？他们有什么特点？
