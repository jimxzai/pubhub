# 查经材料五：属灵争战

> 基于《圣经与资治通鉴时空宇宙观》第九卷
> 6周查经课程

---

## 课程简介

### 目标
帮助信徒认识属灵争战的真实，装备属灵军装，学会分辨仇敌的诡计，靠主站立得稳，活出得胜的生命。

### 适用对象
- 想要更深认识属灵真相的信徒
- 在生命中遭遇属灵争战的基督徒
- 渴望成长为代祷勇士的门徒

### 使用方法
- 每周一课，每课约90-120分钟
- 建议6-12人小组进行
- 课前阅读指定经文和参考材料
- 课中讨论、分享、祷告

---

## 第一周：属灵争战的真实

### 核心经文
> "因我们并不是与属血气的争战，乃是与那些执政的、掌权的、管辖这幽暗世界的，以及天空属灵气的恶魔争战。"（弗6:12）

### 学习目标
1. 认识属灵争战的真实性
2. 理解宇宙性的善恶冲突
3. 避免两个极端：忽视与过度关注

### 核心内容

**一、圣经中的属灵争战**

| 战场 | 经文 | 描述 |
|------|------|------|
| 天上 | 启12:7-9 | 米迦勒与龙争战 |
| 地上 | 创3:15 | 女人后裔与蛇 |
| 人心 | 加5:17 | 情欲与圣灵相争 |
| 教会 | 彼前5:8 | 仇敌如吼叫的狮子 |

**二、属灵争战的本质**

```
┌─────────────────────────────────────────────────────┐
│                  属灵争战的结构                       │
├─────────────────────────────────────────────────────┤
│                                                     │
│      神的国度  ◀══════════════▶  撒但的国度         │
│         ↓                           ↓              │
│      真理                         谎言              │
│      生命                         死亡              │
│      光明                         黑暗              │
│      自由                         捆绑              │
│      爱                           恨                │
│                                                     │
│            信徒在这场争战中选边站                    │
│                                                     │
└─────────────────────────────────────────────────────┘
```

**三、两个危险极端**

| 极端 | 表现 | 危险 |
|------|------|------|
| **忽视** | "没有魔鬼这回事" | 不设防，被动挨打 |
| **过度关注** | 处处看见魔鬼 | 恐惧、迷信、责任外推 |

平衡的态度：
- 承认撒但存在且活跃
- 但不惧怕，因基督已得胜
- 专注于神，而非仇敌

**四、资治通鉴中的"争战"视角**

中国历史充满权力争斗，表面是人与人的斗争，背后有属灵层面：

| 表面 | 深层 |
|------|------|
| 政治斗争 | 善恶之争 |
| 改朝换代 | 神的主权 |
| 忠奸对立 | 光暗对立 |
| 兴衰循环 | 神的审判 |

> "夫兵者，不祥之器，圣人不得已而用之。"——老子

中国智者看到战争的可怕，却未看到背后更深的属灵争战。

### 讨论问题

1. 你相信属灵争战是真实的吗？有什么经历让你这样认为？
2. 你倾向于哪个极端——忽视还是过度关注？为什么？
3. 以弗所书6:12说我们争战的对象"不是属血气的"，这如何改变你看人际冲突的方式？
4. 中国历史中的争战，有哪些可以看出属灵层面的痕迹？

### 生活应用

- **本周默想**：我们的争战不是与属血气的争战
- **具体行动**：列出你目前生命中的"争战"领域，思考其属灵层面
- **祷告焦点**：求神开你的眼睛，看见属灵的真相

## 第二周：认识仇敌

### 核心经文
> "免得撒但趁着机会胜过我们，因我们并非不晓得他的诡计。"（林后2:11）

### 学习目标
1. 认识撒但的本质和策略
2. 理解仇敌的名称所揭示的特性
3. 学会分辨仇敌的诡计

### 核心内容

**一、撒但的名称与本质**

| 名称 | 含义 | 经文 |
|------|------|------|
| 撒但（Satan） | 敌对者、控告者 | 伯1:6 |
| 魔鬼（Diabolos） | 毁谤者 | 太4:1 |
| 古蛇 | 欺骗者 | 创3:1; 启12:9 |
| 说谎者之父 | 一切谎言的源头 | 约8:44 |
| 世界的王 | 暂时统管堕落世界 | 约12:31 |
| 吼叫的狮子 | 寻找可吞吃的人 | 彼前5:8 |
| 光明的天使 | 伪装者 | 林后11:14 |

**二、撒但的起源与堕落**

```
受造的荣美
    ↓
骄傲（"我要升到高云之上，我要与至上者同等"赛14:14）
    ↓
堕落
    ↓
带领三分之一天使反叛（启12:4）
    ↓
被逐出天堂
    ↓
成为神和人的仇敌
```

**三、撒但的策略（诡计）**

| 策略 | 方式 | 例子 |
|------|------|------|
| **欺骗** | 扭曲真理 | "神岂是真说？"（创3:1） |
| **控告** | 定罪、羞耻 | 控告约书亚（亚3:1） |
| **试探** | 引诱犯罪 | 试探耶稣（太4章） |
| **分裂** | 破坏关系 | 该隐杀亚伯 |
| **恐吓** | 制造惧怕 | 歌利亚骂阵 |
| **迷惑** | 混淆是非 | 假先知 |
| **拦阻** | 阻挡善工 | 撒但拦阻保罗（帖前2:18） |

**四、资治通鉴中的"撒但策略"**

中国历史人物使用的诡计，映照出仇敌的作为：

| 撒但策略 | 历史对应 |
|----------|----------|
| 欺骗 | 赵高"指鹿为马" |
| 分裂 | 离间计（周瑜与蒋干） |
| 恐吓 | 残暴震慑（秦始皇） |
| 控告 | 诬陷忠良（岳飞之冤） |
| 试探 | 利诱（李斯受贿） |

> "兵者，诡道也。故能而示之不能，用而示之不用。"——《孙子兵法》

孙子的"诡道"，正是撒但在人间的投影。

### 讨论问题

1. 撒但的七个名称中，哪一个最帮助你理解仇敌的本质？
2. 你个人最容易中的是撒但的哪一种诡计？
3. "我们并非不晓得他的诡计"——为什么"知道"很重要？
4. 中国历史中的"诡诈"，如何帮助我们理解属灵层面的争战？

### 生活应用

- **本周默想**：知己知彼，方能得胜
- **具体行动**：本周特别警醒仇敌最常使用的那一招（针对你个人）
- **祷告焦点**：求神赐分辨的智慧，识破仇敌的诡计

## 第三周：基督的得胜

### 核心经文
> "儿女既同有血肉之体，他也照样亲自成了血肉之体，特要借着死败坏那掌死权的，就是魔鬼。"（来2:14）

### 学习目标
1. 认识基督已经得胜
2. 理解十字架的决定性胜利
3. 建立"已然/未然"的得胜观

### 核心内容

**一、基督得胜的宣告**

| 事件 | 经文 | 意义 |
|------|------|------|
| 道成肉身 | 约1:14 | 神进入战场 |
| 旷野试探 | 太4:1-11 | 得胜试探 |
| 赶鬼医病 | 可3:27 | 捆绑壮士 |
| 十字架 | 西2:15 | 决定性胜利 |
| 复活 | 林前15:54-57 | 死亡被吞灭 |
| 升天 | 弗1:20-21 | 超越一切执政掌权 |

**二、十字架的奥秘**

```
世人眼中          属灵真相
┌─────────┐      ┌─────────────────┐
│ 失败    │  →  │ 最大的胜利      │
│ 软弱    │  →  │ 神大能的彰显    │
│ 羞辱    │  →  │ 仇敌被示众      │
│ 死亡    │  →  │ 死亡被击败      │
└─────────┘      └─────────────────┘

"既将一切执政的、掌权的掳来，
明显给众人看，就仗着十字架夸胜。"
            （西2:15）
```

**三、"已然/未然"的张力**

| 已然（Already） | 未然（Not Yet） |
|-----------------|-----------------|
| 撒但已被击败 | 撒但尚未被完全消灭 |
| 战争已决定胜负 | 战斗仍在继续 |
| 我们在基督里得胜 | 我们仍需争战 |
| 神国已降临 | 神国尚未完全 |

比喻：诺曼底登陆（D-Day）是决定性胜利，但战争直到V-Day才完全结束。十字架是属灵的D-Day。

**四、资治通鉴对照：决定性战役**

| 战役 | 意义 | 属灵类比 |
|------|------|----------|
| 牧野之战 | 周克商，天命转移 | 十字架：新时代开始 |
| 巨鹿之战 | 秦朝覆灭已定 | 撒但权势被击溃 |
| 赤壁之战 | 三分天下格局奠定 | 历史走向被决定 |
| 淝水之战 | 前秦南侵失败 | 仇敌攻势被挡住 |

> "项籍兵败，王翳取其头...五人皆封侯。"

人间战役有决定性时刻，属灵战争也是——十字架是那个时刻。

### 讨论问题

1. "基督已经得胜"——这对你面对属灵争战有什么意义？
2. 西2:15说基督"明显给众人看"仇敌的失败，这如何安慰你？
3. "已然/未然"的张力如何解释我们仍然经历挣扎和失败？
4. D-Day的比喻如何帮助你理解当前的处境？

### 生活应用

- **本周默想**：战争的胜负已定，我是站在得胜的一方
- **具体行动**：每当感到软弱时，宣告基督的得胜
- **祷告焦点**：感谢基督十字架上的得胜，求主帮我活出得胜

## 第四周：全副军装（上）

### 核心经文
> "所以，要拿起神所赐的全副军装，好在磨难的日子抵挡仇敌，并且成就了一切，还能站立得住。"（弗6:13）

### 学习目标
1. 认识属灵军装的重要性
2. 学习前三件军装的含义和应用
3. 操练穿戴军装

### 核心内容

**一、军装的整体意义**

```
┌────────────────────────────────────────┐
│        全副军装（Panoply）              │
├────────────────────────────────────────┤
│  来源：神所赐的                        │
│  目的：抵挡仇敌                        │
│  结果：站立得住                        │
│  时机：磨难的日子                      │
│  要求：全副——不可部分装备              │
└────────────────────────────────────────┘
```

**二、真理的带子（弗6:14a）**

| 层面 | 内容 | 应用 |
|------|------|------|
| **客观真理** | 神的话——圣经 | 认识并持守真道 |
| **主观真诚** | 诚实正直的心 | 活出真实的生命 |
| **基督自己** | 耶稣是真理 | 与基督联合 |

> "你们必晓得真理，真理必叫你们得以自由。"（约8:32）

仇敌用谎言捆绑，真理使人得释放。

**三、公义的护心镜（弗6:14b）**

| 层面 | 内容 | 应用 |
|------|------|------|
| **神的公义** | 因信称义 | 不靠自己的义 |
| **生活的义** | 圣洁生活 | 行为与信仰一致 |
| **保护心** | 胜过控告 | 心不被定罪 |

仇敌攻击我们的"心"：控告、定罪、羞耻。公义的护心镜保护我们。

**四、平安福音的鞋（弗6:15）**

| 层面 | 内容 | 应用 |
|------|------|------|
| **内心平安** | 与神和好 | 在争战中有安息 |
| **传福音预备** | 随时准备 | 分享好消息 |
| **站立稳固** | 福音根基 | 不被动摇 |

> "报福音、传喜信的人，他们的脚踪何等佳美！"（罗10:15）

**五、资治通鉴中的"装备"**

中国古代士兵的装备与属灵军装对照：

| 古代装备 | 功能 | 属灵对应 |
|----------|------|----------|
| 腰带 | 束紧，行动敏捷 | 真理的带子 |
| 铠甲 | 护心，保护要害 | 公义的护心镜 |
| 战靴 | 稳固，行军远征 | 平安福音的鞋 |

> "工欲善其事，必先利其器。"——《论语》

属灵争战也需要装备，不是人造的，而是神赐的。

### 讨论问题

1. 为什么是"全副"军装？部分装备有什么危险？
2. 真理如何抵挡谎言？你生命中有哪些谎言需要被真理取代？
3. "公义的护心镜"如何帮助你面对控告和羞耻？
4. "平安福音的鞋"如何让你在争战中站稳，同时预备传福音？

### 生活应用

- **本周默想**：每天穿戴神所赐的军装
- **具体行动**：每天早晨祷告"穿戴"这三件军装
- **祷告焦点**：求神帮助我活在真理、公义、平安中

## 第五周：全副军装（下）

### 核心经文
> "拿着信德的藤牌，可以灭尽那恶者一切的火箭；并戴上救恩的头盔，拿着圣灵的宝剑，就是神的道。"（弗6:16-17）

### 学习目标
1. 学习后三件军装的含义和应用
2. 认识祷告在争战中的关键作用
3. 整合全副军装的操练

### 核心内容

**一、信德的藤牌（弗6:16）**

| 层面 | 内容 | 应用 |
|------|------|------|
| **信心** | 相信神的应许 | 抵挡怀疑 |
| **藤牌** | 大盾牌，保护全身 | 覆盖所有攻击 |
| **灭火** | 熄灭火箭 | 化解试探 |

```
撒但的火箭              信心的回应
┌─────────────┐        ┌─────────────────┐
│ "神不爱你"  │   →   │ "神爱我到舍命"   │
│ "你不会成功"│   →   │ "靠主凡事都能"   │
│ "神不会饶恕"│   →   │ "基督的血洗净"   │
│ "你太软弱"  │   →   │ "神的能力在软弱显完全" │
└─────────────┘        └─────────────────┘
```

**二、救恩的头盔（弗6:17a）**

| 层面 | 内容 | 应用 |
|------|------|------|
| **保护头脑** | 思想的保护 | 更新心意 |
| **救恩确据** | 知道自己得救 | 不被怀疑动摇 |
| **盼望** | 帖前5:8"救恩盼望的头盔" | 坚定向前 |

仇敌常攻击我们的"头"——思想、理性、救恩确据。头盔保护这一切。

**三、圣灵的宝剑（弗6:17b）**

| 层面 | 内容 | 应用 |
|------|------|------|
| **神的道** | rhema，具体的话 | 在具体处境应用 |
| **攻击性武器** | 唯一的进攻武器 | 主动出击 |
| **耶稣的榜样** | 旷野三次"经上记着" | 用神的话抵挡 |

```
耶稣的榜样（太4:1-11）
┌────────────────────────────────────────────────┐
│ 试探1："将石头变饼"                            │
│ 回应："人活着不是单靠食物..."（申8:3）         │
├────────────────────────────────────────────────┤
│ 试探2："跳下去，天使会保护"                    │
│ 回应："不可试探主你的神"（申6:16）             │
├────────────────────────────────────────────────┤
│ 试探3："拜我就给你万国"                        │
│ 回应："当拜主你的神，单要事奉他"（申6:13）     │
└────────────────────────────────────────────────┘
        每一次都用"神的道"作回应
```

**四、祷告——军装的能源（弗6:18）**

| 祷告特征 | 经文 | 意义 |
|----------|------|------|
| 靠着圣灵 | "靠着圣灵" | 在圣灵里祷告 |
| 多方祷告 | "多方祷告祈求" | 各种形式 |
| 随时祷告 | "随时" | 不断保持连接 |
| 儆醒祷告 | "儆醒不倦" | 警觉、坚持 |
| 代求 | "为众圣徒祈求" | 不只为自己 |

祷告不是第七件军装，而是穿戴军装的方式和争战的持续行动。

**五、资治通鉴中的"武器"**

| 古代武器 | 功能 | 属灵对应 |
|----------|------|----------|
| 盾牌 | 防御 | 信德的藤牌 |
| 头盔 | 护头 | 救恩的头盔 |
| 剑 | 攻击 | 圣灵的宝剑 |

> "匹夫无罪，怀璧其罪。"

有了好装备还不够，还要会使用。属灵军装也是如此。

### 讨论问题

1. 你经历过哪些"火箭"攻击？信心如何帮助你抵挡？
2. 为什么"救恩的确据"对属灵争战如此重要？
3. 如何让"神的道"成为你的宝剑？你熟悉多少可以应用的经文？
4. 祷告在你的属灵争战中扮演什么角色？

### 生活应用

- **本周默想**：神已经赐下一切争战所需的装备
- **具体行动**：背诵3节可以在试探中使用的经文
- **祷告焦点**：操练"靠着圣灵随时多方祷告"

## 第六周：得胜的生活

### 核心经文
> "感谢神，常率领我们在基督里夸胜，并借着我们在各处显扬那因认识基督而有的香气。"（林后2:14）

### 学习目标
1. 整合课程学习，活出得胜生命
2. 认识得胜的原则和实践
3. 委身于持续的属灵成长

### 核心内容

**一、得胜的根基**

```
得胜不是靠...            得胜是靠...
┌─────────────┐        ┌─────────────────┐
│ 自己的能力   │        │ 基督的得胜      │
│ 更多努力     │        │ 在基督里的地位  │
│ 更好方法     │        │ 圣灵的大能      │
│ 经验累积     │        │ 神的话语        │
│ 意志力       │        │ 祷告的能力      │
└─────────────┘        └─────────────────┘
"靠着爱我们的主，在这一切的事上
已经得胜有余了。"（罗8:37）
```

## 附录：带领者指南

### 课程特别注意事项

1. **平衡的态度**
   - 不要忽视属灵争战的真实
   - 也不要过度强调以致造成恐惧
   - 核心信息：基督已经得胜

2. **牧养敏感度**
   - 有些学员可能有属灵创伤
   - 有些可能来自强调"赶鬼"的背景
   - 有些可能完全不相信撒但存在
   - 以爱心和智慧回应不同背景

3. **实践操练**
   - 这是操练性课程，不只是知识
   - 鼓励学员每天"穿戴军装"
   - 在小组中操练祷告争战

### 带领技巧

1. **开场（10分钟）**
   - 分享上周"争战"经历
   - 见证神的帮助和得胜

2. **查经（40分钟）**
   - 朗读核心经文
   - 解释核心概念
   - 强调基督的得胜

3. **讨论（25分钟）**
   - 使用讨论问题
   - 允许分享个人经历
   - 注意保密和安全

4. **祷告（20分钟）**
   - 一起穿戴军装祷告
   - 为彼此的争战领域代祷
   - 宣告基督的得胜

5. **结束（5分钟）**
   - 总结要点
   - 鼓励本周操练
   - 预告下周

### 常见问题

**Q: 如何处理有"被鬼附"经历的学员？**
A: 私下关心了解，如有需要转介给有经验的牧者。小组中不宜深入处理。

**Q: 如果有人说一切问题都是"鬼的工作"怎么办？**
A: 温和指出人的罪性和责任，不是所有问题都是邪灵造成的。

**Q: 如何处理"赶鬼"的问题？**
A: 强调基督的权柄和教会的权柄，不要陷入戏剧化的"赶鬼"，重点是活出得胜的生命。

**Q: 如果有学员感到恐惧怎么办？**
A: 强调基督已经得胜，引导专注于神而非仇敌，必要时私下辅导。
