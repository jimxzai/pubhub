# 查经材料：智慧人生（8周）

> 基于《圣经与资治通鉴对照》第六卷

---

## 课程简介

**适合对象**：所有年龄的基督徒
**时长**：8周，每周1-2小时
**目标**：
1. 认识圣经智慧的本质
2. 学习箴言的生活智慧
3. 理解传道书的人生观
4. 在日常生活中应用智慧

---

## 第一周：智慧的开端

### 本周经文
- 箴言 1:1-7（敬畏耶和华是知识的开端）
- 箴言 9:10（敬畏耶和华是智慧的开端）

### 核心内容

#### 一、智慧的定义

> "敬畏耶和华是智慧的开端；认识至圣者便是聪明。"（箴9:10）

圣经的智慧不只是：
- 聪明（IQ）
- 知识（信息）
- 技能（能力）

而是：**在敬畏神的基础上，正确生活的能力**。

#### 二、敬畏的含义

"敬畏"（希伯来文：יִרְאָה yir'ah）包含：
- 敬重——认识神的伟大
- 畏惧——知道神的圣洁
- 顺服——遵行神的旨意

#### 三、与中国智慧的对比

| 维度 | 圣经智慧 | 中国智慧 |
|------|---------|---------|
| 根基 | 敬畏神 | 人的经验 |
| 来源 | 神的启示 | 圣人的教导 |
| 目标 | 荣耀神 | 成为君子 |
| 标准 | 神的话 | 传统规范 |

### 讨论问题
1. "敬畏神是智慧的开端"对你意味着什么？
2. 现代人追求的"智慧"与圣经智慧有何不同？
3. 如何在日常生活中培养敬畏神的心？

## 第二周：言语的智慧

### 本周经文
- 箴言 10:19-21, 12:18, 15:1-4, 18:21
- 雅各书 3:1-12

### 核心内容

#### 一、言语的力量

> "生死在舌头的权下；爱用舌头的，必吃它的果子。"（箴18:21）

#### 二、箴言论言语

| 经文 | 教导 |
|------|------|
| 箴10:19 | 多言多语难免有过 |
| 箴12:18 | 说话浮躁如刀刺人 |
| 箴15:1 | 回答柔和使怒消退 |
| 箴15:4 | 温良的舌是生命树 |
| 箴18:13 | 未曾听完就回答是愚昧 |

#### 三、当说与不当说

| 当说 | 不当说 |
|------|--------|
| 诚实的话 | 谎言 |
| 造就人的话 | 伤害人的话 |
| 柔和的话 | 暴躁的话 |
| 时机恰当的话 | 时机不当的话 |
| 赞美神的话 | 咒诅的话 |

#### 四、资治通鉴中的言语

- 毛遂自荐——善于言辞
- 商鞅说秦——言语说服
- 触龙说赵太后——智慧劝说

### 讨论问题
1. 你最容易在言语上犯什么错？
2. 如何做到"慢慢地说"（雅1:19）？
3. 社交媒体时代如何管理言语？

### 本周应用

## 第三周：工作的智慧

### 本周经文
- 箴言 6:6-11（向蚂蚁学习）
- 箴言 22:29, 24:30-34
- 歌罗西书 3:23-24

### 核心内容

#### 一、向蚂蚁学习

> "懒惰人哪，你去察看蚂蚁的动作就可得智慧。蚂蚁没有元帅，没有官长，没有君王，尚且在夏天预备食物，在收割时聚敛粮食。"（箴6:6-8）

蚂蚁的特点：
- 勤劳
- 自律
- 有远见
- 不需监督

#### 二、箴言论工作

| 经文 | 教导 |
|------|------|
| 箴10:4 | 手懒的要受贫穷 |
| 箴12:11 | 耕种田地的必得饱食 |
| 箴14:23 | 诸般勤劳都有益处 |
| 箴22:29 | 办事殷勤的必站在君王面前 |

#### 三、懒惰的危害

> "我经过懒惰人的田地、无知人的葡萄园，荆棘长满了地皮，刺草遮盖了田面，石墙也坍塌了。"（箴24:30-31）

#### 四、为主而做

> "无论做什么，都要从心里做，像是给主做的，不是给人做的。"（西3:23）

工作的态度转变：
- 不只为老板——为主
- 不只为薪水——为荣耀神
- 不只是谋生——是服事

### 讨论问题
1. 你如何看待自己的工作？
2. 勤劳与工作狂有何区别？
3. 如何在工作中荣耀神？

## 第五周：关系的智慧

### 本周经文
- 箴言 17:17, 18:24, 27:6, 27:17
- 传道书 4:9-12

### 核心内容

#### 一、真朋友

> "朋友乃时常亲爱，弟兄为患难而生。"（箴17:17）

> "滥交朋友的，自取败坏；但有一朋友比弟兄更亲密。"（箴18:24）

#### 二、朋友的特质

| 经文 | 特质 |
|------|------|
| 箴17:17 | 时常亲爱 |
| 箴18:24 | 亲密忠诚 |
| 箴27:6 | 忠诚责备 |
| 箴27:9 | 真诚劝告 |
| 箴27:17 | 互相磨砺 |

#### 三、交友的智慧

> "与智慧人同行的，必得智慧；和愚昧人作伴的，必受亏损。"（箴13:20）

| 当交的朋友 | 当避的朋友 |

| 诚实的 | 说谎的 |
| 有智慧的 | 嘲笑人的 |

> "有人攻胜孤身一人，若有二人便能敌挡他；三股合成的绳子不容易折断。"（传4:12）

## 第六周：家庭的智慧

### 本周经文
- 箴言 31:10-31（才德的妇人）
- 箴言 22:6（教养孩童）
- 以弗所书 5:22-6:4

### 核心内容

#### 一、才德的妇人（箴31章）

| 特质 | 经节 |
|------|------|
| 丈夫信任 | 31:11 |
| 勤劳工作 | 31:13-19 |
| 帮助穷人 | 31:20 |
| 预备家庭 | 31:21 |
| 言语智慧 | 31:26 |
| 敬畏神 | 31:30 |

#### 二、教养孩童

> "教养孩童，使他走当行的道，就是到老他也不偏离。"（箴22:6）

| 原则 | 说明 |
|------|------|
| 按其本性 | "当行的道"考虑个性 |
| 及早开始 | "孩童"时期 |
| 持续教导 | 到老不偏离 |

#### 三、管教的智慧

> "不忍用杖打儿子的，是恨恶他；疼爱儿子的，随时管教。"（箴13:24）

管教的目的：
- 不是发泄怒气
- 而是塑造品格
- 出于爱，不是恨

#### 四、家庭关系的秩序

以弗所书5-6章：
- 丈夫：爱妻子如基督爱教会
- 妻子：顺服丈夫如教会顺服基督
- 父母：不惹儿女的气，照主的教训养育
- 儿女：在主里听从父母

### 讨论问题
1. 箴言31章的妇人对今天有什么启发？
2. 如何平衡管教与慈爱？
3. 你的家庭最需要什么智慧？

## 第七周：传道书的智慧——人生的虚空

### 本周经文
- 传道书 1:1-11（虚空的虚空）
- 传道书 2:1-11（所罗门的实验）
- 传道书 12:13-14（总意）

### 核心内容

#### 一、虚空的宣告

> "虚空的虚空，虚空的虚空，凡事都是虚空。"（传1:2）

"虚空"（希伯来文：הֶבֶל hevel）意为：
- 气息——转瞬即逝
- 烟雾——抓不住
- 无意义——没有永恒价值

#### 二、所罗门的实验

所罗门尝试了：

| 尝试 | 经文 | 结论 |
|------|------|------|
| 智慧 | 传1:17-18 | 愁烦 |
| 快乐 | 传2:1-2 | 虚空 |
| 酒 | 传2:3 | 愚昧 |
| 建筑 | 传2:4-6 | 暂时 |
| 财富 | 传2:7-8 | 不满足 |
| 享乐 | 传2:8 | 虚空 |

**结论**：
> "后来，我察看我手所经营的一切事和我劳碌所成的功，谁知都是虚空，都是捕风。"（传2:11）

#### 三、日光之下与日光之上

传道书的关键词："日光之下"（出现29次）

| 日光之下 | 日光之上 |
|---------|---------|
| 人的视角 | 神的视角 |
| 暂时 | 永恒 |
| 虚空 | 有意义 |
| 劳碌 | 安息 |

#### 四、传道书的总意

> "这些事都已听见了，总意就是：敬畏神，谨守他的诫命，这是人所当尽的本分。因为人所做的事，连一切隐藏的事，无论是善是恶，神都必审问。"（传12:13-14）

### 讨论问题
1. 你有过"虚空"的感觉吗？
2. 传道书的信息消极还是积极？
3. 如何在虚空的世界中找到意义？

## 第八周：智慧的应用——敬畏神而活

### 本周经文
- 传道书 3:1-15（凡事都有定期）
- 传道书 5:1-7（敬畏神）
- 传道书 9:7-10（享受神所赐的）

### 核心内容

#### 一、接受生命的节奏

> "凡事都有定期，天下万务都有定时。"（传3:1）

| 有时 | 有时 |
|------|------|
| 生 | 死 |
| 栽种 | 拔出 |
| 哭 | 笑 |
| 哀恸 | 跳舞 |
| 寻找 | 失落 |
| 保守 | 舍弃 |
| 静默 | 言语 |
| 争战 | 和好 |

**智慧**：认识时机，顺应神的旨意。

#### 二、敬畏神

> "你到神的殿要谨慎脚步……不可冒失开口……因为神在天上，你在地下，所以你的言语要寡少。"（传5:1-2）

敬畏神的表现：
- 谨慎言语
- 遵守诺言
- 不草率许愿
- 知道自己的位置

#### 三、享受神所赐的

> "你只管去欢欢喜喜吃你的饭，心中快乐喝你的酒，因为神已经悦纳你的作为。"（传9:7）

**享受**不是罪，而是神的赐福——在敬畏神的前提下：
- 享受食物
- 享受工作
- 享受关系
- 享受生命

#### 四、智慧人生的总结

| 原则 | 出处 | 应用 |
|------|------|------|
| 敬畏神 | 箴1:7 | 一切智慧的基础 |
| 管好舌头 | 箴18:21 | 说造就人的话 |
| 殷勤工作 | 箴6:6 | 为主而做 |
| 正确理财 | 箴3:9 | 慷慨给予 |
| 智慧交友 | 箴13:20 | 与智慧人同行 |
| 建造家庭 | 箴22:6 | 传承信仰 |
| 认识虚空 | 传1:2 | 追求永恒 |
| 享受生命 | 传9:7 | 感恩神的赐福 |

### 讨论问题
1. 这八周学习中，什么对你最有帮助？
2. 你计划如何将智慧应用在生活中？
3. 智慧的人生是怎样的？

### 课程总结

```
智慧人生 = 敬畏神 + 遵行神的话 + 认识虚空 + 享受恩典 + 等候永恒
```
