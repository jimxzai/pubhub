# 小组讨论指南

> 《圣经与资治通鉴时空宇宙观》配套材料
> 带领者手册

---

## 使用说明

### 目的
帮助小组带领者有效地使用《圣经与资治通鉴时空宇宙观》各卷进行小组查经和讨论，促进生命成长和信仰应用。

### 适用对象
- 小组带领者
- 查经班教师
- 主日学老师
- 任何使用本材料带领讨论的同工

### 使用方法
- 每卷提供5-8周的讨论大纲
- 每周包含：经文、讨论问题、应用挑战
- 可根据小组情况调整

## 第一卷：人性透视（5周）

### 第一周：人的尊贵与堕落

#### 阅读材料
- 第一卷第1-2组故事
- 创世记1-3章

#### 核心问题
1. **开场**（5分钟）
   - 你认为人是本善还是本恶？为什么？

2. **经文讨论**（20分钟）
   - 创1:26-27说人是"按神的形像"造的，这意味着什么？
   - 创3章记载人的堕落，罪如何影响人性？

3. **对照讨论**（15分钟）
   - 资治通鉴中的人性故事（如赵高指鹿为马）显示什么？
   - 中国传统说"人之初性本善"，圣经怎么说？

4. **应用讨论**（15分钟）
   - 这对你认识自己有什么帮助？
   - 既然人有神的形像，又已堕落，我们该如何看待自己和他人？

#### 应用挑战
本周观察自己内心的挣扎，记录一个"明知是对却不想做"或"明知是错却想做"的经历。

## 带领技巧

### 预备工作

1. **提前阅读**——至少一周前读完材料
2. **选择重点**——不可能讨论所有内容
3. **预备问题**——但灵活调整
4. **祷告预备**——求圣灵引导

### 带领讨论

1. **营造氛围**
   - 安全、接纳
   - 没有愚蠢的问题
   - 保密原则

2. **促进参与**
   - 避免"独角戏"
   - 邀请安静的人
   - 控制发言过多的人

3. **引导讨论**
   - 从开放问题开始
   - 往深处引导
   - 连接到应用

4. **时间管理**
   - 准时开始结束
   - 留足应用时间
   - 不要匆匆结束

### 处理困难情况

| 情况 | 处理方式 |
|------|----------|
| 有人话太多 | 感谢后邀请其他人 |
| 有人沉默 | 不强迫，私下关心 |
| 偏离主题 | 温和引回 |
| 争论激烈 | 承认不同观点，回到经文 |
| 神学错误 | 温和纠正，指向圣经 |

### 结束祷告

1. 总结要点
2. 邀请分享代祷事项
3. 为应用祷告
4. 祝福

## 讨论问题的设计原则

### 好问题的特征

```
┌─────────────────────────────────────────┐
│  好问题：                               │
│  - 开放式（不能只用"是/否"回答）        │
│  - 促进思考                            │
│  - 连接经文与生活                       │
│  - 有多种可能答案                       │
│  - 引向更深讨论                         │
└─────────────────────────────────────────┘
```

### 问题的层次

| 层次 | 类型 | 例子 |
|------|------|------|
| 观察 | 经文说什么？ | "这段经文的主角是谁？" |
| 解释 | 这是什么意思？ | "作者为什么这样说？" |
| 应用 | 这对我有什么意义？ | "这如何影响我的生活？" |
| 回应 | 我该如何行动？ | "本周我可以做什么？" |

### 避免的问题

- 太封闭（是/否）
- 太宽泛（"你对人性有什么看法？"）
- 有标准答案（"考试题"）
- 敏感到难以分享
