# 门训课程二：门徒训练

> 基于《圣经与资治通鉴时空宇宙观》
> 24周进深造就课程

---

## 课程简介

### 目标
建造成熟的基督门徒，在品格、知识、技能上全面成长，预备承担更大的使命。

### 适用对象
- 完成新信徒课程的信徒
- 信主1-3年，渴望进深的基督徒
- 想要系统建立信仰根基的信徒

### 课程结构

| 单元 | 主题 | 周数 |
|------|------|------|
| 一 | 门徒的身份 | 第1-4周 |
| 二 | 门徒的品格 | 第5-10周 |
| 三 | 门徒的生活 | 第11-16周 |
| 四 | 门徒的关系 | 第17-20周 |
| 五 | 门徒的使命 | 第21-24周 |

### 使用方法
- 每周一课，每课约90-120分钟
- 建议4-8人小组进行
- 每课包含：经文研读、教导、讨论、作业、背诵经文
- 强调生命改变和实践应用

## 第一单元：门徒的身份（第1-4周）

### 第一周：什么是门徒

#### 核心经文
> "耶稣对门徒说：'若有人要跟从我，就当舍己，天天背起他的十字架来跟从我。'"（路9:23）

#### 教导大纲

**一、门徒的定义**
- 希腊文 mathetes：学习者、跟随者
- 不只是学生，更是生命的跟随者
- 门徒=学习者+跟随者+效法者

**二、门徒的特征**

| 特征 | 经文 | 应用 |
|------|------|------|
| 跟从耶稣 | 可1:17 | 以耶稣为中心 |
| 舍己 | 路9:23 | 放下自我 |
| 背十字架 | 路9:23 | 愿意受苦 |
| 遵守主道 | 约8:31 | 顺服主话 |
| 彼此相爱 | 约13:35 | 爱的见证 |
| 结出果子 | 约15:8 | 生命改变 |

**三、资治通鉴对照：师徒关系**

| 中国师徒 | 基督门徒 |
|----------|----------|
| 孔子与七十二贤 | 耶稣与十二门徒 |
| 传知识 | 传生命 |
| 学问道德 | 认识神 |
| 师傅会死 | 主永远活着 |

#### 讨论问题
1. 作门徒与"信耶稣"有什么不同？
2. "舍己"对你意味着什么？
3. 你愿意付代价跟随耶稣吗？

#### 本周作业
- **阅读**：马可福音1-2章
- **背诵**：路加福音9:23
- **反思**：我是门徒还是只是信徒？

### 第二周：在基督里的身份

#### 核心经文
> "若有人在基督里，他就是新造的人，旧事已过，都变成新的了。"（林后5:17）

#### 教导大纲

**一、新身份清单**

| 旧身份 | 新身份 | 经文 |
|--------|--------|------|
| 罪人 | 圣徒 | 弗1:1 |
| 仇敌 | 儿女 | 约1:12 |
| 定罪 | 称义 | 罗8:1 |
| 捆绑 | 自由 | 加5:1 |
| 黑暗 | 光明 | 弗5:8 |
| 死亡 | 生命 | 西3:3-4 |

**二、身份决定行为**
- 不是靠行为赢得身份
- 而是从身份活出行为
- "成为你所是"

**三、资治通鉴对照**
- 中国：通过科举改变身份
- 福音：身份先被改变，然后活出来

#### 讨论问题
1. 哪一个"新身份"对你最有意义？
2. 你常忘记自己的新身份吗？什么时候？
3. 知道自己是"神的儿女"如何改变你面对困难的方式？

#### 本周作业
- **阅读**：以弗所书1-2章
- **背诵**：林后5:17
- **写下**：10个你在基督里的新身份

### 第三周：蒙召的确据

#### 核心经文
> "就如神从创立世界以前，在基督里拣选了我们，使我们在他面前成为圣洁，无有瑕疵。"（弗1:4）

#### 教导大纲

**一、神的拣选**
- 神主动拣选我们
- 在创世之前已拣选
- 不是因我们好，是因他的恩典

**二、救恩的确据**

| 确据来源 | 内容 | 经文 |
|----------|------|------|
| 神的应许 | 凡求告主名的必得救 | 罗10:13 |
| 圣灵见证 | 圣灵与我们的心同证 | 罗8:16 |
| 生命改变 | 新造的人 | 林后5:17 |
| 渴慕神 | 爱慕神的话 | 彼前2:2 |

**三、面对怀疑**
- 怀疑是正常的
- 回到神的应许
- 检验生命果子

#### 讨论问题
1. 你确定自己得救了吗？凭什么确定？
2. 当有怀疑时，你怎么办？
3. "神拣选你"这个真理给你什么安慰？

#### 本周作业
- **阅读**：罗马书8章
- **背诵**：罗马书8:16
- **思考**：写下你得救的确据

### 第四周：呼召与命定

#### 核心经文
> "我们原是他的工作，在基督耶稣里造成的，为要叫我们行善，就是神所预备叫我们行的。"（弗2:10）

#### 教导大纲

**一、普遍呼召**
每个信徒都被呼召：
- 成圣（帖前4:3）
- 服事（彼前4:10）
- 见证（徒1:8）
- 荣耀神（林前10:31）

**二、个人呼召**
神对每个人有独特计划：
- 恩赐不同
- 处境不同
- 使命不同

**三、发现呼召的途径**

| 途径 | 说明 |
|------|------|
| 祷告寻求 | 在神面前安静 |
| 认识恩赐 | 神给你什么能力 |
| 观察负担 | 什么使你感动 |
| 开放门户 | 神开什么门 |
| 群体印证 | 他人的看法 |

**四、资治通鉴对照**

| 人物 | 呼召/命定 |
|------|----------|
| 周公 | 辅佐成王 |
| 张良 | 助汉兴 |
| 诸葛亮 | 复兴汉室 |

但这些是人的志向，我们的呼召来自神。

#### 讨论问题
1. 你认为神对你的生命有特别的呼召吗？
2. 你目前感受到的"负担"是什么？
3. 如何在日常生活中活出呼召？

#### 本周作业
- **阅读**：以弗所书2章
- **背诵**：以弗所书2:10
- **反思**：写下你认为神对你的呼召

## 第二单元：门徒的品格（第5-10周）

### 第五周：圣洁

#### 核心经文
> "那召你们的既是圣洁，你们在一切所行的事上也要圣洁。因为经上记着说：'你们要圣洁，因为我是圣洁的。'"（彼前1:15-16）

#### 教导大纲

**一、什么是圣洁**
- 分别出来归神
- 脱离罪恶
- 效法神的性情

**二、圣洁的领域**

| 领域 | 应用 |
|------|------|
| 思想 | 保守心思意念 |
| 言语 | 口舌的节制 |
| 行为 | 远离罪恶 |
| 关系 | 圣洁的交往 |
| 娱乐 | 分辨所看所听 |

**三、追求圣洁的途径**
- 靠圣灵能力
- 神的话洁净
- 团契问责
- 警醒祷告

**四、资治通鉴对照**
- 古人"慎独"——圣洁是在神面前
- "出淤泥而不染"——在世俗中保持圣洁

#### 讨论与作业
- **背诵**：彼前1:15-16
- **问题**：你生命中最难圣洁的领域是什么？
- **行动**：找出一个需要对付的不圣洁领域

---

### 第六周：谦卑

#### 核心经文
> "你们当以基督耶稣的心为心：他本有神的形像...反倒虚己，取了奴仆的形像..."（腓2:5-7）

#### 教导大纲

**一、谦卑的定义**
- 不是自卑或自贬
- 是正确看自己
- 是以神和他人为中心

**二、耶稣的谦卑榜样**
- 道成肉身
- 为门徒洗脚
- 顺服至死

**三、谦卑的实践**

| 表现 | 相反的骄傲 |
|------|------------|
| 承认有限 | 自以为是 |
| 愿意学习 | 固执己见 |
| 服事他人 | 期望被服事 |
| 接受纠正 | 拒绝批评 |
| 归荣耀给神 | 自我夸耀 |

**四、资治通鉴对照**
- 刘备三顾茅庐——谦卑得人
- 项羽刚愎自用——骄傲败亡

#### 讨论与作业
- **背诵**：腓立比书2:5-8
- **问题**：骄傲如何在你生命中表现？
- **行动**：本周刻意做一件谦卑服事他人的事

---

### 第七周：诚实正直

#### 核心经文
> "耶和华啊，谁能寄居你的帐幕？谁能住在你的圣山？就是行为正直、做事公义、心里说实话的人。"（诗15:1-2）

#### 教导大纲

**一、神是诚实的神**
- 神不能说谎
- 他的话可信
- 他信实守约

**二、诚实的层面**

| 层面 | 内容 |
|------|------|
| 言语诚实 | 不说谎、不夸张 |
| 行为诚实 | 不欺骗、不伪装 |
| 关系诚实 | 真诚待人 |
| 金钱诚实 | 不贪不占 |
| 内心诚实 | 自我省察 |

**三、诚实的挑战**
- 职场诚信
- 人际"谎言"
- 自我欺骗

**四、资治通鉴对照**
- 曾子杀猪——言出必行
- 商鞅立木——取信于民
- 秦穆公失马——诚信之君

#### 讨论与作业
- **背诵**：诗篇15:1-2
- **问题**：你在哪方面最容易不诚实？
- **行动**：检视自己，对付一个不诚实的习惯

---

### 第八周：忍耐

#### 核心经文
> "在百般试炼中，你们要以为大喜乐；因为知道你们的信心经过试验，就生忍耐。但忍耐也当成功，使你们成全、完备，毫无缺欠。"（雅1:2-4）

#### 教导大纲

**一、忍耐的定义**
- 不是消极等待
- 是在压力下坚持
- 是信靠神的时间表

**二、需要忍耐的情境**

| 情境 | 忍耐的表现 |
|------|------------|
| 等候应许 | 相信神的时间 |
| 面对苦难 | 不放弃信心 |
| 人际冲突 | 不急躁报复 |
| 成长过程 | 不求速成 |

**三、忍耐的果效**
- 生出品格（罗5:3-4）
- 得到应许（来6:15）
- 蒙神喜悦（来10:36）

**四、资治通鉴对照**
- 勾践卧薪尝胆——忍耐复国
- 司马懿忍功——忍成霸业
- 韩信胯下之辱——忍辱成将

#### 讨论与作业
- **背诵**：雅各书1:2-4
- **问题**：你目前在什么事上需要忍耐？
- **行动**：选一件需要忍耐的事，每天为它祷告

---

### 第九周：饶恕

#### 核心经文
> "你们要彼此包容，彼此饶恕；主怎样饶恕了你们，你们也要怎样饶恕人。"（西3:13）

#### 教导大纲

**一、神的饶恕**
- 因基督的血（弗1:7）
- 完全的饶恕（诗103:12）
- 白白的恩典（罗3:24）

**二、饶恕的误解**

| 饶恕不是 | 饶恕是 |
|----------|--------|
| 忘记 | 选择不追讨 |
| 认可对方行为 | 释放苦毒 |
| 信任对方 | 交托给神 |
| 一次就完成 | 可能需要多次 |

**三、不饶恕的后果**
- 被苦毒捆绑
- 关系破裂
- 祷告受阻

**四、饶恕的操练**
- 数算神对你的饶恕
- 选择饶恕的意志
- 为对方祷告

**五、资治通鉴对照**
- 赵氏孤儿——报仇
- 圣经：饶恕不是软弱，是力量

#### 讨论与作业
- **背诵**：歌罗西书3:13
- **问题**：有谁是你难以饶恕的？
- **行动**：为一个得罪你的人祷告一周

---

### 第十周：喜乐与感恩

#### 核心经文
> "要常常喜乐，不住地祷告，凡事谢恩，因为这是神在基督耶稣里向你们所定的旨意。"（帖前5:16-18）

#### 教导大纲

**一、喜乐的根基**
- 不是环境好才喜乐
- 是因认识神而喜乐
- 是圣灵的果子

**二、喜乐的来源**

| 来源 | 经文 |
|------|------|
| 救恩 | 路10:20 |
| 神的同在 | 诗16:11 |
| 盼望 | 罗12:12 |
| 神的话 | 耶15:16 |

**三、感恩的操练**
- 数算恩典
- 凡事谢恩
- 即使苦难中

**四、资治通鉴对照**
- 颜回"箪食瓢饮"而乐——但缺乏根基
- 基督徒的喜乐有永恒根基

#### 讨论与作业
- **背诵**：帖前5:16-18
- **问题**：什么偷走你的喜乐？
- **行动**：每天写下3件感恩的事

## 第三单元：门徒的生活（第11-16周）

### 第十一周：读经进深

#### 核心经文
> "圣经都是神所默示的，于教训、督责、使人归正、教导人学义都是有益的。"（提后3:16）

#### 教导大纲

**一、读经的方法**

| 方法 | 说明 |
|------|------|
| 灵修式 | 个人亲近神 |
| 研读式 | 深入理解 |
| 默想式 | 反复思想 |
| 背诵式 | 存在心中 |
| 应用式 | 付诸行动 |

**二、归纳式查经**
- 观察：经文说什么？
- 解释：经文是什么意思？
- 应用：我该如何回应？

**三、读经计划**
- 一年读完圣经
- 逐卷研读
- 主题研读

#### 讨论与作业
- **背诵**：提后3:16
- **实践**：制定一个读经计划
- **尝试**：用归纳法查一段经文

---

### 第十二周：祷告进深

#### 核心经文
> "你们亲近神，神就必亲近你们。"（雅4:8）

#### 教导大纲

**一、祷告的层面**

| 层面 | 内容 |
|------|------|
| 赞美敬拜 | 因神的所是 |
| 感恩 | 因神的作为 |
| 认罪 | 诚实面对 |
| 祈求 | 为需要求 |
| 代求 | 为他人求 |
| 倾听 | 安静等候 |

**二、进深的祷告**
- 禁食祷告
- 争战祷告
- 等候神的祷告
- 为国家祷告

**三、祷告的拦阻**
- 不信
- 罪
- 动机不纯
- 不按神旨意

#### 讨论与作业
- **背诵**：雅各书4:8
- **实践**：尝试一次禁食祷告
- **记录**：开始祷告日记

**四、资治通鉴对照**

### 第十六周：属灵争战

#### 核心经文
> "你们要靠着主，倚赖他的大能大力作刚强的人。要穿戴神所赐的全副军装，就能抵挡魔鬼的诡计。"（弗6:10-11）

#### 教导大纲

**一、争战的真实**
- 弗6:12——与空中属灵气的恶魔争战
- 不是与人争战

**二、全副军装**

| 军装 | 含义 |
|------|------|
| 真理带子 | 活在真理中 |
| 公义护心镜 | 因信称义 |

| 信德藤牌 | 信心抵挡 |
| 救恩头盔 | 救恩确据 |
| 圣灵宝剑 | 神的话 |

**三、争战的方式**
- 祷告
- 神的话
- 奉主的名
- 抵挡

**四、资治通鉴对照**
- 古代"知己知彼百战百胜"
- 属灵争战也需认识仇敌

#### 讨论与作业
- **背诵**：弗6:10-11
- **阅读**：以弗所书6:10-18
- **操练**：每天祷告穿戴军装

## 第四单元：门徒的关系（第17-20周）

### 第十七周：与神的关系

#### 核心经文
> "我是葡萄树，你们是枝子。常在我里面的，我也常在他里面，这人就多结果子；因为离了我，你们就不能做什么。"（约15:5）

#### 教导大纲

**一、关系的本质**
- 不是宗教，是关系
- 父与子
- 朋友
- 主与仆

**二、住在基督里**
- 持续的连接
- 依赖他
- 顺服他

**三、深化关系的途径**
- 每日灵修
- 全日与神同行
- 敏锐圣灵引导

#### 讨论与作业
- **背诵**：约15:5
- **反思**：你与神的关系是亲密还是疏远？
- **行动**：本周刻意练习"与神同在"

---

### 第十八周：在家庭中

#### 核心经文
> "你们作儿女的，要在主里听从父母...你们作父亲的，不要惹儿女的气，只要照着主的教训和警戒养育他们。"（弗6:1,4）

#### 教导大纲

**一、圣经的家庭观**
- 家庭是神设立的
- 有秩序的关系
- 爱与责任并重

**二、各人的角色**

| 角色 | 责任 |
|------|------|
| 丈夫 | 爱妻子、带领 |
| 妻子 | 顺服、帮助 |
| 父母 | 教养儿女 |
| 儿女 | 听从、孝敬 |

**三、在家中活出信仰**
- 家庭祭坛
- 以身作则
- 传承信仰

**四、资治通鉴对照**
- 中国家训传统（颜氏家训）
- 基督徒家庭以神为中心

#### 讨论与作业
- **背诵**：弗6:1,4
- **反思**：你在家中如何活出信仰？
- **行动**：计划一次家庭灵修或祷告

---

### 第十九周：在教会中

#### 核心经文
> "我们这许多人，在基督里成为一身，互相联络作肢体。"（罗12:5）

#### 教导大纲

**一、彼此相爱**
- 约13:34-35——新命令
- 爱是门徒的标记

**二、"彼此"的命令**

| 命令 | 经文 |
|------|------|
| 彼此相爱 | 约13:34 |
| 彼此服事 | 加5:13 |
| 彼此包容 | 弗4:2 |
| 彼此认罪 | 雅5:16 |
| 彼此劝勉 | 来3:13 |
| 彼此代祷 | 雅5:16 |

**三、处理冲突**
- 直接沟通
- 寻求和好
- 教会纪律

#### 讨论与作业
- **背诵**：罗12:5
- **反思**：你与教会肢体的关系如何？
- **行动**：主动关心一位少联系的弟兄姐妹

---

### 第二十周：在世界中

#### 核心经文
> "你们是世上的盐...你们是世上的光。"（太5:13-14）

#### 教导大纲

**一、在世但不属世**
- 约17:15-16——不离世界，但不属世界
- 分别为圣，却不与世隔绝

**二、盐与光的角色**

| 比喻 | 功能 |
|------|------|
| 盐 | 调味、防腐、洁净 |
| 光 | 照明、引导、暴露 |

**三、在职场见证**
- 诚实工作
- 卓越服务
- 爱心对人
- 言语见证

**四、资治通鉴对照**
- 古代士大夫"以天下为己任"
- 基督徒在各领域发挥影响

#### 讨论与作业
- **背诵**：太5:13-14
- **反思**：你在工作/学习中如何作盐作光？
- **行动**：想一个具体方式在你的环境中见证基督

## 第五单元：门徒的使命（第21-24周）

### 第二十一周：传福音

#### 核心经文
> "但圣灵降临在你们身上，你们就必得着能力，并要在耶路撒冷、犹太全地和撒玛利亚，直到地极，作我的见证。"（徒1:8）

#### 教导大纲

**一、传福音的命令**
- 大使命（太28:19-20）
- 每个信徒的责任

**二、传福音的方法**

| 方法 | 说明 |
|------|------|
| 生活见证 | 活出来 |
| 个人见证 | 分享经历 |
| 福音信息 | 清楚讲解 |
| 回答疑问 | 护教对话 |

**三、克服恐惧**
- 靠圣灵能力
- 预备好见证
- 抓住机会

#### 讨论与作业
- **背诵**：徒1:8
- **预备**：写下并练习你的3分钟见证
- **行动**：为三个未信者祷告，寻找机会分享

---

### 第二十二周：门徒倍增

#### 核心经文
> "你在许多见证人面前听见我所教训的，也要交托那忠心能教导别人的人。"（提后2:2）

#### 教导大纲

**一、倍增的模式**
- 保罗→提摩太→忠心的人→教导别人
- 四代的传承

**二、门训的过程**

| 阶段 | 内容 |
|------|------|
| 我做你看 | 示范 |
| 我做你帮 | 参与 |
| 你做我帮 | 实习 |
| 你做我看 | 检验 |
| 你做 | 放手 |

**三、寻找可培训的人**
- F.A.T.原则：Faithful（忠心）、Available（可用）、Teachable（受教）

**四、资治通鉴对照**
- 孔子教学法——因材施教
- 耶稣三年培训十二门徒

#### 讨论与作业
- **背诵**：提后2:2
- **思考**：谁是你可以带领的人？
- **行动**：开始为一个人祷告，准备带领他/她

---

### 第二十三周：服事神

#### 核心经文
> "人子来，并不是要受人的服事，乃是要服事人，并且要舍命作多人的赎价。"（可10:45）

#### 教导大纲

**一、服事的榜样——耶稣**
- 洗脚的主
- 舍命的爱

**二、服事的态度**

| 正确态度 | 错误态度 |
|----------|----------|
| 甘心 | 勉强 |
| 谦卑 | 自夸 |
| 为神 | 为人看 |
| 忠心 | 半途而废 |

**三、找到服事岗位**
- 认识恩赐
- 看见需要
- 委身投入

**四、资治通鉴对照**
- 古代忠臣"鞠躬尽瘁"
- 我们服事永恒的主

#### 讨论与作业
- **背诵**：可10:45
- **评估**：你目前参与什么服事？
- **行动**：如还没有，找一个可以服事的岗位

---

### 第二十四周：活出使命

#### 核心经文
> "所以，你们要去，使万民作我的门徒，奉父、子、圣灵的名给他们施洗。凡我所吩咐你们的，都教训他们遵守，我就常与你们同在，直到世界的末了。"（太28:19-20）

#### 教导大纲

**一、回顾24周学习**

```
门徒的身份 → 门徒的品格 → 门徒的生活
       ↓
门徒的关系 → 门徒的使命
```

**二、持续成长的承诺**
- 继续读经祷告
- 委身教会
- 服事他人
- 传福音、带门徒

**三、下一步**
- 参加领袖培训
- 开始带领门训
- 更深投入服事

**四、最后的挑战**

> "你们既然接受了主基督耶稣，就当遵他而行，在他里面生根建造，信心坚固。"（西2:6-7）

做耶稣的门徒是一生的旅程。

#### 讨论与作业
- **背诵**：太28:19-20
- **分享**：24周学习最大的收获是什么？
- **承诺**：写下你接下来的成长计划
- **行动**：找一个人开始门训
