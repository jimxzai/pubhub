# 门训课程四：护教装备

> 基于《圣经与资治通鉴时空宇宙观》
> 16周护教学课程

## 课程简介

### 目标
装备信徒回应信仰质疑，特别是在华人文化背景下的常见问题，建立信仰确据，有效见证基督。

### 适用对象
- 大学生、职场青年
- 经常面对信仰质疑的基督徒
- 想要更深建立信仰根基的信徒
- 有志于护教事工的同工

### 课程结构

| 单元 | 主题 | 周数 |
|------|------|------|
| 一 | 护教学导论 | 第1-2周 |
| 二 | 基督教与中国文化 | 第3-6周 |
| 三 | 信仰核心问题 | 第7-12周 |
| 四 | 当代挑战 | 第13-16周 |

### 核心经文
> "只要心里尊主基督为圣。有人问你们心中盼望的缘由，就要常作准备，以温柔、敬畏的心回答各人。"（彼前3:15）

## 第一单元：护教学导论（第1-2周）

### 第一周：什么是护教学

#### 核心经文
> "有人问你们心中盼望的缘由，就要常作准备，以温柔、敬畏的心回答各人。"（彼前3:15）

#### 教导内容

**一、护教学的定义**

| 希腊文 | apologia |
|--------|----------|
| 含义 | 辩护、答辩 |
| 圣经用法 | 为福音辩护（腓1:7,16） |

护教学 = 理性地为信仰辩护

**二、护教的目的**

| 目的 | 说明 |
|------|------|
| 移除障碍 | 解答疑惑，为信心铺路 |
| 坚固信心 | 帮助信徒确信所信 |
| 荣耀神 | 显明真理的合理性 |
| 预备传福音 | 打开对话的门 |

**三、护教的限制**

```
护教不能...              护教能够...
┌─────────────────┐    ┌─────────────────┐
│ 使人得救（信心）  │    │ 移除理性障碍    │
│ 取代圣灵工作     │    │ 显明信仰合理    │
│ 赢得辩论        │    │ 打开对话        │
│ 强迫人相信      │    │ 预备心接受福音   │
└─────────────────┘    └─────────────────┘
```

**四、护教的态度**

> "以温柔、敬畏的心"（彼前3:15）

- 不是争论、攻击
- 是温柔、尊重
- 目标是人，不是赢

**五、资治通鉴对照**

中国的"辩论"传统：

| 中国传统 | 基督教护教 |
|----------|------------|
| 百家争鸣 | 为真理辩护 |
| 纵横家说服 | 以爱心劝勉 |
| 求胜 | 求真 |

#### 讨论问题
1. 你遇到过哪些对信仰的质疑？你如何回应？
2. 护教的目的和限制分别是什么？
3. "温柔、敬畏"的态度为什么重要？

#### 本周作业
- **背诵**：彼前3:15
- **列出**：你最常遇到的信仰质疑
- **阅读**：使徒行传17:16-34（保罗在雅典）

### 第二周：护教的方法

#### 核心经文
> "我们争战的兵器本不是属血气的，乃是在神面前有能力，可以攻破坚固的营垒，将各样的计谋，各样拦阻人认识神的那些自高之事，一概攻破了。"（林后10:4-5）

#### 教导内容

**一、护教的进路**

| 进路 | 方法 | 代表人物 |
|------|------|----------|
| 古典护教 | 理性论证 | 阿奎那 |
| 证据护教 | 历史证据 | 乔希·麦道卫 |
| 前提护教 | 挑战前提 | 范泰尔 |
| 累积案例 | 综合证据 | 李·史乔贝 |

**二、实用的护教步骤**

```
1. 倾听 ─────→ 理解问题
      ↓
2. 澄清 ─────→ 确认真正问题
      ↓
3. 回应 ─────→ 针对性回答
      ↓
4. 邀请 ─────→ 指向福音
```

**三、对话的技巧**

| 技巧 | 说明 |
|------|------|
| 问问题 | 了解对方立场 |
| 找共同点 | 建立桥梁 |
| 承认不知道 | 诚实谦卑 |
| 讲故事 | 见证的力量 |
| 指向基督 | 核心是福音 |

**四、资治通鉴对照**

孔子的教学法：
- "不愤不启，不悱不发"——等候时机
- 因材施教——针对不同人

保罗的护教：
- 对犹太人讲旧约
- 对希腊人讲未识之神
- 针对不同背景的人

#### 讨论问题
1. 四种护教进路各有什么优缺点？
2. 为什么"倾听"是护教的第一步？
3. 你最需要学习的护教技巧是什么？

#### 本周作业
- **背诵**：林后10:4-5
- **练习**：与一位非基督徒对话，操练倾听
- **预备**：下周开始讨论"基督教是西方宗教"的问题

## 第二单元：基督教与中国文化（第3-6周）

### 第三周：基督教是"西方宗教"吗？

#### 核心经文
> "耶稣基督，昨日、今日、一直到永远，是一样的。"（来13:8）

#### 教导内容

**一、这个质疑的背景**

```
常见说法：
┌──────────────────────────────────────────┐
│ "基督教是西方人的宗教，不适合中国人"       │
│ "信耶稣就是被西方文化洗脑"                │
│ "为什么要信外国人的神？"                  │
└──────────────────────────────────────────┘
```

**二、事实澄清**

| 误解 | 事实 |
|------|------|
| 基督教是西方的 | 起源于中东（以色列） |
| 圣经是西方的书 | 作者全是中东人 |
| 耶稣是西方人 | 耶稣是犹太人 |
| 基督教只在西方 | 全球各洲都有 |

**三、基督教在中国的历史**

| 时期 | 事件 |
|------|------|
| 唐朝（635年） | 景教传入（大秦景教流行中国碑） |
| 元朝 | 也里可温教 |
| 明末清初 | 利玛窦等耶稣会士 |
| 近代 | 新教传教运动 |
| 当代 | 中国教会蓬勃发展 |

基督教在中国已有近1400年历史！

**四、真理不分东西**

> "真理是什么？"——彼拉多

| 类比 | 说明 |
|------|------|
| 数学 | 来自中东/希腊，但全世界通用 |
| 科学 | 无论来源，真理就是真理 |
| 医学 | 不管西医中医，能治病就是好医 |

**五、资治通鉴对照**

中国历史上的"外来"文化：

| 外来文化 | 来源 | 中国态度 |
|----------|------|----------|
| 佛教 | 印度 | 广泛接受 |
| 琵琶、胡琴 | 西域 | 成为国乐 |
| 番茄、辣椒 | 美洲 | 融入饮食 |

为什么对基督教有不同标准？

#### 讨论问题
1. 你或你的朋友是否有过"基督教是西方宗教"的想法？
2. 这些历史事实如何改变这个看法？
3. 真理应该有东西之分吗？

#### 本周作业
- **背诵**：来13:8
- **研究**：景教碑的内容
- **预备**：一个可以使用的回应

### 第四周：圣经与中国经典

#### 核心经文
> "太初有道，道与神同在，道就是神。"（约1:1）

#### 教导内容

**一、"道"的对话**

| 老子的"道" | 约翰的"道" |
|------------|------------|
| 非位格 | 位格（神） |
| 不可名 | 已启示 |
| 自然规律 | 创造者 |
| 无法认识 | 道成肉身 |

> "道可道，非常道。"——老子
> "道成了肉身，住在我们中间。"——约翰

**二、共同点与差异**

| 议题 | 中国经典 | 圣经 |
|------|----------|------|
| 天/神 | 非位格或模糊 | 位格神 |
| 人性 | 性善或可善 | 堕落需救赎 |
| 伦理 | 五常 | 十诫、爱 |
| 来生 | 不明确 | 复活审判 |
| 救法 | 修己 | 恩典信心 |

**三、圣经的独特**

```
圣经的独特之处
┌─────────────────────────────────────────┐
│  1. 神主动启示（不是人寻求）            │
│  2. 位格的神（不是抽象原则）            │
│  3. 罪的严重性（不是小过失）            │
│  4. 恩典的救赎（不是靠修行）            │
│  5. 复活的盼望（不是轮回或消散）        │
└─────────────────────────────────────────┘
```

**四、桥梁与差异**

- 可以用中国概念作桥梁
- 但不能等同
- 最终指向圣经的独特启示

#### 讨论问题
1. "道"在老子和约翰福音中有何异同？
2. 中国文化中有哪些可以成为传福音"桥梁"的概念？
3. 为什么不能说"儒释道和基督教都差不多"？

#### 本周作业
- **背诵**：约1:1
- **阅读**：《道德经》第一章，与约翰福音1章对比
- **思考**：如何用"道"的概念向中国人传福音

### 第五周：信耶稣与孝道冲突吗？

#### 核心经文
> "当孝敬父母，使你的日子在耶和华你神所赐你的地上得以长久。"（出20:12）

#### 教导内容

**一、质疑的内容**

```
常见说法：
┌──────────────────────────────────────────┐
│ "信耶稣就不能拜祖先，这是不孝"           │
│ "基督教破坏中国人的孝道传统"             │
│ "父母信佛，我信耶稣，这是大逆不道"       │
└──────────────────────────────────────────┘
```

**二、圣经怎么说孝道**

| 经文 | 内容 |
|------|------|
| 出20:12 | 当孝敬父母 |
| 弗6:1-3 | 在主里听从父母 |
| 提前5:4 | 在家中学习行孝 |
| 太15:4-6 | 耶稣责备不孝的人 |

**圣经非常强调孝敬父母！**

**三、孝敬 vs 崇拜**

| 孝敬父母 | 崇拜祖先 |
|----------|----------|
| 圣经吩咐 | 圣经禁止 |
| 敬重活着的父母 | 拜已去世的先人 |
| 照顾供养 | 烧纸献祭 |
| 顺服（在主里） | 求祖先保佑 |

**四、如何活出孝道**

| 方式 | 内容 |
|------|------|
| 照顾父母 | 物质和精神 |
| 尊重父母 | 态度和言语 |
| 为父母祷告 | 属灵关怀 |
| 传福音给父母 | 最大的孝 |

**五、资治通鉴对照**

| 中国孝道 | 基督教孝道 |
|----------|------------|
| 二十四孝 | 出于爱的孝敬 |
| 有时愚孝 | 在主里听从 |
| 死后祭拜 | 生前孝敬 |
| 传宗接代 | 属灵传承 |

#### 讨论问题
1. 圣经对孝道的教导是什么？
2. 孝敬父母与崇拜祖先有何分别？
3. 如果父母是非基督徒，你如何活出孝道？

#### 本周作业
- **背诵**：出20:12
- **行动**：本周做一件孝敬父母的事
- **思考**：如何向父母传福音

### 第六周：祭祖与基督信仰

#### 核心经文
> "除了我以外，你不可有别的神。"（出20:3）

#### 教导内容

**一、祭祖的问题**

| 形式 | 问题 |
|------|------|
| 烧纸钱 | 认为祖先需要 |
| 上香跪拜 | 宗教崇拜行为 |
| 求祖先保佑 | 把祖先当神 |

**二、圣经的原则**

1. **独一真神**——出20:3
2. **不可拜偶像**——出20:4-5
3. **不可与死人交流**——申18:10-11

**三、区分：记念 vs 崇拜**

| 可以 | 不可以 |
|------|--------|
| 记念先人 | 崇拜先人 |
| 追思生平 | 烧纸献祭 |
| 感恩神 | 求祖先保佑 |
| 家族团聚 | 宗教仪式 |

**四、实际建议**

| 情境 | 可能的做法 |
|------|------------|
| 清明节 | 扫墓、整理、追思，不烧纸 |
| 家族祭祖 | 事先沟通，尊重但不参与拜祭 |
| 过年团聚 | 参与吃饭，不参与拜祭 |

**五、智慧的平衡**

- 不妥协信仰原则
- 但以爱心对待家人
- 寻找替代方式表达尊重
- 把握机会传福音

#### 讨论问题
1. 记念祖先与崇拜祖先有何分别？
2. 面对家人的祭祖压力，你会怎么处理？
3. 如何在坚守信仰与尊重家人之间取得平衡？

#### 本周作业
- **背诵**：出20:3
- **思考**：写下你个人的处理原则
- **祷告**：为家族中的非基督徒祷告

## 第三单元：信仰核心问题（第7-12周）

### 第七周：神存在吗？

#### 核心经文
> "自从造天地以来，神的永能和神性是明明可知的，虽是眼不能见，但借着所造之物就可以晓得，叫人无可推诿。"（罗1:20）

#### 教导内容

**一、神存在的论证**

| 论证 | 内容 |
|------|------|
| **宇宙论** | 万物必有起源，第一因是神 |
| **设计论** | 宇宙精密设计，必有设计者 |
| **道德论** | 普遍道德观念，来自道德立法者 |
| **本体论** | 最完美存在的概念，必须存在 |

**二、宇宙的精密设计**

```
例子：
- 地球与太阳距离刚刚好
- 宇宙常数极其精确
- DNA信息复杂度惊人
- 人眼比任何相机精密
```

随机进化？还是智慧设计？

**三、中国传统的"天"**

| 概念 | 内容 |
|------|------|
| 天命 | 天有意志 |
| 天理 | 天有公义 |
| 天道 | 天有规律 |

中国人相信"天"，但这个"天"到底是谁？

**四、回应无神论**

| 无神论说法 | 回应 |
|------------|------|
| "没证据证明神存在" | 也没证据证明神不存在；有设计的证据 |
| "科学解释一切" | 科学解释"如何"，不解释"为何" |
| "邪恶证明没有神" | 邪恶也需要解释，没有神如何定义邪恶 |

#### 讨论问题
1. 哪一个神存在的论证对你最有说服力？
2. 中国人的"天"与圣经的神有何异同？
3. 如何回应说"神是人想象出来的"的人？

#### 本周作业
- **背诵**：罗1:20
- **观察**：本周注意大自然中设计的证据
- **练习**：向一个人分享神存在的证据

### 第八周：圣经可靠吗？

#### 核心经文
> "圣经都是神所默示的。"（提后3:16）

#### 教导内容

**一、圣经可靠的证据**

| 证据类型 | 内容 |
|----------|------|
| **手抄本证据** | 超过24,000份新约手抄本 |
| **考古证据** | 考古不断证实圣经记载 |
| **预言应验** | 300多条弥赛亚预言应验 |
| **内在一致** | 40位作者，1500年，高度一致 |
| **生命改变** | 无数人因圣经生命改变 |

**二、与其他古籍比较**

| 古籍 | 最早抄本与原著时间差 | 抄本数量 |
|------|---------------------|----------|
| 新约圣经 | 25-50年 | 24,000+ |
| 荷马史诗 | 500年 | 643 |
| 柏拉图著作 | 1,200年 | 7 |
| 资治通鉴 | 几百年 | 较少 |

圣经是古代最有文献支持的书！

**三、回应质疑**

| 质疑 | 回应 |
|------|------|
| "圣经被篡改" | 手抄本证据显示高度一致 |
| "圣经有矛盾" | 大多数可以调和；小差异不影响核心 |
| "圣经是人写的" | 人写，但神默示 |

**四、资治通鉴对照**

司马光编《资治通鉴》的严谨，但圣经的可靠性更有支持。

#### 讨论问题
1. 圣经可靠的证据中，哪一个最打动你？
2. 如何回应说"圣经都被改过了"的人？
3. 你个人相信圣经的原因是什么？

#### 本周作业
- **背诵**：提后3:16
- **研究**：死海古卷的发现及意义
- **分享**：向一个人介绍圣经的可靠性

### 第九周：耶稣是神吗？

#### 核心经文
> "太初有道，道与神同在，道就是神...道成了肉身，住在我们中间。"（约1:1,14）

#### 教导内容

**一、耶稣自称是神**

| 宣告 | 经文 |
|------|------|
| "我与父原为一" | 约10:30 |
| "还没有亚伯拉罕就有了我" | 约8:58 |
| 接受敬拜 | 约20:28 |
| 赦罪 | 可2:5-7 |

**二、路易斯的三难困境**

```
耶稣说自己是神，那么他只能是：
┌───────────────────────────────────────┐
│  1. 疯子（Lunatic）— 精神错乱        │
│  2. 骗子（Liar）— 故意欺骗           │
│  3. 主（Lord）— 他真的是神           │
└───────────────────────────────────────┘
            没有第四个选项：
           "好老师但不是神"
```

**三、耶稣是神的证据**

| 证据 | 内容 |
|------|------|
| 自我宣称 | 多次宣称与神同等 |
| 无罪生命 | 连仇敌也找不出错 |
| 神迹奇事 | 医病、赶鬼、控制自然 |
| 复活 | 死后三天复活（下周详述） |
| 门徒改变 | 为他殉道 |

**四、资治通鉴对照**

中国历史上有许多"圣人"，但没有一个敢说自己是神，因为他们不是。耶稣敢宣称，因为他是。

#### 讨论问题
1. 耶稣宣称自己是神的证据有哪些？
2. "三难困境"如何帮助回应"耶稣只是好老师"的说法？
3. 你相信耶稣是神吗？为什么？

#### 本周作业
- **背诵**：约1:1,14
- **阅读**：约翰福音8章，注意耶稣的宣称
- **练习**：向人解释"三难困境"

### 第十周：耶稣真的复活了吗？

#### 核心经文
> "基督若没有复活，我们所传的便是枉然，你们所信的也是枉然。"（林前15:14）

#### 教导内容

**一、复活的重要性**

如果复活是真的：
- 耶稣的宣称被证实
- 基督教信仰成立
- 我们有永生盼望

如果复活是假的：
- 基督教就是谎言
- 保罗说"我们比众人更可怜"

**二、复活的证据**

| 证据 | 说明 |
|------|------|
| **空坟墓** | 犹太人、罗马人都无法出示尸体 |
| **目击者** | 500多人同时看见（林前15:6） |
| **门徒改变** | 从胆怯变为殉道 |
| **教会兴起** | 从耶路撒冷开始，迅速传开 |
| **安息日改变** | 犹太人改在主日聚会 |

**三、回应替代解释**

| 替代解释 | 问题 |
|----------|------|
| 门徒偷尸体 | 为什么愿意为谎言殉道？ |
| 耶稣没死（昏过去） | 罗马兵确认死亡；如何移开大石？ |
| 集体幻觉 | 500人同时？太夫妻不认识也看见？ |
| 传说演变 | 太快传开，且有目击者在世 |

**四、资治通鉴对照**

中国历史记载的"死而复生"都是传说；耶稣复活有充分历史证据支持。

#### 讨论问题
1. 复活为什么是基督教的核心？
2. 四个替代解释各有什么问题？
3. 复活对你个人有什么意义？

#### 本周作业
- **背诵**：林前15:14
- **阅读**：哥林多前书15章
- **练习**：能够说出复活的四个证据

### 第十一周：为什么要相信三位一体？

#### 核心经文
> "奉父、子、圣灵的名给他们施洗。"（太28:19）

#### 教导内容

**一、三位一体的定义**

```
一位神，三个位格
┌───────────────────────────────────┐
│  父不是子，子不是圣灵             │
│  父是神，子是神，圣灵是神         │
│  但只有一位神                     │
└───────────────────────────────────┘
```

**二、圣经根据**

| 真理 | 经文 |
|------|------|
| 神是一 | 申6:4 |
| 父是神 | 弗1:3 |
| 子是神 | 约1:1,14 |
| 圣灵是神 | 徒5:3-4 |
| 三位同时出现 | 太3:16-17 |

**三、回应质疑**

| 质疑 | 回应 |
|------|------|
| "这不合逻辑" | 超越逻辑不等于违反逻辑 |
| "这是异教多神论" | 一神三位格，不是三神 |
| "圣经没有'三位一体'这词" | 概念在圣经中，词是后来归纳的 |

**四、为什么重要**

- 关乎我们认识神的本质
- 关乎救恩（子成肉身，圣灵重生）
- 关乎神是爱（神自身有爱的关系）

#### 讨论问题
1. 三位一体的教义从何而来？
2. 如何回应"三位一体不合逻辑"的质疑？
3. 这个教义对你认识神有什么帮助？

#### 本周作业
- **背诵**：太28:19
- **研究**：找出三位同时出现的经文
- **思考**：如何向人解释三位一体

### 第十二周：既然神是全能全善的，为什么有苦难？

#### 核心经文
> "我们晓得万事都互相效力，叫爱神的人得益处。"（罗8:28）

#### 教导内容

**一、苦难问题**

```
传统的"苦难论证"：
1. 神若全能，能除去苦难
2. 神若全善，愿除去苦难
3. 但苦难存在
4. 所以神不存在或不是全能全善的
```

**二、基督教的回应**

| 回应 | 内容 |
|------|------|
| **自由意志** | 神给人选择，人选择带来后果 |
| **堕落世界** | 罪进入世界，带来苦难 |
| **灵魂塑造** | 苦难可以塑造品格 |
| **更大美善** | 神可以用苦难成就更大旨意 |
| **永恒视角** | 今生苦难相对永恒是短暂的 |

**三、神自己进入苦难**

- 十字架：神不是远离苦难，而是进入苦难
- 耶稣受苦：与我们的苦难同在
- 复活：苦难不是终局

**四、资治通鉴对照**

中国对苦难的回应：
- 儒家：承受天命
- 佛家：苦是四谛之一
- 道家：顺应自然

基督教独特之处：神进入苦难，最终除去苦难。

#### 讨论问题
1. 这个问题为什么这么难回答？
2. 以上回应中，哪一个对你最有帮助？
3. 你自己经历苦难时，信仰如何帮助你？

#### 本周作业
- **背诵**：罗8:28
- **阅读**：约伯记38-42章
- **默想**：十字架如何回应苦难问题

## 第四单元：当代挑战（第13-16周）

### 第十三周：科学与信仰冲突吗？

#### 核心经文
> "诸天述说神的荣耀，穹苍传扬他的手段。"（诗19:1）

#### 教导内容

**一、误解澄清**

| 误解 | 事实 |
|------|------|
| 科学证明没有神 | 科学研究自然界，不研究神 |
| 科学家都是无神论者 | 许多伟大科学家是基督徒 |
| 信仰反对科学 | 基督教促进了现代科学兴起 |

**二、科学与信仰的关系**

```
科学回答：            信仰回答：
"如何"(How)           "为何"(Why)
机制、过程             意义、目的
┌─────────┐          ┌─────────┐
│ 宇宙如何运作│          │ 宇宙为何存在│
│ 生命如何繁衍│          │ 生命的意义  │
│ 大脑如何运作│          │ 意识是什么  │
└─────────┘          └─────────┘
        不同领域，可以互补
```

**三、著名基督徒科学家**

| 科学家 | 贡献 |
|--------|------|
| 牛顿 | 经典力学 |
| 法拉第 | 电磁学 |
| 开尔文 | 热力学 |
| 帕斯卡 | 数学、物理 |
| 当代：柯林斯 | 人类基因组计划 |

**四、进化论讨论**

| 可以接受 | 需要区分 |
|----------|----------|
| 微进化（物种内变化） | 宏进化（跨物种变化） |
| 自然选择存在 | 能否产生复杂信息 |
| 地球历史 | 人是神创造的形像 |

核心：人有神的形像，人不只是进化的动物。

#### 讨论问题
1. 科学和信仰真的冲突吗？
2. 进化论的讨论中，什么是核心问题？
3. 如何向"只信科学"的人传福音？

#### 本周作业
- **背诵**：诗19:1
- **研究**：一位基督徒科学家的生平
- **思考**：科学指向还是否定神？

### 第十四周：为什么基督教排斥其他宗教？

#### 核心经文
> "耶稣说：'我就是道路、真理、生命；若不借着我，没有人能到父那里去。'"（约14:6）

#### 教导内容

**一、质疑的内容**

```
常见说法：
┌──────────────────────────────────────────┐
│ "所有宗教都是殊途同归"                   │
│ "基督教太排外、太傲慢"                   │
│ "为什么只有你们对？"                     │
└──────────────────────────────────────────┘
```

**二、所有宗教真的"差不多"吗？**

| 议题 | 基督教 | 佛教 | 伊斯兰教 | 印度教 |
|------|--------|------|----------|--------|
| 神 | 位格三一神 | 无神 | 一神（安拉） | 多神 |
| 耶稣 | 神子、救主 | 开悟者 | 先知 | 化身之一 |
| 救法 | 恩典、信心 | 自力、八正道 | 顺服、行为 | 轮回、业报 |
| 结局 | 天堂/地狱 | 涅槃 | 天堂/地狱 | 轮回/梵我合一 |

各宗教核心教义有根本差异，不可能都对。

**三、不是"傲慢"，而是忠于耶稣的话**

- 耶稣自己说的（约14:6）
- 不是基督徒发明的
- 这是真理的问题，不是傲慢的问题

**四、真理的本质**

| 例子 | 说明 |
|------|------|
| 数学 | 2+2=4，不是"傲慢"，是事实 |
| 医学 | 正确诊断才能治病 |
| 路标 | 指向唯一正确方向 |

如果基督教是真的，那么"排他"是爱的表现——指向唯一的救法。

#### 讨论问题
1. 各宗教真的"殊途同归"吗？
2. 说"耶稣是唯一道路"是傲慢吗？
3. 如何以爱心传达这个真理？

#### 本周作业
- **背诵**：约14:6
- **比较**：列出基督教与一个其他宗教的核心差异
- **练习**：以温和方式说明基督教的独特

### 第十五周：教会有这么多丑闻，还能信吗？

#### 核心经文
> "人人都有罪，亏缺了神的荣耀。"（罗3:23）

#### 教导内容

**一、承认问题存在**

教会历史上的黑暗：
- 十字军东征
- 宗教裁判所
- 性侵丑闻
- 假先知、敛财

**不能否认，不应辩护**

**二、如何理解**

| 角度 | 回应 |
|------|------|
| 人的罪 | 基督徒也是罪人（被救的罪人） |
| 名义与真实 | 并非所有自称基督徒的都是 |
| 不代表基督 | 人的失败不等于基督的失败 |
| 好果子 | 也有无数美好的见证 |

**三、基督教的正面贡献**

| 领域 | 贡献 |
|------|------|
| 医疗 | 医院起源于基督教 |
| 教育 | 大学源于教会 |
| 慈善 | 孤儿院、救济 |
| 人权 | 废奴运动 |
| 科学 | 许多科学家是基督徒 |

**四、核心问题**

```
不是问"基督徒做了什么"
而是问"耶稣是谁？他说的是真的吗？"
         ↓
     信仰的对象是基督
     不是基督徒
```

**五、资治通鉴对照**

历史上许多"儒士"也有丑行，但这不能否定儒学的价值。

#### 讨论问题
1. 教会的丑闻对你的信心有影响吗？
2. 如何回应用教会丑闻来否定信仰的人？
3. 你如何区分"基督"与"基督徒"？

#### 本周作业
- **背诵**：罗3:23
- **研究**：基督教在一个领域的正面贡献
- **反思**：我的生命是好见证吗？

### 第十六周：如何与人分享信仰

#### 核心经文

#### 教导内容

**一、课程回顾**

```
护教装备总览
┌─────────────────────────────────────────┐
│ 导论：什么是护教、护教的方法            │
│ 中国文化：西方宗教？孝道？祭祖？         │
│ 核心问题：神存在？圣经？耶稣？复活？     │
│           三一？苦难？                   │
│ 当代挑战：科学？排他？丑闻？             │
└─────────────────────────────────────────┘
```

**二、护教的原则总结**

| 原则 | 内容 |
|------|------|
| 态度温柔 | 不是争论，是对话 |
| 倾听为先 | 先理解问题 |
| 回到圣经 | 圣经是最终权威 |
| 指向基督 | 福音是核心 |
| 祷告倚靠 | 是圣灵的工作 |
| 活出见证 | 生命是最好的护教 |

**三、实践对话**

| 步骤 | 内容 |
|------|------|
| 建立关系 | 成为朋友 |
| 倾听问题 | 真正理解 |
| 澄清问题 | 确认重点 |
| 回应问题 | 用所学回答 |
| 邀请决定 | 指向福音 |
| 持续跟进 | 不是一次对话 |

**四、处理回答不了的问题**

- 诚实说"我不知道，但我会去查"
- 核心是福音，不是每个问题都有答案
- 邀请继续对话

**五、最后的挑战**

> "常作准备"——彼前3:15

- 继续学习
- 建立关系
- 抓住机会
- 活出见证

#### 讨论问题
1. 这16周学习中，最有帮助的是什么？
2. 你最需要继续深入的领域是什么？
3. 你准备好与人分享信仰了吗？
