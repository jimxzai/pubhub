# 门训课程一：新信徒课程

> 基于《圣经与资治通鉴时空宇宙观》
> 12周初信造就课程

---

## 课程简介

### 目标
帮助初信者建立信仰根基，认识基督教基本教义，开始过敬虔生活，融入教会群体。

### 适用对象
- 刚接受耶稣基督为救主的初信者
- 信主多年但从未受过系统栽培的信徒
- 想要重新扎根的基督徒

### 课程结构

| 单元 | 主题 | 周数 |
|------|------|------|
| 一 | 认识救恩 | 第1-3周 |
| 二 | 认识神 | 第4-6周 |
| 三 | 灵命成长 | 第7-9周 |
| 四 | 融入教会 | 第10-12周 |

### 使用方法
- 每周一课，每课约60-90分钟
- 建议一对一或小组（3-6人）进行
- 每课包含：经文、教导、讨论、作业

## 第一单元：认识救恩（第1-3周）

### 第一周：我为什么需要救恩？

#### 核心经文
> "因为世人都犯了罪，亏缺了神的荣耀。"（罗3:23）
> "罪的工价乃是死；惟有神的恩赐，在我们的主基督耶稣里，乃是永生。"（罗6:23）

#### 教导内容

**一、神造人的目的**

神按自己的形像造人，目的是：
- 与神有美好的关系
- 管理受造世界
- 荣耀神、享受神

**二、人的问题——罪**

| 罪的含义 | 说明 |
|----------|------|
| 不中目标 | 偏离神的标准 |
| 悖逆 | 违反神的命令 |
| 过犯 | 越过神的界限 |

```
神的计划          人的光景
┌─────────┐      ┌─────────┐
│ 与神同在 │  →  │ 与神隔绝 │
│ 享受丰盛 │  →  │ 内心空虚 │
│ 活出意义 │  →  │ 迷失方向 │
│ 永远生命 │  →  │ 面对死亡 │
└─────────┘      └─────────┘
      罪带来分隔
```

**三、东西方对人性的认识**

| 圣经 | 中国传统 | 对话 |
|------|----------|------|
| 人有神的形像 | 人之初性本善 | 人有高贵尊严 |
| 人已堕落 | 近墨者黑 | 人容易走偏 |
| 靠己无法解决 | 修身养性 | 自救的限度 |

> "人之初，性本善。性相近，习相远。"——《三字经》

中国人相信人性本善可以恢复，但历史证明人无法靠自己变好。

#### 讨论问题

1. 在认识耶稣之前，你觉得自己是"好人"还是"罪人"？为什么？
2. 罪如何影响你与神、与人、与自己的关系？
3. 你尝试过什么方法解决内心的问题？有效吗？

#### 本周作业

- **阅读**：创世记第3章
- **思考**：我生命中有哪些"罪"需要面对？
- **祷告**：为认识自己的真实光景感谢神

### 第二周：神如何拯救我？

#### 核心经文
> "神爱世人，甚至将他的独生子赐给他们，叫一切信他的，不致灭亡，反得永生。"（约3:16）

#### 教导内容

**一、神的爱——救恩的根源**

| 神可以... | 但神选择... |
|-----------|-------------|
| 忽视人类 | 主动寻找 |
| 任凭灭亡 | 差遣救主 |
| 远离罪人 | 道成肉身 |
| 公义审判 | 慈爱赦免 |

**二、耶稣基督——救恩的成就**

```
耶稣是谁？
┌────────────────────────────────────────┐
│  完全的神 ──── 有能力拯救               │
│  完全的人 ──── 能代替人受罚             │
│  无罪的生命 ── 有资格作祭物             │
│  复活的主 ──── 胜过死亡                 │
└────────────────────────────────────────┘
```

**三、十字架——救恩的中心**

| 十字架做了什么 | 经文 |
|----------------|------|
| 代替我们受罚 | 赛53:5 |
| 担当我们的罪 | 彼前2:24 |
| 使我们与神和好 | 罗5:10 |
| 赐我们新生命 | 罗6:4 |

**四、资治通鉴对照**

中国历史中的"代替"故事：

| 故事 | 内容 | 与基督的比较 |
|------|------|--------------|
| 程婴救孤 | 以己子代赵氏孤儿 | 基督以命代罪人 |
| 苏武牧羊 | 为国受苦不屈 | 基督为世人受苦 |

但人的代替有限，只有神的儿子能完全解决罪的问题。

#### 讨论问题

1. 约3:16是最著名的经文之一，它告诉我们关于神什么重要的事？
2. 为什么只有耶稣能拯救我们？为什么不是其他宗教领袖？
3. 十字架对你个人意味着什么？

#### 本周作业

- **阅读**：约翰福音3:1-21
- **默想**：耶稣为我做了什么？
- **祷告**：感谢耶稣的牺牲

### 第三周：我如何领受救恩？

#### 核心经文
> "你们得救是本乎恩，也因着信；这并不是出于自己，乃是神所赐的；也不是出于行为，免得有人自夸。"（弗2:8-9）

#### 教导内容

**一、不是靠行为**

| 不能靠... | 经文 |
|-----------|------|
| 好行为 | 弗2:9 |
| 宗教仪式 | 赛1:11 |
| 道德修养 | 罗3:20 |
| 功德积累 | 加2:16 |

**二、乃是靠恩典因着信**

```
        恩典
         ↓
 神的作为：差遣基督，成就救恩
         ↓
        信心
         ↓
 人的回应：承认、相信、接受
         ↓
        得救
```

**三、信心的三个层面**

| 层面 | 内容 | 例子 |
|------|------|------|
| **知道** | 头脑的认知 | 知道耶稣是救主 |
| **相信** | 心里的认同 | 相信他为我死 |
| **接受** | 意志的委身 | 接受他作我的主 |

> "你若口里认耶稣为主，心里信神叫他从死里复活，就必得救。"（罗10:9）

**四、中国文化的对话**

| 中国观念 | 福音回应 |
|----------|----------|
| 靠行善积德 | 恩典白白赐予 |
| 靠修行悟道 | 真理已经启示 |
| 靠自己努力 | 信靠基督成就 |

> "天行健，君子以自强不息。"——《易经》

自强是好的，但在救恩上，我们需要谦卑领受。

#### 讨论问题

1. 为什么神不让我们靠行为得救？
2. "信心"对你来说意味着什么？你真的相信吗？
3. 如果有人问你"你确定自己得救了吗？"，你会怎么回答？

#### 本周作业

- **阅读**：罗马书第5章
- **确认**：你是否已经真正相信并接受耶稣？
- **练习**：用几句话说出你的信主见证

## 第二单元：认识神（第4-6周）

### 第四周：认识三位一体的神

#### 核心经文
> "所以，你们要去，使万民作我的门徒，奉父、子、圣灵的名给他们施洗。"（太28:19）

#### 教导内容

**一、什么是三位一体**

```
         一位神
     ┌─────────────┐
     │             │
   ┌─┴─┐  ┌───┐  ┌─┴─┐
   │父 │  │子 │  │圣灵│
   └───┘  └───┘  └───┘
     ↑      ↑      ↑
   同质、同等、永恒
   但各有位格、各有工作
```

| 位格 | 工作重点 | 与我们的关系 |
|------|----------|--------------|
| 父神 | 创造、计划 | 天父 |
| 子神 | 救赎、启示 | 救主、朋友 |
| 圣灵 | 更新、引导 | 保惠师、内住 |

**二、圣经中的三位一体**

| 经文 | 显示 |
|------|------|
| 太3:16-17 | 耶稣受洗：父说话、子受洗、灵降下 |
| 约14:26 | "父因我的名所要差来的圣灵" |
| 林后13:14 | "主耶稣基督的恩惠、神的慈爱、圣灵的感动" |

**三、这奥秘对我们的意义**

- 神是关系性的——神自身就有爱的团契
- 神邀请我们进入这关系——参与神圣团契
- 每一位都与我们相关——父爱我、子救我、灵住在我里面

**四、回应中国文化**

| 中国神观 | 基督教神观 |
|----------|------------|
| 天——非位格 | 神——有位格 |
| 多神杂拜 | 独一真神 |
| 遥不可及 | 亲近可知 |

> "天何言哉？四时行焉，百物生焉，天何言哉？"——孔子

天不说话，但圣经的神说话，并且住在我们中间。

#### 讨论问题

1. 三位一体是个奥秘，你觉得难以理解吗？这影响你的信心吗？
2. 你与父神、子神、圣灵各自的关系如何？
3. 知道神是关系性的，对你的信仰生活有什么帮助？

#### 本周作业

- **阅读**：约翰福音14-16章
- **默想**：父、子、圣灵如何在你生命中工作？
- **祷告**：操练分别向父、向子、向圣灵祷告

### 第五周：认识神的属性

#### 核心经文
> "耶和华，耶和华，是有怜悯有恩典的神，不轻易发怒，并有丰盛的慈爱和诚实。"（出34:6）

#### 教导内容

**一、神的超越属性**

| 属性 | 含义 | 经文 |
|------|------|------|
| 永恒 | 神无始无终 | 诗90:2 |
| 全能 | 神无所不能 | 太19:26 |
| 全知 | 神无所不知 | 诗139:1-4 |
| 全在 | 神无处不在 | 诗139:7-10 |
| 自有 | 神不依赖任何 | 出3:14 |
| 不变 | 神永不改变 | 玛3:6 |

**二、神的道德属性**

| 属性 | 含义 | 经文 |
|------|------|------|
| 圣洁 | 完全分别、纯净 | 赛6:3 |
| 公义 | 公正、正直 | 申32:4 |
| 慈爱 | 恩慈、怜悯 | 约壹4:8 |
| 信实 | 守约、可靠 | 申7:9 |
| 智慧 | 全然美善的旨意 | 罗11:33 |

**三、平衡的神观**

```
        只强调圣洁公义          只强调慈爱怜悯
            ↓                      ↓
        恐惧、律法主义            放纵、廉价恩典
            ↘         ↙
            平衡的神观
         ┌─────────────┐
         │ 神是公义的   │
         │ 也是慈爱的   │
         │ 在十字架完美结合│
         └─────────────┘
```

**四、资治通鉴对照**

中国对"天"的理解：

| 概念 | 内容 | 与圣经对比 |
|------|------|------------|
| 天命 | 天有意志 | 神有主权 |
| 天理 | 天有公义 | 神是公义的 |
| 皇天 | 天至高无上 | 神是超越的 |

但圣经的神更加位格化、更加可认识。

#### 讨论问题

1. 神的哪一个属性对你最有吸引力？为什么？
2. 神的圣洁公义与慈爱怜悯如何在十字架上完美结合？
3. 认识神的属性如何影响你的日常生活？

#### 本周作业

- **阅读**：诗篇139篇
- **默想**：神的全知全在对我意味着什么？
- **祷告**：敬拜赞美神的伟大属性

### 第六周：认识圣经

#### 核心经文
> "圣经都是神所默示的，于教训、督责、使人归正、教导人学义都是有益的，叫属神的人得以完全，预备行各样的善事。"（提后3:16-17）

#### 教导内容

**一、圣经是什么**

```
┌─────────────────────────────────────────────┐
│                圣经的独特性                   │
├─────────────────────────────────────────────┤
│  来源：神所默示（神呼出的）                   │
│  内容：66卷书，40多位作者，1500年写成         │
│  中心：耶稣基督                               │
│  目的：使人因信得救，并指导生活               │
│  权威：最高的真理标准                         │
└─────────────────────────────────────────────┘
```

**二、圣经的结构**

| 部分 | 卷数 | 内容 |
|------|------|------|
| 旧约 | 39卷 | 律法、历史、诗歌、先知 |
| 新约 | 27卷 | 福音书、使徒行传、书信、启示录 |

**三、为什么信任圣经**

| 证据 | 说明 |
|------|------|
| 预言应验 | 300多条弥赛亚预言在耶稣身上应验 |
| 历史可靠 | 考古学不断证实圣经记载 |
| 内在一致 | 1500年写成，却高度一致 |
| 生命改变 | 无数人因圣经生命改变 |
| 圣灵见证 | 圣灵在心里印证真理 |

**四、对比中国经典**

| 圣经 | 中国经典 |
|------|----------|
| 神的启示 | 人的智慧 |
| 绝对权威 | 参考价值 |
| 一贯中心（基督） | 多元思想 |
| 改变生命 | 提升修养 |

> "述而不作，信而好古。"——孔子

孔子承认自己只是"述"而非"作"。圣经的作者则是神的代言人。

#### 讨论问题

1. 你以前怎么看圣经？现在有什么新的认识？
2. 什么帮助你相信圣经是神的话？
3. 圣经在你生命中扮演什么角色？

#### 本周作业

- **阅读**：诗篇119:1-48
- **开始**：建立每天读经的习惯（从约翰福音开始）
- **实践**：用SOAP方法读经（经文、观察、应用、祷告）

## 第三单元：灵命成长（第7-9周）

### 第七周：祷告生活

#### 核心经文
> "你们祷告，无论求什么，只要信，就必得着。"（太21:22）
> "不住地祷告。"（帖前5:17）

#### 教导内容

**一、什么是祷告**

```
祷告是...
┌─────────────────────────────────────┐
│  与神说话                          │
│  与神交通                          │
│  与神同行                          │
│  不只是求东西，更是建立关系         │
└─────────────────────────────────────┘
```

**二、祷告的种类**

| 种类 | 内容 | 例子 |
|------|------|------|
| 赞美 | 敬拜神的属性 | "主啊，你真伟大" |
| 感谢 | 为神的作为感恩 | "感谢你的供应" |
| 认罪 | 承认并悔改罪 | "求你赦免我" |
| 祈求 | 为需要祷告 | "求你帮助我" |
| 代求 | 为他人祷告 | "求你保守他" |

**三、主祷文——祷告的模式**

| 内容 | 焦点 |
|------|------|
| "我们在天上的父" | 关系——我们是神的儿女 |
| "愿人都尊你的名为圣" | 神的荣耀 |
| "愿你的国降临" | 神的国度 |
| "愿你的旨意行在地上" | 神的旨意 |
| "我们日用的饮食今日赐给我们" | 我们的需要 |
| "免我们的债如同我们免了人的债" | 我们的关系 |
| "不叫我们遇见试探；救我们脱离凶恶" | 我们的保护 |

**四、资治通鉴对照**

中国古人的"祷告"：

| 形式 | 对象 | 与基督教比较 |
|------|------|--------------|
| 祭祀 | 祖先、天地 | 我们祷告独一真神 |
| 占卜 | 求问吉凶 | 我们寻求神的旨意 |
| 求签 | 神明 | 我们与神对话 |

> "祭如在，祭神如神在。"——孔子

孔子说"如神在"，但我们知道神确实在、听我们祷告。

#### 讨论问题

1. 你目前的祷告生活如何？有什么挑战？
2. 主祷文教导我们祷告应该先关注什么？这改变你的祷告吗？
3. 祷告对你来说是"任务"还是"关系"？

#### 本周作业

- **阅读**：马太福音6:5-15
- **实践**：每天固定时间祷告15分钟
- **操练**：用主祷文的结构祷告一周

### 第八周：读经生活

#### 核心经文
> "你的话是我脚前的灯，是我路上的光。"（诗119:105）
> "惟喜爱耶和华的律法，昼夜思想，这人便为有福。"（诗1:2）

#### 教导内容

**一、为什么要读经**

| 原因 | 经文 |
|------|------|
| 认识神 | 约5:39 |
| 生命成长 | 彼前2:2 |
| 抵挡试探 | 太4:4 |
| 引导方向 | 诗119:105 |
| 心意更新 | 罗12:2 |

**二、如何读经**

```
SOAP读经法
┌─────────────────────────────────────────┐
│ S - Scripture（经文）：写下经文          │
│ O - Observation（观察）：注意到什么      │
│ A - Application（应用）：如何应用        │
│ P - Prayer（祷告）：回应神                │
└─────────────────────────────────────────┘
```

**三、读经的障碍与突破**

| 障碍 | 突破 |
|------|------|
| 没时间 | 安排固定时间，从5分钟开始 |
| 读不懂 | 使用注释、参加查经班 |
| 没兴趣 | 求圣灵帮助、找读经伙伴 |
| 没坚持 | 建立习惯、问责伙伴 |

**四、资治通鉴对照**

中国的读书传统：

| 中国读经典 | 基督徒读圣经 |
|------------|--------------|
| 为考功名 | 为认识神 |
| 死记硬背 | 明白应用 |
| 个人修养 | 生命改变 |
| 人的智慧 | 神的启示 |

> "书中自有颜如玉，书中自有黄金屋。"

中国人读书有功利目的，但读圣经是为认识神、改变生命。

#### 讨论问题

1. 你目前的读经状况如何？有什么困难？
2. SOAP读经法对你有帮助吗？试着分享一段你的SOAP。
3. 圣经中哪一卷书或哪一段经文对你最有意义？

#### 本周作业

- **阅读**：诗篇119:49-96
- **实践**：每天用SOAP读约翰福音一章
- **背诵**：诗篇119:105

### 第九周：胜过试探

#### 核心经文
> "你们所遇见的试探，无非是人所能受的。神是信实的，必不叫你们受试探过于所能受的；在受试探的时候，总要给你们开一条出路，叫你们能忍受得住。"（林前10:13）

#### 教导内容

**一、试探的来源**

| 来源 | 说明 | 经文 |
|------|------|------|
| 世界 | 世俗价值观的压力 | 约壹2:15-16 |
| 肉体 | 内在的私欲 | 加5:17 |
| 魔鬼 | 撒但的引诱 | 彼前5:8 |

**二、试探的过程**

```
私欲 → 引诱 → 怀胎 → 罪 → 死亡
        ↑
     可以在此拒绝
   "在受试探的时候，总要给你们开一条出路"
```

**三、胜过试探的策略**

| 策略 | 经文 | 应用 |
|------|------|------|
| 逃避 | 提后2:22 | 远离试探的环境 |
| 抵挡 | 雅4:7 | 顺服神，抵挡魔鬼 |
| 神的话 | 太4:4 | 用神的话回应 |
| 祷告 | 太26:41 | 警醒祷告 |
| 团契 | 加6:1-2 | 彼此扶持 |

**四、耶稣胜过试探的榜样**

| 试探 | 耶稣的回应 |
|------|------------|
| 石头变饼 | "人活着不是单靠食物" |
| 跳下殿顶 | "不可试探主你的神" |
| 拜撒但得万国 | "当拜主你的神，单要事奉他" |

每次都用"经上记着说"——神的话是得胜武器。

**五、资治通鉴对照**

中国历史中的试探与失败：

| 人物 | 试探 | 结果 |
|------|------|------|
| 纣王 | 酒色 | 亡国 |
| 李斯 | 权位 | 被杀 |
| 吴三桂 | 美色 | 降清 |

> "色即是空，空即是色。"——佛教试图否定欲望

圣经不否定欲望，而是引导欲望走向正确方向。
