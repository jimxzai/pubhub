# 视频课程大纲

> 《圣经与资治通鉴时空宇宙观》视频系列
> 三套课程：概览、深度解读、动画精华

---

## 系列一：圣经与资治通鉴概览

> 12集，每集15-20分钟
> 平台：YouTube、B站

### 系列目标
让观众快速了解整套材料的框架和核心内容。

### 视觉风格
- 现代简洁
- 信息图表
- 历史图片/画作
- 地图动画

---

### 第1集：为什么要对照圣经与资治通鉴？

**时长**：18分钟

**内容大纲**：

1. **开场**（2分钟）
   - 两本书的分量
   - 东西方智慧的结晶
   - 为什么要放在一起？

2. **这套材料是什么**（5分钟）
   - 护教的工具
   - 门训的材料
   - 传福音的桥梁
   - 灵命塑造的资源

3. **四大主题预览**（8分钟）
   ```
   透视人性 → 寻找出路 → 认识神的计划 → 明白属灵争战
   ```

4. **如何使用这套材料**（2分钟）
   - 个人研读
   - 小组查经
   - 教会门训

5. **结语**（1分钟）
   - 邀请订阅
   - 预告下集

**视觉元素**：
- 圣经与资治通鉴的书封
- 世界地图（东西对照）
- 四大主题图示

### 动画制作指南

**每集结构**（6分钟版）：

```
0:00-0:30  吸引眼球的开场问题/场景
0:30-2:00  圣经故事动画
2:00-3:30  中国历史故事动画
3:30-5:00  对比与核心信息
5:00-5:30  金句/应用
5:30-6:00  结尾CTA（关注、分享）
```

**视觉规范**：
- 竖屏9:16（短视频平台）
- 横屏16:9（YouTube）
- 大字幕，易读
- 品牌一致性

**音频规范**：
- 配音清晰
- 背景音乐轻快
- 音效适当
