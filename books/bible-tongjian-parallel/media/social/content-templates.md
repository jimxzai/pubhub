**尺寸**：
- 微信/微博：1080×1080px（正方形）
- Instagram：1080×1080px 或 1080×1350px

**视觉风格**：
```

│  背景：渐变色/水墨/简洁纯色            │
│  字体：经文用衬线体，出处用无衬线体    │
│  品牌：右下角放Logo                    │
```

**配色方案**：
- 主色：深蓝 #1a365d、暖金 #d69e2e

- 强调：红色 #c53030（谨慎使用）

## 四、互动问答

### 问答形式

**类型一：投票问答**
```
📊 今日投票

[问题]

A. [选项A]
B. [选项B]
C. [选项C]
D. 其他（留言）

💬 投完票，留言说说你的想法！

#互动 #信仰问答
```

**类型二：开放问题**
```
💭 今日讨论

[开放式问题]

例如：
"如果你是约瑟，你会原谅出卖你的兄弟吗？"

💬 在评论区分享你的想法！

#讨论 #信仰生活
```

**类型三：快问快答**
```
⚡ 快问快答

Q: [问题]

🅰️ [简短答案]

💡 详细解释：[补充说明]

📖 参考经文：[经文]

#信仰问答
```
