# 家庭與職場的新秩序
## Colossians 3:18-4:1 — The Household Code

**經文核對**：[ai-eden.com/bible/colossians/3](https://www.ai-eden.com/bible/colossians/3?t=CUV,ESV&cols=2) ｜ [colossians/4](https://www.ai-eden.com/bible/colossians/4?t=CUV,ESV&cols=2)

## 基督焦點 (Christ at the Center)

六種身分、六段關係，卻只有一位「主」貫穿全段——「在主裡面」「主所喜悅」「敬畏主」「給主做的」「你們所事奉的乃是主基督」「你們也有一位主在天上」。基督不是在教會裡才作主，祂要進到婚姻裡、進到親子關係裡、進到僕人與主人的日常勞動裡。祂不只是你敬拜的對象，祂是廚房裡、辦公室裡那位看得見一切、卻從不偏待人的主。

## 精義一句話

保羅剛講完「穿上新人」（3:12-17），立刻接著講「妻子、丈夫、兒女、父親、僕人、主人」——這不是話題跳開了，是刻意的安排：新人不是關在密室裡的個人靈修狀態，是要穿進家裡、穿進職場，活給最親近的人看的。這六組身分，被同一句話框住——「在主裡」「主所喜悅」「敬畏主」「給主做的」——保羅不是給你一套新的行為規條，是把每一段最日常的關係，重新掛在基督的名下。

## 黃長老這樣帶你讀

### 1. 六個身分，一個框架——「在主裡」重新定義一切關係

18-4:1這段家庭守則，列了三組共六種身分：妻子/丈夫、兒女/父親、僕人/主人。每一組命令的結尾或核心，都有一句框住全句的話：「這在主裡面是相宜的」（18節）、「這是主所喜悅的」（20節）、「敬畏主」（22節）、「像是給主做的」（23節）、「你們也有一位主在天上」（4:1）。留意：這六種身分在保羅那個時代的社會裡，地位天差地別——自由人與奴隸、丈夫與妻子——但保羅用同一個「主」字，把它們放進同一個框架裡衡量。不是「你的身分決定你怎麼被對待」，是「你們共同的主決定每個人怎麼行」。

帶領提問：**你在家裡、在職場的角色，通常是被誰的眼光定義的——是別人怎麼看你，還是「這是給主做的」？**

### 2. 妻子與丈夫——相宜的順服，不苦待的愛

18-19節是一組互相制衡的命令：「妻子……當順服自己的丈夫，這在主裡面是相宜的」——「順服」不是無條件的服從，是「在主裡面」才「相宜」（ἀνῆκεν，恰如其分、合宜）的順服，暗示丈夫的要求若不合乎主的心意，這順服就失去了根基。而丈夫的命令更重——「愛你們的妻子，不可苦待他們」——麥克阿瑟特別指出「愛」在原文是持續進行式：不是一次性下決心去愛，是每天不斷更新的動作，用來對治容易累積的苦毒（見下方〈一位牧者的洞見〉）。兩句命令合起來看，保羅要拆的正是歌羅西文化裡「丈夫可以隨意苦待妻子」的權力結構。

帶領提問：**如果「順服」與「愛」都是「在主裡面」才成立的命令，你今天在婚姻或親密關係裡的要求，是以自己的方便為出發點，還是以「這樣做討主喜悅」為出發點？**

### 3. 兒女與父親——凡事聽從，卻不可惹氣

20-21節同樣是一組互相制衡的命令：「兒女的，要凡事聽從父母，因為這是主所喜悅的」——但緊接著限制了父母的權柄：「你們作父親的，不要惹兒女的氣，恐怕他們失了志氣」。「惹氣」（ἐρεθίζετε）原文有「持續刺激、挑動」的意味——可能是過度嚴苛，也可能（麥克阿瑟特別指出）是過度保護、偏心、情緒失控（見下方〈一位牧者的洞見〉）。保羅給父母的權柄，同時附帶了一個限制：你的權柄若使孩子「失了志氣」（灰心喪志、失去盼望），你就用錯了這權柄。

帶領提問：**你在管教或帶領晚輩、下屬時，有沒有想過：你手上的權柄，是用來使人「更靠近主」，還是不小心使人「灰心喪志」？**

### 4. 僕人與主人——給主做的，不是給人做的

22-25節與4:1篇幅最長，這是因為當時教會裡確實有奴僕與主人同席敬拜的處境——這是初代教會一個極為激進的處境：社會地位懸殊的兩種人，在基督裡稱彼此為弟兄。23節是這整段最重要的一句：「無論做甚麼，都要從心裡做，像是給主做的，不是給人做的」——保羅沒有廢除奴僕制度（那是另一個歷史議題），但他徹底顛覆了工作的動機：你眼前服事的人是誰不重要，重要的是你知道自己「真正」在服事誰。24節說得更直接：「你們所事奉的乃是主基督」。

帶領提問：**你今天在職場上遇到一位不公平、甚至不值得尊敬的上司或客戶——「像是給主做的，不是給人做的」這句話，如果真的落實在你身上，你今天的工作態度會有什麼不同？**

## 一位牧者的洞見

### 麥克阿瑟：新人在家裡活出來 (John MacArthur)

麥克阿瑟把18-19節讀成一組配對命令：「你們作妻子的，要順服自己的丈夫，這在主裡面是相宜的」，對照「你們作丈夫的，要持續愛你們的妻子，不要苦毒相待」——他特別強調「持續」（keep on loving）的時態：愛不是一次性的決定，是每天更新的動作，用來對治積累的苦毒。論20-21節，他把兒女的命令濃縮為一個字：「兒女，要凡事聽從父母……這是主所喜悅的」；父親的命令則是一連串警告的總和：「不要惹你兒女的氣，免得他們灰心喪志」——過度保護、偏心、情緒失控，都可能是「惹氣」的具體形式。

## 整本聖經的連結

- **以弗所書 5:22-6:9 ↔ 歌羅西書 3:18-4:1**：這對姊妹書信的家庭守則幾乎並列——以弗所書更詳細地把丈夫愛妻子與「基督愛教會，為教會捨己」（弗5:25）連在一起，把婚姻本身變成福音的活圖畫；歌羅西書的版本更簡潔，卻同樣把每一組關係鎖進「在主裡」的框架。

- **腓利門書 ↔ 歌羅西書 3:22-4:1**：歌羅西書寫「僕人的，要凡事聽從你們肉身的主人」，同一批送信人（推基古、阿尼西母，西4:7-9）也帶著腓利門書——那封信正是保羅為一位逃奴阿尼西母向他的主人腓利門說情，求腓利門「不再是奴僕，乃是……親愛的兄弟」（門1:16）。歌羅西書的家庭守則，不是空談，是正在被實踐的活生生的個案。

## 讓話語住在裡面

**默想問題**：「無論做甚麼，都要從心裡做，像是給主做的，不是給人做的」（23節）——今天有哪一件你正在應付、敷衍、或帶著怨氣去做的事，如果換一個念頭：這是給主做的——你會不會換一種心態去做？

**今天的生命實踐**：想一個你目前關係中最容易起摩擦的角色（配偶、父母、兒女、上司、下屬）。今天在與這個人互動之前，先默想「這在主裡面是相宜的」或「像是給主做的」，然後才進入那段互動。晚上回想：這個框架有沒有改變你當下的反應？

## 你看見耶穌了嗎？

這段家庭守則表面上是行為規範，實際上是把耶穌基督放進了最日常的關係裡——婚姻、親子、職場。祂不只是你主日敬拜的對象，祂是你家裡的那位「主」，是你辦公桌前的那位「主」。新人的信仰不是週日一小時的事，是廚房裡、餐桌上、會議室裡，隨時都在服事的那一位。

## 反思問題 (Reflection Questions)

1. 「這在主裡面是相宜的」（3:18）——你在最親密的關係裡提出的要求，是以自己的方便為出發點，還是以「這樣做討主喜悅」為出發點？

2. 「不要惹兒女的氣」（3:21）——你手上對晚輩、下屬的權柄，是用來使人更靠近主，還是不小心使人灰心喪志？

3. 「像是給主做的，不是給人做的」（3:23）——你今天最應付、最帶著怨氣去做的一件工作，如果換一個念頭會怎樣？

4. 保羅沒有廢除奴僕制度，卻徹底顛覆了工作的動機（3:22-24）——這對你今天在職場上「該改變什麼、不該等什麼」，有什麼提醒？

5. 六種身分被同一位「主」框住——你今天在家裡、在職場上扮演的角色，哪一個最需要被「這是給主做的」重新定義？
