## 編排說明

- **分組脈絡**:1-5 為馬太13章「天國七喻」;6、17 為路加15章「失而復得」;7、8 論愛與饒恕;9、13、18 論恩典與自義;10-12 為橄欖山「末世三喻」;14、19、20 論錢財與永恆;15-16 論成長與禱告;21-26 為受難週與門徒代價;27 為登山寶訓終章,也是全卷收尾。
- **每章結構**:念經文 → 找鑰匙(主句/字義)→ 提問引導 → 以經解經(整本聖經貫通)→ 住(默想)→ 行(每日活祭)→ 回到核心(看見耶穌是誰)。
- **經文版本**:中文以和合本修訂版(RCUV)為主,英文以 ESV 為主。
- **聖經查證**:所有經文與引用應對照 [ai-eden.com](https://ai-eden.com) 核實(見專案記憶)。

## 第二輪・深掘

[`28-part2-deep-dive-greek-and-john.md`](28-part2-deep-dive-greek-and-john.md) — 一份整合的第二輪文件:

- **Part A・約翰福音貫通總綱**:約翰沒有比喻,卻用「七個我是 (ἐγώ εἰμι)」「七個兆頭 (σημεῖον)」把比喻的謎底直接寫出;含 27 個比喻 → 我是/兆頭的總對照表,及四條最強貫通線(好牧人、真葡萄樹、復活與生命、道路真理生命)。
- **Part B・逐章深掘**:全 27 章,每章【原文字義】(關鍵希臘文 + 轉寫)+【約翰連結】。
- ⚠️ 希臘文須於出版前對照 NA28/UBS5 與 BDAG 核實。
