# 舊約歷史書精讀

## The History Books — 以色列的史實骨幹與屬靈領受

舊約「歷史書」共 **十二卷**，記以色列從**進入應許之地**到**被擄歸回**約一千年的史實。這套筆記以**冷靜的史實（dates, events, what actually happened）為骨**，再帶出黃長老指向基督的讀法。

## 二、史實骨幹：以色列走過的一千年

| 年代 (BC) | 階段 | 大事 | 書卷 |
|------|------|------|------|
| ~1406 | 進入 | 過約旦河，耶利哥傾倒，分地 | 約書亞記 |
| ~1380–1050 | 士師 | 犯罪—受壓—哀求—拯救的循環 | 士師記·路得記 |
| ~1050 | 立王 | 撒母耳膏掃羅，以色列有了王 | 撒上 |
| ~1010–970 | 大衛 | 統一王國，大衛之約，定都耶路撒冷 | 撒下·代上 |
| ~970–931 | 所羅門 | 建第一聖殿（~959），國勢極盛 | 王上·代下 |
| **~931** | **分裂** | 南國猶大 / 北國以色列 | 王上 12 |
| ~931–722 | 分立 | 先知（以利亞、以利沙）對抗偶像 | 王上下 |
| **722** | **北國亡** | 以色列亡於亞述，撒瑪利亞被擄 | 王下 17 |
| ~701 | 存留 | 希西家禱告，亞述退兵 | 王下 18–19 |
| ~622 | 復興 | 約西亞尋見律法書，全國改革 | 王下 22–23 |
| **586** | **南國亡** | 猶大亡於巴比倫，聖殿被焚，被擄 | 王下 25·代下 36 |
| **538** | **歸回令** | 古列下詔，准猶大人歸回 | 拉 1 |
| 536–516 | 重建殿 | 所羅巴伯建第二聖殿 | 以斯拉記 |
| ~483–473 | 散居 | 以斯帖在波斯，普珥的拯救 | 以斯帖記 |
| ~458 | 重建信 | 以斯拉教導律法 | 以斯拉記 |
| ~445 | 重建城 | 尼希米建耶路撒冷城牆 | 尼希米記 |

**一句話串起來**：神**應許**地（約書亞）→ 人**悖逆**陷循環（士師）→ 神**立王**並向大衛**起誓**（撒上下）→ 王國**分裂、墮落**（列王紀）→ 因不聽而**被擄**（列王紀下、代下）→ 神**信實領回**餘民，重建殿、牆、心，並在暗中**保全**子民（以斯拉、尼希米、以斯帖）。

## 三、貫穿全套的一條線：指向基督

黃長老：「你研讀聖經，卻沒有看見耶穌——那本書等於白讀了。」十二卷歷史書，箭頭都指向同一位：

| 影子 | 應驗 |
|------|------|
| 約書亞領進迦南的安息 | 耶穌領進真安息（來 4） |
| 喇合、路得（外邦人）入家譜 | 救恩臨到萬民（太 1:5） |
| 士師都不完全的拯救者 | 完全的拯救者—基督 |
| 大衛之約「國位直到永遠」 | 「大衛的子孫」作王到永遠（路 1:32-33） |
| 聖殿的榮耀充滿又流失 | 道成肉身，榮耀永住（約 1:14） |
| 被擄與歸回 | 被擄的得釋放（路 4:18） |
| 以斯帖保全猶大人 | 保全那通向彌賽亞的線 |

> 整段歷史，是神為「大衛的子孫，耶穌基督」（太 1:1）所預備的舞台。
