# 主線貫穿：兩條線，一位王

## Two Threads, One King

黃長老說：「以經解經——用聖經解釋聖經。」逐卷讀完，再抽出**一條線**從頭拉到尾，整套書就活了。撒母耳記與歷代志裡有兩條最粗的線，最後在同一個人身上打結——**耶穌基督**。

---

## 線一：王 (Kingship) — 從「像列國的王」到「萬王之王」

### 追蹤這條線

| 站 | 經文 | 這一站在說什麼 |
|----|------|--------------|
| 1. 無王 | 士 21:25 | 「各人任意而行」——人要一個王的心已經發動 |
| 2. 求王 | 撒上 8:5 | 「立一個王治理我們，像列國一樣」 |
| 3. 厭棄真王 | 撒上 8:7 | 「他們厭棄我，不要我作他們的王」 |
| 4. 人要的王 | 撒上 9–15 | 掃羅：身量高大，卻悖逆被棄 |
| 5. 神揀的王 | 撒上 16:7 | 大衛：神看內心，合神心意 |
| 6. 受膏卻受苦 | 撒上 16–31 | 受膏多年，曠野山洞，被追殺 |
| 7. 登基 | 撒下 2, 5 | 希伯崙七年，再統一全國 |
| 8. 永遠的王位 | 撒下 7:16 | 「你的國位必堅立，直到永遠」 |
| 9. 王也跌倒 | 撒下 11–18 | 拔示巴、押沙龍——最好的王也是罪人 |
| 10. 坐神的位 | 代上 29:23 | 所羅門「坐在耶和華的位上」 |
| 11. 王位被擄 | 代下 36 | 聖殿被焚，王朝中斷 |
| 12. 一線存留 | 代下 36:23 | 「上去吧」——大衛的根還在 |

### 這條線的歸宿

整條「王」的線，是一個漸進的問題：**真正的王是誰？**
- 不是身量最高的（掃羅）；
- 也不是合神心意卻仍會跌倒的（大衛）；
- 而是那位「大衛的子孫」——

> 「他要為大，稱為至高者的兒子；主神要把他祖大衛的位給他。他要作雅各家的王，直到永遠；他的國也沒有窮盡。」（路 1:32-33）

最終在啟示錄收尾：「在他衣服和大腿上有名寫著說：**萬王之王，萬主之主。**」（啟 19:16）

> **帶領提問**：以色列要一個「像列國一樣」的王，結果換來掃羅。你心裡有沒有也立過一個「像列國一樣」的王——某個看得見、能倚靠的東西，把那看不見的真王擠到一邊？

## 線二：大衛之約 (The Davidic Covenant) — 整套書的心臟

### 約的內容（撒下 7 / 代上 17）

神向大衛起的誓，有三個「永遠」：

1. **永遠的後裔**：「我必興起你的後裔……」（撒下 7:12）
2. **永遠的國**：「我必堅定他的國位，直到永遠。」（撒下 7:13）
3. **永遠的家**：「你的家和你的國，必在我面前永遠堅立。」（撒下 7:16）

關鍵的翻轉：大衛要為神建「殿」（house），神卻說要為大衛建「家」（house）——同一個字，方向相反。**不是人為神做，是神為人做。** 這是恩典的方向。

### 追蹤這條線怎樣展開

| 站 | 經文 | 約如何延續 |
|----|------|-----------|
| 立約 | 撒下 7:16 | 神起誓，無條件 |
| 大衛領受 | 撒下 7:18-29 | 「主耶和華啊，我是誰？」 |
| 詩中傳唱 | 詩 89, 132 | 「我與我所揀選的人立了約」（詩 89:3） |
| 重述安慰 | 代上 17 | 被擄餘民聽見：約沒有廢 |
| 危機 | 代下 36 | 王朝中斷——約看似落空 |
| 應驗 | 太 1:1；路 1:32-33 | 「大衛的子孫，耶穌基督」 |
| 永遠成全 | 啟 22:16 | 「我是大衛的根，又是他的後裔」 |

### 這約為什麼是「福音」的根

馬太福音第一句：「亞伯拉罕的後裔，**大衛的子孫**，耶穌基督的家譜。」（太 1:1）——新約一開卷，就站在撒下 7 章上。你不讀撒下 7，就不懂太 1:1 為什麼要強調「大衛的子孫」。

黃長老的話：「**你們若不信摩西、不讀舊約的這些書，怎能信我的話？** 大衛之約是地基，福音是建在這地基上的房子。地基不認識，房子就懸空。」

> **帶領提問**：大衛之約是「無條件」的——神起誓，不靠人守得住。這和你平時對神的理解一樣嗎？你是不是常以為，神的應許要看「我表現得好不好」？恩典的約，是神為自己的名作的，不是為我的好行為。

## 兩條線的交匯：那位受膏君（彌賽亞）

「王」這條線在問：誰是真王？
「約」這條線在應許：大衛家必有一位永遠的王。

兩條線在一個稱號上交匯——**彌賽亞 / 基督（受膏者，Anointed One）**。撒上裡，撒母耳用油膏掃羅、膏大衛；那膏立的動作，預表那位終極的「受膏者」。

> 「**基督（Christ）**」這個字，就是希臘文的「受膏者」；希伯來文是「**彌賽亞（Messiah）**」。撒母耳手裡的那瓶膏油，一路指向各各他——那位真正受膏、為我們受苦、又得榮耀作王到永遠的耶穌。

**一個禱告**：主耶穌，你是大衛的根，又是大衛的子孫；是那位受膏的君王，國位直到永遠。求你拆掉我心裡一切「像列國一樣」的假王，獨坐我心的寶座。你的約靠你自己堅立，我就安息在你的信實裡。阿們。
