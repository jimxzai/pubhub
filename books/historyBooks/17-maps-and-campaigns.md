# 地圖與戰役

## The Geography of Israel's Fall, Exile, and Return

歷史書的審判，是在真實的地理上發生的。這份用文字地圖把**兩國覆亡、被擄路線、歸回路線**畫出來。

> 經文據和合本；地名沿用和合本譯名，括註現代/英文名。

## 一、肥沃月灣全景（為什麼被擄要繞遠路）

被擄不是橫穿阿拉伯沙漠（無水），而是沿著**肥沃月灣（Fertile Crescent）**這道弧形綠帶北上、再南下。這解釋了為什麼亞述、巴比倫的大軍與被擄的隊伍，都走同一條弧線。

```
                          ┌─────── 亞述 Assyria ───────┐
                          │  尼尼微 Nineveh ·          │
   地中海                 │         · 迦拉 Calah        │   瑪代
 Mediterranean            │  哈博河 Habor ·            │   Media
      │                   │                            │  (米底亞)
 ┌────┴─────┐        哈馬 Hamath                       │
 │  以色列   │         · 利比拉 Riblah                  │
 │  Israel   │              │                          │
 │ 撒瑪利亞· │              │      幼發拉底河            │
 │ 耶路撒冷· │           大馬色 ·   Euphrates           │
 │  Judah    │          Damascus    ╲                  │
 └────┬──────┘                       ╲   · 巴比倫       │
      │                               ╲   Babylon       │
   猶大曠野                            ╲    · 吾珥 Ur    │
      │                                ╲________________│
    埃及 Egypt（南）                波斯灣 Persian Gulf
```

> 弧線的左端是以色列／猶大，右下端是巴比倫。被擄的人，沿弧線「上去」（往北經利比拉）再「下到」巴比倫——所以聖經說「**被擄到巴比倫**」，路途上千公里。

## 二、王國分裂後的南北地理（~931 BC）

```
            ┌──────────── 北國以色列 Israel ────────────┐
            │  但 Dan ·（金牛犢）                        │
            │        · 夏瑣 Hazor                        │
   地中海   │   · 多坍                                   │
   Med. Sea │        · 撒瑪利亞 Samaria（首都）         │ 約旦河
            │   · 米吉多 Megiddo                         │ Jordan
            │        · 示劍 Shechem                      │   │
            │   · 伯特利 Bethel ·（金牛犢）             │   │
            ├───────────────────────────────────────────┤ 死海
            │  · 耶路撒冷 Jerusalem（首都）             │ Dead
            │        · 伯利恆                            │ Sea
            │   南國猶大 Judah                           │
            │        · 希伯崙 Hebron                     │
            │   · 拉吉 Lachish                           │
            └───────────────────────────────────────────┘
                          · 別是巴 Beersheba
```

- **北國以色列**：地大、人多、土肥，但無聖殿，金牛犢在**但**（最北）與**伯特利**（最南近猶大邊界）——耶羅波安要攔住百姓南下耶路撒冷敬拜。
- **南國猶大**：地小、山多，卻有**耶路撒冷聖殿**與**大衛王朝**——這是它多存 136 年的兩個原因。

## 三、北國覆亡（722 BC）——亞述的鐵蹄

```
   亞述進攻方向（由北而南）
   提革拉毗列色三世(734-732) → 撒縵以色五世(725-722圍城) → 撒珥根二世(722陷城)

        亞述 Assyria
            │ ↓
        大馬色 Damascus（732 先亡，亞蘭被滅）
            │ ↓
   ┌────────┴────────┐
   │  · 但 / 夏瑣      │  ← 加利利、約旦河東先被擄（王下 15:29）
   │                  │
   │  · 撒瑪利亞 ★    │  ← 圍城三年，722 陷落（王下 17:5-6）
   │                  │
   └──────────────────┘
            │
       北國亡，十支派被擄往：
       哈臘 Halah、哈博河 Habor（歌散 Gozan）、瑪代的城邑
       （王下 17:6）
```

**關鍵經文**：王下 17:6「亞述王……將以色列人擄到亞述，把他們安置在哈臘與歌散的哈博河邊，並瑪代人的城邑。」
**亡國判詞**：王下 17:7-23——因拜偶像、不聽眾先知勸戒。
**後果**：亞述移外族入撒瑪利亞，與餘民混居 → 後來的**撒瑪利亞人**（約 4 章背景）。

## 四、亞述未能拿下耶路撒冷（701 BC）——希西家蒙拯救

北國亡後二十一年，亞述西拿基立揮軍南下猶大：

```
   西拿基立 Sennacherib（701）
        │ ↓
   · 拉吉 Lachish ★（攻陷，亞述浮雕著名戰役）
        │ ↓
   · 耶路撒冷 ——圍而未陷！
     希西家展信禱告（王下 19:14-19）
     一夜之間，耶和華的使者擊殺亞述營中 185,000 人（王下 19:35）
     西拿基立撤回尼尼微，後被己子所弒
```

> 南國這次得存，不是因軍力，是因**禱告**。這也成了以賽亞「餘民」信息的活見證。

## 五、南國覆亡（605 / 597 / 586 BC）——巴比倫三擊

巴比倫先在 **迦基米施（Carchemish, 605 BC）** 擊敗埃及，取得近東霸權，然後三次南下打猶大：

```
   巴比倫 尼布甲尼撒 Nebuchadnezzar
        │ ↓ 沿肥沃月灣南下，經利比拉 Riblah（指揮部）
        │
   ① 605 約雅敬年：擄走貴胄少年（但以理等）   ┐
   ② 597 約雅斤年：擄約一萬人（以西結等）、聖殿器皿 │ 三梯次
   ③ 586 西底家年：圍城破城 ★                    ┘
        │
   · 耶路撒冷陷落（586）
     - 聖殿被焚（王下 25:9）
     - 城牆拆毀
     - 西底家在利比拉被剜眼、鎖往巴比倫（王下 25:7）
     - 護衛長尼布撒拉旦擄掠百姓
        │ ↓
   被擄之民沿弧線北上經利比拉，再東南下到巴比倫
```

**關鍵經文**：王下 25 全章；耶 52；代下 36:17-21。
**被擄年限**：七十年（耶 25:11；29:10 → 應驗於代下 36:21；但 9:2）。
**逃亡支線**：部分餘民（含耶利米）被擄往**埃及**（耶 43）。

## 六、歸回路線（538 BC 起）——「上去吧」

波斯古列 539 BC 滅巴比倫，538 BC 下詔准歸回。歸回是**逆著被擄的弧線走回來**：

```
   巴比倫 Babylon
        │ ↑ 沿肥沃月灣西北上，再南下迦南
        │
   ① 538 所羅巴伯率約 5 萬人歸回 → 重建祭壇、聖殿（516 完工）
   ② 458 以斯拉率文士、利未人歸回 → 重建律法
   ③ 445 尼希米歸回 → 52 天重建耶路撒冷城牆
        │ ↓
   · 耶路撒冷 ——殿、牆、群體，逐步重建
```

> 注意方向詞：被擄是「**下到**巴比倫」，歸回是「**上去**耶路撒冷」（拉 1:3「可以上去」）。聖城在山上——屬靈上，回到神面前永遠是「上行」（參上行之詩，詩 120–134）。

## 八、一句屬靈的領受（黃長老）

> 「你看這地圖——亞述從北邊下來，巴比倫從北邊下來，連被擄都往北繞。先知早說了：『必有災禍從北方發出』（耶 1:14）。神的管教，不是地理的巧合，是祂親手安排的路。但你再看歸回——同一條路，反著走回來，叫『上去』。**神能叫你被擄的路，成為你歸回的路。**」

**默想**：你生命中有沒有一條「被擄的路」——失去、流放、繞遠？求神叫它有一天成為你「上去」的歸回之路。
