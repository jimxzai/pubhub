# 附錄六：歷史背景——羅馬奴隸制度
**Historical Background: Roman Slavery**

---

## 一世紀羅馬帝國的奴隸制度

| 方面 | 描述 |
|------|------|
| **人口比例** | 羅馬帝國約 30-40% 人口為奴隸 |
| **奴隸來源** | 戰俘、債務、出生、被遺棄的嬰兒 |
| **法律地位** | 奴隸是「會說話的工具」(instrumentum vocale)，是財產而非人 |
| **主人權力** | 主人對奴隸有生殺大權 |
| **逃奴命運** | 可被處死、烙印、鞭打、釘十字架 |

---

## 逃奴法 (Lex Fugia)

| 法律規定 | 處罰 |
|---------|------|
| 逃跑 | 可被主人處死或出售 |
| 協助逃奴 | 嚴厲處罰協助者 |
| 抓獲逃奴 | 有獎金鼓勵舉報 |
| 逃奴標記 | 額頭烙上 "F" (Fugitivus) |

### 阿尼西母的危險處境

按照羅馬法，腓利門有權對阿尼西母：
- 處死
- 額頭烙印
- 嚴厲鞭打
- 賣給更殘忍的主人
- 送去做礦工（等於死刑）

這就是為什麼保羅的代求如此重要。

---

## 奴隸的日常生活

### 家庭奴隸 (Familia Urbana)

- 管家、秘書、教師、醫生
- 生活條件相對較好
- 可能受過良好教育
- 阿尼西母可能屬於此類

### 農業奴隸 (Familia Rustica)

- 在莊園中勞動
- 生活條件惡劣
- 經常被鏈子鎖住

### 礦工奴隸

- 最惡劣的處境
- 平均壽命極短
- 逃奴常被送到礦場

---

## 奴隸的權利（或缺乏權利）

| 方面 | 情況 |
|------|------|
| 婚姻 | 法律上不承認 |
| 子女 | 屬於主人的財產 |
| 財產 | 法律上不能擁有（但實際上有 peculium 制度）|
| 證詞 | 只有受刑訊後才能作證 |
| 宗教 | 通常跟隨主人的宗教 |

---

## 釋放奴隸 (Manumissio)

奴隸可以通過以下方式獲得自由：

1. **遺囑釋放**：主人死後在遺囑中釋放
2. **購買自由**：用積蓄贖身
3. **因功釋放**：特殊服務後獲釋
4. **法律程序**：正式的法律行為

釋放後成為「自由人」(libertus)，但仍對前主人有義務。

---

## 福音的革命

保羅沒有直接呼籲廢除奴隸制，但他的信息播下了廢奴的種子：

| 保羅的教導 | 長遠影響 |
|-----------|---------|
| 「不再是奴僕，是兄弟」(門16) | 挑戰奴隸制的道德基礎 |
| 「在基督裡不分主僕」(加3:28) | 確立人在神面前的平等 |
| 「收納他如同收納我」(門17) | 要求主人改變對待奴隸的方式 |

### 為什麼保羅沒有直接呼籲廢奴？

1. **時機問題**：立即廢奴會造成社會崩潰
2. **策略問題**：從內在改變比外在革命更持久
3. **優先順序**：福音的核心是靈魂的拯救
4. **信任過程**：種子會成長為大樹

---

## 基督教與廢奴運動

早期教會的做法：
- 鼓勵主人善待奴隸
- 鼓勵主人釋放奴隸
- 接納奴隸為教會成員
- 有些奴隸成為教會領袖（如阿尼西母的傳統）

後來的影響：
- 威廉·威伯福斯（英國廢奴運動）
- 美國廢奴主義者多是基督徒
- 「不再是奴僕，是兄弟」成為廢奴的聖經依據
