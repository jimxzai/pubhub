# 第十一章：饒恕的極限
**The Limits of Forgiveness: Enemies, Hatred, and Reconciliation**

---

> 「\textcolor{RedemptionRed}{父阿！赦免他們；因為他們所做的，他們不曉得。}」
> — 路加福音 23:34

> « Toute sa vie, il avait fait consister la bonté dans le respect de la loi ; et voilà qu'il venait de voir une bonté qui était hors la loi, au-dessus de la loi. »
>
> 中文翻譯：「他一生都相信良善在於遵守律法；但現在他看見了一種在律法之外、超越律法的良善。」
> — 雨果，《悲慘世界》

---

## 引言：饒恕有極限嗎？

這是人類歷史上最困難的問題之一。

- 大屠殺的受害者要饒恕納粹嗎？
- 被強暴的人要饒恕施暴者嗎？
- 家人被殺害的人要饒恕兇手嗎？
- 被出賣的人要饒恕背叛者嗎？

當彼得問耶穌這個問題時，他以為自己已經很慷慨了：

> 彼得問：「主啊，我弟兄得罪我，我當饒恕他幾次呢？到七次可以嗎？」
> 耶穌說：「\textcolor{RedemptionRed}{我對你說，不是到七次，乃是到**七十個七次**。}」— 馬太福音 18:21-22

**七十個七次**——490次？還是**無限次**？

MacArthur 在「饒恕的時代」講道中觀察到：我們活在一個充滿憤怒的時代，人們不願饒恕，以為記仇是一種力量。但正如第五章所探討的，饒恕恰恰是最像神的時刻，是屬靈力量的最高展示。

耶穌的回答不是給出一個數字，而是徹底顛覆了彼得的計算邏輯：

彼得以為自己已經很慷慨了——七次！猶太拉比傳統通常要求饒恕三次，他願意加倍。但耶穌的回答徹底粉碎了這種「計算式饒恕」的邏輯：不是七次，而是七十個七次——意思不是490次然後停止，而是**根本不要計數**。饒恕不是交易，不是記帳本上的一筆賬目，而是一種生命態度。正如保羅所說：「愛是不計算人的惡」（林前13:5）。

然而，聖經對饒恕的教導令人震驚的不只是「七十個七次」，而是這個警告：

> 「\textcolor{RedemptionRed}{你們若不饒恕人，你們的天父也不饒恕你們的過犯。}」— 馬太福音 6:15

這是否意味著饒恕是簡單的、廉價的、或不需要過程的？

絕對不是。

## 第一節：十字架——饒恕的源頭

### 人類歷史上最震撼的饒恕宣言

> 「\textcolor{RedemptionRed}{父阿！赦免他們；因為他們所做的，他們不曉得。}」— 路加福音 23:34

這是耶穌在十字架上的第一句話，也是人類歷史上最震撼的饒恕宣言。

| 處境 | 人的反應 | 耶穌的回應 |
|------|---------|-----------|
| 被門徒離棄 | 怨恨、報復 | 「\textcolor{RedemptionRed}{父啊，赦免他們}」 |
| 被猶大出賣 | 咒詛、論斷 | 「\textcolor{RedemptionRed}{朋友，你來要做的事，就做吧}」(太26:50) |
| 被彼得否認 | 傷心、絕交 | 復活後三次問「\textcolor{RedemptionRed}{你愛我嗎？}」(約21) |
| 被兵丁戲弄 | 反擊、控訴 | 沉默如羊 (賽53:7) |
| 被釘十字架 | 苦毒、報仇 | 為殺害祂的人代求 |

### 「他們不曉得」——饒恕的基礎

耶穌沒有說「他們沒有錯」，而是說「他們不明白自己在做什麼」。

饒恕不是否認傷害，而是超越傷害。

這正是保羅對腓利門的暗示：阿尼西母「從前沒有益處」，但那是因為他還未認識基督。現在，他已經是新造的人了。

### 饒恕的鏈條——從十架到腓利門

| 階段 | 饒恕的傳遞 | 經文 |
|:----:|-----------|------|
| 1 | 天父饒恕世人 | 約3:16 |
| ↓ | | |
| 2 | 基督在十架上代求：「\textcolor{RedemptionRed}{父啊，赦免他們}」 | 路23:34 |
| ↓ | | |
| 3 | 司提反為逼迫者禱告：「主啊，不要將這罪歸於他們」 | 徒7:60 |
| ↓ | | |
| 4 | 掃羅（保羅）成為使徒——司提反禱告的果效 | 徒9 |
| ↓ | | |
| 5 | 保羅為阿尼西母代求：「都歸在我的賬上」 | 門18 |
| ↓ | | |
| 6 | 腓利門被呼召饒恕阿尼西母 | 門17 |
| ↓ | | |
| 7 | **我們被呼召饒恕那些虧負我們的人** | 太6:14 |

**司提反的饒恕，間接催生了寫腓利門書的保羅。**

當司提反被石頭打死時，「看守那些打死他之人的衣裳」的，正是年輕的掃羅（徒7:58）。特土良說得好：「殉道者的血是教會的種子。」司提反的禱告如種子落在掃羅心中，後來在大馬士革路上發芽。

## 第二節：沙威——無法接受恩典的悲劇

### 沙威的世界觀——律法主義的化身

雨果筆下的沙威探長（Javert）是文學史上最悲劇的人物之一。他代表了一種完美的律法主義——以及律法主義的終極崩潰。

**沙威的背景**：
- 生於監獄（母親是犯人）
- 從小選擇站在「秩序」這一邊
- 一生執法，從不例外
- 深信法律即正義

**沙威的信條**：

| 信條 | 問題 | 聖經視角 |
|------|------|---------|
| 「法律是最高準則」 | 沒有恩典的空間 | 「字句叫人死，精意叫人活」(林後3:6) |
| 「犯罪必受懲罰」 | 沒有悔改的可能 | 「神的恩慈是領你悔改」(羅2:4) |
| 「一次罪犯，永遠罪犯」 | 沒有更新的盼望 | 「若有人在基督裡，他就是新造的人」(林後5:17) |
| 「我是公義的執行者」 | 自以為義 | 「神阻擋驕傲的人」(雅4:6) |

### 沙威與冉阿讓的對峙

沙威追捕冉阿讓二十年。對他來說，冉阿讓永遠是24601——那個偷麵包的罪犯，永遠不可能改變。

但當革命爆發，冉阿讓有機會殺死被俘虜的沙威時，他選擇放他走：

> « Vous êtes libre. »
>
> 中文翻譯：「你自由了。」

這句話的深刻含義：
1. **物理層面**：我不殺你
2. **關係層面**：我不恨你
3. **神學層面**：我釋放你脫離我們之間的仇恨
4. **屬靈層面**：我盼望你也能得自由

### 沙威的崩潰

當冉阿讓饒恕沙威時，沙威的整個世界觀崩塌了：

**沙威的困境**：
1. 冉阿讓是罪犯 → 但他行善
2. 我執行法律 → 但他饒恕我
3. 法律是最高準則 → 但有比法律更高的東西
4. 我一生的意義 → 被徹底顛覆

**雨果描寫沙威的內心掙扎**：

> « Toute sa vie, il avait fait consister la bonté dans le respect de la loi ; et voilà qu'il venait de voir une bonté qui était hors la loi, au-dessus de la loi. »
>
> 「他一生都相信良善在於遵守律法；但現在他看見了一種**在律法之外、超越律法**的良善。」

**沙威的最後獨白**：

> « Il y avait donc quelque chose au-dessus de la loi? »
>
> 中文翻譯：「難道真有**比法律更高**的東西嗎？」

沙威無法整合這個認知。他的選擇是投河自殺——**拒絕恩典的終極悲劇**。

### 沙威的神學警示

沙威代表了一種「宗教性」但沒有恩典的生命：

| 沙威的特質 | 宗教性的扭曲 | 恩典的對比 |
|-----------|-------------|-----------|
| 嚴格遵守規條 | 律法主義 | 「字句是叫人死，精意是叫人活」(林後3:6) |
| 不容許例外 | 缺乏憐憫 | 「我喜愛憐恤，不喜愛祭祀」(太9:13) |
| 自以為義 | 驕傲 | 「自以為義的...神阻擋他們」(雅4:6) |
| 無法接受恩典 | 拒絕福音 | 「你們得救是本乎恩，也因著信」(弗2:8) |

**沙威的悲劇提醒我們**：單靠律法不能得救，甚至不能活下去。人需要恩典。

他的問題不是別人無法饒恕他，而是**他無法接受饒恕**。

### 瑪絲洛娃——被饒恕者的抗拒

如果沙威代表「無法給出恩典的執法者」，托爾斯泰筆下的瑪絲洛娃則代表「無法接受恩典的受害者」。

聶赫留朵夫在法庭上認出瑪絲洛娃後，試圖贖罪。但瑪絲洛娃的反應不是感激，而是冰冷的拒絕：

> « Вы мной в этой жизни воспользовались, вы мной хотите и в будущей жизни спастись! »
>
> 中文翻譯：「你在今生利用了我，現在你想用我來為你的來世贖罪！」

她拒絕饒恕聶赫留朵夫，不是因為驕傲（像沙威），而是因為**太深的傷痛使她不敢再信任**：

| 瑪絲洛娃的想法 | 內心的傷痛 |
|--------------|----------|
| 「你只是為了自己的良心」 | 被利用的恐懼 |
| 「我不需要你的施捨」 | 尊嚴的自我保護 |
| 「你改變不了什麼」 | 絕望的麻木 |

但托爾斯泰沒有停在這裡。在小說的後半部分，瑪絲洛娃漸漸解凍——不是因為聶赫留朵夫的努力感動了她，而是因為她在西伯利亞流放途中遇見了真正無私的政治犯西蒙松，看見了純粹善良的可能性。她的心牆被慢慢融化，最終選擇了寬恕與新生。

**饒恕的雙重極限——沙威與瑪絲洛娃的對照**：

| 面向 | 沙威的極限 | 瑪絲洛娃的極限 |
|------|----------|-------------|
| 障礙本質 | 無法給出恩典 | 無法接受恩典 |
| 根源 | 律法的僵化 | 傷痛的防衛 |
| 攔阻 | 驕傲 | 恐懼 |
| 結局 | 自我毀滅 | 漸漸解凍，重獲生命 |

瑪絲洛娃最終的「復活」提醒我們：饒恕不只是饒恕者的掙扎，也是被饒恕者的掙扎。有時候，**接受饒恕比給出饒恕更難**。腓利門書的場景也暗示了這一點——阿尼西母願意冒死回到腓利門面前，正是他接受保羅的恩典、準備好接受腓利門饒恕的表現。

## 第三節：約瑟——被害卻饒恕的典範

### 約瑟的苦難與饒恕

與沙威形成鮮明對比的，是舊約中的約瑟。他有充分的理由不饒恕：

| 約瑟的苦難 | 人的惡意 | 神的美意 |
|-----------|---------|---------|
| 被兄弟出賣為奴 | 嫉妒、仇恨 | 送往埃及 |
| 被波提乏之妻誣陷 | 欲望、謊言 | 下到監獄 |
| 被酒政遺忘 | 忘恩負義 | 等候神的時間 |
| 二十多年的苦難 | 人的惡意累積 | 神的計劃展開 |

當約瑟成為埃及宰相，兄弟們因饑荒來求糧時，他完全有權力報復。

但他選擇了饒恕：

> 「從前你們的意思是要害我，但**神的意思原是好的**，要保全許多人的性命，成就今日的光景。」— 創世記 50:20

### 約瑟與沙威的對比

| 約瑟 | 沙威 |
|------|------|
| 是受害者 | 是執法者 |
| 被人虧欠 | 追捕「虧欠者」 |
| 選擇饒恕 | 無法接受被饒恕 |
| 看見神的主權 | 只看見法律 |
| 生命結局：全家和好 | 生命結局：自殺 |

**約瑟的智慧**：他不是說「你們做的沒關係」，而是說「神把你們的惡意轉化為美意」。饒恕不是否認傷害，而是將審判權交給神。

## 第四節：苦毒的根——不饒恕的毒害

### 聖經的警告

> 「又要謹慎，恐怕有人失了神的恩；恐怕有**毒根生出來擾亂你們**，因此叫眾人沾染污穢。」— 希伯來書 12:15

「苦毒的根」（πικρίας ῥίζα）是什麼？就是**埋在心裡的不饒恕**。

### 苦毒的生長過程

| 階段 | 內心狀態 | 外在表現 |
|------|---------|---------|
| 種子 | 「他傷害了我」 | 記住傷害 |
| 發芽 | 「他必須付代價」 | 開始怨恨 |
| 成長 | 「我不會忘記」 | 反覆回想 |
| 開花 | 「我要讓他知道」 | 尋找報復機會 |
| 結果 | 「我永遠不會原諒」 | 苦毒充滿生命 |
| 擴散 | 向他人訴說委屈 | 污染他人 |
| 毒害 | 拒絕釋放對方 | **其實是囚禁自己** |

### 苦毒 vs. 饒恕

| 苦毒 | 饒恕 |
|------|------|
| 「他不配被饒恕」 | 「我也不配被饒恕，但神饒恕了我」 |
| 「我要他付出代價」 | 「基督已經付了代價」 |
| 「我永遠不會忘記」 | 「我選擇不再用這事定罪他」 |
| 「等他道歉我才饒恕」 | 「饒恕是我的決定，不取決於他」 |
| 「我享受受害者身份」 | 「我選擇從受害者變成得勝者」 |

### 文學中的苦毒

| 作品 | 人物 | 苦毒的表現 | 結局 |
|------|------|-----------|------|
| 悲慘世界 | 沙威 | 無法接受冉阿讓的改變 | 自殺 |
| 復活 | 瑪絲洛娃（起初） | 對聶赫留朵夫的冷漠與敵意 | 後來釋放、重生 |
| 基督山恩仇記 | 唐泰斯 | 一生追求報仇 | 最終領悟「等待與希望」 |

保羅以弗所書的呼籲：

> 「一切苦毒、惱恨、忿怒、嚷鬧、毀謗，並一切的惡毒，都當從你們中間除掉。並要以恩慈相待，存憐憫的心，彼此饒恕，**正如神在基督裡饒恕了你們一樣**。」— 以弗所書 4:31-32

## 第五節：基督山恩仇記——復仇的終極反思

### 唐泰斯的復仇之路

大仲馬的《基督山恩仇記》提供了另一種處理背叛的方式。

愛德蒙·唐泰斯（Edmond Dantès）被三個人陷害入獄十四年：
- **鄧格拉斯**：出於嫉妒
- **費爾南**：為了奪取他的未婚妻
- **維爾福**：為了保護自己的仕途

出獄後，他獲得巨額財富，化身為「基督山伯爵」，展開精密的復仇計劃。

### 復仇 vs. 饒恕

| 基督山伯爵（唐泰斯） | 保羅 |
|-------------------|------|
| 花一生復仇 | 選擇代求 |
| 「我是上帝的使者，執行公義」 | 「都歸在我的賬上」 |
| 親手毀滅仇人 | 為「仇人」代求 |
| 「以眼還眼」 | 「不要自己伸冤」(羅12:19) |

### 基督山伯爵的領悟

在小說結尾，當復仇完成後，基督山伯爵並沒有得到預期的滿足。他寫給馬克西米利安的信中說：

> « Toute la sagesse humaine sera résumée dans ces deux mots : **attendre et espérer**. »
>
> 中文翻譯：「全部人類的智慧都包含在這兩個詞中：**等待和希望**。」

這是復仇者最終的領悟：復仇並不能帶來真正的平安。

**對比**：保羅對腓利門說的是「**永遠得著他**」(門15)——這不是「等待和希望」有一天能報復，而是盼望永恆的和好。

## 第六節：饒恕的艱難與真諦

### 饒恕為何如此困難？

| 障礙 | 內心的聲音 | 聖經的回應 |
|------|-----------|-----------|
| 公義的要求 | 「有人虧負我，必須償還」 | 「伸冤在我，我必報應」(羅12:19) |
| 驕傲的攔阻 | 「我是受害者，憑什麼我要饒恕？」 | 「你也是蒙恩的罪人」(門19) |
| 恐懼的綑綁 | 「饒恕會不會縱容罪惡？」 | 「饒恕不是說沒事，是說我不再定罪」 |
| 記憶的傷痛 | 「我無法忘記他做過的事」 | 「饒恕不是忘記，是選擇不再用它定罪」 |
| 苦毒的纏繞 | 「我享受這種受害者的身份」 | 「苦毒的根會污染眾人」(來12:15) |

### 饒恕不是什麼 vs. 饒恕是什麼

| 饒恕不是 | 饒恕是 |
|---------|--------|
| 假裝沒事發生 | 承認傷害，但選擇釋放 |
| 忘記過去 | 記得，但不再用它來定罪 |
| 允許繼續傷害 | 設立健康的界線 |
| 立即信任 | 信任需要時間重建 |
| 感覺 | **意志的決定** |
| 廉價的恩典 | 真實面對傷害後的選擇 |
| 對方配得 | 因為我先被饒恕 |

### 饒恕 vs. 和解

| 饒恕 | 和解 |
|------|------|
| 單方面的決定 | 需要雙方參與 |
| 我選擇釋放對方 | 對方也需要回應 |
| 可以不需要對方道歉 | 需要某種程度的悔改 |
| 為了自己的自由 | 為了關係的修復 |
| 我可以獨自完成 | 需要中保的幫助 |

**這正是沙威的問題**：冉阿讓已經饒恕他了，但沙威無法接受這饒恕，所以和解無法發生。

---

## 第七節：保羅的智慧——創造饒恕的空間

保羅在腓利門書中展現了創造饒恕空間的極大智慧：

### 1. 不用權威命令
> 「靠著基督能放膽吩咐你合宜的事，然而還是**憑著愛心求**你。」(門8-9)

他有權力命令，但選擇放棄權力，用愛請求。

### 2. 承認對方的主權
> 「不知道你的意見，我就不願意這樣行，叫你的善行**不是出於勉強，乃是出於甘心**。」(門14)

尊重腓利門的選擇權——真正的饒恕必須是甘心的。

### 3. 願意代償
> 「他若虧負你，或欠你甚麼，**都歸在我的賬上**。」(門18)

移除不饒恕的藉口——如果是錢的問題，我來付。

### 4. 提醒對方也是蒙恩的罪人
> 「我並不用對你說，**連你自己也是虧欠於我**。」(門19)

這是最微妙的提醒：你也曾經欠債，你也曾經被饒恕。

### 5. 強調未來的祝福
> 「他暫時離開你，或者是叫你**永遠得著他**。」(門15)

轉移焦點從過去的傷害到未來的祝福。

這五步對應 MacArthur 的饒恕框架（詳見第五章）：品格（門1-7）奠定信任基礎，行動（門8-18）包含接納、恢復、代償，動機（門19-25）提醒對方也是蒙恩的人。保羅不是隨意堆砌詞句，而是按照精密的邏輯展開——**先鋪墊關係，再提出請求，最後激發動機。**這是愛的智慧，值得每一個需要促進和解的人學習。

---

## 第八節：和解的必要條件

保羅在腓利門書中示範了和解的五個要素：

### 1. 承認傷害（阿尼西母的「虧負」）
> 「他若**虧負**你，或**欠**你甚麼...」— 腓利門書 18

保羅沒有否認阿尼西母的過錯，而是承認有傷害發生。

### 2. 悔改的意願（回到腓利門面前）

阿尼西母願意回去面對他逃離的主人——這需要極大的勇氣和悔改。一個沒有悔改的罪犯，只是在逃避。

### 3. 中保的介入（保羅的代求）

保羅作為中保，承擔風險，為雙方搭橋。每一個和解都需要中保——最終的中保是基督。

### 4. 饒恕的決定（腓利門的選擇）

保羅不是命令，而是請求：

> 「叫你的善行**不是出於勉強，乃是出於甘心**。」— 腓利門書 14

### 5. 關係的重建（「永遠得著他」）

目標不只是不處罰，而是建立新的關係——從奴僕到兄弟，從虧欠者到家人。

---

## 結論：饒恕的終極源頭

饒恕有極限嗎？

從人的角度看，是的。沙威無法接受饒恕。瑪絲洛娃起初也無法接受。我們每一個人都有「饒恕不了」的時刻。

但從神的角度看，**饒恕是無限的**——因為饒恕的源頭是那位在十字架上說「\textcolor{RedemptionRed}{父啊，赦免他們}」的基督。

保羅對腓利門的提醒，也是對我們每一個人的提醒：

> 「連你自己也是虧欠於我。」(門19)

我們能夠饒恕人，不是因為對方配得，而是因為**我們先被饒恕**。

雨果說得好：

> « La plus haute fonction de l'homme est de pardonner. »
>
> 中文翻譯：「人最崇高的職能是**饒恕**。」

而這饒恕的能力，唯獨來自那位先饒恕我們的神。
