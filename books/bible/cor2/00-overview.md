## 為什麼讀哥林多後書？— 三重啟示

哥林多後書是保羅最私密的書信。不同於前書以教義和教會秩序為主，後書是**保羅的生命見證**：他如何在聖靈裡服事基督、在軟弱中經歷神的大能。

### 神的福音設計 (God's Design of the Gospel)

```
神的主權設計
   │
   ├── 1. 福音的內容：在基督裡與神和好 (5:17-21)
   │     「若有人在基督裡，他就是新造的人」
   │
   ├── 2. 福音的器皿：瓦器中的寶貝 (4:7)
   │     「要顯明這莫大的能力是出於神，不是出於我們」
   │
   └── 3. 福音的動力：聖靈的工作 (3:6, 3:17-18)
         「字句叫人死，精意（聖靈）叫人活」
         「主的靈在哪裡，那裡就得以自由」
```

### 聖靈的啟示與工作 (The Holy Spirit's Revelation)

哥林多後書是保羅書信中對**聖靈職事**最深刻的闡述之一：

| 章節 | 聖靈的工作 | 原文 |
|------|-----------|------|
| **3:3** | 用靈寫在心版上 | πνεύματι θεοῦ ζῶντος — Spirit of the living God |
| **3:6** | 賜生命的靈 | τὸ πνεῦμα ζῳοποιεῖ — the Spirit gives life |
| **3:8** | 聖靈的職事 | ἡ διακονία τοῦ πνεύματος — ministry of the Spirit |
| **3:17** | 主就是那靈 | ὁ κύριος τὸ πνεῦμά ἐστιν — the Lord is the Spirit |
| **3:17** | 靈帶來自由 | ἐλευθερία — liberty/freedom |
| **3:18** | 靈使人變像 | ἀπὸ κυρίου πνεύματος — from the Lord, the Spirit |
| **5:5** | 聖靈為憑據 | τὸ ἀρραβῶνα τοῦ πνεύματος — guarantee of the Spirit |
| **13:14** | 聖靈的感動 | κοινωνία τοῦ ἁγίου πνεύματος — fellowship of the Spirit |

> **Campbell Morgan**: "The letter reveals, the Spirit realizes." — 字句顯示應然，聖靈成就實然。律法揭露人的不能，聖靈糾正人的不能並賜下生命。

### 保羅的生命見證 (Paul's Personal Testimony)

哥林多後書是保羅**最深刻的自傳**。他不隱藏自己的：

| 保羅的經歷 | 經文 | 神的回應 |
|-----------|------|---------|
| 在亞細亞的極大患難 | 1:8-9 | 叫死人復活的神安慰我們 |
| 流淚寫信的痛苦 | 2:4 | 提多帶回好消息 (7:6) |
| 身上的刺 | 12:7 | 恩典夠你用的 |
| 被打、被石頭打、船壞 | 11:24-27 | 軟弱中的能力 |
| 被假使徒攻擊 | 11:13 | 真使徒的記號 (12:12) |

**MacArthur 的洞見**：一個真正的事奉者有五個記號 —
1. 無需薦信的無瑕品格
2. 改變生命的事奉果效
3. 對恩賜和呼召的堅定信心
4. 對神能力的謙卑倚靠
5. 傳講新約福音的信息

## 使徒行傳背景：保羅建立哥林多教會 (Acts 18:1-18)

> **使徒行傳 18:9-10**
> 「夜間，主在異象中對保羅說：**『不要怕，只管講，不要閉口，有我與你同在，必沒有人下手害你，因為在這城裡我有許多的百姓。』**」
> — 紅字經文：主耶穌的直接話語

### 哥林多教會的誕生 (約主後 50-51)

| 事件 | 經文 | 意義 |
|------|------|------|
| 保羅從雅典來到哥林多 | 徒 18:1 | 從哲學之城到商業之城 |
| 與亞居拉、百基拉同住做工 | 徒 18:2-3 | 織帳棚為業，自養傳道 |
| 每逢安息日在會堂辯論 | 徒 18:4 | 向猶太人和希臘人作見證 |
| 西拉、提摩太從馬其頓來 | 徒 18:5 | 保羅為道迫切，證明耶穌是基督 |
| 猶太人抵擋，保羅轉向外邦人 | 徒 18:6 | 福音戰略的轉折 |
| 管會堂的基利司布全家信主 | 徒 18:8 | 連帶許多哥林多人信而受洗 |
| **主在異象中親自說話** | 徒 18:9-10 | 「不要怕⋯在這城裡我有許多的百姓」 |
| 保羅住了一年零六個月 | 徒 18:11 | 扎實建立教會根基 |

這就是保羅寫信的對象 — 他親手建立、用生命澆灌的教會。所以後書中流露的眼淚和心腸，不是修辭，是真實的牧者之愛。

## 寫作背景：從前書到後書

### 保羅與哥林多教會的關係時間線

```
AD 50-51    保羅建立教會（徒18:1-18, 住了一年半）
     │
AD 54-55    哥林多前書（從以弗所寫，處理教會問題）
     │
AD 55       痛苦的造訪（2:1, 短暫到訪，遭遇衝突）
     │
AD 55       流淚的信（2:4, 已佚失）
     │
AD 55-56    提多帶回好消息（7:6-7, 教會悔改！）
     │
AD 55-56    哥林多後書（從馬其頓寫，保羅的心腸流露）
```

### 前書 vs 後書

| 面向 | 哥林多前書 | 哥林多後書 |
|------|-----------|-----------|
| **語氣** | 使徒的教導權柄 | 牧者的心腸流露 |
| **焦點** | 教會的問題 | **保羅的生命見證** |
| **處理** | 分爭、淫亂、恩賜、復活 | 患難、安慰、聖靈、使徒權柄 |
| **核心** | 十字架的智慧 (1:18) | **十字架的能力** (12:9) |
| **名句** | 「愛是恆久忍耐」(13:4) | 「恩典夠你用的」(12:9) |
| **風格** | 系統性教導 (Doctrine) | **個人性見證 (Testimony)** |
| **聖靈** | 恩賜的分配 (12章) | **生命的更新** (3章) |

## 聖靈在哥林多後書的七重工作

> **MacArthur**: 聖靈在舊約和新約中有四個主要功能：(1)創造、(2)賦能、(3)啟示、(4)重生。在新約中，聖靈不再只是外在的能力，而是內住的生命。

```
1. 寫在心版上 (3:3)
   └→ 不是用墨，乃是用永生神的靈
       舊約：石版（外在律法）→ 新約：心版（內在生命）
       背景：耶利米 31:33, 以西結 36:26-27

2. 賜下生命 (3:6)
   └→ 字句叫人死，精意（聖靈）叫人活
       Morgan: "The letter reveals, the Spirit realizes."

3. 帶來自由 (3:17)
   └→ 主的靈在哪裡，那裡就得以自由
       MacArthur: 從律法的束縛、自力稱義的重擔中釋放

4. 使人變像 (3:18)
   └→ 從榮耀到榮耀，變成主的形狀
       聖靈持續的更新工作，不是一次性的

5. 作為憑據 (5:5)
   └→ 聖靈是我們得天上居所的保證
       ἀρραβών (arrabōn) = 訂金、首付、保證金

6. 賦能爭戰 (10:3-5)
   └→ 屬靈兵器拆毀堅固營壘
       不是屬血氣的爭戰，乃是在聖靈裡的爭戰

7. 三一團契 (13:14)
   └→ 聖靈的感動（κοινωνία）與眾人同在
       全書以三位一體的祝福作結
```

## 新約 vs 舊約：MacArthur 的八重對比

| 面向 | 舊約（字句） | 新約（聖靈） |
|------|------------|------------|
| 1. 結果 | 叫人死 | **賜人生命** |
| 2. 定性 | 定罪的職事 | **稱義的職事** |
| 3. 持續 | 漸漸退去的榮耀 | **長存的榮耀** |
| 4. 盼望 | 無望（帕子蒙蔽） | **充滿盼望** |
| 5. 清晰 | 預表和影子 | **揭開帕子，直見基督** |
| 6. 中心 | 象徵性的期待 | **以基督為中心** |
| 7. 自由 | 外在律法的束縛 | **聖靈裡的釋放** |
| 8. 更新 | 只揭露罪 | **改變生命** |

> **Morgan 的精闢總結**: 律法「有榮耀，且是崇高的，但對犯罪的人沒有盼望；所以它退去，因為它留人在罪中。」恩典「有更崇高的榮耀，因為對犯罪的人有盼望；所以它永不退去，因為它使人成為義。」
