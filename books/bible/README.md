## 阅读计划

**旧约**（39卷）：
- 摩西五经：创世记 → 出埃及记 → 利未记 → 民数记 → 申命记
- 历史书：约书亚记 → ... → 以斯帖记
- 诗歌智慧书：约伯记、诗篇、箴言、传道书、雅歌
- 先知书：以赛亚 → ... → 玛拉基

**新约**（27卷）：
- 福音书：马太 → 马可 → 路加 → 约翰
- 使徒行传
- 保罗书信
- 普通书信
- 启示录
