## 一冊書，一條弧線，一個整體

四福音書從四個角度看同一位基督。
本卷把它們**按時序編織成一個整體**——從**永恆的太初**，經**伯利恆的馬槽**、**加利利的會堂**、**髑髏地的十字架**，到**空墳墓**、**橄欖山的升天**，**指向主再來與永恆**。

中心是**十字架**。
中心的中心是**架上七言**。

## 中心結構：弧線、軸心、回響

### 弧線（從永恆到永恆）
```
太初有道 → 道成肉身 → 隱藏歲月 → 公開事奉 → 受難週 → ✟十字架✟ → 復活 → 升天 → 教會時代 → 主再來
```

### 軸心（基督生平的中心）
**第十一章 — 十字架與架上七言**
週五上午 9 時到下午 3 時的 6 小時，是人類歷史的**中心**。
這 6 小時內所說的 7 句話，是**全部福音的 7 個音符**。

## 信、望、愛 — 整部基督生平的三和弦

- **信** — 神的信實：4000 年的應許都應驗了
- **望** — 復活的盼望：死亡不是終點
- **愛** — 神愛世人：愛到底，愛永遠

> 「**如今常存的有信、有望、有愛這三樣，其中最大的是愛**」（林前 13:13）

## 給讀者的話

這部編年志是**為了「整體看見」而寫**。

你可以在書架上找到許多單獨討論基督生平片段的書——這本書的不同之處是：**它把四福音的場景按時序縫合成一個整體，並讓十字架與七言成為中心的中心**。

讀完之後，你應該能：
1. **講出基督生平的完整弧線**（從太初到永恆）
2. **背出十架七言的順序與意義**
3. **看見每一個事件如何指向、預備、或回應十字架**
4. **把基督的故事讀成自己的故事**
