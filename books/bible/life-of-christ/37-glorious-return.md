## 37.1 經文 (Scripture)

### 寶座與羔羊（啟示錄 5:6-9）

**中文 — RCUV**

6 我又看見寶座與四活物，並長老之中有羔羊站著，像是被殺過的，有七個角七個眼睛——就是神的七靈，奉差遣往普天下去的。

9 他們唱新歌，說：「你配拿書卷，配揭開它的七印，因為你曾被殺，用自己的血從各支派、各方言、各民族、各邦國中買了人來，使他們歸於神，10 又使他們成為國民和祭司，歸於我們的神；他們將要在地上掌權。」

### 萬王之王（啟示錄 19:11-16）

11 後來我看見天開了。有一匹白馬，騎在馬上的稱為「忠信」、「真實」，他審判和爭戰都憑著公義。12 他的眼睛如火焰，頭上戴著許多冠冕……13 他穿著浸過血的衣服；他的名稱為「神之道」。

14 在天上的眾軍，騎著白馬，穿著潔白的細麻衣跟隨他。

16 在他衣服和大腿上有名字寫著：「萬王之王，萬主之主」。

### 新天新地（啟示錄 21:3-5）

3 我聽見有大聲音從寶座出來，說：「看哪，神的帳幕在人間！他要與人同住，他們要作他的子民。神要親自與他們同在，作他們的神。4 神要擦去他們一切的眼淚；不再有死亡，也不再有悲哀、哭號、疼痛，因為先前的事都過去了。」

5 那位坐在寶座上的說：*「看哪，我把一切都更新了！」*

### 末了的應許（啟示錄 22:20）

20 證明這些事的說：*「是的，我必快來！」*阿們！主耶穌啊，我願你來！

**English — NKJV**

20 *"Surely I am coming quickly."* Amen. Even so, come, Lord Jesus!

### 「**羔羊**」(*arnion*)

啟示錄用 *arnion*（**小**羊）—— 親密、柔弱的稱呼。
**寶座上的不是猛獅，是小羊**——但**這小羊**「**像是被殺過的**」**有七角七眼**——**完全的權柄、完全的洞察**。

**小羊與獅子是同一位**：**獅子**得勝因為他**先作小羊被殺**（啟 5:5-6）。

## 37.4 解經洞見 (Commentary)

### John MacArthur 論再來的確定性

「『**我必快來**』—— 這應許在啟示錄結尾**重複三次**（啟 22:7, 12, 20）。**「快**」(*tachy*)**不只是時間上的近**，**也是「迅速地、突然地**」**——當主再來，是無預警的**。**自啟示錄寫成已 1900+ 年**，許多人質疑『**他何時來**』。彼得回答：『**主看一日如千年，千年如一日**……主所應許的尚未成就，有人以為他是耽延，其實不是耽延，乃是寬容你們**』（彼後 3:8-9）。**「耽延」是恩典**——**讓更多人有機會悔改**。」 *(adapted from MacArthur, Revelation 12-22)*

### G. Campbell Morgan 論「**主耶穌啊，我願你來**」

「**整本聖經以這個禱告結束**——『**主耶穌啊，我願你來**』。**這應該是每一個基督徒最深的禱告**。但很多基督徒**不太想他來**——因為**還想多享受幾年世界的好處**。**「主，我願你來」**的真誠程度，**測量**你**對基督的愛**和**對世界的態度**。**真正愛主的人**，**自然渴望見主**。**真正看世界為過客之地**的人，**自然渴望回家**。」 *(adapted from Morgan, Revelation)*

## 37.5 神學要點 (Theological Points)

### 1. 基督生平的最後弧線

```
永恆 → 道成肉身 → 受難 → 復活 → 升天 → 教會 → 再來 → 永恆
```

**第一次降臨**完成救贖；**第二次降臨**完成國度。

### 2. 新天新地的核心應許

| 「不再有」 | 「將有」 |
|---|---|
| 死亡 | 永生 |
| 悲哀 | 喜樂 |
| 哭號 | 讚美 |
| 疼痛 | 安息 |
| 海（混亂的記號）| 平靜 |
| 罪 | 聖潔 |
| 黑暗 | 神就是光 |
| 殿（不需要）| 神和羔羊就是殿 |

### 3. 神的帳幕在人間

> 「**看哪，神的帳幕在人間**！他要**與人同住**」（啟 21:3）

**整本聖經的最大主題**：**神與人同住**。
- **創 1-2**：神與人在伊甸園
- **出 25**：神在會幕中
- **約 1:14**：神道成肉身
- **林前 3:16**：神住在信徒裡
- **啟 21:3**：**神永遠與人同住** —— **以馬內利的最終實現**

### 4. 雙重的「已然/未然」

| 已然 | 未然 |
|---|---|
| 基督已來（道成肉身）| 基督**將**來（榮耀再臨）|
| 國度已開始 | 國度**將**完成 |
| 撒但已被擊敗 | 撒但**將**被丟在火湖 |
| 信徒已得永生 | 信徒**將**得復活之體 |
| 教會已存在 | 教會**將**作羔羊之新婦 |

**基督徒活在「已然/未然」的張力之間** —— 已被救贖，等候完全的救贖。
