# 哥林多前書主日學（2025-2026）

## 第三课：属灵与属肉体

**（哥林多前书 3:1-23）**

---

## 一、引言

哥林多教会问题的表徵是信徒结党纷争、高举人、彼此嫉妒，但实际上是他们灵命不成熟，仍然是「属肉体」的婴孩。保罗在前两章论述了神的智慧与世人的智慧之别，指出十字架的道理是神的大能，并解释了属灵人与属血气人的分别——属灵人有基督的心，能参透万事（林前2:15-16）。

进入第三章，保罗直言不讳地指出哥林多信徒虽已信主，却仍照着世人的样子行。他们当中的嫉妒、纷争、结党，正是属肉体的明证。保罗以此为起点，教导他们正确认识：

* 神的工人只是仆人，真正使人成长的是神
* 教会是神的田地、神的房屋、神的殿
* 信徒当在基督的根基上用正确的材料建造
* 不可自欺，因万有全是你们的，你们是属基督的，基督是属神的

 **警惕信徒「不要自欺！」** （林前3:18）——不可以为自己有智慧，乃要变作愚拙，好成为有智慧的。

---

## 二、属灵与属肉体的分别（林前3:1-4）

> **（林前3:1-4）**
> 1 弟兄们，我从前对你们说话，不能把你们当作属灵的，只得把你们当作属肉体，在基督里为婴孩的。2 我是用奶喂你们，没有用饭喂你们。那时你们不能吃，就是如今还是不能。3 你们仍是属肉体的，因为在你们中间有嫉妒、纷争，这岂不是属乎肉体、照着世人的样子行吗？4 有说「我是属保罗的」；有说「我是属亚波罗的」。这岂不是你们和世人一样吗？

### 1. 属灵与属肉体的对比

| 属肉体的       | 属灵的                   | 参考经文                   |
| -------------- | ------------------------ | -------------------------- |
| 在基督里为婴孩 | 在基督里成熟长大         | 当长大成人（来5:14）       |
| 只能吃奶       | 能吃干粮                 | 干粮是为成熟的人（来5:14） |
| 有嫉妒、纷争   | 有圣灵的果子：仁爱、和平 | 加5:22-23                  |
| 照世人的样子行 | 照圣灵引导而行           | 加5:16                     |
| 高举人、结党   | 只夸主耶稣基督的十字架   | 加6:14                     |
| 体贴肉体       | 体贴圣灵                 | 体贴肉体的就是死（罗8:6）  |

### 2. 属肉体的表现

* **嫉妒** ：不能容忍别人比自己好
* **纷争** ：因意见不合而争吵分裂
* **结党** ：「我是属保罗的」「我是属亚波罗的」

### 3. 属肉体的实质

* 虽已信主，却没有在灵命上成长
* 仍以世界的标准来看事情
* 高举人过于高举神
* 没有让圣灵掌管生命

> **（罗8:5-8）**
> 因为随从肉体的人体贴肉体的事；随从圣灵的人体贴圣灵的事。体贴肉体的就是死；体贴圣灵的乃是生命、平安。原来体贴肉体的，就是与神为仇；因为不服神的律法，也是不能服。而且属肉体的人不能得神的喜欢。

## 三、神的工人与神的工作（林前3:5-9）

> **（林前3:5-9）**
> 5 亚波罗算什么？保罗算什么？无非是执事，照主所赐给他们各人的，引导你们相信。6 我栽种了，亚波罗浇灌了，惟有神叫他生长。7 可见栽种的算不得什么，浇灌的也算不得什么；只在那叫他生长的神。8 栽种的和浇灌的，都是一样，但将来各人要照自己的工夫得自己的赏赐。9 因为我们是与神同工的；你们是神所耕种的田地，所建造的房屋。

### 1. 神的工人的本质

* **是执事/仆人** （διάκονοι）：不是主人，只是服事的人
* **照主所赐给各人的** ：各有不同的恩赐和职分
* **引导人相信** ：工人的角色是引导，真正使人信的是神

### 2. 神的工作的比喻

| 比喻 | 保罗的角色 | 亚波罗的角色 | 神的角色   |
| ---- | ---------- | ------------ | ---------- |
| 田地 | 栽种       | 浇灌         | 叫他生长   |
| 房屋 | 立根基     | 继续建造     | 房屋的主人 |

### 3. 重要真理

* **栽种的算不得什么，浇灌的也算不得什么** ——工人不应被高举
* **只在那叫他生长的神** ——荣耀当归给神
* **各人要照自己的工夫得赏赐** ——神看重忠心的服事
* **我们是与神同工的** ——何等荣耀的呼召！

## 四、建造的根基与材料（林前3:10-15）

> **（林前3:10-15）**
> 10 我照神所给我的恩，好像一个聪明的工头，立好了根基，有别人在上面建造；只是各人要谨慎怎样在上面建造。11 因为那已经立好的根基就是耶稣基督，此外没有人能立别的根基。12 若有人用金、银、宝石、草木、禾秸在这根基上建造，13 各人的工程必然显露，因为那日子要将它表明出来，有火发现；这火要试验各人的工程怎样。14 人在那根基上所建造的工程若存得住，他就要得赏赐。15 人的工程若被烧了，他就要受亏损，自己却要得救；虽然得救，乃像从火里经过的一样。

### 1. 唯一的根基：耶稣基督

* 根基已经立好——不是人能立的，是神所立的
* 此外没有别的根基——只有耶稣基督
* 任何离开基督的教导都是错误的根基

### 2. 建造的材料

| 建造前的材料 | 建造后的结果   | 属灵意义                   |
| ------------ | -------------- | -------------------------- |
| 金           | 经火试验存得住 | 出于神、有永恒价值的工作   |
| 银           | 经火试验存得住 | 符合真理、经得起考验的服事 |
| 宝石         | 经火试验存得住 | 圣灵引导的事工             |
| 草           | 被火烧毁       | 出于肉体、没有永恒价值     |
| 木           | 被火烧毁       | 人意的工作、属世的方法     |
| 禾秸         | 被火烧毁       | 虚浮的工程、没有生命的事奉 |

### 3. 火的试验

* **那日子** ：指基督审判台前的日子（林后5:10）
* **有火发现** ：火代表神的审判，显明各人工程的真实性
* **工程若存得住** ：就要得赏赐
* **工程若被烧了** ：受亏损，但自己仍然得救（因在基督根基上）

### 4. 实际应用

**各人要谨慎怎样在上面建造：**

* 我们事奉的动机是什么？出于爱神还是爱自己？
* 我们用什么方法事奉？属灵的方式还是属世的手段？
* 我们追求什么果效？神的荣耀还是自己的名声？

## 五、神的殿与神的同在（林前3:16-17）

> **（林前3:16-17）**
> 16 岂不知你们是神的殿，神的灵住在你们里头吗？17 若有人毁坏神的殿，神必要毁坏那人；因为神的殿是圣的，这殿就是你们。

### 1. 教会是神的殿

* **「你们」是复数** ：指整个教会群体，不只是个人
* **神的灵住在里头** ：圣灵内住，教会是神居住的所在
* **神的殿是圣的** ：分别为圣，属于神

### 2. 毁坏神殿的严重性

* **毁坏** （φθείρω）：破坏、败坏、腐蚀
* 包括：制造分裂、传播异端、引入罪恶、破坏合一
* **神必要毁坏那人** ：严肃的警告，神必追讨

### 3. 与15章荣耀身体的联系

| 今生：神的殿（教会） | 将来：荣耀的身体         |
| -------------------- | ------------------------ |
| 神的灵住在里头       | 复活成为灵性的身体       |
| 是圣的               | 是荣耀的、不朽坏的       |
| 当保守合一、追求圣洁 | 将来与基督荣耀的身体相似 |

## 六、不可自欺——真智慧与假智慧（林前3:18-23）

> **（林前3:18-23）**
> 18 人不可自欺。你们中间若有人在这世界自以为有智慧，倒不如变作愚拙，好成为有智慧的。19 因这世界的智慧，在神看是愚拙。如经上记着说：「主叫有智慧的，中了自己的诡计」；20 又说：「主知道智慧人的意念是虚妄的。」21 所以无论谁，都不可拿人夸口，因为万有全是你们的。22 或保罗，或亚波罗，或矶法，或世界，或生，或死，或现今的事，或将来的事，全是你们的；23 并且你们是属基督的，基督又是属神的。

### 1. 不可自欺！

* **自以为有智慧** ：用世界的标准来判断
* **变作愚拙** ：承认自己的有限，谦卑在神面前
* **好成为有智慧的** ：得着从神而来的真智慧

### 2. 世界的智慧 vs 神的智慧

| 世界的智慧     | 神的智慧       |
| -------------- | -------------- |
| 在神看是愚拙   | 在世人看是愚拙 |
| 中了自己的诡计 | 十字架的道理   |
| 意念是虚妄的   | 使人得救的大能 |
| 导致高举人     | 导致高举基督   |

### 3. 万有全是你们的

**不要高举人，因为：**

* 保罗、亚波罗、矶法——他们是为你们服事的
* 世界、生、死——都在神的掌管之下
* 现今的事、将来的事——都为着信徒的益处

**荣耀的次序：**

| 归属 | 属于 |
|------|------|
| 万有 | 属于信徒 |
| 信徒 | 属于基督 |
| 基督 | 属于神 |

这是何等大的恩典！我们在基督里承受万有，却不是为了高举自己，而是因为我们是属基督的，而基督是属神的。

## 七、结语

属灵生命的成长是每个信徒的责任。保罗严厉指出哥林多信徒的问题——他们虽然信主，却仍是属肉体的婴孩，因嫉妒纷争、结党高举人。这一章教导我们：

1. **要从婴孩长大成人** ：离开属肉体的样式，追求属灵的成熟
2. **要正确认识神的工人** ：他们只是仆人，真正使人成长的是神
3. **要谨慎建造生命** ：在基督的根基上，用金银宝石建造
4. **要保守教会的圣洁** ：因为教会是神的殿，神的灵住在其中
5. **不要自欺** ：放下世界的智慧，得着神的真智慧
6. **要认识自己的身份** ：万有全是你们的，你们是属基督的

## ***反思与讨论***

1. **你认为自己目前的灵命是「婴孩」还是「成熟」的？**有什么具体的表现？
2. **在你的生活中，是否有「嫉妒、纷争、结党」的情况？**这如何影响你与神、与人的关系？
3. **你在教会中的服事，是用什么「材料」建造的？**是金银宝石，还是草木禾秸？
4. **「万有全是你们的」这句话对你有什么意义？**既然如此，为何还要高举人或属世的事物？
5. **读完哥林多前书第三章，你对「属灵」与「属肉体」有什么新的认识？**这如何改变你今天的生活？
