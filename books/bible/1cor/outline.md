# 導論：哥林多前書的背景與主題

## 一、哥林多城的背景

哥林多是古希臘最繁華的商業城市之一，位於連接東西方的戰略要地。這座城市以三件事聞名：

1. **財富與商業** ：作為重要港口城市，商賈雲集
2. **希臘哲學** ：崇尚「言語的智慧」（wisdom of words）
3. **道德敗壞** ：「哥林多化」（to corinthianize）成為放縱淫亂的代名詞

> **MacArthur 指出** ：哥林多的道德腐敗如此嚴重，以至於城市的名字本身就成了道德墮落的同義詞。亞弗洛狄忒（愛神）神廟中有上千名廟妓，公開招攬生意，市場上充斥著祭偶像之物。

> **Morgan 觀察** ：哥林多教會的問題正是當時文化環境的反映——他們被世界的智慧所迷惑，被肉體的情慾所轄制，卻自以為屬靈。

## 二、哥林多教會的問題

保羅在哥林多前書中處理的問題可分為以下幾類：

| 章節    | 問題       | 核心原因               |
| ------- | ---------- | ---------------------- |
| 1-4章   | 結黨紛爭   | 高舉人、追求世界的智慧 |
| 5章     | 容忍淫亂   | 對罪缺乏敏感           |
| 6章     | 信徒互告   | 屬世的處事方式         |
| 7章     | 婚姻問題   | 對聖潔生活的誤解       |
| 8-10章  | 祭偶像之物 | 愛心與自由的張力       |
| 11章    | 崇拜的混亂 | 不尊重神的秩序         |
| 12-14章 | 恩賜的濫用 | 驕傲、缺乏愛心         |
| 15章    | 否認復活   | 信仰根基的動搖         |

## 三、哥林多前書的中心主題

 **MacArthur 總結** ：哥林多前書處理的是「教會的敬虔問題解決方案」——世俗化、屬靈的不合一、性犯罪、婚姻離婚的問題、女性角色、屬靈恩賜。

 **Morgan 強調** ：整卷書信的根基命題是「教會被召進入與耶穌基督的團契」。所有的問題都源於他們未能活在這團契的真實中。

# 第一部分：神的智慧與世界的智慧（1-4章）

## 第一課：十字架的道理——神的智慧（1:18-2:16）

### 經文要義

> **（林前1:18-21）**
> 因為十字架的道理，在那滅亡的人為愚拙，在我們得救的人卻為神的大能。就如經上所記：「我要滅絕智慧人的智慧，廢棄聰明人的聰明。」智慧人在哪裡？文士在哪裡？這世上的辯士在哪裡？神豈不是叫這世上的智慧變成愚拙嗎？

### MacArthur 的解釋

保羅對比了兩種智慧：

| 世界的智慧       | 神的智慧           |
| ---------------- | ------------------ |
| 以人的理性為根基 | 以十字架為根基     |
| 追求言語的巧妙   | 宣講基督被釘十字架 |
| 靠人的能力       | 靠聖靈的大能       |
| 導致驕傲         | 導致謙卑           |
| 結果是分裂       | 結果是合一         |

### Morgan 的洞見

> 「這些由『言語的智慧』爭論所造成的派系，是因為愚昧地未能領會那偉大『十字架之道』的奇妙智慧——這道正是他們信仰的根基，使他們與耶穌基督聯合，因此也彼此聯合。」

### 🌐 當代應用：矽谷與AI時代的智慧觀

**哥林多的「智慧崇拜」與今天矽谷的文化驚人地相似：**

| 古代哥林多       | 當代矽谷                      |
| ---------------- | ----------------------------- |
| 希臘哲學的智慧   | 科技創新的智慧                |
| 雄辯家受追捧     | 科技領袖被神化                |
| 「我是屬保羅的」 | 「我是屬 Elon/Sam/Jensen 的」 |
| 用言語的智慧傳道 | 用市場策略傳福音              |
| 追求人的認可     | 追求 likes、followers、估值   |

**挑戰與試探：**

1. **將技術等同於救贖** ：AI 被視為解決人類問題的終極方案
2. **數據驅動的信仰決策** ：用 metrics 來衡量事工「成功」
3. **效率崇拜** ：用企業管理方式經營教會
4. **知識精英主義** ：看不起「不懂技術」的弟兄姊妹

**在基督裡得自由的路徑：**

* 十字架的「愚拙」仍是神的大能——不是靠更好的演算法，而是靠聖靈
* 承認人類智慧的有限——AI 也無法解決罪的問題
* 在專業卓越中保持謙卑——我們的身份不在於職稱或公司
* 用神的智慧來評估一切——不是看是否「創新」，而是看是否合神心意

## 第二課：屬靈與屬肉體（3:1-23）

### 經文要義

> **（林前3:1-4）**
> 弟兄們，我從前對你們說話，不能把你們當作屬靈的，只得把你們當作屬肉體，在基督裡為嬰孩的。我是用奶餵你們，沒有用飯餵你們。那時你們不能吃，就是如今還是不能。你們仍是屬肉體的，因為在你們中間有嫉妒、紛爭，這豈不是屬乎肉體、照著世人的樣子行嗎？

### 屬靈成熟的標記

| 屬肉體的表現   | 屬靈的表現   |
| -------------- | ------------ |
| 在基督裡為嬰孩 | 在基督裡成熟 |
| 只能吃奶       | 能吃乾糧     |
| 嫉妒、紛爭     | 聖靈的果子   |
| 高舉人         | 高舉基督     |
| 結黨分門       | 在基督裡合一 |

### 建造的根基與材料（3:10-15）

> **（林前3:11-13）**
> 因為那已經立好的根基就是耶穌基督，此外沒有人能立別的根基。若有人用金、銀、寶石、草木、禾稭在這根基上建造，各人的工程必然顯露，因為那日子要將它表明出來，有火發現；這火要試驗各人的工程怎樣。

### MacArthur 的教導

根基是基督——這是不可改變的。問題在於我們用什麼材料建造：

* **金、銀、寶石** ：出於神、有永恆價值的工作
* **草、木、禾稭** ：出於肉體、沒有永恆價值的工作

### 🌐 當代應用：在矽谷建造生命

**試問自己：**

1. 我事奉的動機是什麼？
   * 是為了神的榮耀，還是為了建立個人品牌？
   * 是要討神喜悅，還是要討老闆/投資人喜悅？
2. 我用什麼方法事奉？
   * 是靠聖靈的大能，還是靠人脈關係？
   * 是按神的時間，還是追求「快速增長」？
3. 我追求什麼果效？
   * 是生命的改變，還是數字的增長？
   * 是門徒的建立，還是觀眾的累積？

**在AI時代持守真道的原則：**

| 試探             | 持守真道           |
| ---------------- | ------------------ |
| 用AI生成講道     | 在聖靈裡深入研經   |
| 追求viral內容    | 忠實傳講全備福音   |
| 用演算法定義成功 | 以神的標準衡量忠心 |
| 效率至上         | 以愛為首要原則     |

# 第二部分：聖潔的生活（5-7章）

## 第三課：分別為聖——處理罪的問題（5-6章）

### 經文要義

> **（林前5:6-8）**
> 你們這自誇是不好的。豈不知一點麵酵能使全團發起來嗎？你們既是無酵的麵，應當把舊酵除淨，好使你們成為新團；因為我們逾越節的羔羊基督已經被殺獻祭了。所以，我們守這節不可用舊酵，也不可用惡毒、邪惡的酵，只用誠實真正的無酵餅。

### MacArthur 的解釋

哥林多教會對罪的態度是致命的：

* 容忍明顯的罪（亂倫）
* 以「恩典」為藉口
* 不執行教會紀律

這反映了一個核心問題： **他們沒有真正理解福音的聖潔要求** 。

### 自由與聖潔的平衡（6:12-20）

> **（林前6:12, 19-20）**
> 凡事我都可行，但不都有益處；凡事我都可行，但無論哪一件，我總不受它的轄制……豈不知你們的身子就是聖靈的殿嗎？這聖靈是從神而來，住在你們裡頭的；並且你們不是自己的人，因為你們是重價買來的，所以要在你們的身子上榮耀神。

### 🌐 當代應用：在放縱文化中活出聖潔

**矽谷文化的試探：**

1. **工作倫理的灰色地帶**
   * 公司利益 vs 誠信
   * 「Everyone does it」的合理化
2. **性道德的模糊**
   * hookup culture
   * 色情的普及化
   * 同居被視為常態
3. **物質主義的轄制**
   * 股票期權成為偶像
   * 生活方式的攀比
   * 「財務自由」的追求

**得自由的真理：**

* **你是重價買來的** ——你的價值不在於薪資或股票
* **你的身體是聖靈的殿** ——如何使用身體是敬拜的問題
* **真自由不是放縱** ——而是不受罪的轄制
* **聖潔帶來真正的喜樂** ——不是限制，而是保護

## 第四課：婚姻與獨身（7章）

### 經文要義

保羅在此章回答哥林多信徒關於婚姻的問題，平衡地教導：

1. 婚姻是美好的——不是「次等」的靈性選擇
2. 獨身是恩賜——為了更專心事奉主
3. 婚姻中的忠貞——不可離棄
4. 信與不信者婚姻的處理

### MacArthur 的原則

> 「婚姻是神設立的，是神聖的。但在末世的緊迫中，保羅提醒我們，地上的關係不是終極的——我們的眼目當注視永恆。」

### 🌐 當代應用：在矽谷建立敬虔的家庭

**挑戰：**

* 高壓工作與家庭的張力
* 雙薪家庭的時間管理
* 子女教育的屬世標準
* 「事業優先」的文化

**持守真道的原則：**

1. **婚姻是盟約，不是合同** ——不以「不合適」為由輕易離婚
2. **配偶比career重要** ——願意為家庭放下升遷機會
3. **在婚姻中服事基督** ——家是最重要的事工禾場
4. **教養敬虔後代** ——不只追求Stanford，更追求屬靈成熟

# 第三部分：愛與自由的平衡（8-11章）

## 第五課：知識叫人自高，惟有愛心造就人（8-10章）

### 經文要義

> **（林前8:1-3）**
> 論到祭偶像之物，我們曉得我們都有知識。但知識是叫人自高自大，惟有愛心能造就人。若有人以為自己知道什麼，按他所當知道的，他仍是不知道。若有人愛神，這人乃是神所知道的。

### 自由的原則與限制

| 我有自由…   | 但愛心要求我…   |
| ------------ | ---------------- |
| 吃祭偶像之物 | 不絆倒軟弱的弟兄 |
| 主張我的權利 | 為福音放下權利   |
| 凡事都可行   | 凡事都為榮耀神   |

### Morgan 的洞見

> 「基督徒的自由不是為了自己的享受，而是為了服事他人。保羅說『凡事都可行』，但立刻加上『不都有益處』和『不都造就人』。愛是自由的引導原則。」

### 保羅的榜樣（9章）

保羅有權利：

* 靠福音養生
* 娶信主的姊妹為妻
* 享用教會的供應

 **但他放下這些權利** ，為要多得人。

> **（林前9:19, 22-23）**
> 我雖是自由的，無人轄管，然而我甘心作了眾人的僕人，為要多得人……向軟弱的人，我就作軟弱的人，為要得軟弱的人。向什麼樣的人，我就作什麼樣的人。無論如何總要救些人。凡我所行的，都是為福音的緣故，為要與人同得這福音的好處。

### 🌐 當代應用：在多元文化中活出愛

**矽谷的「祭偶像之物」：**

1. **財務上的決定**
   * 可以投資某些公司嗎？（如賭博、大麻）
   * 可以為某些項目工作嗎？（如違背信仰的內容）
2. **社交上的參與**
   * 可以參加有酒的公司活動嗎？
   * 可以去某些娛樂場所嗎？
3. **文化上的適應**
   * 如何與LGBT同事相處？
   * 如何參與公司的「多元文化」活動？

**愛心的原則：**

* **對軟弱的肢體** ：不絆倒他們（即使我有自由）
* **對未信者** ：作眾人的僕人，為要得人
* **對神** ：凡事為榮耀神而行

**實際應用：**

| 情境                   | 可能的回應                     |
| ---------------------- | ------------------------------ |
| 同事邀請去酒吧         | 可以去，點非酒精飲料，建立關係 |
| 被要求做違背良心的項目 | 有禮貌但堅定地拒絕，說明原因   |
| 公司Pride活動          | 不參與慶祝，但繼續愛和尊重同事 |
| 與不信者團隊合作       | 以卓越工作見證基督，不妥協真理 |

# 第四部分：敬拜的秩序（11-14章）

## 第六課：主的晚餐——記念與合一（11:17-34）

### 經文要義

> **（林前11:23-26）**
> 我當日傳給你們的，原是從主領受的，就是主耶穌被賣的那一夜，拿起餅來，祝謝了，就擘開，說：「這是我的身體，為你們捨的，你們應當如此行，為的是記念我。」飯後，也照樣拿起杯來，說：「這杯是用我的血所立的新約，你們每逢喝的時候，要如此行，為的是記念我。」你們每逢吃這餅，喝這杯，是表明主的死，直等到他來。

### 哥林多的問題

* 富人先吃，窮人挨餓
* 有人醉酒
* 分門別類，破壞合一

### MacArthur 的教導

主的晚餐要求我們：

1. **省察自己** ——不是審判別人
2. **記念基督** ——不是社交聚會
3. **表明合一** ——不是製造分裂
4. **等候主來** ——有末世的盼望

### 🌐 當代應用：在個人主義文化中活出群體

**矽谷的個人主義：**

* 「我的信仰是私人的」
* 教會被視為「可選的服務」
* 群體崇拜讓位給個人靈修podcast

**主的晚餐提醒我們：**

* 信仰是群體性的——「我們」擘餅
* 基督的身體是合一的——不分貧富、種族、職位
* 我們互相需要——不能「單獨」作基督徒

## 第七課：屬靈恩賜與愛的道路（12-14章）

### 經文要義

> **（林前12:4-7）**
> 恩賜原有分別，聖靈卻是一位；職事也有分別，主卻是一位；功用也有分別，神卻是一位，在眾人裡面運行一切的事。聖靈顯在各人身上，是叫人得益處。

### 恩賜的原則

| 原則   | 說明                             |
| ------ | -------------------------------- |
| 多樣性 | 恩賜各有不同，是神主權的分配     |
| 合一性 | 都是同一位聖靈所賜，為著一個身體 |
| 功用性 | 是為了造就人，不是為了炫耀       |
| 相互性 | 身體的各部分互相需要             |

### 更大的恩賜——愛的道路（13章）

> **（林前13:1-3）**
> 我若能說萬人的方言，並天使的話語，卻沒有愛，我就成了鳴的鑼、響的鈸一般。我若有先知講道之能，也明白各樣的奧祕、各樣的知識，而且有全備的信，叫我能夠移山，卻沒有愛，我就算不得什麼。我若將所有的賙濟窮人，又捨己身叫人焚燒，卻沒有愛，仍然與我無益。

### Morgan 論愛

> 「使徒論到愛，它的價值（1-3節）、它的美德（4-7節）、和它的得勝（8-13節）。沒有愛，最大的恩賜也毫無價值。愛是永不止息的——當一切恩賜都過去，愛仍然長存。」

### MacArthur 論方言

MacArthur 對哥林多方言問題的解釋：

* 方言是真實的語言，不是「ecstatic utterance」
* 主要功用是向不信的人作記號
* 在聚會中必須有翻譯
* 先知講道更能造就教會

### 🌐 當代應用：在恩賜文化中行走愛的道路

**矽谷的「恩賜崇拜」：**

* 天才工程師被神化
* 10x developer 的迷思
* 「顛覆性創新」的偶像
* 口才、表達能力被高度重視

**教會中的試探：**

* 高舉有「講道恩賜」的牧者
* 追求「超自然經歷」
* 用恩賜來衡量屬靈程度
* 忽略「不顯眼」的服事

**愛的道路：**

| 恩賜沒有愛…       | 愛使恩賜…     |
| ------------------ | -------------- |
| 成了鳴的鑼、響的鈸 | 帶來生命的改變 |
| 算不得什麼         | 有永恆的價值   |
| 與我無益           | 榮耀神、造就人 |

**實際操練：**

1. **承認自己的有限** ——我不需要有所有恩賜
2. **感謝別人的恩賜** ——身體的每個部分都重要
3. **以愛運用恩賜** ——問「這是否造就人？」
4. **追求更大的恩賜** ——但以愛為首要

# 第五部分：榮耀的盼望（15章）

## 第八課：復活的福音——信仰的根基（15:1-34）

### 經文要義

> **（林前15:1-4）**
> 弟兄們，我如今把先前所傳給你們的福音告訴你們知道。這福音你們也領受了，又靠著站立得住；並且你們若不是徒然相信，能以持守我所傳給你們的，就必因這福音得救。我當日所領受又傳給你們的，第一，就是基督照聖經所說，為我們的罪死了，而且埋葬了，又照聖經所說，第三天復活了。

### 福音的核心內容

1. **基督為我們的罪死了** ——代贖
2. **埋葬了** ——真正的死亡
3. **第三天復活了** ——身體的復活
4. **照聖經所說** ——應驗預言

### MacArthur 的強調

> 「身體的復活是福音信息的核心。任何自稱跟隨耶穌的人都必須相信身體的復活。」

### 若基督沒有復活的後果（15:12-19）

如果沒有復活：

* 我們所傳的便是枉然
* 你們的信也是枉然
* 我們便是妄作見證
* 你們仍在罪裡
* 已經睡了的人也滅亡了
* 我們就比眾人更可憐

### 🌐 當代應用：在懷疑主義時代持守復活真理

**矽谷的懷疑主義：**

* 「科學不能證明復活」
* 「死而復活違反自然法則」
* 「這只是神話或比喻」
* AI 時代的 materialist worldview

**持守真道的回應：**

1. **復活是歷史事件** ——有眼見證人、空墳墓、初代教會的存在
2. **復活是信仰根基** ——沒有復活，一切都是枉然
3. **復活是理性的盼望** ——創造宇宙的神當然能使人復活
4. **復活改變一切** ——死亡不是終點，而是新生命的開始

---

## 第九課：榮耀身體——復活的本質（15:35-58）

### 經文要義

> **（林前15:42-44）**
> 死人復活也是這樣：所種的是必朽壞的，復活的是不朽壞的；所種的是羞辱的，復活的是榮耀的；所種的是軟弱的，復活的是強壯的；所種的是血氣的身體，復活的是靈性的身體。若有血氣的身體，也必有靈性的身體。

### 復活身體的本質

| 復活前的身體 | 復活後的身體 | 意義                 |
| ------------ | ------------ | -------------------- |
| 必朽壞的     | 不朽壞的     | 不再衰老、疾病、死亡 |
| 羞辱的       | 榮耀的       | 不再有罪的羞恥       |
| 軟弱的       | 強壯的       | 不再有身體的限制     |
| 血氣的       | 靈性的       | 完全順服聖靈         |
| 屬地的       | 屬天的       | 適合永恆的國度       |

### 榮耀的奧秘（15:50-57）

> **（林前15:51-52, 54-55）**
> 我如今把一件奧祕的事告訴你們：我們不是都要睡覺，乃是都要改變，就在一霎時，眨眼之間，號筒末次吹響的時候。因號筒要響，死人要復活，成為不朽壞的，我們也要改變……那時經上所記「死被得勝吞滅」的話就應驗了。「死啊，你得勝的權勢在哪裡？死啊，你的毒鉤在哪裡？」

### MacArthur 論榮耀身體

> 「基督復活的身體是我們復活身體的模型。我們將像祂一樣——可以被認出，但已經改變；是物質的，但已經榮耀。腓立比書 3:21 說，祂『要按著那能叫萬有歸服自己的大能，將我們這卑微的身體改變形狀，和祂自己榮耀的身體相似。』」

### 得勝的確據

> **（林前15:57-58）**
> 感謝神，使我們藉著我們的主耶穌基督得勝！所以，我親愛的弟兄們，你們務要堅固，不可搖動，常常竭力多做主工，因為知道，你們的勞苦在主裡面不是徒然的。

### 🌐 當代應用：在死亡恐懼中活出復活盼望

**矽谷對死亡的回應：**

* 生物科技追求「延長壽命」
* AI 追求「意識上傳」
* 冷凍技術希望「未來復活」
* Transhumanism 的「超越人類限制」

**這些都反映人對死亡的恐懼和對永生的渴望——但只有基督能真正勝過死亡。**

**復活盼望帶來的自由：**

| 沒有復活盼望   | 有復活盼望           |
| -------------- | -------------------- |
| 恐懼死亡       | 死亡是入口，不是終點 |
| 抓緊今生       | 為永恆投資           |
| 勞苦是徒然的   | 勞苦在主裡不是徒然   |
| 追求今生的舒適 | 甘心為主受苦         |
| 害怕失去       | 得著基督就得著一切   |

**實際影響：**

1. **面對職場失敗** ——這不是最終的結局
2. **面對健康問題** ——這個身體會改變
3. **面對親人離世** ——我們還會再見
4. **面對逼迫患難** ——至暫至輕的苦楚要成就極重無比永遠的榮耀

# 結論：在基督裡得自由

## 哥林多前書的核心信息

### Morgan 的總結

> 「這封書信的根基命題是：教會被召進入與耶穌基督的團契。第一部分是糾正性的，處理屬肉體本性主導的證據。最後以榮耀的盼望結束——復活的確據和得勝的宣告。」

### MacArthur 的總結

> 「保羅處理十字架、神的智慧與人的智慧、聖靈在光照中的工作、屬肉體、永恆的獎賞、救恩的轉化、成聖、基督的本性、與祂的聯合、神為女性設立的角色、婚姻與離婚、聖靈的洗禮、內住與恩賜、教會在一個身體中的合一、愛的神學、以及復活的教義。」

## 十大原則：在AI時代持守真道

基於哥林多前書，我們可以歸納出以下原則，幫助我們在當今矽谷的挑戰中得勝：

### 1. 十字架是唯一的智慧

* 不被世界的智慧迷惑
* AI 不能解決罪的問題
* 技術不能取代福音

### 2. 屬靈成熟比專業成就重要

* 從屬肉體的嬰孩成長為屬靈的成人
* 用乾糧餵養自己
* 生命品格比履歷重要

### 3. 在基督的根基上建造

* 用金銀寶石，不是草木禾稭
* 追求永恆的價值
* 事奉的動機決定果效

### 4. 身體是聖靈的殿

* 在身體上榮耀神
* 逃避淫行
* 不受任何事轄制

### 5. 愛心比知識更重要

* 知識叫人自高，愛心造就人
* 為弟兄放下自由
* 以他人的益處為念

### 6. 甘心作眾人的僕人

* 效法保羅，為福音放下權利
* 向什麼樣的人，就作什麼樣的人
* 服事比成功更重要

### 7. 在群體中成長

* 身體的各部分互相需要
* 不可忽視聚會
* 在愛中運用恩賜

### 8. 追求愛的道路

* 愛是最大的
* 沒有愛，恩賜毫無價值
* 愛是永不止息的

### 9. 持守復活的盼望

* 這是福音的核心
* 這是事奉的動力
* 這是面對死亡的答案

### 10. 堅固、不可搖動、竭力多做主工

* 因為我們的勞苦在主裡面不是徒然的
* 萬有全是我們的
* 我們是屬基督的，基督是屬神的
