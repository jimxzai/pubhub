## 写作标准

每篇完整注疏应包含：
1. 原文（繁体 + 简体）
2. 英译对照（至少两版）
3. 历代注疏精选（曹操、杜牧、梅尧臣等）
4. AI时代案例（3-5个真实案例）
5. 个人心得（500-1000字）

- 英文：Lionel Giles (1910), Ralph Sawyer (1994), Victor Mair (2007)
