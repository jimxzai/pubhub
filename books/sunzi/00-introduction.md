## 核心概念

### 一、五事七計 (始計篇)

**五事**（戰略評估五要素）：
| 要素 | 內容 | 現代對應 |
|------|------|----------|
| **道** | 令民與上同意 | 政治認同、企業文化 |
| **天** | 陰陽寒暑時制 | 時機、環境因素 |
| **地** | 遠近險易廣狹 | 地理、市場條件 |
| **將** | 智信仁勇嚴 | 領導力 |
| **法** | 曲制官道主用 | 組織制度 |

**七計**（比較分析）：
1. 主孰有道？
2. 將孰有能？
3. 天地孰得？
4. 法令孰行？
5. 兵眾孰強？
6. 士卒孰練？
7. 賞罰孰明？

### 二、全勝思想 (謀攻篇)

> **「百戰百勝，非善之善者也；不戰而屈人之兵，善之善者也。」**

**攻擊優先序**：
```
1. 伐謀 (最上) → 瓦解敵人的計劃
2. 伐交 (次之) → 破壞敵人的外交
3. 伐兵 (又次) → 在戰場上擊敗敵軍
4. 攻城 (最下) → 圍攻城池（代價最大）
```

### 三、知己知彼

> **「知彼知己，百戰不殆；不知彼而知己，一勝一負；不知彼不知己，每戰必殆。」**

### 四、奇正之變 (兵勢篇)

> **「凡戰者，以正合，以奇勝。」**

- **正**：正面、常規、預期的行動
- **奇**：出其不意、變化、創新

### 五、虛實之道 (虛實篇)

> **「兵之形，避實而擊虛。」**

核心原則：致人而不致於人（掌握主動權）

## 三書對照視角

### 孫子與聖經

| 孫子兵法 | 聖經 | 對照分析 |
|----------|------|----------|
| 「兵者，詭道也」 | 「你們要靈巧像蛇」(太10:16) | 智慧與誠實的平衡 |
| 「不戰而屈人」 | 基督的得勝方式 | 犧牲的愛 vs 武力征服 |
| 「知彼知己」 | 「鑒察人心」(耶17:10) | 人的知識 vs 神的全知 |
| 「五事」之「道」 | 「道路真理生命」(約14:6) | 政治認同 vs 永恆之道 |
| 將帥之德 | 基督的品格 | 人的領導 vs 神的牧養 |

## 四維透視框架

### 1. 人性透視

**孫子的人性觀**：
- 人心可察、可測、可用
- 利害驅動行為
- 恐懼與貪婪是弱點

**聖經的補充**：
- 人心比萬物都詭詐 (耶17:9)
- 唯有神能真正鑒察人心
- 人需要救贖，不只是管理

### 2. 出路透視

**孫子的出路**：知識 + 策略 + 執行
**聖經的出路**：悔改 + 信靠 + 跟隨

孫子提供的是**戰術智慧**，聖經提供的是**生命道路**。

### 3. 神的計劃透視

孫子兵法是**普遍恩典**的結晶——
- 人類理性的成就
- 觀察經驗的總結
- 不處理永恆問題

聖經揭示**特殊啟示**——
- 神主動說話
- 救贖計劃展開
- 指向永恆

### 4. 爭戰透視

| 層面 | 孫子兵法 | 聖經 |
|------|----------|------|
| 軍事 | 詳細論述 | 屬靈軍裝 (弗6) |
| 政治 | 廟算決策 | 神掌管萬邦 |
| 屬靈 | 沉默 | 光暗對抗 |
| 終極 | 一時勝敗 | 羔羊得勝 |

## 信・望・愛

### 信
孫子的智慧值得學習，但不是終極信靠。
唯有基督是道路、真理、生命。

### 望
孫子盼望的是一時勝利，
我們盼望的是基督再來、新天新地。

### 愛
孫子因愛國而著書，
我們因神的愛而研讀，
愛永遠長存。
