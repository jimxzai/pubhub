# 約翰一書 4:7-21：神就是愛
## 1 John 4:7-21 — God Is Love

---

## 經文

### 和合本修訂版 (RCUV)

**愛的源頭 (4:7-10)**

> ⁷ 親愛的，我們要彼此相愛，因為愛是從神來的。凡有愛的都是由神而生，並且認識神。
> ⁸ 沒有愛的就不認識神，因為神就是愛。
> ⁹ 神差他獨一的兒子到世上來，使我們藉著他得生命；神的愛就在此向我們顯明了。
> ¹⁰ 不是我們愛神，而是神愛我們，差他的兒子為我們的罪作了贖罪祭；這就是愛。

**住在愛裡 (4:11-16)**

> ¹¹ 親愛的，神既然這樣愛我們，我們也要彼此相愛。
> ¹² 從來沒有人見過神，我們若彼此相愛，神就住在我們裡面，他的愛在我們裡面得以完全了。
> ¹³ 因為神把他的靈賜給我們，由此我們知道我們是住在他裡面，他也住在我們裡面。
> ¹⁴ 父差子作世人的救主，這是我們所看見並且作見證的。
> ¹⁵ 凡宣認耶穌是神兒子的，神就住在他裡面，他也住在神裡面。
> ¹⁶ 神愛我們的心，我們也知道，也信。神就是愛；住在愛裡面的，就是住在神裡面，神也住在他裡面。

**愛裡沒有懼怕 (4:17-21)**

> ¹⁷ 這樣，愛在我們裡面得以完全，使我們可以在審判的日子坦然無懼。因為他怎樣，我們在這世上也怎樣。
> ¹⁸ 愛裡沒有懼怕；完全的愛把懼怕驅逐出去，因為懼怕含有刑罰，懼怕的人在愛裡未得完全。
> ¹⁹ 我們愛，因為神先愛我們。
> ²⁰ 若有人說「我愛神」，卻恨他的弟兄，他就是說謊的；不愛他所看見的弟兄，怎能愛沒有看見的神呢？
> ²¹ 愛神的，也要愛弟兄，這是我們從神所受的命令。

## 原文研讀

### 關鍵詞彙

| 希臘文 | 音譯 | 意義 |
|--------|------|------|
| ἀγάπη | agapē | 愛（名詞）|
| ἀγαπάω | agapaō | 愛（動詞）|
| μονογενής | monogenēs | 獨一的、獨生的 |
| ἱλασμός | hilasmos | 贖罪祭、挽回祭 |
| τελειόω | teleioō | 使完全、成全 |
| φόβος | phobos | 懼怕、恐懼 |
| κόλασις | kolasis | 刑罰 |
| μένω | menō | 住、常在 |

### 重要原文解析

#### 1. 「神就是愛」(ὁ θεὸς ἀγάπη ἐστίν)

- **語法結構**：主語 + 謂語，沒有冠詞在 ἀγάπη 前
- **神學意義**：愛是神的本質，不只是神的屬性之一
- **對比**：約翰說「神就是愛」，但沒有說「愛就是神」
  - 順序不可逆轉：神定義愛，不是愛定義神
- **平行宣告**：
  - 約一 1:5「神就是光」(ὁ θεὸς φῶς ἐστιν)
  - 約 4:24「神是靈」(πνεῦμα ὁ θεός)

#### 2. 「獨一的兒子」(τὸν υἱὸν αὐτοῦ τὸν μονογενῆ)

- **μονογενής**：μόνος（唯一）+ γένος（種類）
- **不是「獨生」**：不強調出生方式，而是獨特地位
- **與約翰福音一致**：約 1:14, 18; 3:16, 18
- **神學意義**：耶穌與父的關係是獨一無二的

#### 3. 「贖罪祭」(ἱλασμός)

- **詞根**：ἱλάσκομαι（使和好、平息怒氣）
- **舊約背景**：贖罪日的儀式（利 16 章）
- **新約用法**：
  - 羅 3:25 ἱλαστήριον（施恩座、挽回祭）
  - 來 2:17 ἱλάσκεσθαι（為罪獻祭）
- **神學爭議**：
  - 「挽回祭」(propitiation)：平息神的怒氣
  - 「贖罪祭」(expiation)：除去罪的污穢
  - 兩者皆有：基督的死既平息神的怒氣，也潔淨人的罪

#### 4. 「愛在我們裡面得以完全」(ἡ ἀγάπη αὐτοῦ ἐν ἡμῖν τετελειωμένη ἐστίν)

- **τελειόω**：達到目標、使完全
- **完成被動式**：神的愛已經在我們裡面達到目標
- **「完全」的含義**：不是無缺點，而是達到預定目的
- **條件**：「我們若彼此相愛」——愛弟兄是愛完全的管道

#### 5. 「完全的愛把懼怕驅逐出去」(ἡ τελεία ἀγάπη ἔξω βάλλει τὸν φόβον)

- **ἔξω βάλλει**：扔出去、趕出去（強烈動詞）
- **φόβος**：懼怕，特指對審判/刑罰的恐懼
- **邏輯**：懼怕與愛不能共存
  - 懼怕聚焦於自我保護
  - 愛聚焦於他者的益處
- **實際應用**：對神審判的懼怕被神愛的確據所取代

#### 6. 「我們愛，因為神先愛我們」(ἡμεῖς ἀγαπῶμεν, ὅτι αὐτὸς πρῶτος ἠγάπησεν ἡμᾶς)

- **πρῶτος**：首先、第一
- **神學次序**：神的愛是因，我們的愛是果
- **恩典的本質**：不是我們配得，而是神主動
- 這是約翰對「愛」的最終定義：愛的源頭在神，不在人

## 歷代註釋

### 奧古斯丁 (Augustine, 354-430)

> 「『神就是愛』——這句話說得多麼簡短，多麼重要！如果在整封書信中什麼都沒說，只說這一句，這就夠了。」
> — *Homilies on 1 John* 7.4

**現代理解**：奧古斯丁把「神就是愛」視為約翰神學的核心。這不是說愛是神眾多屬性之一，而是愛就是神的本質。

### 馬丁·路德 (Martin Luther, 1483-1546)

> 「『不是我們愛神，乃是神愛我們』——這是對一切功德主義的致命一擊。我們無法藉著愛神來賺取救恩，因為我們連愛的能力都沒有。一切始於神的愛。」
> — *Commentary on 1 John*

**現代理解**：路德從這段經文看見「唯獨恩典」的真理。人的愛是對神愛的回應，不是救恩的條件。

### 加爾文 (John Calvin, 1509-1564)

> 「『愛裡沒有懼怕』——這不是說信徒沒有敬畏神的心，而是說他們不再有奴隸的恐懼，擔心被神拒絕。完全的愛帶來兒子的確據，不是奴僕的戰兢。」
> — *Commentary on 1 John*

**現代理解**：加爾文區分「奴僕的懼怕」(servile fear) 和「兒子的敬畏」(filial reverence)。前者被愛驅逐，後者與愛並存。

### 約翰·歐文 (John Owen, 1616-1683)

> 「神的愛在基督裡的彰顯，是整個福音的總綱。約翰在這裡告訴我們愛是什麼：不是人上升到神那裡的努力，而是神下降到人這裡的恩典。」
> — *Communion with God*

**現代理解**：歐文強調愛的方向是「神→人」，這顛覆了所有宗教的努力方向。基督教的獨特在於神主動尋找人。

## 神學主題

### 主題一：神的本質是愛

4:8, 16 兩次宣告「神就是愛」，這是聖經中最深刻的神學陳述之一。

**「神就是愛」的含義**：

| 是 | 不是 |
|-----|------|
| 愛是神的本質 | 愛只是神的一個屬性 |
| 神定義什麼是愛 | 人的愛情定義神 |
| 三位一體內的永恆相愛 | 神需要受造物才能愛 |
| 愛必須有對象 | 愛是抽象概念 |

**三位一體與愛**：
- 父愛子（約 3:35; 5:20）
- 子愛父（約 14:31）
- 聖靈是愛的聯繫（羅 5:5）
- 神在永恆中就是愛，不需要創造人才能愛

### 主題二：愛的顯明——差遣獨生子

4:9-10 定義了愛的內容：

| 愛的要素 | 經文 |
|----------|------|
| 主動性 | 神差祂的兒子 |
| 犧牲性 | 獨一的兒子 |
| 目的性 | 使我們得生命 |
| 代贖性 | 為我們的罪作贖罪祭 |

**與約翰福音 3:16 的平行**：

| 約 3:16 | 約一 4:9-10 |
|---------|-------------|
| 神愛世人 | 神愛我們 |
| 甚至將祂的獨生子賜給他們 | 差祂獨一的兒子到世上 |
| 叫一切信祂的不致滅亡 | 使我們藉著祂得生命 |
| 反得永生 | 為我們的罪作贖罪祭 |

### 主題三：愛的次序——神先愛我們

4:10, 19 強調愛的次序：

```
神的愛 (因) → 我們的愛 (果)
     ↓
  差遣基督
     ↓
  贖罪祭
     ↓
  我們得生命
     ↓
  我們愛神
     ↓
  我們彼此相愛
```

**神學要點**：
- 人的愛不是救恩的條件，而是救恩的果子
- 我們無法藉著愛神來賺取恩典
- 恩典在先，回應在後

### 主題四：愛的完全——驅逐懼怕

4:17-18 描述愛的完全狀態：

| 完全的愛 | 未完全的愛 |
|----------|------------|
| 審判日坦然無懼 | 懼怕審判 |
| 確信被神接納 | 擔心被神拒絕 |
| 聚焦於愛他人 | 聚焦於自我保護 |
| 兒子的自由 | 奴僕的恐懼 |

**「他怎樣，我們在這世上也怎樣」(4:17)**：
- 基督是被父所愛的
- 我們在基督裡，也被父所愛
- 祂在審判中站立得住，我們也能

### 主題五：愛的測試——愛弟兄

4:20-21 再次把愛神與愛弟兄連結：

> 「不愛他所看見的弟兄，怎能愛沒有看見的神呢？」

**邏輯**：
- 弟兄是看得見的
- 神是看不見的
- 如果連看得見的都不能愛，怎能愛看不見的？
- 愛弟兄是愛神的具體表達

## 實際應用

### 1. 重新定義愛

現代文化對愛的誤解：
- 愛是感覺 → 約翰：愛是行動（差遣、捨命、給予）
- 愛是找到對的人 → 約翰：愛是成為對的人
- 愛是被愛 → 約翰：愛是去愛
- 愛是吸引力 → 約翰：愛是犧牲

按照 4:10 重新定義：
> 「不是我們愛神，而是神愛我們，差祂的兒子為我們的罪作了贖罪祭；這就是愛。」

### 2. 處理懼怕

4:18 的應用：

**懼怕的根源**：
- 擔心不被接納
- 害怕審判和刑罰
- 對未來的不確定

**愛的解藥**：
- 確信神的愛（4:16「我們也知道，也信」）
- 住在愛裡（4:16「住在愛裡面的，就是住在神裡面」）
- 讓愛「驅逐」懼怕——不是壓抑，而是被取代

### 3. 愛的優先次序

4:19 的應用：

```
錯誤次序：我愛神 → 神愛我
正確次序：神愛我 → 我愛神 → 我愛弟兄
```

**實際意義**：
- 不要試圖藉著愛來賺取神的接納
- 先領受神的愛，再流露出對人的愛
- 愛是果子，不是根

### 4. 愛的具體化

4:20 的挑戰：

| 抽象的愛神 | 具體的愛弟兄 |
|------------|--------------|
| 感覺良好 | 行動付出 |
| 私下禱告 | 公開服事 |
| 無法驗證 | 可被觀察 |
| 可以偽裝 | 難以假冒 |

**應用問題**：
- 我說「愛神」，但我如何對待教會中的弟兄姊妹？
- 我對最難相處的人有愛嗎？
- 我的愛停留在言語還是落實在行動？

## AI 時代反思

### 1. 愛的不可複製性

AI 可以模擬很多東西，但無法真正「愛」：

| AI 能做的 | AI 不能做的 |
|-----------|-------------|
| 說出愛的語言 | 發出愛的心意 |
| 分析愛的定義 | 體驗愛的關係 |
| 模仿愛的行為 | 犧牲自己 |
| 提供陪伴感 | 真正與人同在 |

**神學反思**：「神就是愛」意味著愛是位格性的。只有位格才能愛，機器沒有位格。

### 2. 「先愛」的主動性

4:19「神先愛我們」的獨特性：

- AI 是被動的：等待輸入，然後回應
- 神是主動的：在我們尋找祂之前，祂已經差遣基督
- 愛的本質是主動尋找，不是被動回應

### 3. 懼怕與控制

現代人的懼怕常與失控感有關：
- AI 帶來的工作不安全感
- 資訊爆炸帶來的認知超載
- 社會變遷帶來的身份焦慮

4:18 的安慰依然有效：
- 完全的愛驅逐懼怕
- 神掌管一切，包括 AI 時代
- 在神的愛中，我們有終極的安全感

### 4. 看得見 vs 看不見

4:20 的邏輯在數位時代有新的應用：

- 我們花大量時間在「看不見」的虛擬世界
- 卻忽略「看得見」的身邊的人
- 約翰的挑戰：先愛「看得見的」弟兄

## 「神就是愛」vs「神就是光」

約翰一書有兩個核心宣告，形成平衡：

| 神就是光 (1:5) | 神就是愛 (4:8, 16) |
|----------------|-------------------|
| 神的聖潔 | 神的慈愛 |
| 不能與罪妥協 | 主動拯救罪人 |
| 審判的根據 | 救恩的動力 |
| 使罪顯露 | 使罪得赦 |
| 行在光中 | 住在愛裡 |

**兩者不可分割**：
- 沒有光的愛是濫愛，縱容罪惡
- 沒有愛的光是冷酷，令人絕望
- 十字架是光與愛的交匯：神的聖潔審判了罪，神的愛承擔了刑罰
