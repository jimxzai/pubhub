# 約翰一書 2:28-3:10：神的兒女
## 1 John 2:28-3:10 — Children of God

## 結構分析

```
神的兒女 (2:28-3:10)

A. 住在主裡的呼召 (2:28-29)
   ├── 目的：主顯現時坦然無懼
   └── 標記：行公義 = 從神生的

B. 神兒女的身份 (3:1-3)
   ├── 現在：已經是神的兒女
   ├── 將來：必像祂
   └── 回應：潔淨自己

C. 罪的本質 (3:4-6)
   ├── 罪 = 違背律法
   ├── 基督顯現除掉罪
   └── 住在祂裡面不犯罪

D. 兩種源頭 (3:7-10)
   ├── 行義 = 屬神
   ├── 犯罪 = 屬魔鬼
   └── 神的種子 vs 魔鬼的作為
```

## 原文研讀

### 關鍵詞彙

| 希臘文 | 音譯 | 意義 |
|--------|------|------|
| τέκνα θεοῦ | tekna theou | 神的兒女 |
| παρρησία | parrēsia | 坦然、膽量、放膽 |
| φανερόω | phaneroō | 顯現、顯明 |
| ἁμαρτία | hamartia | 罪 |
| ἀνομία | anomia | 不法、違背律法 |
| σπέρμα | sperma | 種子 |
| διάβολος | diabolos | 魔鬼、控告者 |

### 重要原文解析

#### 1. 「何等的慈愛」(ποταπὴν ἀγάπην)

ποταπός 是驚嘆詞，表示「何等的！什麼樣的！」約翰對神的愛發出驚嘆。

#### 2. 「神的兒女」(τέκνα θεοῦ)

- τέκνα：孩子（強調出生關係）
- 對比 υἱοί：兒子（強調地位身份）

約翰用 τέκνα 強調我們是從神「生」的。

#### 3. 「不犯罪」的動詞時態

3:6 和 3:9 的「不犯罪」：
- οὐχ ἁμαρτάνει (3:6)：現在式，不持續犯罪
- ἁμαρτίαν οὐ ποιεῖ (3:9)：現在式，不以犯罪為習慣

**重要**：約翰不是說信徒絕不會犯罪（這與 1:8-10 矛盾），而是說信徒不會以犯罪為生活方式。

#### 4. 「神的種子」(σπέρμα αὐτοῦ)

σπέρμα 可指：
- 聖靈
- 神的道
- 新生命的本質

最可能指**神所賜的新生命本質**，使信徒有不犯罪的傾向。

## 神學要點

### 1. 神兒女的確據

約翰給出三重時態的確據：

| 時態 | 內容 | 經文 |
|------|------|------|
| 過去 | 父賜給我們慈愛 | 3:1a |
| 現在 | 我們現在是神的兒女 | 3:2a |
| 將來 | 我們必像祂 | 3:2b |

**現在就是**——不是將來才成為，而是現在已經是神的兒女。

### 2. 榮耀的盼望

> 「他若顯現，我們必會像他，因為我們必得見他的真面目。」(3:2)

這是基督教末世論的核心盼望：

```
現在                    將來
  │                      │
神的兒女 ──────────→ 像基督
（身份已定）           （完全變化）
```

「像他」(ὅμοιοι αὐτῷ)：道德上、榮耀上像基督。

### 3. 盼望的倫理效應

> 「凡對他有這指望的，就潔淨自己」(3:3)

末世盼望不是逃避現實，而是改變現在：
- 因為將來要像祂，所以現在就開始潔淨
- 盼望產生聖潔的動力

### 4. 罪的定義

> 「罪就是違背律法」(ἡ ἁμαρτία ἐστὶν ἡ ἀνομία) (3:4)

約翰給出罪的本質定義：
- ἀνομία：不法、無律法、違背神的標準
- 罪不只是錯誤，而是對神律法的叛逆

### 5. 「不犯罪」的難題

3:6 和 3:9 說信徒「不犯罪」，但 1:8-10 說「若說沒有罪就是自欺」。如何調和？

| 解釋 | 說明 |
|------|------|
| 時態解釋 | 不持續犯罪（現在式表習慣性） |
| 本質解釋 | 新生命的本質是不犯罪的 |
| 理想解釋 | 這是神兒女應有的標準 |
| 末世解釋 | 在將來完全實現 |

**綜合理解**：真信徒可能會犯罪，但不會以犯罪為生活模式。新生命的本質是抗拒罪的。

### 6. 兩種源頭、兩種生命

| 從神生的 | 從魔鬼來的 |
|----------|------------|
| 行公義 | 犯罪 |
| 神的種子在裡面 | 魔鬼的作為 |
| 神的兒女 | 魔鬼的兒女 |

沒有中間地帶。行為顯明身份。

### 7. 基督顯現的目的

約翰提到基督顯現的兩個目的：
- 「除掉罪」(ἵνα τὰς ἁμαρτίας ἄρῃ) (3:5)
- 「毀滅魔鬼的作為」(ἵνα λύσῃ τὰ ἔργα τοῦ διαβόλου) (3:8)

基督的工作是雙重的：對付罪，對付撒但。

## 歷代注疏

### 奧古斯丁 (Augustine)

> 「約翰說『凡從神生的就不犯罪』，不是說他永不犯任何罪，而是說他不持續活在罪中。若有人跌倒，他會起來；若有人犯罪，他會認罪悔改。這就是神的種子在他裡面的證據。」
> — *Homilies on 1 John* 5.3

### 加爾文 (John Calvin)

> 「『我們必像他』是何等甜蜜的應許！現在我們背負著肉體的軟弱，但那日我們要完全像基督。這盼望應當激勵我們現在就追求聖潔，因為我們正走向那榮耀的目標。」
> — *Commentary on 1 John*

### 斯托得 (John Stott)

> 「約翰把人類分為兩個家族：神的兒女和魔鬼的兒女。這種非黑即白的劃分在現代人聽來很刺耳，但這正是聖經的觀點。沒有人可以同時屬於兩個家族。」
> — *The Letters of John* (TNTC)

## 現代應用

### 1. 身份的確據

在焦慮的時代，約翰給我們確據：
- 你**現在就是**神的兒女（不是將來、不是也許）
- 這身份是神**賜給**的（不是賺來的）
- 這是**何等的**慈愛（值得驚嘆）

### 2. 盼望帶來改變

「對他有這指望的，就潔淨自己」——基督徒的倫理動力來自盼望，不是恐懼。

問自己：我對將來的盼望如何影響我今天的生活？

### 3. 行為顯明身份

約翰的測試很直接：
- 行公義 → 屬神
- 犯罪 → 屬魔鬼

這不是靠行為得救，而是真信心必有真行為。

### 4. 兩個家族的現實

現代文化模糊善惡界線，但約翰堅持：只有兩個家族，沒有第三選項。

## 反思問題

1. 「你看父賜給我們的是何等的慈愛」——你有沒有為自己是神兒女的身份感到驚嘆？

2. 知道「將來必像祂」，這如何影響你今天面對試探的態度？

3. 你的生活模式（不是偶爾的失敗）顯明你屬於哪個家族？

4. 「神的種子」在你裡面，你如何經歷這新生命抵擋罪的力量？
