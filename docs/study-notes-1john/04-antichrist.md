## 原文研讀

### 關鍵詞彙

| 希臘文 | 音譯 | 意義 |
|--------|------|------|
| ἀντίχριστος | antichristos | 敵基督、假基督 |
| ἐσχάτη ὥρα | eschatē hōra | 末時、最後的時刻 |
| χρῖσμα | chrisma | 恩膏、膏油 |
| ἀρνέομαι | arneomai | 否認、不認 |
| ψεῦστης | pseustēs | 說謊者 |
| μένω | menō | 住、常在、持續 |

### 重要原文解析

#### 1. 「敵基督」(ἀντίχριστος)

這個詞只出現在約翰書信中（約壹 2:18, 22; 4:3; 約貳 7）。

ἀντί 有兩個意義：
- **對抗** (against)：與基督敵對
- **代替** (instead of)：冒充基督

敵基督既是基督的敵人，也是假冒的基督。

#### 2. 「末時」(ἐσχάτη ὥρα)

- ἐσχάτη：最後的、末了的
- ὥρα：時刻、時段

約翰說「如今是末世的時候」——從基督升天到再來的整個時期都是「末世」。我們現在就活在末世。

#### 3. 「恩膏」(χρῖσμα)

- 詞根與「基督」(Χριστός = 受膏者) 相同
- 舊約背景：君王、祭司、先知受膏
- 新約應用：聖靈的內住

信徒都從「那聖者」(基督) 領受了恩膏 (聖靈)。

#### 4. 「不認」(ἀρνέομαι)

- 意義：否認、拒絕承認、背棄
- 同一詞用於彼得「不認」主（太 26:70）

敵基督的核心特徵是「不認」——否認耶穌是基督，否認父與子的關係。

## 神學要點

### 1. 末世論的視角

約翰的末世觀：

```
基督第一次來          基督第二次來
      │                    │
      ▼                    ▼
──────┼────────────────────┼──────
      │←──── 末世/末時 ────→│
      │    （我們在這裡）    │
```

「末時」不是未來某個時刻，而是現在進行中的時代。敵基督的出現證明我們活在末世。

### 2. 敵基督的多重性

約翰區分：
- **那敵基督** (ὁ ἀντίχριστος)：單數，末世要來的那位
- **好些敵基督** (ἀντίχριστοι πολλοί)：複數，已經出現的假教師

| 單數 | 複數 |
|------|------|
| 末世終極的敵基督 | 現在活躍的假教師 |
| 將要來 | 已經來了 |
| 啟示錄中的獸？ | 教會中的異端 |

### 3. 離開 = 顯明本相

> 「他們從我們中間出去，卻不是屬我們的」(2:19)

這是一個重要的神學原則：
- 離開不是使他們「變成」不屬我們的
- 離開是**顯明**他們本來就不屬我們

真信徒會「仍舊與我們同在」(μεμενήκεισαν ἂν μεθ' ἡμῶν)。

### 4. 敵基督的神學特徵

約翰定義敵基督的核心錯誤：

| 否認 | 神學含義 |
|------|----------|
| 耶穌是基督 | 否認耶穌是神所差的彌賽亞 |
| 父與子 | 否認三位一體的關係 |
| 子 | 否認基督的神性或人性 |

這針對的是當時的異端（可能是克林妥主義或早期諾斯底主義）。

### 5. 恩膏的教導

> 「這恩膏怎樣教導你們，你們就怎樣住在他裡面」(2:27)

恩膏（聖靈）的功能：
- 教導真理（參約 16:13）
- 分辨錯謬
- 使信徒能住在基督裡

**注意**：這不是說不需要人的教導，而是說聖靈是真理的終極教師，能幫助信徒分辨真假。

### 6. 父與子不可分割

> 「凡不認子的，就沒有父；認子的，連父也有了。」(2:23)

這是基督教的獨特宣告：
- 不能繞過子去認識父
- 承認子 = 擁有父
- 否認子 = 失去父

對現代多元宗教對話的挑戰：「所有宗教最終都敬拜同一位神」的說法與這節經文矛盾。

## 歷代注疏

### 愛任紐 (Irenaeus, c. 130-202)

> 「約翰所說的敵基督，就是那些否認主道成肉身的人。他們聲稱認識神，卻否認神在肉身中來到我們中間。這樣的人是最危險的，因為他們從教會內部破壞信仰。」
> — *Against Heresies* III.16.8

### 奧古斯丁 (Augustine)

> 「敵基督不只是一個人，而是一個身體，就如基督有祂的身體（教會），敵基督也有他的身體（所有反對基督的人）。每一個時代都有敵基督的靈在運作。」
> — *Homilies on 1 John* 3.4

### 加爾文 (John Calvin)

> 「約翰說信徒有恩膏，不需要人教導，這不是廢除教會的教導職分，而是說聖靈使我們能分辨真假。若沒有聖靈的光照，即使最好的教導也是無用的；有了聖靈，我們就能分辨誰在教導真理。」
> — *Commentary on 1 John*

## 現代應用

### 1. 辨別「敵基督的靈」

現代的「敵基督」特徵：
- 否認基督的神性（自由派神學）
- 否認基督的人性（某些靈恩極端）
- 否認基督的獨一性（多元宗教主義）
- 否認基督的救贖（成功神學的扭曲）

### 2. 聖靈的分辨

在資訊爆炸的時代，如何分辨真假？
- 依靠聖靈的教導
- 對照聖經的真理
- 在教會群體中印證

### 3. 「從中間出去」的現象

教會歷史上不斷有人「從我們中間出去」：
- 異端領袖
- 背道的信徒
- 分裂教會的人

約翰的安慰：這顯明他們本來就不屬我們。真信徒會持守到底。

### 4. 末世的生活態度

既然我們活在末世：
- 警醒（敵基督的靈在運作）
- 持守（讓起初聽見的常存）
- 盼望（主必再來）
