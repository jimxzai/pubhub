# 約翰二書：在真理中行走
## 2 John — Walking in the Truth

---

## 經文

### 和合本修訂版 (RCUV)

**問候 (1-3)**

> ¹ 長老寫信給蒙揀選的太太和她的兒女，就是我在真理中所愛的；不只是我愛你們，也是一切認識真理的人愛你們，
> ² 因為真理存在我們裡面，也必與我們永遠同在。
> ³ 從父神和他兒子耶穌基督來的恩惠、憐憫、平安，在真理和愛中必與我們同在。

**在真理中行走 (4-6)**

> ⁴ 我見你的兒女有人按照我們從父所領受的命令，在真理中行走，我就非常歡喜。
> ⁵ 太太啊，我現在請求你：我們要彼此相愛。這不是我寫給你一條新命令，而是我們從起初就有的命令。
> ⁶ 我們若照他的命令行，這就是愛。你們從起初所聽見的，就是這命令，叫你們行在愛中。

**警告假教師 (7-11)**

> ⁷ 因為世上有許多迷惑人的出來，他們不認耶穌基督是成了肉身來的。這樣的人就是迷惑人的，也是敵基督的。
> ⁸ 你們要小心，不要失去我們所做的工，卻要得著滿足的賞賜。
> ⁹ 凡越過基督的教訓而不持守的，就沒有神；持守這教訓的，就有父又有子。
> ¹⁰ 若有人到你們那裡，不傳這教訓，不要接他到家裡，也不要向他問安；
> ¹¹ 因為向他問安的，就在他的惡行上有份。

**結語 (12-13)**

> ¹² 我還有許多話要寫給你們，卻不願意用紙和墨寫出來，但盼望到你們那裡，與你們當面談論，使我們的喜樂滿足。
> ¹³ 你那蒙揀選之姊妹的兒女都問候你。

## 原文研讀

### 關鍵詞彙

| 希臘文 | 音譯 | 意義 |
|--------|------|------|
| πρεσβύτερος | presbyteros | 長老 |
| ἐκλεκτός | eklektos | 蒙揀選的 |
| κυρία | kyria | 太太、女主人 |
| ἀλήθεια | alētheia | 真理 |
| περιπατέω | peripateō | 行走 |
| ἐντολή | entolē | 命令 |
| πλάνος | planos | 迷惑人的、欺騙者 |
| ἀντίχριστος | antichristos | 敵基督 |
| διδαχή | didachē | 教訓、教導 |

### 重要原文解析

#### 1. 「長老」(ὁ πρεσβύτερος)

- **字義**：年長者、長老
- **約翰的自稱**：不稱「使徒」，而稱「長老」
- **可能原因**：
  - 謙卑的自稱
  - 強調牧養關係
  - 晚年的身份（約翰是最後一位離世的使徒）

#### 2. 「蒙揀選的太太」(τῇ ἐκλεκτῇ κυρίᾳ)

**兩種主要解釋**：

| 解釋 | 支持論據 |
|------|----------|
| **某位姊妹** | 人名 Kyria 或「太太」的稱呼 |
| **某個教會** | v.13「姊妹」也指教會；「兒女」指信徒 |

- **多數學者**認為指教會，因為：
  - 「兒女」(τέκνα) 在約翰書信中常指信徒
  - v.13 的「姊妹」暗示是另一個教會
  - 全信的內容更像寫給教會而非個人

#### 3. 「在真理中行走」(ἐν ἀληθείᾳ περιπατοῦντας)

- **περιπατέω**：行走，比喻生活方式
- **ἐν ἀληθείᾳ**：在真理中/裡
- **神學意義**：
  - 真理不只是頭腦知識，更是生活實踐
  - 「行走」強調持續的、日常的生活方式
- **與約翰一書的連結**：
  - 約一 1:6「行在光明中」
  - 約一 2:6「照祂所行的去行」

#### 4. 「不認耶穌基督是成了肉身來的」(μὴ ὁμολογοῦντες Ἰησοῦν Χριστὸν ἐρχόμενον ἐν σαρκί)

- **ἐρχόμενον**：現在分詞，「正來的」或「曾來的」
- **與約一 4:2 的比較**：
  - 約一 4:2 用完成時態 ἐληλυθότα（已經來了）
  - 約二 7 用現在分詞 ἐρχόμενον
- **可能解釋**：
  - 強調持續性：基督「持續」以人性與我們同在
  - 指向再來：「將要來的」基督也是道成肉身的
  - 針對不同異端表述

#### 5. 「越過」(προάγων)

- **字義**：往前走、超越、越過
- **負面含義**：走得太前面，離開了基礎
- **神學意義**：
  - 「進步」不一定是好事
  - 離開基督的教訓就是「越過」
  - 持守 (μένων) vs 越過 (προάγων)

#### 6. 「不要接他到家裡」(μὴ λαμβάνετε αὐτὸν εἰς οἰκίαν)

- **οἰκία**：房子、家
- **背景**：
  - 早期教會在家庭中聚會
  - 接待巡迴教師是常見做法
  - 「接到家裡」意味著提供平台
- **不是個人待客**：
  - 重點是不給假教師傳播異端的機會
  - 不是不許對他們表達基本禮貌

## 歷代註釋

### 奧古斯丁 (Augustine, 354-430)

> 「約翰在這裡教導我們，愛必須有真理作基礎。沒有真理的愛是盲目的縱容；沒有愛的真理是冷酷的律法。二者必須並行。」
> — *Homilies on 1 John*

**現代理解**：奧古斯丁看見真理與愛的張力和平衡。約翰二書正是這個平衡的典範。

### 加爾文 (John Calvin, 1509-1564)

> 「『不要接他到家裡』——這不是說我們要粗魯地對待每一個錯誤的人，而是說我們不應該給傳異端的人提供傳播謬誤的機會。教會的責任是保護羊群。」
> — *Commentary on 2 John*

**現代理解**：加爾文區分個人禮貌和教會責任。拒絕接待假教師是保護，不是排斥。

### 馬丁·路德 (Martin Luther, 1483-1546)

> 「『越過基督的教訓』——今天許多人以為自己比使徒更聰明，要『進步』到基督以外的地方。但約翰說得清楚：離開基督教訓的，就沒有神。」
> — *Commentary on 2 John*

**現代理解**：路德警告「進步神學」的危險。真正的進步是更深入基督，不是越過基督。

### 約翰·斯托得 (John Stott, 1921-2011)

> 「約翰二書教導我們一個不舒服但必要的真理：愛有界限。真正的愛不會對謊言視而不見。愛要求真理，真理要求分辨。」
> — *The Letters of John* (TNTC)

**現代理解**：斯托得指出現代教會常忽略的真理：愛不是無條件的接納所有教導。

## 神學主題

### 主題一：真理與愛的平衡

約翰二書的核心張力是**真理**與**愛**的關係：

| 維度 | 經文 | 內容 |
|------|------|------|
| 真理中的愛 | v.1 | 「在真理中所愛的」|
| 真理和愛中的恩典 | v.3 | 「在真理和愛中」與我們同在 |
| 真理中的行走 | v.4 | 「在真理中行走」|
| 愛的命令 | v.5-6 | 彼此相愛 = 行在命令中 |
| 真理的保護 | v.7-11 | 不接待否認真理的 |

**神學要點**：
- 愛不能脫離真理
- 真理不能缺少愛
- 愛真理 = 愛神 = 愛弟兄
- 保護真理也是愛的表現

### 主題二：基督論的核心測試

v.7 提供了分辨真假教師的標準：

> 「他們不認耶穌基督是成了肉身來的」

**為什麼道成肉身如此重要？**

| 層面 | 含義 |
|------|------|
| **創造** | 神造的物質世界是好的 |
| **救贖** | 基督必須成為真正的人才能代贖 |
| **復活** | 肉身復活是救恩的一部分 |
| **關係** | 神真正進入人的處境 |

**否認道成肉身的後果**：
- 失去真正的救贖
- 失去身體復活的盼望
- 基督教變成抽象哲學

### 主題三：「持守」vs「越過」

v.9 呈現兩種態度：

| 持守 (μένω) | 越過 (προάγω) |
|-------------|---------------|
| 停留在基督教訓中 | 超越基督教訓 |
| 有父又有子 | 沒有神 |
| 保守主義的美德 | 「進步」的危險 |

**神學要點**：
- 真正的進步是更深入基督，不是離開基督
- 「古舊」的福音不需要「更新」
- 基礎性的教義不能妥協

### 主題四：接待的責任

v.10-11 的教導看似嚴厲，但有特定背景：

**早期教會的背景**：
- 巡迴教師依靠信徒接待
- 教會在家庭中聚會
- 接待 = 提供傳播平台

**「不要接他到家裡」的含義**：
- 不是不給水喝
- 不是不表達基本人道
- 而是**不給他教導的機會**

**現代應用**：
- 不邀請異端教師來教會講道
- 不推薦他們的書籍
- 不提供他們傳播謬誤的平台

## 愛永遠長存：四維透視

### 一、人性實況：真理的需要

約翰二書揭示人性的處境：
- **迷惑人的出來** (v.7)：人容易被欺騙
- **可能失去獎賞** (v.8)：信徒也會跌倒
- **有人越過基督** (v.9)：甚至在教會中

**人需要真理**，因為人心容易偏離。

### 二、神的出路：基督的教訓

神的出路是**基督的教訓** (ἡ διδαχὴ τοῦ Χριστοῦ)：
- 不是人的智慧
- 不是新的啟示
- 而是道成肉身的基督所啟示的真理

**持守這教訓的，就有父又有子** (v.9)。

### 三、神的計劃：漸進啟示的完成

約翰二書在救贖歷史中的位置：

```
舊約先知         基督道成肉身         使徒教訓         教會持守
   │                  │                  │               │
  預言      →       成就       →      記錄      →     傳承
   │                  │                  │               │
影子的真理    實體的啟示    完整的教訓    信徒的責任
```

**「從起初的命令」**(v.5-6) 連結了整個救贖歷史。

### 四、屬靈爭戰：真理與謬誤

約翰二書呈現屬靈爭戰的實況：

| 真理的一方 | 謬誤的一方 |
|------------|------------|
| 承認基督道成肉身 | 不認基督道成肉身 |
| 持守基督教訓 | 越過基督教訓 |
| 有父又有子 | 沒有神 |
| 接待真教師 | 迷惑人的、敵基督的 |

**愛要求分辨，分辨保護愛。**

| 主題 | 約翰一書 | 約翰二書 |

| **真理** | 1:6「行在光中」| 4「在真理中行走」|
| **愛的命令** | 2:7-8「舊命令也是新命令」| 5「從起初就有的命令」|
| **愛的定義** | 5:3「遵守祂的命令」| 6「照祂的命令行」|
| **道成肉身** | 4:2「認耶穌基督成肉身來的」| 7「不認耶穌基督成肉身來的」|
| **敵基督** | 2:18, 22; 4:3 | 7「迷惑人的、敵基督的」|

**約翰二書是約翰一書的「實踐版」**：
- 約一解釋**什麼是**真理和愛
- 約二教導**如何**在真理和愛中行動

## 實際應用

### 1. 真理與愛的平衡

**自我檢視**：
- 我強調真理時，是否失去了愛？
- 我強調愛時，是否妥協了真理？
- 我如何在真理中表達愛？

**應用原則**：
- 對人：愛和尊重
- 對教義：清晰和堅持
- 對行為：按真理行

### 2. 分辨假教師

**v.7 的標準**依然適用：

| 問題 | 思考 |
|------|------|
| 這教導是否承認基督的神人二性？ | 核心基督論 |
| 這教導是否越過基督的教訓？ | 「進步」的危險 |
| 這教導是否有使徒教訓的根據？ | 聖經為本 |

### 3. 接待與拒絕

**v.10-11 的現代應用**：

| 情境 | 應用 |
|------|------|
| 講壇邀請 | 不邀請否認核心教義的講員 |
| 書籍推薦 | 不推薦否認基督的書籍 |
| 個人交往 | 可以與人交往，但不給傳播異端的機會 |
| 基本禮貌 | 仍要表達基本的人道和尊重 |

### 4. 面談的喜樂

**v.12 的提醒**：
- 文字不能取代面對面的團契
- 書信/電郵有限，當面更完整
- 喜樂滿足需要真實的相交

## AI 時代反思

### 1. 真理的稀缺

在資訊爆炸的時代：
- AI 可以生成無數「看似真理」的內容
- 分辨真假比以往更困難
- 「在真理中行走」需要更多刻意

**約翰的呼召依然有效**：持守基督的教訓。

### 2. 「越過」的試探

現代人傾向「進步」：
- 更新的理論
- 更時髦的神學
- 「超越」傳統的教導

**約翰的警告**：「越過基督的教訓而不持守的，就沒有神。」

### 3. 數位時代的接待

v.10 在數位時代的應用：
- 社交媒體的「分享」= 某種形式的「接待」
- 訂閱/推薦 = 提供平台
- 不是所有「寬容」都是美德

### 4. 面對面的不可取代

v.12 提醒我們：
- 視訊不等於真正的同在
- 文字交流有其限制
- 真正的團契需要實體相遇
