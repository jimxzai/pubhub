# 第五單元：進前來的呼召

> **經文：希伯來書 10:19-39**

---

## 經文概覽

### 結構大綱

```
希伯來書 10:19-39 結構

第一段 (10:19-25) — 三重呼召
├── 10:19-21  根基：有新道路和大祭司
├── 10:22     第一呼召：進前來 (信心)
├── 10:23     第二呼召：持定 (盼望)
├── 10:24-25  第三呼召：彼此相顧 (愛心)

├── 10:26-27  故意犯罪，再沒有贖罪的祭

├── 10:35-36  不可丟棄勇敢的心
```

## 一、三重呼召 (10:19-25)

### 1.1 根基：新的道路 (10:19-21)

> **「弟兄們，我們既因耶穌的血得以坦然進入至聖所，是藉著他給我們開了一條又新又活的路，從幔子經過，這幔子就是他的身體。又有一位大祭司治理神的家。」**

**三個恩典根基**：

| 根基 | 內容 | 舊約對比 |
|------|------|----------|
| **耶穌的血** | 進入的資格 | 牛羊的血只能外在潔淨 |
| **又新又活的路** | 進入的通道 | 舊約每年一次 |
| **大祭司** | 進入的保證 | 亞倫祭司會死 |

**「又新又活」(πρόσφατον καὶ ζῶσαν)**：

```
又新又活的路

「新」(πρόσφατον)
  ↓
剛剛宰殺的 → 基督剛獻上的
永不過時 → 歷久彌新

「活」(ζῶσαν)
  ↓
有生命的 → 活的道路
持續有效 → 永遠有功效
```

**幔子的撕裂**：

> **「這幔子就是他的身體。」**

| 事件 | 意義 |
|------|------|
| 基督身體裂開 | 幔子撕裂 (太27:51) |
| 進入至聖所的路打開 | 神人隔絕的障礙除去 |
| 一次永遠 | 通路永遠敞開 |

### 1.2 第一呼召：進前來 — 信心 (10:22)

> **「並我們心中天良的虧欠已經灑去，身體用清水洗淨了，就當存著誠心和充足的信心來到神面前。」**

**進前來的條件**：

| 條件 | 希臘文 | 意義 |
|------|--------|------|
| **誠心** | ἀληθινῆς καρδίας | 真誠的心 |
| **充足的信心** | πληροφορίᾳ πίστεως | 完全的確信 |
| **心被灑淨** | ῥεραντισμένοι τὰς καρδίας | 良心的潔淨 |
| **身體洗淨** | λελουμένοι τὸ σῶμα | 外在的潔淨 |

**「進前來」(προσερχώμεθα) 的命令**：

這是希伯來書的核心呼召，在4:16已經出現：
> 「所以，我們只管坦然無懼地來到施恩的寶座前。」

### 1.3 第二呼召：持定 — 盼望 (10:23)

> **「也要堅守我們所承認的指望，不至搖動，因為那應許我們的是信實的。」**

**堅守的對象**：

| 對象 | 希臘文 | 說明 |
|------|--------|------|
| **所承認的指望** | τὴν ὁμολογίαν τῆς ἐλπίδος | 公開宣告的盼望 |

**堅守的根據**：

> **「因為那應許我們的是信實的。」**

- 不是靠我們的堅持
- 而是因為神的信實

### 1.4 第三呼召：彼此相顧 — 愛心 (10:24-25)

> **「又要彼此相顧，激發愛心，勉勵行善。你們不可停止聚會，好像那些停止慣了的人，倒要彼此勸勉，既知道那日子臨近，就更當如此。」**

**三個行動**：

| 行動 | 希臘文 | 說明 |
|------|--------|------|
| **彼此相顧** | κατανοῶμεν ἀλλήλους | 仔細注意彼此 |
| **激發** | εἰς παροξυσμὸν | 刺激、激勵 |
| **彼此勸勉** | παρακαλοῦντες | 安慰、鼓勵 |

**不可停止聚會**：

```
聚會的重要性

停止聚會 ────→ 孤立 ────→ 軟弱 ────→ 跌倒
     ↑                                    ↓
     └──────────── 惡性循環 ←────────────┘

彼此聚會 ────→ 團契 ────→ 剛強 ────→ 堅忍
     ↑                                    ↓
     └──────────── 良性循環 ←────────────┘
```

### 1.5 三重呼召的總結

| 呼召 | 動詞 | 對象 | 美德 |
|------|------|------|------|
| **進前來** | προσερχώμεθα | 神 | 信心 |
| **持定** | κατέχωμεν | 盼望 | 盼望 |
| **彼此相顧** | κατανοῶμεν | 彼此 | 愛心 |

**與林前13:13的連結**：

> **「如今常存的有信、有望、有愛這三樣，其中最大的是愛。」**

## 二、第四次警告 (10:26-31)

### 2.1 故意犯罪的嚴重性 (10:26-27)

> **「因為我們得知真道以後，若故意犯罪，贖罪的祭就再沒有了；唯有戰懼等候審判和那燒滅眾敵人的烈火。」**

**「故意犯罪」(ἑκουσίως ἁμαρτανόντων)**：

| 不是指 | 是指 |
|--------|------|
| 偶然跌倒 | 蓄意背棄 |
| 軟弱犯罪 | 悖逆離棄 |
| 無知過犯 | 明知故犯 |

**後果**：
- 「贖罪的祭就再沒有了」
- 不是說基督的救贖不夠
- 而是說棄絕基督後沒有別的救法

### 2.2 踐踏神的兒子 (10:28-29)

> **「人干犯摩西的律法，憑兩三個見證人，尚且不得憐恤而死，何況人踐踏神的兒子，將那使他成聖之約的血當作平常，又褻慢施恩的聖靈，你們想，他要受的刑罰該怎樣加重呢！」**

**三重罪行**：

| 罪行 | 對象 | 意義 |
|------|------|------|
| **踐踏** | 神的兒子 | 輕視基督的位格 |
| **當作平常** | 立約的血 | 輕視基督的工作 |
| **褻慢** | 施恩的聖靈 | 輕視聖靈的見證 |

**從小到大的論證**：

```
論證邏輯

干犯摩西律法 → 憑見證人定罪 → 死 (申17:6)
        ↓
    何況...
        ↓
踐踏神的兒子 → 更嚴重的背棄 → 更重的刑罰
```

### 2.3 落在永生神手中 (10:30-31)

> **「因為我們知道誰說：『伸冤在我，我必報應』；又說：『主要審判他的百姓。』落在永生神的手裡，真是可怕的！」**

引用申命記32:35-36。

**「可怕」(φοβερὸν) 的原因**：
- 神是活的神——不是偶像
- 神是公義的神——必定審判
- 神是永恆的神——無法逃脫

## 三、鼓勵堅忍 (10:32-39)

### 3.1 回想從前的爭戰 (10:32-34)

> **「你們要追念往日，蒙了光照以後所忍受大爭戰的各樣苦難：一面被毀謗，遭患難，成了戲景，叫眾人觀看；一面陪伴那些受這樣苦難的人。因為你們體恤了那些被捆鎖的人，並且你們的家業被人搶去，也甘心忍受，知道自己有更美長存的家業。」**

**從前的見證**：

| 苦難 | 回應 |
|------|------|
| 被毀謗、遭患難 | 忍受 |
| 成了戲景 | 不以為恥 |
| 陪伴受苦的人 | 團契 |
| 家業被搶 | 甘心忍受 |

**「更美長存的家業」**：
- 地上的財產可以被奪
- 天上的基業永不朽壞

### 3.2 不可丟棄勇敢的心 (10:35-36)

> **「所以，你們不可丟棄勇敢的心；存這樣的心必得大賞賜。你們必須忍耐，使你們行完了神的旨意，就可以得著所應許的。」**

**勇敢的心 (παρρησία)**：
- 4:16 用來描述「坦然無懼」
- 這裡是同一個詞
- 不是自己的勇氣，是因基督而有的確據

**忍耐的目的**：

```
忍耐 → 行完神的旨意 → 得著所應許的
```

### 3.3 主必快來 (10:37-39)

> **「因為還有一點點時候，那要來的就來，並不遲延。只是義人必因信得生。他若退後，我心裡就不喜歡他。我們卻不是退後入沉淪的那等人，乃是有信心以致靈魂得救的人。」**

引用哈巴谷書2:3-4。

**兩條路**：

| 路 | 結果 | 神的態度 |
|----|------|----------|
| **退後** | 入沉淪 | 「我心裡不喜歡」 |
| **有信心** | 靈魂得救 | 「義人必因信得生」 |

**積極的宣告 (10:39)**：

> **「我們卻不是退後入沉淪的那等人，乃是有信心以致靈魂得救的人。」**

作者相信讀者會選擇信心的路。

## 四、神學綜合

### 4.1 教義與應用的轉折

希伯來書10:19是全書的轉折點：

```
結構

1:1-10:18      →      10:19-13:25
──────────────────────────────────
教義部分              應用部分
基督的超越            信徒的回應
祂為我們成就的        我們當如何行
```

### 4.2 三重呼召與三位一體

| 呼召 | 美德 | 對象 | 三一神 |
|------|------|------|--------|
| 進前來 | 信心 | 父神 | 聖父 |
| 持定 | 盼望 | 主的應許 | 聖子 |
| 彼此相顧 | 愛心 | 弟兄姊妹 | 聖靈團契 |

### 4.3 與約翰著作的連結

| 主題 | 希伯來書 10 | 約翰著作 |
|------|------------|----------|
| 又新又活的路 | 10:20 | 「我就是道路」(約14:6) |
| 幔子撕裂 | 10:20 | 「成了」(約19:30) |
| 彼此相愛 | 10:24 | 「新命令」(約13:34) |
| 不可停止聚會 | 10:25 | 「使他們合而為一」(約17:21) |

## 五、愛永遠長存的視角

### 四維透視

| 維度 | 希伯來書 10:19-39 的呈現 |
|------|-------------------------|
| **人性實況** | 容易退後、需要勉勵、面對逼迫 |
| **神的出路** | 又新又活的路，大祭司治理神家 |
| **神的計劃** | 那要來的就來，並不遲延 |
| **屬靈爭戰** | 大爭戰的苦難、堅忍到底 |

### 信望愛的呼召

```
信、望、愛的呼召

        10:22 進前來 (信)
             │
        10:23 持定 (望)
             │
        10:24-25 彼此相顧 (愛)
             │
             ↓
      「其中最大的是愛」(林前13:13)
```

## 六、應用與反思

### 6.1 進前來

- 你是否經常來到神面前？
- 有什麼攔阻你親近神？
- 你相信道路已經打開了嗎？

### 6.2 持定

- 你的盼望是什麼？
- 在困難中是否動搖？
- 你信靠神的信實嗎？

### 6.3 彼此相顧

- 你有固定的聚會生活嗎？
- 你如何激勵弟兄姊妹？
- 你是否孤立自己？

### 6.4 堅忍到底

- 回想你信仰路上的爭戰
- 你曾如何經歷神的保守？
- 「還有一點點時候」——這如何激勵你？
