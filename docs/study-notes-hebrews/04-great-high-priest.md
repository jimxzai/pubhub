## 一、坦然無懼到施恩寶座 (4:14-16)

### 1.1 偉大的大祭司 (4:14)

> **「我們既然有一位已經升入高天尊榮的大祭司，就是神的兒子耶穌，便當持定所承認的道。」**

**關鍵詞彙**：

| 希臘文 | 意義 |
|--------|------|
| μέγαν ἀρχιερέα | 偉大的大祭司 |
| διεληλυθότα τοὺς οὐρανούς | 經過諸天的 |
| Ἰησοῦν τὸν υἱὸν τοῦ θεοῦ | 神的兒子耶穌 |

**與亞倫祭司的對比**：

| 亞倫祭司 | 基督 |
|----------|------|
| 進入地上的至聖所 | 升入高天 |
| 一年一次 | 一次永遠 |
| 為自己的罪獻祭 | 無罪 |
| 會死 | 長遠活著 |

### 1.2 能體恤我們的軟弱 (4:15)

> **「因我們的大祭司並非不能體恤我們的軟弱。他也曾凡事受過試探，與我們一樣，只是他沒有犯罪。」**

**關鍵神學**：

```
基督的獨特性

完全的神性                    完全的人性
    │                            │
    └──────────┬─────────────────┘
               │
        能體恤軟弱
        凡事受試探
        只是沒有犯罪
               │
               ↓
        完美的大祭司
```

**「體恤」(συμπαθῆσαι, sympathēsai)**：
- 不只是同情
- 是同感、同受
- 祂真實經歷過我們的掙扎

### 1.3 施恩的寶座 (4:16)

> **「所以，我們只管坦然無懼地來到施恩的寶座前，為要得憐恤，蒙恩惠，作隨時的幫助。」**

**三個動作**：

| 動作 | 希臘文 | 意義 |
|------|--------|------|
| **坦然無懼** | μετὰ παρρησίας | 帶著放膽、確據 |
| **來到** | προσερχώμεθα | 進前、靠近 |
| **得** | λάβωμεν | 領受、得到 |

**三重恩典**：

```
施恩寶座的供應

        來到施恩寶座
             │
    ┌────────┼────────┐
    ↓        ↓        ↓
  憐恤      恩惠     隨時的幫助
  (ἔλεος)  (χάρις)  (εὔκαιρον βοήθειαν)
    │        │        │
  過去的罪  現在的需要  未來的試探
```

## 二、大祭司的資格 (5:1-10)

### 2.1 亞倫等次的條件 (5:1-4)

> **「凡從人間挑選的大祭司，是奉派替人辦理屬神的事，為要獻上禮物和贖罪祭。他能體諒那愚蒙的和失迷的人，因為他自己也是被軟弱所困。故此，他理當為百姓和自己獻祭贖罪。這大祭司的尊榮，沒有人自取。唯要蒙神所召，像亞倫一樣。」**

**亞倫祭司的三個條件**：

| 條件 | 內容 |
|------|------|
| 1. 從人中挑選 | 必須是人，才能代表人 |
| 2. 能體諒軟弱 | 因為自己也軟弱 |
| 3. 蒙神呼召 | 不是自取尊榮 |

### 2.2 基督的任命 (5:5-6)

> **「如此，基督也不是自取榮耀作大祭司，乃是在乎向他說『你是我的兒子，我今日生你』的那一位；就如經上又有一處說：『你是照著麥基洗德的等次永遠為祭司。』」**

**基督超越亞倫**：

| 亞倫等次 | 麥基洗德等次 |
|----------|--------------|
| 利未支派 | 猶大支派 |
| 律法設立 | 起誓設立 |
| 會死的人 | 長遠活著 |
| 軟弱的 | 完全的 |
| 預表 | 實體 |

### 2.3 藉苦難學了順從 (5:7-10)

> **「基督在肉體的時候，既大聲哀哭，流淚禱告，懇求那能救他免死的主，就因他的虔誠蒙了應允。他雖然為兒子，還是因所受的苦難學了順從。他既得以完全，就為凡順從他的人成了永遠得救的根源，並蒙神照著麥基洗德的等次稱他為大祭司。」**

**客西馬尼園的禱告**：

```
苦難與順從

大聲哀哭、流淚禱告 (路22:44)
          │
    「不要照我的意思，
     只要照你的意思」
          │
          ↓
     學了順從
          │
          ↓
    得以完全 (τελειωθείς)
          │
          ↓
  成了永遠得救的根源
```

**「學了順從」的意義**：
- 不是從不順從到順從
- 而是在人性中經歷順從的代價
- 「雖然為兒子」強調祂的神性沒有豁免祂的苦難

## 三、第三次警告：不可停滯不前 (5:11-6:12)

### 3.1 屬靈的嬰孩 (5:11-14)

> **「論到他，我們有好些話要說，只是難以講解，因為你們聽不進去。看你們學習的工夫，本該作師傅，誰知還得有人將神聖言小學的開端另教導你們，並且成了那必須吃奶、不能吃乾糧的人。凡只能吃奶的都不熟練仁義的道理，因為他是嬰孩；唯獨長大成人的才能吃乾糧；他們的心竅習練得通達，就能分辨好歹了。」**

**屬靈成長的對比**：

| 嬰孩 | 成人 |
|------|------|
| 吃奶 | 吃乾糧 |
| 不熟練仁義的道理 | 心竅習練通達 |
| 需要別人教導 | 能作師傅 |
| 不能分辨 | 能分辨好歹 |

### 3.2 離開道理的開端 (6:1-3)

> **「所以，我們應當離開基督道理的開端，竭力進到完全的地步，不必再立根基，就如那懊悔死行、信靠神、各樣洗禮、按手之禮、死人復活，以及永遠審判各等教訓。神若許我們，我們必如此行。」**

**六項基礎教導**：

| # | 教導 | 類別 |
|---|------|------|
| 1 | 懊悔死行 | 悔改 |
| 2 | 信靠神 | 信心 |
| 3 | 各樣洗禮 | 禮儀 |
| 4 | 按手之禮 | 禮儀 |
| 5 | 死人復活 | 末世 |
| 6 | 永遠審判 | 末世 |

### 3.3 最嚴重的警告 (6:4-8)

> **「論到那些已經蒙了光照、嘗過天恩的滋味、又於聖靈有份，並嘗過神善道的滋味、覺悟來世權能的人，若是離棄道理，就不能叫他們重新懊悔了。因為他們把神的兒子重釘十字架，明明地羞辱他。」**

**五重經歷**：

| 經歷 | 意義 |
|------|------|
| 蒙了光照 | 接受了真理的亮光 |
| 嘗過天恩的滋味 | 經歷過神的恩典 |
| 於聖靈有份 | 與聖靈有交通 |
| 嘗過神善道的滋味 | 品味過神話語的美好 |
| 覺悟來世權能 | 經歷過神能力的彰顯 |

**解釋這段經文的不同觀點**：

```
主要解釋

1. 假信徒觀點：
   這些人只是外在經歷，從未真正重生

2. 真信徒警告觀點：
   這是對真信徒的嚴重警告，
   但神會保守祂的兒女不至於此

3. 假設性警告觀點：
   作者用極端假設來警戒讀者
   不是說這真的會發生
```

### 3.4 更好的盼望 (6:9-12)

> **「親愛的弟兄們，我們雖是這樣說，卻深信你們的行為強過這些，而且近乎得救。因為神並非不公義，竟忘記你們所做的工和你們為他名所顯的愛心，就是先前伺候聖徒，如今還是伺候。我們願你們各人都顯出這樣的殷勤，使你們有滿足的指望，一直到底。並且不懈怠，總要效法那些憑信心和忍耐承受應許的人。」**

**鼓勵的話**：
- 「深信你們的行為強過這些」
- 「近乎得救」(σωτηρίας，屬於救恩的)
- 神不會忘記他們的愛心服事

## 四、神應許的確據 (6:13-20)

### 4.1 神用起誓確認 (6:13-18)

> **「當初神應許亞伯拉罕的時候，因為沒有比自己更大可以指著起誓的，就指著自己起誓，說：『論福，我必賜大福給你；論子孫，我必叫你的子孫多起來。』這樣，亞伯拉罕既恆久忍耐，就得了所應許的。人都是指著比自己大的起誓，並且以起誓為實據，了結各樣的爭論。照樣，神願意為那承受應許的人格外顯明他的旨意是不更改的，就起誓為證。藉這兩件不更改的事，神決不能說謊，好叫我們這逃往避難所、持定擺在我們前頭指望的人可以大得勉勵。」**

**雙重確據**：

| 確據 | 內容 |
|------|------|
| 1. 神的應許 | 神說話 |
| 2. 神的起誓 | 神確認 |

### 4.2 靈魂的錨 (6:19-20)

> **「我們有這指望，如同靈魂的錨，又堅固又牢靠，且通入幔內。作先鋒的耶穌，既照著麥基洗德的等次成了永遠的大祭司，就為我們進入幔內。」**

**錨的比喻**：

```
盼望如同靈魂的錨

          船 (我們)
           │
           │ 錨鏈 (信心)
           │
           ▼
    ━━━━━━━━━━━━━━━ 海面 (今生)
           │
           │
           ▼
    ┌─────────────┐
    │   幔內      │  ← 錨 (盼望)
    │ (至聖所)    │  ← 耶穌已進入
    │ (神的同在)  │
    └─────────────┘
```

## 五、麥基洗德等次的祭司 (7:1-28)

### 5.1 麥基洗德的超越性 (7:1-10)

> **「這麥基洗德就是撒冷王，又是至高神的祭司，本是長遠為祭司的。他當亞伯拉罕殺敗諸王回來的時候，就迎接他，給他祝福。亞伯拉罕也將自己所得來的，取十分之一給他。他頭一個名翻出來就是仁義王，他又是撒冷王，就是平安王的意思。他無父，無母，無族譜，無生之始，無命之終，乃是與神的兒子相似。」**

**麥基洗德的七個特點**：

| 特點 | 希伯來文/希臘文 | 意義 |
|------|----------------|------|
| 1. 撒冷王 | מַלְכִּי־צֶדֶק | 公義王 |
| 2. 至高神祭司 | | 不屬利未 |
| 3. 無父無母 | | 無族譜記載 |
| 4. 無生之始 | | 無起始記載 |
| 5. 無命之終 | | 無結束記載 |
| 6. 亞伯拉罕向他納十一 | | 比亞伯拉罕更大 |
| 7. 給亞伯拉罕祝福 | | 大的為小的祝福 |

**麥基洗德超越利未的論證**：

```
論證邏輯

1. 亞伯拉罕向麥基洗德納十一
2. 利未是亞伯拉罕的後代
3. 所以利未（在亞伯拉罕腰中）也向麥基洗德納十一
─────────────────────────────────────────────
結論：麥基洗德的祭司職分超越利未
```

### 5.2 利未祭司制度的不足 (7:11-19)

> **「從前百姓在利未人祭司職任以下受律法，倘若藉這職任能得完全，又何用另外興起一位祭司，照麥基洗德的等次，不照亞倫的等次呢？祭司的職任既已更改，律法也必須更改。」**

**利未制度的問題**：

| 問題 | 說明 |
|------|------|
| 不能使人完全 | 7:11 |
| 律法的軟弱無益 | 7:18 |
| 只是引進更美指望 | 7:19 |
| 祭司會死 | 7:23 |

### 5.3 基督憑神起誓作大祭司 (7:20-22)

> **「再者，耶穌為祭司，並不是不起誓立的。至於那些祭司，原不是起誓立的，只有耶穌是起誓立的；因為那立他的對他說：主起了誓，決不後悔，你是永遠為祭司。既是起誓立的，耶穌就作了更美之約的中保。」**

**起誓的意義**：

| 亞倫祭司 | 基督 |
|----------|------|
| 不是起誓立的 | 起誓立的 |
| 可更改的 | 不後悔的 |
| 暫時的 | 永遠的 |

### 5.4 長遠活著，長遠代求 (7:23-25)

> **「那些成為祭司的，數目本來多，是因為有死阻隔，不能長久。這位既是永遠常存的，他祭司的職任就長久不更換。凡靠著他進到神面前的人，他都能拯救到底；因為他是長遠活著，替他們祈求。」**

**基督代求的榮耀**：

```
基督的代求

永遠常存 ─→ 祭司職任不更換
    │
    ↓
長遠活著 ─→ 長遠代求
    │
    ↓
凡靠他進到神面前的 ─→ 拯救到底
```

**「拯救到底」(σῴζειν εἰς τὸ παντελές)**：
- 完全的拯救
- 永遠的拯救
- 沒有任何缺欠

### 5.5 完全的大祭司 (7:26-28)

> **「像這樣聖潔、無邪惡、無玷污、遠離罪人、高過諸天的大祭司，原是與我們合宜的。他不像那些大祭司，每日必須先為自己的罪，後為百姓的罪獻祭；因為他只一次將自己獻上，就把這事成全了。律法本是立軟弱的人為大祭司；但在律法以後起誓的話，是立兒子為大祭司，乃是成全到永遠的。」**

**基督的五重聖潔**：

| 特質 | 希臘文 | 意義 |
|------|--------|------|
| 聖潔 | ὅσιος | 虔誠、分別為聖 |
| 無邪惡 | ἄκακος | 無惡意 |
| 無玷污 | ἀμίαντος | 無污穢 |
| 遠離罪人 | κεχωρισμένος ἀπὸ τῶν ἁμαρτωλῶν | 與罪分離 |
| 高過諸天 | ὑψηλότερος τῶν οὐρανῶν | 升到至高 |

## 七、應用與反思

### 7.1 坦然無懼來到寶座前

- 你是否常常來到施恩寶座？
- 你相信基督能體恤你的軟弱嗎？
- 你需要什麼「隨時的幫助」？

### 7.2 追求屬靈長進

- 你仍在吃奶還是已能吃乾糧？
- 你的心竅是否習練得通達？
- 有什麼攔阻你成長？

### 7.3 抓住盼望的錨

- 你的靈魂有穩固的錨嗎？
- 在風浪中，你靠什麼站立？
- 基督在幔內為你代求——這給你什麼確據？
