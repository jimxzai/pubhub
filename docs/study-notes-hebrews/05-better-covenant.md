## 一、更美之約 (8:1-13)

### 1.1 我們有這樣的大祭司 (8:1-2)

> **「我們所講的事，其中第一要緊的，就是我們有這樣的大祭司，已經坐在天上至大者寶座的右邊，在聖所，就是真帳幕裡，作執事；這帳幕是主所支的，不是人所支的。」**

**關鍵對比**：

| 地上帳幕 | 天上帳幕 |
|----------|----------|
| 人所支的 | 主所支的 |
| 影兒 | 真像 |
| 暫時的 | 永恆的 |
| 亞倫祭司服事 | 基督坐在右邊 |

### 1.2 更美之約的中保 (8:3-6)

> **「如今耶穌所得的職任是更美的，正如他作更美之約的中保；這約原是憑更美之應許立的。」**

**「更美」(κρείττων) 的三重層次**：

```
更美的層次

更美的職任 ────→ 大祭司的服事
     │
更美之約 ─────→ 新約取代舊約
     │
更美的應許 ───→ 新約的內容
```

### 1.3 新約的應許 (8:7-12)

作者引用耶利米書31:31-34，這是舊約中最完整的新約預言：

> **「那些日子以後，我與以色列家所立的約乃是這樣：我要將我的律法放在他們裡面，寫在他們心上；我要作他們的神；他們要作我的子民。他們不用各人教導自己的鄉鄰和自己的弟兄，說：你該認識主；因為他們從最小的到至大的，都必認識我。我要寬恕他們的不義，不再記念他們的罪愆。」**

**新約的四個特點**：

| 特點 | 內容 | 與舊約對比 |
|------|------|-----------|
| 1. 內在化 | 律法寫在心上 | 舊約寫在石版上 |
| 2. 關係化 | 我作他們的神 | 直接的關係 |
| 3. 普遍化 | 從最小到最大都認識神 | 需要中介教導 |
| 4. 赦免化 | 不再記念罪愆 | 祭物不斷獻上 |

### 1.4 舊約漸漸衰老 (8:13)

> **「既說新約，就以前約為舊了；但那漸舊漸衰的，就必快歸無有了。」**

**神學意義**：
- 「舊」不是貶義，是預備性的
- 新約來了，舊約的功能完成
- 如同影子在陽光下消失

## 二、更美的帳幕 (9:1-14)

### 2.1 地上帳幕的結構 (9:1-5)

```
地上帳幕的結構

                ┌─────────────────────────────────┐
                │           至聖所                 │
                │                                  │
                │    ┌─────────────────────┐     │
                │    │      約櫃           │     │
                │    │  (施恩座/基路伯)    │     │
                │    └─────────────────────┘     │
                │                                  │
                ├──────────── 幔子 ────────────────┤
                │           聖所                   │
                │                                  │
                │  ┌──────┐  ┌──────┐  ┌──────┐  │
                │  │金燈臺│  │金壇  │  │陳設餅│  │
                │  └──────┘  └──────┘  └──────┘  │
                │                                  │
                └─────────────────────────────────┘
```

### 2.2 舊約禮儀的限制 (9:6-10)

> **「這些事既如此預備齊了，眾祭司就常進頭一層帳幕，行拜神的禮。至於第二層帳幕，唯有大祭司一年一次獨自進去，沒有不帶著血為自己和百姓的過犯獻上。」**

**舊約制度的三重限制**：

| 限制 | 說明 | 經文 |
|------|------|------|
| **進入限制** | 只有大祭司能進至聖所 | 9:7 |
| **時間限制** | 一年只能一次 | 9:7 |
| **效果限制** | 不能叫禮拜的人良心完全 | 9:9 |

### 2.3 基督進入真帳幕 (9:11-14)

> **「但現在基督已經來到，作了將來美事的大祭司，經過那更大更全備的帳幕，不是人手所造、也不是屬乎這世界的；並且不用山羊和牛犢的血，乃用自己的血，只一次進入聖所，成了永遠贖罪的事。」**

**基督獻祭的超越**：

| 舊約祭司 | 基督 |
|----------|------|
| 進入地上帳幕 | 進入天上帳幕 |
| 用牛羊的血 | 用自己的血 |
| 每年一次 | 只一次 |
| 暫時除罪 | 永遠除罪 |
| 外在潔淨 | 洗淨良心 |

**洗淨良心 (9:14)**：

> **「何況基督藉著永遠的靈，將自己無瑕無疵獻給神，他的血豈不更能洗淨你們的良心，除去你們的死行，使你們事奉那永生神嗎？」**

```
基督獻祭的果效

        基督的血
            │
    ┌───────┼───────┐
    ↓       ↓       ↓
洗淨良心  除去死行  事奉永生神
```

## 三、更美的祭 (9:15-28)

### 3.1 約與血的關係 (9:15-22)

> **「為此，他作了新約的中保，既然受死贖了人在前約之時所犯的罪過，便叫蒙召之人得著所應許永遠的產業。凡有遺命必須等到留遺命的人死了。因為人死了，遺命才有效力。」**

**希臘文 διαθήκη 的雙重意義**：

| 意義 | 應用 |
|------|------|
| **約 (Covenant)** | 神與人立約 |
| **遺命 (Testament)** | 需要立遺命的人死後才生效 |

**血的必要性 (9:22)**：

> **「若不流血，罪就不得赦免了。」**

### 3.2 基督一次獻上 (9:23-26)

> **「如今在這末世顯現一次，把自己獻為祭，好除掉罪。」**

**「一次」(ἅπαξ, hapax) 的強調**：

| 經文 | 強調 |
|------|------|
| 9:12 | 一次進入聖所 |
| 9:26 | 一次獻上自己 |
| 9:28 | 一次被獻 |
| 10:10 | 一次獻上身體 |

### 3.3 基督再來 (9:27-28)

> **「按著定命，人人都有一死，死後且有審判。像這樣，基督既然一次被獻，擔當了多人的罪，將來要向那等候他的人第二次顯現，並與罪無關，乃是為拯救他們。」**

**基督的兩次顯現**：

| 第一次顯現 | 第二次顯現 |
|-----------|-----------|
| 擔當多人的罪 | 與罪無關 |
| 為贖罪 | 為拯救 |
| 受苦受死 | 得榮耀 |
| 已經成就 | 將來成就 |

## 四、一次永遠的獻祭 (10:1-18)

### 4.1 律法的影兒 (10:1-4)

> **「律法既是將來美事的影兒，不是本物的真像，總不能藉著每年常獻一樣的祭物叫那近前來的人得以完全。若不然，獻祭的事豈不早已止住了嗎？因為禮拜的人，良心既被潔淨，就不再覺得有罪了。但這些祭物是叫人每年想起罪來；因為公牛和山羊的血，斷不能除罪。」**

**影兒與本物**：

```
影兒 (律法)                 本物 (基督)
────────────────────────────────────────
祭物 ─────────────────→ 基督的身體
祭壇 ─────────────────→ 十字架
大祭司 ───────────────→ 基督
帳幕 ─────────────────→ 天上聖所
牛羊的血 ─────────────→ 基督的血
```

**舊約祭物的矛盾 (10:3)**：
- 每年獻祭 → 叫人「想起罪來」
- 如果有效 → 應該不用再獻
- 持續獻祭 → 證明不能除罪

### 4.2 基督的順服 (10:5-10)

> **「所以基督到世上來的時候，就說：神啊，祭物和禮物是你不願意的；你曾給我預備了身體。燔祭和贖罪祭是你不喜歡的。那時我說：神啊，我來了，為要照你的旨意行；我的事在經卷上已經記載了。」**

引用詩篇40:6-8，基督用順服取代祭物：

| 神不願意的 | 基督獻上的 |
|-----------|-----------|
| 祭物和禮物 | 順服的心 |
| 燔祭贖罪祭 | 預備的身體 |
| 外在儀式 | 照神的旨意行 |

**「一次獻上」的果效 (10:10)**：

> **「我們憑這旨意，靠耶穌基督，只一次獻上他的身體，就得以成聖。」**

### 4.3 坐下與站著的對比 (10:11-14)

> **「凡祭司天天站著事奉神，屢次獻上一樣的祭物，這祭物永不能除罪。但基督獻了一次永遠的贖罪祭，就在神的右邊坐下了。從此，等候他仇敵成了他的腳凳。因為他一次獻祭，便叫那得以成聖的人永遠完全。」**

**站著 vs 坐下**：

| 舊約祭司 | 基督 |
|----------|------|
| 天天站著 | 坐下了 |
| 屢次獻上 | 一次獻上 |
| 永不能除罪 | 永遠完全 |
| 工作未完 | 工作完成 |

### 4.4 聖靈的見證 (10:15-18)

> **「聖靈也對我們作見證；因為他既已說過：主說：那些日子以後，我與他們所立的約乃是這樣：我要將我的律法寫在他們心上，又要放在他們裡面。以後就說：我不再記念他們的罪愆和他們的過犯。這些罪過既已赦免，就不用再為罪獻祭了。」**

**新約的結論**：

```
聖靈見證新約的果效

律法寫在心上 ────→ 內在化
不再記念罪愆 ────→ 完全赦免
        │
        ↓
不用再為罪獻祭 ────→ 基督一次成全
```

## 七、應用與反思

### 7.1 活在新約中

- 你是否活在「新約」的自由中？
- 還是仍被律法主義捆綁？
- 神的律法寫在你心上了嗎？

### 7.2 良心的潔淨

- 你的良心是否被基督的血潔淨？
- 還有什麼「死行」需要除去？
- 你是否自由地事奉永生神？

### 7.3 等候再來

- 你是「等候他」的人嗎？
- 基督第二次顯現時，你會如何？
- 這盼望如何影響你現在的生活？
