# 第二單元：基督超越摩西與約書亞

> **經文：希伯來書 3:1-4:13**

---

## 經文概覽

### 結構大綱

```
希伯來書 3:1-4:13 結構

第一段 (3:1-6) — 基督超越摩西
├── 3:1-2    思想耶穌：忠信的使者和大祭司
├── 3:3-4    基督比摩西更尊榮：建造者超越房屋
├── 3:5      摩西是忠心的僕人
└── 3:6      基督是治理神家的兒子

第二次警告 (3:7-19) — 不可硬著心
├── 3:7-11   引用詩篇 95：曠野世代的失敗
├── 3:12-14  警告：不可有不信的惡心
└── 3:15-19  結論：不信使他們不能進安息

第二段 (4:1-13) — 進入神的安息
├── 4:1-5    安息的應許仍然存留
├── 4:6-10   另有一安息日的安息為神的子民存留
├── 4:11     勸勉：竭力進入那安息
└── 4:12-13  神的道是活潑的、有功效的
```

## 一、基督超越摩西 (3:1-6)

### 1.1 思想耶穌 (3:1-2)

> **「同蒙天召的聖潔弟兄啊，你們應當思想我們所認為使者、為大祭司的耶穌。他為那設立他的盡忠，如同摩西在神的全家盡忠一樣。」**

**關鍵詞彙**：

| 希臘文 | 音譯 | 中文 | 意義 |
|--------|------|------|------|
| κατανοήσατε | katanoēsate | 思想 | 仔細注視、深思默想 |
| ἀπόστολος | apostolos | 使者 | 被差遣者 |
| ἀρχιερεύς | archiereus | 大祭司 | 最高祭司 |

**雙重職分**：

```
耶穌的雙重職分

        使者 (Apostle)              大祭司 (High Priest)
             ↓                              ↓
        從神到人                        從人到神
        傳遞啟示                        獻上祭物
        摩西的職分                      亞倫的職分
             ↓                              ↓
        基督超越摩西                    基督超越亞倫
        (第3章)                         (第4-7章)
```

### 1.2 建造者與房屋的對比 (3:3-4)

> **「他比摩西算是更配多得榮耀，好像建造房屋的比房屋更尊榮。因為房屋都必有人建造，但建造萬物的就是神。」**

**論證邏輯**：

```
論證結構

前提1：建造房屋的人比房屋本身更尊榮
前提2：摩西是房屋的一部分（神家中的僕人）
前提3：基督是建造房屋的（建造萬物的神）
─────────────────────────────────────────
結論：基督比摩西更配得榮耀
```

**神學意義**：

| 摩西 | 基督 |
|------|------|
| 房屋的一部分 | 建造房屋的 |
| 受造物 | 創造者 |
| 服事者 | 主人 |
| 在房屋之內 | 在房屋之上 |

### 1.3 僕人與兒子的對比 (3:5-6)

> **「摩西為僕人，在神的全家誠然盡忠，為要證明將來必傳說的事。但基督為兒子，治理神的家；我們若將可誇的盼望和膽量堅持到底，便是他的家了。」**

**核心對比**：

| 摩西 | 基督 |
|------|------|
| θεράπων (therapōn) 僕人 | υἱός (huios) 兒子 |
| 在家中服事 | 治理神的家 |
| 見證將來的事 | 成就所見證的事 |
| 預表 | 實體 |

**條件性的警告**：

> 「我們**若**將可誇的盼望和膽量**堅持到底**，便是他的家了。」

這引出接下來的警告段落。

## 二、第二次警告：不可硬著心 (3:7-19)

### 2.1 引用詩篇95 (3:7-11)

> **「聖靈有話說：你們今日若聽他的話，就不可硬著心，像在曠野惹他發怒、試探他的時候一樣。在那裡，你們的祖宗試我探我，並且觀看我的作為有四十年之久。所以，我厭煩那世代的人，說：他們心裡常常迷糊，竟不曉得我的作為！我就在怒中起誓說：他們斷不可進入我的安息。」**

**歷史背景**：民數記13-14章

```
曠野世代的失敗

加低斯巴尼亞
      ↓
十二探子回報
      ↓
十人報惡信：「我們是蚱蜢」
      ↓
百姓哭號、埋怨
      ↓
想回埃及
      ↓
神發怒：四十年曠野漂流
      ↓
那一代人不得進入應許之地
```

### 2.2 三個「今日」(3:7, 13, 15)

作者三次強調「今日」：

| 經文 | 內容 | 意義 |
|------|------|------|
| 3:7 | 「你們今日若聽他的話」 | 聖靈的呼召 |
| 3:13 | 「總要趁著還有今日，天天彼此相勸」 | 信徒的責任 |
| 3:15 | 「你們今日若聽他的話」 | 重複強調 |

**神學意義**：
- 「今日」是恩典的時刻
- 機會不是無限的
- 需要立即回應

### 2.3 「不信的惡心」(3:12-14)

> **「弟兄們，你們要謹慎，免得你們中間或有人存著不信的惡心，把永生神離棄了。總要趁著還有今日，天天彼此相勸，免得你們中間有人被罪迷惑，心裡就剛硬了。我們若將起初確實的信心堅持到底，就在基督裡有份了。」**

**警告的核心**：

```
不信的漸進過程

聽見 → 不信 → 心剛硬 → 被罪迷惑 → 離棄神
  ↑                                    ↓
  └────────── 不能進入安息 ←───────────┘
```

**「不信」(ἀπιστία, apistia) 的本質**：
- 不是缺乏知識
- 而是拒絕信靠
- 根源是「惡心」

### 2.4 曠野世代的教訓 (3:15-19)

> **「誰聽見他的話惹他發怒呢？豈不是跟著摩西從埃及出來的眾人嗎？神四十年之久，又厭煩誰呢？豈不是那些犯罪、屍首倒在曠野的人嗎？又向誰起誓，不容他們進入他的安息呢？豈不是向那些不順從的人嗎？這樣看來，他們不能進入安息是因為不信的緣故了。」**

**三個問題與答案**：

| 問題 | 答案 | 關鍵詞 |
|------|------|--------|
| 誰惹神發怒？ | 從埃及出來的眾人 | 惹怒 (παραπικραίνω) |
| 神厭煩誰？ | 犯罪的人 | 厭煩 (προσοχθίζω) |
| 神向誰起誓？ | 不順從的人 | 不順從 (ἀπειθέω) |

**結論**：他們不能進入，是因為**不信**。

## 三、進入神的安息 (4:1-13)

### 3.1 安息的應許仍然存留 (4:1-5)

> **「我們既蒙留下，有進入他安息的應許，就當畏懼，免得我們中間或有人似乎是趕不上了。因為有福音傳給我們，像傳給他們一樣；只是所聽見的道與他們無益，因為他們沒有信心與所聽見的道調和。」**

**「安息」(κατάπαυσις, katapausis) 的多重含義**：

```
安息的多重層次

1. 創造的安息 (4:4)
   └── 神第七日安息

2. 迦南的安息 (4:8)
   └── 約書亞帶領進入

3. 屬靈的安息 (4:9-10)
   └── 信徒現今進入

4. 永恆的安息 (4:11)
   └── 將來完全實現
```

### 3.2 約書亞的安息不是終點 (4:6-8)

> **「既有必進安息的人，那先前聽見福音的，因為不順從，不得進去。所以過了多年，就在大衛的書上，又限定一日，如以上所引的說：你們今日若聽他的話，就不可硬著心。若是約書亞已叫他們享了安息，後來神就不再提別的日子了。」**

**論證邏輯**：

```
約書亞的安息不是終極安息

前提1：約書亞帶領以色列人進入迦南
前提2：多年後，大衛仍說「今日若聽他的話」(詩95)
─────────────────────────────────────────────────
結論：約書亞給的安息不是終極的安息
      另有一安息為神的子民存留
```

**希伯來文雙關**：
- 「約書亞」(יְהוֹשֻׁעַ, Yehoshua) 與「耶穌」(Ἰησοῦς, Iēsous) 在希臘文中是同一個詞
- 約書亞是預表，耶穌是實體

### 3.3 安息日的安息 (4:9-10)

> **「這樣看來，必另有一安息日的安息為神的子民存留。因為那進入安息的，乃是歇了自己的工，正如神歇了他的工一樣。」**

**σαββατισμός (sabbatismos)**：
- 這個詞在新約只出現一次
- 指「安息日式的安息」
- 是終極的、永恆的安息

**安息的本質**：

| 不是 | 是 |
|------|---|
| 無所事事 | 歇了自己的工 |
| 疲倦的休息 | 完成後的享受 |
| 暫時的停頓 | 永恆的狀態 |

### 3.4 竭力進入 (4:11)

> **「所以，我們務必竭力進入那安息，免得有人學那不順從的樣子跌倒了。」**

**看似矛盾？**
- 安息 = 不靠自己
- 竭力 = 努力追求

**解答**：
- 「竭力」不是靠行為得救
- 而是積極回應恩典
- 堅持信心到底
- 警醒不要跌倒

### 3.5 神的道是活潑的 (4:12-13)

> **「神的道是活潑的，是有功效的，比一切兩刃的劍更快，甚至魂與靈，骨節與骨髓，都能刺入、剖開，連心中的思念和主意都能辨明。並且被造的沒有一樣在他面前不顯然的；原來萬物在那與我們有關係的主眼前，都是赤露敞開的。」**

**神的道的四個特質**：

| 特質 | 希臘文 | 意義 |
|------|--------|------|
| **活潑** | ζῶν (zōn) | 有生命的、活的 |
| **有功效** | ἐνεργής (energēs) | 有能力運行的 |
| **鋒利** | τομώτερος (tomōteros) | 比兩刃劍更快 |
| **辨明** | κριτικός (kritikos) | 能審判、分辨 |

**神的道的雙重功能**：

```
神的道

刺入剖開 ──→ 揭露人心隱藏的
辨明審判 ──→ 判斷思念和主意
     │
     ↓
沒有人能在神面前隱藏
萬物都赤露敞開
```

## 四、神學綜合

### 4.1 三重超越的結構

| 超越對象 | 經文 | 核心對比 |
|----------|------|----------|
| 天使 | 1:5-2:18 | 兒子 vs 僕役的靈 |
| **摩西** | **3:1-6** | **兒子 vs 僕人** |
| **約書亞** | **4:1-13** | **真安息 vs 預表安息** |

### 4.2 警告的模式

希伯來書的警告有固定模式：

```
警告模式

1. 舊約的例子 (負面)
   ↓
2. 應用於讀者
   ↓
3. 「若...就...」的條件
   ↓
4. 正面的勸勉
```

### 4.3 「信」與「不信」的對比

| 信 | 不信 |
|---|------|
| 聽見的道有益 | 聽見的道無益 |
| 進入安息 | 不能進入安息 |
| 在基督裡有份 | 離棄永生神 |
| 堅持到底 | 隨流失去 |

## 六、應用與反思

### 6.1 思想耶穌

- 不是膚淺地想想，而是深深注視
- 讓基督的超越性影響你的優先順序
- 基督比你所依靠的任何事物都更可靠

### 6.2 警惕不信

- 不信不是一夜之間發生的
- 從忽略開始，到心裡剛硬，到離棄神
- 需要天天彼此相勸

### 6.3 進入安息

- 不是將來才有的，現在就可以進入
- 不是無所事事，而是信靠基督
- 歇了自己的工，享受祂的工

### 6.4 面對神的道

- 神的道能看透你
- 不要隱藏，要敞開
- 讓神的道光照、潔淨、更新

## 七、研讀問題

### 觀察題
1. 作者用什麼比喻來說明基督比摩西更尊榮？
2. 曠野世代為什麼不能進入安息？
3. 4:12-13描述神的道有哪些特質？

### 解釋題
1. 「僕人」和「兒子」的對比有什麼神學意義？
2. 為什麼約書亞給的安息不是終極的安息？
3. 「竭力進入安息」這句話如何理解？

### 應用題
1. 你有沒有經歷過「心裡剛硬」的危險？如何警醒？
2. 你現在是否享受著神的安息？還是仍在靠自己的工？
3. 你如何讓神的道更深地光照你的內心？
