# 正典脈絡導論：從創世到完成

> 「神既在古時藉著眾先知多次多方地曉諭列祖，就在這末世藉著他兒子曉諭我們。」
> — 希伯來書 1:1-2

---

## 引言：時空中的有限與永恆的視角

### 人的困境

人是**時空中的存在**。我們被困在：

- **時間的河流中**：只能看見過去的碎片，無法預見未來
- **空間的局限中**：只能從一個角度觀看，無法縱覽全局
- **認知的框架中**：被文化、經驗、罪性所蒙蔽

因此，**視角** (perspective) 對人極其重要。同一件事，從不同視角看，意義截然不同。

瞎子摸象的故事告訴我們：人沒有神的視角，就無法認識真相。

### 神的恩典：漸進啟示

神是**超越時空的存在**：

> 「我是阿拉法，我是俄梅戛；我是首先的，我是末後的；我是初，我是終。」(啟22:13)

神從永恆的視角看歷史，但祂選擇**進入時空**，在人的歷史中漸進地啟示自己。這不是神改變主意，而是神**按照人能承受的程度**逐步揭開奧秘。

就像父親教導幼兒，先用圖畫，再用故事，最後用道理。

## 聖經的大敘事：一個故事

### 四幕劇

聖經不是一本規條彙編，而是**一個宏大的故事**：

```
     創造                 墮落                 救贖                 完成
   (Creation)           (Fall)           (Redemption)        (Consummation)
      ↓                   ↓                   ↓                   ↓
   創1-2章              創3章             創3:15-啟20章          啟21-22章
      ↓                   ↓                   ↓                   ↓
  「甚好」              咒詛               應許→成就             「成了」
      ↓                   ↓                   ↓                   ↓
  伊甸園             被逐出園            漫長的歸途            新耶路撒冷
```

### 一條主線：神與人同在

貫穿聖經的主線是：**神要與人同在**。

| 階段 | 同在的形式 | 經文 |
|------|-----------|------|
| 伊甸園 | 神在園中行走 | 創3:8 |
| 列祖時代 | 神顯現、立約 | 創17、26、28 |
| 出埃及 | 會幕——神住在人中間 | 出25:8 |
| 進入迦南 | 聖殿——神的居所 | 王上8:13 |
| 道成肉身 | 道成了肉身，住在我們中間 | 約1:14 |
| 教會時代 | 聖靈住在信徒心中 | 林前6:19 |
| 新天新地 | 神的帳幕在人間 | 啟21:3 |

**以馬內利**——「神與我們同在」——是聖經的框架。馬太福音以此開始 (1:23)，又以此結束 (28:20)。

## 渐进启示的脈絡

### 舊約：影兒與應許

> 「這些原是後事的影兒；那形體卻是基督。」(西2:17)

舊約不是過時的歷史，而是**指向基督的預表**：

| 舊約的影兒 | 新約的實體 | 完成於啟示錄 |
|-----------|-----------|-------------|
| 伊甸園的生命樹 | 基督是生命 | 新耶路撒冷的生命樹 (啟22:2) |
| 逾越節羔羊 | 神的羔羊基督 | 寶座上的羔羊 (啟5:6) |
| 大祭司年年進入至聖所 | 基督一次進入天上聖所 | 不再有殿 (啟21:22) |
| 摩西的會幕 | 道成肉身住在人間 | 神的帳幕在人間 (啟21:3) |
| 大衛的寶座 | 基督坐在天上寶座 | 萬王之王的寶座 (啟19:16) |

### 希伯來書的神學框架

希伯來書為我們提供了理解**渐进启示**的鑰匙：

**第一章**：神從前藉先知，如今藉兒子

> 「神既在古時藉著眾先知**多次多方**地曉諭列祖，就在**這末世**藉著他**兒子**曉諭我們。」(來1:1-2)

**「多次多方」**(πολυμερῶς καὶ πολυτρόπως)：
- 多次：不是一次傾倒，而是歷史中逐步展開
- 多方：律法、先知、詩歌、智慧、歷史

**「藉著他兒子」**：最終的、完全的、不可超越的啟示。

**核心對比**：

| 舊約 | 新約（基督） |
|------|-------------|
| 天使頒布律法 | 神子超越天使 (1-2章) |
| 摩西是僕人 | 基督是兒子 (3章) |
| 約書亞的安息 | 真正的安息 (4章) |
| 亞倫的祭司職 | 麥基洗德的等次 (5-7章) |
| 舊約的條例 | 更美的約 (8章) |
| 帳幕的禮儀 | 天上的實體 (9章) |
| 牛羊的血 | 基督的血 (10章) |

### 信心的見證人

希伯來書第11章是**信心名人堂**——那些在渐进启示中活出信心的人：

> 「這些人都是存著信心死的，並沒有得著所應許的；卻從遠處望見，且歡喜迎接。」(來11:13)

他們看見的是**影兒**，我們看見的是**實體**。他們憑信心等候，我們活在應驗之中。

## 四大主題的展開

### 一、透視人性

聖經毫不留情地揭露人性：

**創世記的人性畫像**：
- 亞當夏娃：推卸責任
- 該隱：嫉妒殺人
- 挪亞世代：終日所思想的盡都是惡
- 巴別塔：驕傲自大

**舊約歷史的模式**：
```
拯救 → 感恩 → 忘記 → 悖逆 → 審判 → 呼求 → 拯救...
```

士師記的循環、列王的敗壞、先知的控訴——人性從未改變。

**新約的確認**：
> 「沒有義人，連一個也沒有。」(羅3:10)

### 二、出路何在

人的嘗試都失敗了：

| 嘗試 | 代表 | 結局 |
|------|------|------|
| 靠己努力 | 巴別塔 | 混亂分散 |
| 靠律法 | 法利賽人 | 定罪絕望 |
| 靠宗教 | 異教祭祀 | 虛空偶像 |
| 靠哲學 | 希臘智慧 | 世上的智慧神看為愚拙 |

**出路只有一條**：

> 「神愛世人，甚至將他的獨生子賜給他們，叫一切信他的，不致滅亡，反得永生。」(約3:16)

這出路不是一套系統，不是一種方法，而是**一個人**——耶穌基督。

### 三、神的計劃

神的計劃從創世以前就定好了：

> 「神從創立世界以前，在基督裡揀選了我們。」(弗1:4)

**計劃的展開時間線**：

```
創造 (Creation)
    ↓
墮落 & 原始福音 (創3:15 - 女人的後裔)
    ↓
挪亞之約 (創9 - 普世之約)
    ↓
亞伯拉罕之約 (創12 - 萬國蒙福)
    ↓
摩西之約 (出19-24 - 律法與聖潔)
    ↓
大衛之約 (撒下7 - 永恆王國)
    ↓
新約應許 (耶31 - 寫在心版上)
    ↓
基督來臨 (加4:4 - 及至時候滿足)
    ↓
教會時代 (徒2 - 聖靈降臨)
    ↓
基督再來 (帖前4 - 號筒吹響)
    ↓
新天新地 (啟21 - 成了！)
```

### 四、宇宙爭戰

聖經揭示了一場看不見的爭戰：

> 「我們並不是與屬血氣的爭戰，乃是與那些執政的、掌權的、管轄這幽暗世界的，以及天空屬靈氣的惡魔爭戰。」(弗6:12)

**爭戰的軌跡**：

| 舊約 | 新約 | 啟示錄 |
|------|------|--------|
| 蛇引誘人 | 撒但試探基督 | 龍攻擊婦人 |
| 法老抵擋神 | 希律殺嬰孩 | 獸敵擋羔羊 |
| 假先知欺騙 | 法利賽人拒絕 | 假先知迷惑 |
| 巴比倫征服 | 羅馬逼迫 | 大巴比倫傾倒 |

**得勝的確據**：

> 「女人的後裔要傷你的頭。」(創3:15)

這是聖經第一個福音——**原始福音** (Protoevangelium)。蛇的頭最終在十字架上被傷，在啟示錄中被扔進火湖。

## 從馬太福音到啟示錄

### 馬太福音：天國君王來了

馬太福音是**新約的門檻**，連接舊約與新約：

- **向後看**：超過60次引用舊約，宣告「這是要應驗...」
- **向前看**：「使萬民作門徒」的使命，直到世界末了

**主題**：耶穌是大衛的子孫、亞伯拉罕的後裔，是舊約預言的彌賽亞君王。

**核心經文**：
> 「天國近了，你們應當悔改！」(太4:17)

### 啟示錄：天國完全實現

啟示錄是**聖經的終點**，回應創世記的每一個主題：

| 創世記 | 啟示錄 |
|--------|--------|
| 「起初神創造天地」| 「我將一切都更新了」|
| 蛇進入伊甸園 | 撒但被扔進火湖 |
| 死亡進入世界 | 不再有死亡 |
| 咒詛臨到大地 | 不再有咒詛 |
| 人被逐出樂園 | 人進入新耶路撒冷 |

**主題**：基督是萬王之王，祂必再來，使萬物更新。

**核心經文**：
> 「看哪，我必快來！」(啟22:7)
> 「阿們！主耶穌啊，我願你來！」(啟22:20)

### 馬太與啟示錄的呼應

| 馬太福音（開始）| 啟示錄（完成）|
|----------------|---------------|
| 以馬內利——神與我們同在 (1:23) | 神的帳幕在人間 (21:3) |
| 天國近了 (4:17) | 世上的國成了主的國 (11:15) |
| 登山寶訓：天國子民的品格 | 新婦妝飾整齊 (21:2) |
| 十字架的羔羊 | 寶座上的羔羊 |
| 大使命：萬民作門徒 | 各族各民敬拜 (7:9) |
| 我常與你們同在 (28:20) | 他們要見他的面 (22:4) |

## 信、望、愛：三重維度

### 信 (πίστις)：回應過去

信心是對**神已經成就之事**的回應。

我們相信：
- 神創造了世界
- 神揀選了亞伯拉罕
- 神差遣了基督
- 基督死了、復活了

> 「信就是所望之事的實底，是未見之事的確據。」(來11:1)

### 望 (ἐλπίς)：面向未來

盼望是對**神將要成就之事**的確信。

我們盼望：
- 身體的復活
- 基督的再來
- 新天新地
- 與主永遠同在

> 「我們得救是在乎盼望...我們若盼望那所不見的，就必忍耐等候。」(羅8:24-25)

### 愛 (ἀγάπη)：活在當下

愛是**信心和盼望的果子**，是**當下的行動**。

> 「如今常存的有信，有望，有愛這三樣，其中最大的是愛。」(林前13:13)

**為什麼愛最大？**

- 在天國裡，**信心**會變成眼見——「我們必見他的真體」
- 在天國裡，**盼望**會變成實現——「成了！」
- 唯有**愛**永遠長存——因為「神就是愛」(約壹4:8)

這就是為什麼保羅說「其中最大的是愛」——愛是永恆的本質。

## 寫作的目的：透視與轉化

### 我們寫作是為了：

1. **幫助讀者看見**
   - 提供神的視角，超越有限的人眼
   - 揭開幔子，看見屬靈的實況

2. **幫助讀者明白**
   - 解釋神在歷史中的作為
   - 連接舊約預言與新約成就

3. **幫助讀者回應**
   - 呼召信心：相信神已經成就的
   - 呼召盼望：盼望神將要成就的
   - 呼召愛：在當下活出神的愛

4. **幫助讀者得勝**
   - 認識爭戰的實況
   - 穿戴神的全副軍裝
   - 在患難中持守信心

## 結語：從這裡開始

帶著這個框架，我們進入馬太福音和啟示錄的研讀：

**馬太福音**告訴我們：**君王來了**——祂是誰，祂做了什麼，祂要我們做什麼。

**啟示錄**告訴我們：**君王要再來**——祂必得勝，祂必審判，祂必更新萬物。

我們活在「已然」與「未然」之間：
- **已然**：基督已經得勝，救恩已經成就
- **未然**：基督尚未再來，新天新地尚未實現

這張力不是問題，而是**恩典**——給人悔改的機會，給教會見證的使命。

> 「所以，你們要去，使萬民作我的門徒...我就常與你們同在，直到世界的末了。」(太28:19-20)

> 「阿們！主耶穌啊，我願你來！」(啟22:20)
