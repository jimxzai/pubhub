# Claude Code 配置說明

本目錄包含約翰福音研讀項目的 Claude Code AI 輔助配置。

---

## 目錄結構

```
.claude/
├── README.md           # 本文件：配置說明
├── settings.json       # 項目核心配置
└── commands/           # AI Agent 命令
    ├── master-editor.md   # Agent 1: 總編輯
    ├── annotate.md        # Agent 2: 注疏師
    ├── proofread.md       # Agent 4: 校對神
    └── publish.md         # Agent 5: 出書總管
```

---

## AI Agent 命令

### `/master-editor` - 總編輯
**功能**: 分析和評估研讀筆記
- 內容分類（經卷、章節、主題）
- 質量評估（1-5分制）
- 改進建議
- 標籤推薦

### `/annotate` - 注疏師
**功能**: 提供深度注疏
- 中英希三語對照
- 歷代注疏精華（教父、改革宗、當代）
- 希臘文關鍵詞解析
- 歷史文化背景

### `/proofread` - 校對神
**功能**: 校對和潤色
- 經文引用核對
- 語言潤色
- 邏輯檢查
- 神學準確性驗證

### `/publish` - 出書總管
**功能**: 生成報告和書稿
- 周報（1000-2000字）
- 月報（3000-5000字）
- 書稿章節（5000-10000字）

---

## 使用流程

### 1. 撰寫研讀筆記
在 `Gospel of John.md` 中記錄你的研讀心得。

### 2. 評估筆記質量
```
/master-editor
```
獲取質量評分和改進建議。

### 3. 添加深度注疏
```
/annotate
```
為特定經文添加歷代注疏和原文分析。

### 4. 校對潤色
```
/proofread
```
檢查經文引用、語言和神學準確性。

### 5. 生成報告
```
/publish
```
生成周報、月報或書稿章節。

## 研讀框架

### 核心經文
> 「道成了肉身，住在我們中間，充充滿滿地有恩典有真理。」
> （約翰福音 1:14）

### 結構公式
**榮耀 = 恩典 + 真理**
- **恩典** → 七個神蹟 (Signs)
- **真理** → 七個「我是」宣告 (I AM)

---

## 質量標準

### 必須遵守
- ✅ 經文引用 100% 準確
- ✅ 注疏來源可查證
- ✅ 希臘文解釋有根據
- ✅ 神學立場正統

### 禁止行為
- ❌ 杜撰聖經內容
- ❌ 編造歷代注疏
- ❌ 空洞屬靈術語
- ❌ 脫離經文過度發揮
