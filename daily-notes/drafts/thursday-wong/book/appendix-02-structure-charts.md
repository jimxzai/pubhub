## 一、約翰福音整體結構

### 傳統分段

```
約翰福音結構

┌─────────────────────────────────────────────────────────────┐
│  序言 Prologue (1:1-18)                                      │
│  「太初有道」— 道成肉身的宣告                                 │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│  神蹟之書 Book of Signs (1:19-12:50)                         │

                              ↓
┌─────────────────────────────────────────────────────────────┐
```

### 詳細大綱

```
I. 序言 (1:1-18)
   A. 道的永恆存在 (1:1-2)
   B. 道與創造 (1:3-5)
   C. 施洗約翰的見證 (1:6-8)
   D. 道來到世界 (1:9-13)
   E. 道成肉身 (1:14-18)

II. 神蹟之書 (1:19-12:50)
    A. 見證與呼召 (1:19-51)
    B. 迦拿到迦拿 (2:1-4:54)
       - 神蹟 1: 水變酒
       - 潔淨聖殿
       - 尼哥底母
       - 撒瑪利亞婦人
       - 神蹟 2: 醫治大臣之子
    C. 耶路撒冷衝突 (5:1-47)
       - 神蹟 3: 醫治畢士大癱子
    D. 加利利的高峰 (6:1-71)
       - 神蹟 4: 五餅二魚
       - 神蹟 5: 海面行走
       - 生命之糧講論
    E. 住棚節爭議 (7:1-10:42)
       - 神蹟 6: 醫治生來瞎眼者
       - 好牧人講論
    F. 拉撒路與受難預告 (11:1-12:50)
       - 神蹟 7: 拉撒路復活
       - 馬利亞膏主
       - 榮耀進城

III. 榮耀之書 (13:1-20:31)
     A. 臨別講論 (13:1-17:26)
        1. 洗腳與新命令 (13章)
        2. 安慰與道路 (14章)
        3. 葡萄樹與逼迫 (15章)
        4. 聖靈與喜樂 (16章)
        5. 大祭司禱告 (17章)
     B. 受難敘事 (18:1-19:42)
        1. 被捕 (18:1-11)
        2. 審問 (18:12-27)
        3. 彼拉多審判 (18:28-19:16)
        4. 釘十字架 (19:17-37)
        5. 埋葬 (19:38-42)
     C. 復活敘事 (20:1-31)
        1. 空墳墓 (20:1-10)
        2. 向馬利亞顯現 (20:11-18)
        3. 向門徒顯現 (20:19-23)
        4. 多馬的認信 (20:24-29)
        5. 目的宣言 (20:30-31)

IV. 跋 (21:1-25)
    A. 加利利海顯現 (21:1-14)
    B. 彼得的恢復 (21:15-19)
    C. 愛徒的命運 (21:20-23)
    D. 結語 (21:24-25)
```

## 四、約翰福音與符類福音對比

### 內容對比

| 主題 | 符類福音 | 約翰福音 |
|------|----------|----------|
| 事工時間 | 約1年 | 至少3年（3個逾越節）|
| 地理重心 | 加利利 | 耶路撒冷 |
| 教導形式 | 比喻、登山寶訓 | 講論、對話 |
| 神蹟數量 | 約20-35個 | 7個精選 |
| 神蹟稱呼 | δύναμις (能力) | σημεῖον (記號) |
| 最後晚餐 | 設立聖餐 | 洗腳 |
| 基督論 | 漸進啟示 | 從開始就宣告神性 |
