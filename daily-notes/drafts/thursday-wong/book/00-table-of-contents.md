## 閱讀建議

### 初學者路線
1. 先讀 [完整導讀](./chapter-FINAL-complete.md) 掌握全貌
2. 按順序閱讀第一至五章
3. 參考附錄深化理解

### 主題研究路線
- **神蹟研究**：第二章 + 附錄二的神蹟對照表
- **基督論研究**：第三章 + 第一章的 λόγος 神學
- **門徒訓練**：第四章的臨別講論
- **受難週默想**：第五章的逐段研讀

### 靈修閱讀路線
- 每日選讀一個神蹟或「我是」宣告
- 配合經文禱讀
- 使用每章結尾的「對我們的意義」反思

## 本書特色

| 特色 | 說明 |
|------|------|
| 神學框架 | 以「榮耀 = 恩典 + 真理」(約 1:14) 為核心 |
| 希臘文解析 | 關鍵詞彙的原文分析與神學意涵 |
| 歷史註釋 | 引用教父至當代學者的註釋 |
| 結構分析 | 七個神蹟、七個「我是」的對應關係 |
| 生命應用 | 每個主題都連結實際生活 |
