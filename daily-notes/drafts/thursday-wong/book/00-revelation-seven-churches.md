# 啟示錄導讀：七教會書信
## Revelation Introduction: Letters to the Seven Churches

---

## 引言：約翰的第二卷書

> **「約翰寫信給亞細亞的七個教會。但願從那昔在、今在、以後永在的神，和祂寶座前的七靈，並那誠實作見證的、從死裡首先復活、為世上君王元首的耶穌基督，有恩惠、平安歸與你們！」**
> — 啟示錄 1:4-5

約翰福音記載道成肉身的基督，啟示錄則啟示榮耀再來的基督。兩卷書首尾呼應，共同構成約翰對基督的完整見證。

## 二、七封書信的統一結構

### 每封書信的七個元素

```
1. 收信人   → 「你要寫信給...的使者說」
      ↓
2. 基督的自稱 → 「那...的說」(取自第一章異象)
      ↓
3. 稱讚     → 「我知道你的...」
      ↓
4. 責備     → 「然而有一件事我要責備你...」
      ↓
5. 勸勉     → 「所以你當...」
      ↓
6. 呼召     → 「聖靈向眾教會所說的話，凡有耳的就應當聽」
      ↓
7. 應許     → 「得勝的，我必...」
```

## 三、七教會詳解

### 1. 以弗所：離棄起初的愛心 (2:1-7)

> **「然而有一件事我要責備你，就是你把起初的愛心離棄了。」** — 啟 2:4

| 項目 | 內容 |
|------|------|
| **基督自稱** | 右手拿著七星、在七個金燈臺中間行走的 |
| **稱讚** | 勞碌、忍耐、不容忍惡人、試驗假使徒、恨惡尼哥拉黨 |
| **責備** | 離棄起初的愛心 |
| **應許** | 吃生命樹的果子 |

**約翰福音連結**：
- 「生命樹」呼應約翰福音的「生命在祂裡頭」(約 1:4)
- 「起初的愛心」呼應「神就是愛」的教導 (約壹 4:8)

**四維透視**：
- 人性：事奉的熱忱可以取代愛的關係
- 出路：悔改，回到起初
- 計劃：生命樹——伊甸園的恢復
- 爭戰：純正教義 vs. 愛心冷淡

---

### 2. 士每拿：至死忠心 (2:8-11)

> **「你務要至死忠心，我就賜給你那生命的冠冕。」** — 啟 2:10

| 項目 | 內容 |
|------|------|
| **基督自稱** | 那首先的、末後的、死過又活的 |
| **稱讚** | 在患難貧窮中卻是富足的 |
| **責備** | 無 |
| **應許** | 不受第二次死的害 |

**約翰福音連結**：
- 「死過又活的」呼應「復活在我，生命也在我」(約 11:25)
- 「生命的冠冕」呼應「永生」的應許 (約 3:16)

**四維透視**：
- 人性：外在貧窮可以是內在富足
- 出路：至死忠心——效法基督
- 計劃：勝過第二次死
- 爭戰：撒但會派 vs. 教會

---

### 3. 別迦摩：撒但座位之城 (2:12-17)

> **「我知道你的居所，就是有撒但座位之處。」** — 啟 2:13

| 項目 | 內容 |
|------|------|
| **基督自稱** | 有兩刃利劍的 |
| **稱讚** | 在撒但座位處堅守信仰、沒有棄絕信心 |
| **責備** | 容讓巴蘭和尼哥拉黨的教訓 |
| **應許** | 隱藏的嗎哪、白石新名 |

**約翰福音連結**：
- 「兩刃利劍」象徵神的道——「道就是神」(約 1:1)
- 「隱藏的嗎哪」呼應「生命的糧」(約 6:35)

**四維透視**：
- 人性：在敵意環境中妥協的試探
- 出路：持守真道
- 計劃：隱藏的嗎哪——天上的供應
- 爭戰：撒但座位 vs. 基督寶座

---

### 4. 推雅推喇：容讓耶洗別 (2:18-29)

> **「然而，有一件事我要責備你，就是你容讓那自稱是先知的婦人耶洗別。」** — 啟 2:20

| 項目 | 內容 |
|------|------|
| **基督自稱** | 眼目如火焰、腳像光明銅的神之子 |
| **稱讚** | 愛心、信心、勤勞、忍耐，末後所行比起初更多 |
| **責備** | 容讓假先知「耶洗別」|
| **應許** | 權柄制伏列國、晨星 |

**約翰福音連結**：
- 「神之子」呼應約翰福音的核心主題 (約 20:31)
- 「晨星」預示基督的再來

**四維透視**：
- 人性：善良可以成為容忍罪惡的藉口
- 出路：分辨真假先知
- 計劃：與基督一同作王
- 爭戰：耶洗別精神 vs. 真先知

---

### 5. 撒狄：按名是活的卻是死的 (3:1-6)

> **「你按名是活的，其實是死的。」** — 啟 3:1

| 項目 | 內容 |
|------|------|
| **基督自稱** | 有神的七靈和七星的 |
| **稱讚** | 有幾名是未曾污穢衣服的 |
| **責備** | 按名是活卻是死、行為不完全 |
| **應許** | 穿白衣、名錄生命冊、主必承認 |

**約翰福音連結**：
- 「神的七靈」呼應聖靈的工作 (約 14:26)
- 「生命冊」呼應「因祂的名得生命」(約 20:31)

**四維透視**：
- 人性：屬靈的名聲可以掩蓋內在的死亡
- 出路：儆醒、堅固將要死的
- 計劃：名錄生命冊
- 爭戰：外在名聲 vs. 內在生命

---

### 6. 非拉鐵非：敞開的門 (3:7-13)

> **「我知道你的行為。看哪，我在你面前給你一個敞開的門，是無人能關的。」** — 啟 3:8

| 項目 | 內容 |
|------|------|
| **基督自稱** | 聖潔真實的、拿著大衛的鑰匙 |
| **稱讚** | 略有一點力量、遵守我的道、不棄絕我的名 |
| **責備** | 無 |
| **應許** | 作神殿中的柱子、刻上神的名和新耶路撒冷的名 |

**約翰福音連結**：
- 「大衛的鑰匙」呼應「我就是門」(約 10:9)
- 「敞開的門」呼應耶穌是唯一的道路 (約 14:6)

**四維透視**：
- 人性：力量微小卻蒙悅納
- 出路：持守所有的
- 計劃：新耶路撒冷——永恆的城
- 爭戰：猶太會堂 vs. 真教會

---

### 7. 老底嘉：不冷不熱 (3:14-22)

> **「你既如溫水，也不冷也不熱，所以我必從我口中把你吐出去。」** — 啟 3:16

| 項目 | 內容 |
|------|------|
| **基督自稱** | 阿們、誠信真實見證的、神創造萬物之首 |
| **稱讚** | 無 |
| **責備** | 不冷不熱、自以為富足實則貧窮瞎眼赤身 |
| **應許** | 與基督同坐寶座 |

**約翰福音連結**：
- 「誠信真實見證」呼應「道...充充滿滿地有恩典有真理」(約 1:14)
- 「同坐寶座」呼應門徒與基督的合一 (約 17:22)
- 「在門外叩門」呼應基督來到自己的地方 (約 1:11)

**四維透視**：
- 人性：物質富足導致屬靈貧窮
- 出路：買金子、白衣、眼藥——從基督得真富足
- 計劃：與基督同坐寶座
- 爭戰：自滿 vs. 真實

## 四、七教會的整體信息

### 得勝者的七重應許

| 教會 | 應許 | 對應的永恆實現 (啟 21-22) |
|------|------|---------------------------|
| 以弗所 | 生命樹 | 「生命樹...結十二樣果子」(22:2) |
| 士每拿 | 不受第二次死的害 | 「第二次的死...無份」(20:6) |
| 別迦摩 | 隱藏的嗎哪、白石新名 | 「祂的名字必寫在他們的額上」(22:4) |
| 推雅推喇 | 制伏列國、晨星 | 「基督作王直到永永遠遠」(11:15) |
| 撒狄 | 白衣、生命冊 | 「穿白衣與祂同行」(3:4) |
| 非拉鐵非 | 神殿柱子、刻神的名 | 「神的帳幕在人間」(21:3) |
| 老底嘉 | 同坐寶座 | 「他們要作王直到永永遠遠」(22:5) |

### 七教會的時代解讀

有學者認為七教會代表教會歷史的七個時期：

| 教會 | 可能代表的時期 | 特徵 |
|------|----------------|------|
| 以弗所 | 使徒時代 (30-100) | 純正但漸冷 |
| 士每拿 | 逼迫時代 (100-313) | 殉道忠心 |
| 別迦摩 | 國教時代 (313-600) | 與世界混合 |
| 推雅推喇 | 中世紀 (600-1517) | 假教導盛行 |
| 撒狄 | 改教時代 (1517-1700) | 有名無實 |
| 非拉鐵非 | 宣教時代 (1700-1900) | 宣教敞開門 |
| 老底嘉 | 現代/末世 (1900-?) | 不冷不熱 |

> **註**：此解讀為歷史詮釋法之一種，不是唯一正確解法。七封書信對每個時代的教會都有直接教導意義。

## 五、愛永遠長存：四維透視總論

### 1. 人性實況

七教會顯示人性的各種軟弱：
- **愛心冷淡** (以弗所)
- **懼怕逼迫** (士每拿)
- **妥協世俗** (別迦摩)
- **容忍罪惡** (推雅推喇)
- **有名無實** (撒狄)
- **力量微小** (非拉鐵非)
- **自滿驕傲** (老底嘉)

### 2. 神的出路

基督對每個問題都有回應：
- 悔改回轉 (以弗所、別迦摩、推雅推喇、撒狄、老底嘉)
- 至死忠心 (士每拿)
- 持守所有 (非拉鐵非)

### 3. 神的計劃

七重應許指向新天新地：生命樹、永生、隱藏嗎哪、晨星、白衣、新名、寶座——一切在基督再來時完全成就。

### 4. 屬靈爭戰

```
假教師 (尼哥拉黨、耶洗別) ←→ 真教會
撒但的會堂              ←→ 基督的教會
巴比倫的淫亂            ←→ 新婦的貞潔
獸的印記                ←→ 神的印記
```

## 七、信望愛的應用

### 信 (Faith)
七教會書信呼召我們信靠那位「在七個金燈臺中間行走」的基督。祂看見、祂知道、祂掌管。我們可以信靠祂的判斷和應許。

### 望 (Hope)
每封書信都以「得勝者」的應許結束——指向新天新地的終極盼望。無論現在經歷什麼，我們都有榮耀的將來。

### 愛 (Love)
- **以弗所**的警告：不要失去起初的愛心
- **推雅推喇**的稱讚：愛心比起初更多
- **老底嘉**的應許：與主同坐寶座——愛的終極團契

> **「我責備管教我所愛的人；所以你要發熱心，也要悔改。看哪，我站在門外叩門，若有聽見我聲音就開門的，我要進到他那裡去，我與他，他與我一同坐席。」** — 啟 3:19-20

這是愛的呼喚。基督不是站在遠處審判，而是站在門外叩門，渴望與我們恢復愛的關係。

**愛永遠長存。**

## 結語

七教會書信是復活基督對祂教會的直接說話。每封信都：
1. 揭示基督的榮耀
2. 診斷教會的光景
3. 呼召悔改或堅持
4. 應許永恆的賞賜

這七封信不只是歷史文獻——它們是活潑的道，今天仍在對我們說話：

> **「聖靈向眾教會所說的話，凡有耳的，就應當聽！」**
