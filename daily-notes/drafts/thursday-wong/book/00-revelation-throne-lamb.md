# 啟示錄導讀：天上寶座與羔羊
## Revelation Introduction: The Throne and the Lamb (Chapters 4-5)

---

## 引言：從七教會到天上寶座

> **「此後，我觀看，見天上有門開了。我初次聽見好像吹號的聲音，對我說：『你上到這裡來，我要將以後必成的事指示你。』」**
> — 啟示錄 4:1

啟示錄 1-3 章是地上的視角——復活基督對七教會說話。
啟示錄 4-5 章轉向天上——約翰被提升，看見宇宙的實況：
**神坐在寶座上，羔羊配得打開書卷。**

這是全本啟示錄的神學核心，也是約翰福音「神的羔羊」的榮耀成全。

---

                              │

           │                  │                  │
           ▼                  ▼                  ▼
    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐

### 寶座周圍的元素

| 元素 | 經文 | 象徵意義 | 舊約背景 |
|------|------|----------|----------|
| **碧玉、紅寶石** | 4:3 | 神的榮耀與聖潔 | 出 28:17-20 大祭司胸牌 |
| **虹如翡翠** | 4:3 | 神的信實與約 | 創 9:13 挪亞之約 |
| **二十四長老** | 4:4 | 教會代表/天使長老 | 代上 24 祭司班次 |
| **七燈** | 4:5 | 神的七靈 | 亞 4:2 金燈臺 |
| **玻璃海** | 4:6 | 神的超越與分隔 | 出 30:18 銅海 |
| **四活物** | 4:6-8 | 受造界的代表 | 結 1:5-14 四活物 |

### 四活物的意義

| 活物 | 形象 | 象徵 | 四福音對應 |
|------|------|------|------------|
| 第一 | 獅子 | 威嚴、王權 | 馬太：君王基督 |
| 第二 | 牛犢 | 服事、勞苦 | 馬可：僕人基督 |
| 第三 | 人面 | 智慧、人性 | 路加：人子基督 |
| 第四 | 飛鷹 | 超越、神性 | 約翰：神子基督 |

> **「聖哉！聖哉！聖哉！主神是昔在、今在、以後永在的全能者。」** — 啟 4:8

### 二十四長老的敬拜

> **「我們的主，我們的神，你是配得榮耀、尊貴、權柄的，因為你創造了萬物，並且萬物是因你的旨意被創造而有的。」** — 啟 4:11

**敬拜的焦點**：神是創造主 (Creator)
- 萬物因祂被創造
- 萬物因祂的旨意存在
- 祂配得一切榮耀

## 二、羔羊與書卷 (第5章)

### 書卷的奧秘

> **「我看見坐寶座的右手中有書卷，裡外都寫著字，用七印封嚴了。」** — 啟 5:1

| 問題 | 答案 |
|------|------|
| 書卷是什麼？ | 神對歷史的計劃、救贖的旨意 |
| 為何七印封嚴？ | 表示完全封閉，需要有權柄者打開 |
| 裡外都寫著字？ | 內容豐富完整，無可增減 |

### 約翰的哭泣

> **「我就大哭，因為沒有配展開、觀看那書卷的。」** — 啟 5:4

**為何哭泣？**
- 如果無人能打開書卷，神的計劃就無法成就
- 歷史就沒有意義，邪惡就永遠得勝
- 人類就永遠沒有盼望

### 猶大支派的獅子

> **「長老中有一位對我說：『不要哭！看哪，猶大支派中的獅子，大衛的根，祂已得勝，能以展開那書卷，揭開那七印。』」** — 啟 5:5

| 稱號 | 舊約來源 | 意義 |
|------|----------|------|
| **猶大支派的獅子** | 創 49:9-10 | 君王彌賽亞 |
| **大衛的根** | 賽 11:1, 10 | 大衛之約的應驗 |

### 羔羊的顯現

> **「我又看見寶座與四活物，並長老之中有羔羊站立，像是被殺過的，有七角七眼，就是神的七靈，奉差遣往普天下去的。」** — 啟 5:6

**約翰「聽見」獅子，卻「看見」羔羊！**

這是啟示錄最震撼的神學轉折：
- **獅子** = 期待中威武的征服者
- **羔羊** = 實際的拯救方式——犧牲的愛

```
舊約期待           啟示錄揭示           約翰福音印證
────────────────────────────────────────────────────
猶大的獅子    →    被殺的羔羊    ←    神的羔羊 (約1:29)
武力征服      →    犧牲捨命      ←    為世人死 (約3:16)
政治彌賽亞    →    受苦僕人      ←    好牧人捨命 (約10:11)
```

### 羔羊的七重特徵

| 特徵 | 經文 | 意義 |
|------|------|------|
| **站立** | 5:6 | 復活得勝 |
| **像被殺過的** | 5:6 | 十架犧牲的永恆印記 |
| **七角** | 5:6 | 完全的能力 |
| **七眼** | 5:6 | 完全的知識（神的七靈）|
| **配得** | 5:9 | 有權柄打開書卷 |
| **被讚美** | 5:12 | 與神同受敬拜 |
| **在寶座中間** | 5:6 | 與神同等 |

## 三、天上的敬拜 (5:8-14)

### 三重敬拜的擴展

```
第一圈：四活物 + 二十四長老
        「你配拿書卷...因為你曾被殺」(5:9-10)
              ↓
第二圈：千千萬萬天使
        「曾被殺的羔羊是配得...」(5:11-12)
              ↓
第三圈：一切受造之物
        「願頌讚、尊貴、榮耀、權勢都歸給...羔羊」(5:13)
              ↓
        四活物說「阿們」，長老俯伏敬拜 (5:14)
```

### 新歌的內容

> **「你配拿書卷，配揭開七印；因為你曾被殺，用自己的血從各族、各方、各民、各國中買了人來，叫他們歸於神，又叫他們成為國民，作祭司歸於神，在地上執掌王權。」** — 啟 5:9-10

**四重普世性**: 各族 (φυλῆς)、各方 (γλώσσης)、各民 (λαοῦ)、各國 (ἔθνους)

**三重身份**:
1. 歸於神的子民
2. 作祭司事奉神
3. 在地上作王

### 羔羊的七重頌讚

> **「曾被殺的羔羊是配得權柄、豐富、智慧、能力、尊貴、榮耀、頌讚的。」** — 啟 5:12

| 頌讚 | 希臘文 | 意義 |
|------|--------|------|
| 權柄 | δύναμιν | 能力、大能 |
| 豐富 | πλοῦτον | 財富、資源 |
| 智慧 | σοφίαν | 智慧、謀略 |
| 能力 | ἰσχὺν | 力量、剛強 |
| 尊貴 | τιμὴν | 榮譽、價值 |
| 榮耀 | δόξαν | 榮光、輝煌 |
| 頌讚 | εὐλογίαν | 祝福、稱頌 |

## 四、約翰福音與啟示錄的羔羊主題

### 「神的羔羊」的首尾呼應

| 約翰福音 | 啟示錄 |

| 「看哪，神的羔羊，除去世人罪孽的！」(1:29) | 「曾被殺的羔羊」(5:6) |
| 施洗約翰見證 | 天上見證 |
| 地上出現 | 天上顯現 |
| 即將被殺 | 已經得勝 |
| 一人認出 | 萬物敬拜 |

### 羔羊在啟示錄中的角色

| 經文 | 角色 |
|------|------|
| 5:5-6 | 打開書卷的得勝者 |
| 6:1, 16 | 揭開七印、施行審判 |
| 7:17 | 牧養群羊、引到生命泉 |
| 12:11 | 聖徒得勝的根據 |
| 14:1-4 | 十四萬四千人的元首 |
| 17:14 | 萬主之主、萬王之王 |

### 從約翰福音到啟示錄的基督論

## 五、愛永遠長存：四維透視

### 1. 人性實況

**人無法打開書卷**

約翰大哭，因為天上地下沒有配展開書卷的。這揭示人性的根本困境：
- 我們無法掌控歷史
- 我們無法戰勝邪惡
- 我們無法自我拯救
- 我們需要一位救主

### 2. 神的出路

**羔羊配得打開書卷**

神的出路不是派天使、不是靠人的努力，而是：
- **犧牲的愛**：被殺的羔羊
- **復活的能力**：站立的羔羊
- **完全的主權**：七角七眼

> 獅子的能力，透過羔羊的犧牲彰顯。
> 這是十字架的智慧——以愛得勝。

### 3. 神的計劃

**從創造到新創造**

| 啟示錄 4 章 | 啟示錄 5 章 |
|-------------|-------------|
| 神是創造主 | 羔羊是救贖主 |
| 萬物因祂被創造 | 萬民被祂的血買贖 |
| 頌讚創造的榮耀 | 頌讚救贖的大功 |

**救贖歷史的樞紐**：羔羊打開書卷，歷史才能向前推進，直到新天新地。

### 4. 屬靈爭戰

**羔羊的得勝 vs. 龍的失敗**

```
啟示錄的核心衝突

羔羊 (5章)              龍/獸 (12-13章)
────────────────────────────────────────
被殺卻站立              被扔下地上
七角七眼               七頭十角
萬物敬拜               世人跟從
寶座中間               模仿寶座
生命冊                 獸的印記
新婦                   大淫婦

結局：羔羊得勝，龍被扔在火湖
```

## 六、信望愛的應用

### 信 (Faith)

**我們可以信靠那位「曾被殺的羔羊」**

- 祂已經得勝
- 祂掌管歷史
- 祂打開書卷
- 一切都在祂手中

當世界看起來失控時，記得：羔羊在寶座上。

### 望 (Hope)

**我們盼望羔羊的完全得勝**

啟示錄 4-5 章是「預覽」——讓我們看見結局：
- 萬物都要敬拜羔羊
- 每個膝蓋都要跪拜
- 每條舌頭都要承認
- 新天新地必定來臨

### 愛 (Love)

**羔羊的愛是犧牲的愛**

> 祂「用自己的血從各族、各方、各民、各國中買了人來」(5:9)

這不是抽象的愛，而是付代價的愛：
- 被殺的痕跡永遠留在羔羊身上
- 這愛「買」了我們——我們屬於祂
- 這愛使我們成為「祭司」——事奉神
- 這愛使我們「作王」——與祂同掌權

**愛永遠長存**——因為羔羊永遠站立在寶座中間。

## 七、敬拜的啟示

### 天上敬拜的特徵

| 特徵 | 經文 | 今日應用 |
|------|------|----------|
| **以神為中心** | 4:2 「有一個寶座」 | 敬拜不是關於我們的感受 |
| **俯伏敬拜** | 4:10, 5:14 | 承認神的主權 |
| **放下冠冕** | 4:10 | 將一切榮耀歸神 |
| **晝夜不停** | 4:8 | 持續的敬拜生活 |
| **唱新歌** | 5:9 | 救贖帶來新的讚美 |
| **萬物同聲** | 5:13 | 合一的敬拜 |

### 敬拜的焦點

```
創造 (第4章)    +    救贖 (第5章)    =    完全的敬拜
    ↓                    ↓                    ↓
  神是誰              神做了什麼           我們的回應
  (本體)              (作為)               (頌讚)
```

## 結語：看見寶座

啟示錄 4-5 章邀請我們「上到這裡來」——從地上的困境仰望天上的實況：

1. **神仍在寶座上** — 無論地上多混亂
2. **羔羊已經得勝** — 無論仇敵多囂張
3. **書卷正在展開** — 歷史有意義、有方向
4. **敬拜是宇宙的中心** — 我們被邀請加入

> **「坐寶座的和羔羊配得頌讚、尊貴、榮耀、權勢，直到永永遠遠！」**
> — 啟示錄 5:13

這是約翰福音「神的羔羊」的榮耀成全。
這是「愛永遠長存」的天上頌歌。

**阿們！**
