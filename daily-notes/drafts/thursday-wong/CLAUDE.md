# 生命之道 — 約翰福音研讀項目

本項目專注於使徒約翰的著作研讀，以「道」為主題，深入理解耶穌基督的神性與救恩信息。

---

## 寫作的靈魂：愛永遠長存

> **「如今常存的有信、有望、有愛這三樣，其中最大的是愛。」**
> — 哥林多前書 13:13

### 核心視角

**人是時空中的受造物**——我們需要時間與空間的座標來理解事物。神的啟示也配合人的處境，**漸進地**展開祂永恆的計劃：

```
古時           末世           永恆
 │              │              │
眾先知    →    神的兒子    →    愛永遠長存
(多次多方)     (一次完全)       (林前13:13)
 │              │              │
影子      →    實體       →    完成
應許      →    成就       →    享受
```

### 信、望、愛的時間結構

| 維度 | 方向 | 核心 | 歸宿 |
|------|------|------|------|
| **信** | 過去→現在 | 基督已成就的救贖 | 見面時不再需要 |
| **望** | 現在→未來 | 基督再來的應許 | 成就時不再需要 |
| **愛** | 永恆→永恆 | 神的本質與關係 | **永遠長存** |

### 這是我們寫作的靈魂

透過約翰的著作，我們要看見：
- **人性的實況**：墮落、隔絕、被轄制
- **神的出路**：基督的救贖、聖靈的重生
- **神的計劃**：從創造到新創造的救贖歷史
- **屬靈爭戰**：光與暗、真理與謊言、基督與敵基督

**而貫穿一切的，是愛——神就是愛，愛永遠長存。**

### 約翰福音作為第四福音

約翰福音不是符類福音的重複，而是神聖啟示的完成：

```
四福音的四重見證

馬太 (獅子) → 君王基督 → 以色列的彌賽亞
馬可 (牛)   → 僕人基督 → 受苦的服事者
路加 (人)   → 人子基督 → 完美的人性
約翰 (鷹)   → 神子基督 → 完全的神性 ← 最高視角
```

### 承上啟下的橋樑

約翰福音在正典中的獨特位置：

```
舊約聖經              約翰福音              新約/啟示錄
─────────────────────────────────────────────────────────

創世記「起初」  ←───  「太初有道」  ───→  啟示錄「都成了」
逾越節羔羊     ←───  「神的羔羊」  ───→  寶座上的羔羊

先知的預言     ←───  應驗成就      ───→  完全實現
```

## 研讀原則

### 核心方法
1. **正典為本**: 將約翰福音放在全本聖經的脈絡中理解
2. **原文為本**: 參考希臘原文，理解關鍵詞彙
3. **經以釋經**: 以聖經解釋聖經，注意上下文
4. **歷代注疏**: 參考教父、改革宗、當代福音派學者
5. **生命應用**: 將真理落實於日常生活

### 引用規範
- 聖經引文必須標明書卷、章節
- 注疏引用必須標明出處和作者
- 希臘文詞彙需提供音譯和解釋

### 禁止行為
- 不可杜撰聖經內容或注疏
- 不可脫離經文過度發揮
- 避免空洞的屬靈術語
