# 每日流程（极简版）

**总时间：45-60分钟**

---

## 步骤 1：读（20分钟）

```
打开原文，读一小节
- 孙子兵法：1节约200字
- 资治通鉴：1段约500字
- 圣经：1章约30节
```

## 步骤 2：写（20分钟）

```bash
npm run new-note
```

只写三件事：
1. **原文抄写**（中英对照，50字）
2. **我的理解**（用自己的话说，200字）
3. **AI映射**（一句话类比，50字）

**示例**：

```markdown
# 2025-12-05

**书**: 孙子兵法 | **章**: 始计篇

---

## 原文

> 兵者，国之大事，死生之地，存亡之道，不可不察也。
>
> *War is a matter of vital importance to the State.*

## 心得

孙子开篇就把战争提到国家存亡的高度。这不是好战，而是因为
后果太严重所以必须慎重。对应到现在：AI竞争也是"国之大事"，
OpenAI、Google、Anthropic 的每一步都影响整个行业格局...

## AI映射

**庙算** → **战略规划会议**
孙子说"多算胜，少算不胜"。Sam Altman 在 GPT-4 发布前
肯定也是"庙算"了无数次。

---

`#孙子兵法` `#始计`
```

## 步骤 3：存（5分钟）

```bash
git add . && git commit -m "📝 $(date +%Y-%m-%d)" && git push
```

---

## 可选：NotebookLM（+15分钟）

如果有时间：
1. 打开 NotebookLM
2. 点 "Generate Audio Overview"
3. 边听边补充笔记

**没时间就跳过**——这是锦上添花，不是必须。

---

## 每周（周六，30分钟）

```bash
npm run weekly-summary
# 或者在 Claude Code 中运行 /publish
```

看一眼生成的周总结，改几句，提交。

---

## 常见问题

**Q: 没灵感怎么办？**
A: 随便写。写"今天读不懂"也行。坚持比质量重要。

**Q: 时间不够怎么办？**
A: 只写100字也行。核心是每天动笔。

**Q: 漏了一天怎么办？**
A: 第二天补上，标注日期。别补太多，会崩。

---

## 关键心态

```
完成 > 完美
坚持 > 爆发
300字 × 2500天 = 75万字
```

7年后回头看，数量自然变质量。
